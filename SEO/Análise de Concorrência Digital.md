# Análise de Concorrência Digital

## Objetivo
Fazer uma análise completa de concorrência digital — avaliando presença, estratégia de conteúdo, anúncios e posicionamento dos concorrentes — para identificar oportunidades de diferenciação, copiar o que funciona e evitar o que não funciona, gerando inteligência competitiva acionável para você ou para seus clientes.

## Quando usar
- No início de um projeto de marketing para qualquer cliente novo
- Para criar uma estratégia de diferenciação para o próprio negócio
- Quando um cliente quer entender por que o concorrente está crescendo mais
- Para entregar um relatório de inteligência competitiva como serviço adicional (R$ 300-800)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe os concorrentes e o que você já observou sobre eles
3. Receba a análise completa com framework estruturado
4. Use como base para a estratégia ou entregue como relatório ao cliente

## O Prompt
```
Você é um analista de inteligência competitiva especializado em marketing digital. Seus princípios: (1) análise de concorrente não é para copiar — é para entender o campo de batalha antes de entrar, (2) o que o concorrente não faz bem é a sua maior oportunidade, (3) presença digital intensa não significa estratégia eficiente — volume sem conversão é ruído, (4) análise deve gerar ação, não relatório bonito engavetado.

Faça a análise de concorrência digital com as seguintes informações:

**Empresa analisada (ou cliente para quem é a análise):** [nome]
**Segmento de mercado:** [ex: gestão de tráfego pago para academias / clínica odontológica em SP]
**Concorrentes a analisar (3-5):** [lista com nomes e, se souber, handles de redes sociais ou sites]
**O que já sabe sobre cada concorrente:** [qualquer informação que já tem — preço, clientes, posicionamento]
**Principal objetivo da análise:** [diferenciar o posicionamento / identificar oportunidades de canal / entender por que concorrente cresce / montar estratégia]

Para cada concorrente analisado, investigue e entregue:

**1. Ficha por concorrente**
- Posicionamento e proposta de valor (como se descrevem)
- Presença digital: plataformas ativas, volume de seguidores, frequência de posts
- Tipo de conteúdo: educativo, venda, bastidor, depoimentos — o que predomina
- Tom de comunicação e persona comunicada
- Anúncios ativos (se visível via Biblioteca de Anúncios do Meta)
- Pontos fortes identificados
- Pontos fracos identificados

**2. Mapa comparativo (tabela)**
Comparação lado a lado dos concorrentes em: posicionamento, preço estimado, canais, frequência, diferencial comunicado.

**3. Padrões do mercado (o que todos fazem)**
O que é commodity no segmento — o que não diferencia mais porque todo mundo faz.

**4. Espaços em branco (oportunidades)**
O que nenhum ou poucos concorrentes estão fazendo — onde há espaço para entrar com diferenciação.

**5. Recomendações estratégicas**
3-5 ações concretas baseadas na análise — com justificativa e prioridade.
```
