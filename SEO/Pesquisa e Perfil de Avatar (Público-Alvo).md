# Pesquisa e Perfil de Avatar (Público-Alvo)

## Objetivo
Criar o perfil completo e detalhado do cliente ideal (avatar) para profissionais de saúde e bem-estar — com dores, desejos, objeções, linguagem e comportamento de compra — para fundamentar toda a comunicação, conteúdo e estratégia de captação em dados reais sobre quem você quer atrair.

## Quando usar
- Antes de criar qualquer campanha, conteúdo ou proposta comercial
- Quando sente que fala para todo mundo e não converte ninguém
- Para revisar o posicionamento e entender por que os clientes que chegam não são os ideais
- Para criar comunicação que faz a pessoa sentir "parece que escreveram isso para mim"

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o que você sabe sobre seus clientes atuais e ideais
3. Receba o perfil completo do avatar com insights de comunicação
4. Use como documento de referência para todo conteúdo e copy que criar

## O Prompt
```
Você é um especialista em pesquisa de mercado e desenvolvimento de personas para o nicho de saúde e bem-estar. Seus princípios: (1) avatar não é dado demográfico — é mapa emocional, (2) a linguagem que o cliente usa para descrever a própria dor é o melhor copy que existe — sua tarefa é encontrá-la, (3) objeção não é resistência — é informação sobre o que o cliente ainda não entendeu ou não acredita, (4) quem tenta falar com todo mundo não fala com ninguém — nicho específico converte mais.

Crie o perfil completo de avatar para este contexto:

**Minha especialidade e serviço:** [ex: nutricionista funcional — programa de emagrecimento hormonal]
**Clientes que já tenho (descreva os melhores):** [quem são os clientes com quem o trabalho flui melhor]
**Clientes que você NÃO quer atender:** [perfil problemático ou desalinhado]
**Faixa de preço do seu serviço:** [para inferir poder aquisitivo do avatar]
**Onde seu cliente ideal provavelmente está:** [Instagram, LinkedIn, Google, indicação, etc.]
**O que você já observou sobre o comportamento deles:** [comportamentos, dúvidas recorrentes, objeções que aparecem]

Entregue:

**1. Perfil demográfico**
Idade, gênero, localização, escolaridade, renda estimada, ocupação mais comum.

**2. Perfil psicográfico**
Valores, crenças, estilo de vida, medos mais profundos, desejos mais íntimos.

**3. A dor central (em 3 camadas)**
- Dor superficial: o que ele diz que tem (o sintoma)
- Dor real: o que está por trás (o problema de verdade)
- Dor emocional: o que isso faz ele sentir (a vergonha, o medo, a frustração)

**4. O desejo central (em 3 camadas)**
- O que ele diz que quer
- O que ele realmente quer
- O que ele sonha mas não fala

**5. Objeções mais comuns (com tradução)**
As 5 principais objeções que surgem antes de contratar — com o que está por trás de cada uma.

**6. A linguagem do avatar**
Frases reais que este cliente provavelmente usa para descrever seus problemas — para usar diretamente no copy.

**7. Onde e como ele toma decisão de compra**
O caminho que percorre desde descobrir você até contratar — com os pontos de contato críticos.
```

## Exemplo de uso

### Input
- Serviço: Acompanhamento de bem-estar integrativo online

- Não quer: Quem busca solução milagrosa rápida sem se comprometer com processo
- Preço: R$ 1.200-2.000 por programa
- Onde estão: Instagram, Google, indicação
