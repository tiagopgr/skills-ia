# Pesquisa de Mercado e Concorrência para Briefing de Branding

## Objetivo
Estruturar pesquisa de mercado e análise de concorrência visual para projetos de branding — mapeando o que concorrentes comunicam visualmente, identificando padrões saturados no setor e localizando oportunidades de diferenciação — para que o designer entre na fase criativa com direção estratégica clara, não com uma tela em branco e intuição.

## Quando usar
- No início de qualquer projeto de branding antes de criar qualquer conceito
- Quando o cliente não sabe como se diferenciar visualmente no mercado
- Quando quiser fundamentar as escolhas criativas em pesquisa, não só em gosto
- Quando precisar mostrar ao cliente por que a direção escolhida faz sentido estrategicamente
- Quando quiser elevar o nível do briefing entregue ao cliente antes de começar

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do cliente e do mercado em que atua
3. Receba a análise de mercado e concorrência pronta para guiar a criação
4. Use como base para o briefing estratégico e para justificar decisões criativas

## O Prompt
```
Você é um estrategista de marca com especialização em posicionamento visual e análise de identidade de marcas competitivas. Seus princípios: (1) design sem pesquisa de concorrência é design que pode copiar sem saber — análise visual do mercado é parte do processo criativo, não opcional, (2) padrão visual do setor é mapa do que evitar — diferenciação começa por entender o que todo mundo já está fazendo, (3) oportunidade de branding é o espaço vazio no mapa visual da categoria — quem ocupa primeiro, domina a percepção, (4) pesquisa de mercado para branding não é sobre produto — é sobre como as marcas se comunicam, o que prometem visualmente e para quem.

Faça a pesquisa de mercado e concorrência para o seguinte projeto de branding:

**Cliente e negócio:** [nome e o que faz]
**Setor e categoria:** [ex: clínica de saúde / escritório de advocacia / restaurante / startup de tech]
**Público-alvo do cliente:** [quem ele quer atingir — perfil, idade, renda, comportamento]
**Posicionamento desejado:** [como o cliente quer ser percebido — premium / acessível / inovador / tradicional]
**Concorrentes diretos que o cliente mencionou:** [nomes ou perfis de concorrentes conhecidos]
**Referências visuais que o cliente trouxe:** [marcas que ele admira — mesmo fora do setor]
**O que o cliente quer evitar:** [estilos, cores, sensações que ele não quer]
**Geolocalização:** [onde atua — local / nacional / digital]

Entregue:

**1. Panorama visual do setor**
O que a maioria das marcas nesse setor faz visualmente — cores predominantes, estilos tipográficos, símbolos recorrentes, tom de comunicação. O que está saturado.

**2. Análise de concorrentes**
Para cada concorrente (ou perfil de concorrente):
| Marca | Paleta | Estilo | Promessa visual | Público percebido | Ponto fraco |

**3. Mapa de oportunidade**
O que nenhum (ou poucos) concorrentes estão fazendo visualmente — onde há espaço de diferenciação.

**4. Direções estratégicas possíveis**
2-3 caminhos de posicionamento visual com:
- Nome do território (ex: "Sofisticação Clínica" / "Humanidade Acessível")
- O que esse território comunica
- Por que funciona para este cliente
- Referências visuais de outros setores que exploram esse território

**5. O que evitar neste projeto**
Armadilhas visuais do setor — o que faria a marca parecer mais um do mesmo.
```

## Exemplo de uso

### Input
- Cliente: Studio de pilates — Forma Studio
- Setor: Fitness / bem-estar / pilates
- Público: Mulheres 28-45, classe B+/A, urbanas, buscam corpo e saúde com sofisticação
- Posicionamento desejado: Premium, diferente das academias, mais próximo de spa e lifestyle
- Concorrentes: Studios de pilates da região, academias premium
- Referências: Lululemon, Glossier, marcas minimalistas
- Evitar: Rosa clichê, fontes cursivas excessivas, visual de academia

### Output
"ESPAÇO LIVRE: O território 'Movimento como Estilo de Vida'
A maioria dos studios de pilates comunica saúde funcional (corpo que funciona, postura, reabilitação) ou leveza feminina (rosa, cursivo, flores). Nenhum dos players locais ocupa o território de lifestyle urbano sofisticado — onde movimento é identidade, não obrigação.
Esse território permite paleta neutra/quente (areia, terracota, off-white), tipografia editorial com personalidade, tom de comunicação que fala de pertencimento e não de resultado físico — muito mais alinhado com como o público-alvo consome marcas como Lululemon e Alo Yoga."
