# Kit Visual para Campanhas Apollo

## Objetivo
Criar o kit completo de assets visuais para campanhas de prospecção automatizadas no Apollo.io — incluindo templates de email, imagens personalizadas, materiais de apoio para cada step da sequência e peças para LinkedIn.

## Quando usar
- Ao configurar uma nova campanha/sequência no Apollo.io
- Para criar assets visuais que acompanham cada step da automação
- Quando quer elevar o nível visual das campanhas outbound
- Para padronizar os materiais de prospecção do time de vendas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a campanha, ICP e sequência planejada
3. Receba o kit completo com especificações de design para cada peça
4. Execute no Figma/Canva e configure no Apollo

## O Prompt
```
Você é um designer de campanhas outbound que cria kits visuais completos para sequências de prospecção no Apollo.io. Você entende que: (1) uma campanha outbound moderna é multicanal — email + LinkedIn + conteúdo de apoio, (2) cada touchpoint precisa de material visual adequado, (3) consistência visual entre os touchpoints constrói reconhecimento mesmo com leads frios, (4) no outbound, design serve para REDUZIR atrito e AUMENTAR percepção de valor.

Crie o kit visual para a campanha Apollo:

**Nome/objetivo da campanha:** [ex: prospectar heads de marketing em SaaS]
**ICP:** [cargo, setor, tamanho da empresa]
**Oferta:** [o que está vendendo/propondo]
**Steps da sequência no Apollo:** [liste os steps]
**Identidade visual:** [cores da marca, fontes, estilo]
**Materiais de apoio existentes:** [tem case study, one-pager, landing page?]
**Ferramenta de design:** [Figma, Canva]

Entregue: mapa visual da campanha em diagrama; assets para cada step (email, LinkedIn connection, LinkedIn message, materiais de apoio); imagens personalizadas para email (600x300px com variáveis dinâmicas); banner de LinkedIn do vendedor (1584x396px); e checklist de configuração no Apollo.
```

## Exemplo de uso

### Input
Campanha: prospectar CTOs de fintechs para serviço de design de produto
ICP: CTO/CPO de fintechs com 30-100 funcionários
Steps: email 1 → linkedin connection (dia 2) → email 2 (dia 4) → linkedin message (dia 6) → break-up (dia 10)
Identidade: dark theme, azul #3B82F6 + branco

### Output
Mapa visual da campanha com diagrama dia a dia (D0 Email 1 / D2 LinkedIn Connect / D4 Email 2 / D6 LinkedIn Message + One-pager / D10 Break-up), specs de cada asset (template email, nota 300 chars, banner perfil, headline perfil, imagem personalizada 600x300px).
