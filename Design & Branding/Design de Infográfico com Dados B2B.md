# Design de Infográfico com Dados B2B

## Objetivo
Transformar dados brutos, métricas e estatísticas em infográficos visuais de alto impacto — com hierarquia de informação que torna dados complexos instantaneamente compreensíveis — para usar em apresentações, relatórios, redes sociais, propostas e materiais de marketing B2B.

## Quando usar
- Para apresentar resultados e métricas em relatórios ou reuniões
- Para criar posts de dados no LinkedIn que geram autoridade
- Em propostas comerciais para visualizar ROI e projeções
- Para transformar pesquisas e reports em conteúdo visual compartilhável

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Forneça os dados brutos que precisa visualizar
3. Receba o design completo do infográfico com especificações
4. Execute no Figma, Canva ou ferramenta de data visualization

## O Prompt
```
Você é um designer de informação que transforma dados em histórias visuais para contexto B2B. Seus princípios: (1) dados sem contexto são ruído — o infográfico precisa contar UMA história clara, (2) o olho processa visual 60.000x mais rápido que texto — use isso, (3) cada tipo de dado tem um gráfico ideal — bar chart para comparação, line para tendência, donut para proporção, (4) menos é mais — se precisa de legenda para entender, o gráfico falhou, (5) cores informam — verde = bom, vermelho = ruim, accent = destaque.

Crie o infográfico para:

**Dados a visualizar:** [cole os dados brutos, números, métricas]
**Mensagem principal:** [qual conclusão os dados devem comunicar]
**Público:** [quem vai ver — CEO, investidor, lead, time interno]
**Contexto de uso:** [apresentação, relatório, LinkedIn, proposta, email]
**Formato:** [vertical story/poster, horizontal slide/banner, quadrado social]
**Dimensões:** [1080x1920, 1920x1080, 1080x1080, A4, custom]
**Identidade visual:** [cores, fontes — ou sugerir]
**Estilo desejado:** [clean/minimal, editorial, data-heavy, infographic classic]
**Ferramenta:** [Figma, Canva, PowerPoint, Illustrator]

Entregue: estrutura narrativa (headline, contexto, fluxo visual, takeaway); wireframe do layout com posição de cada elemento; tipo de gráfico recomendado por dado (com justificativa); design detalhado de cada componente (dimensões, cores hex, tipografia, espaçamento); variações de formato (slide 16:9, LinkedIn 1080x1350, story 1080x1920, A4 landscape, card email 600x400); e checklist de qualidade (mensagem em 5s, sem legenda, hierarquia por tamanho, sem gráficos 3D, máx 7 dados).
```

## Exemplo de uso

### Input
Dados: Resultados Q1 2025 — Leads +171% (312→847), CAC -47% (R$89→47), CVR +133% (1.8→4.2%), Revenue +169% (R$87k→234k)
Mensagem: "A nova estratégia de outbound triplicou resultados em 1 trimestre"
Público: board/investidores
Formato: slide 16:9 para deck
Estilo: dark, minimal, tech

### Output
4 cards KPI em linha: [↑171% Leads 312→847] [↓47% CAC R$89→47] [↑133% CVR 1.8→4.2%] [↑169% Revenue R$87k→234k]. Todos em verde #22C55E. Bar chart horizontal abaixo: Q4 (cinza #333) vs Q1 (azul #3B82F6). Fundo #0A0A0B, cards #161618, borda #222. Fonte: Apollo.io + HubSpot CRM, Jan-Mar 2025.
