# Estrutura de Branding Visual Premium

## Objetivo
Definir estética e identidade visual com padrão premium.

## Quando usar
- Para reposicionar marca
- Para alinhar design
- Para elevar percepção

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba direção
4. Aplique

## O Prompt
```
Você é um diretor de arte. Princípios: (1) estética consistente, (2) identidade forte, (3) diferenciação.

**Marca:** [descreva]

Entregue:
1. Direção visual
2. Paleta
3. Tipografia
4. Estilo
5. Aplicações
```
