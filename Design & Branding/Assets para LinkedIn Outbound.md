# Assets para LinkedIn Outbound

## Objetivo
Criar o kit completo de assets visuais para prospecção outbound no LinkedIn — banner de perfil, carrosséis de autoridade, imagens para mensagens e posts estratégicos — para transformar o perfil do vendedor em uma máquina de gerar conversas com decisores B2B.

## Quando usar
- Ao estruturar a presença de LinkedIn do time de vendas para outbound
- Para criar conteúdo visual que apoia a prospecção ativa
- Quando o perfil do vendedor não transmite autoridade profissional
- Para acompanhar campanhas Apollo/outbound com presença social

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o contexto de vendas, ICP e identidade visual
3. Receba todas as especificações de design
4. Execute no Figma/Canva e implemente no LinkedIn

## O Prompt
```
Você é um designer de social selling que cria assets visuais para prospecção B2B no LinkedIn. Você sabe que: (1) quando um lead recebe seu cold email, a primeira coisa que faz é olhar seu LinkedIn — o perfil É a landing page do vendedor, (2) conteúdo visual no LinkedIn tem 2x mais engajamento que texto puro, (3) carrosséis são o formato com maior alcance orgânico, (4) o banner do perfil é o billboard mais subvalorizado — 1584x396px de espaço gratuito.

Crie os assets de LinkedIn outbound para:

**Quem vai usar:** [vendedor, SDR, founder, consultor]
**O que vende:** [produto/serviço]
**ICP:** [cargo, setor, tamanho da empresa dos leads]
**Tom de posicionamento:** [especialista técnico, consultor estratégico, líder visionário]
**Identidade visual:** [cores, fontes, estilo da empresa]
**Ferramenta de design:** [Figma, Canva]
**Tem fotos profissionais?** [sim/não]

Entregue: banner do perfil 1584x396px (proposta de valor + prova social, safe zone mobile); guidelines de foto de perfil; template carrossel de autoridade 1080x1350px 8-10 slides (capa, conteúdo 1 ideia/slide, resumo, CTA); imagens para posts e DMs; kit de conteúdo semanal (calendário 80% valor / 20% oferta); e guia de consistência visual.
```

## Exemplo de uso

### Input
Quem: SDR de agência de design B2B
Vende: serviço de design de materiais de vendas
ICP: Head de Vendas de SaaS com 50-200 funcionários
Tom: consultor estratégico, moderno
Cores: dark #0F0F11, accent laranja #F97316, branco
Ferramenta: Canva

### Output
Banner 1584x396px: fundo #0F0F11. Esquerda: "Ajudo times de vendas B2B a fechar 40% mais com materiais que impressionam decisores" 28pt bold #FAFAFA + CTA laranja "→ Agende diagnóstico gratuito". Direita: logos grayscale 40px SAP / RD / VTEX / Hotmart. Safe zone: conteúdo nos 60% centrais.
