# Roteiro de Case Study de Projeto de Branding

## Objetivo
Estruturar case studies de projetos de branding que vão além de mostrar o resultado visual — documentando o problema, o processo estratégico, as decisões criativas e o impacto — para que o designer construa portfólio que vende próximos projetos, não só portfólio bonito que ninguém sabe como foi feito.

## Quando usar
- Após finalizar um projeto de branding que ficou muito bom
- Quando quiser criar conteúdo de portfólio para Instagram, LinkedIn ou site
- Quando quiser mostrar processo além de resultado para atrair cliente que valoriza estratégia
- Quando precisar de case para apresentar em uma proposta comercial
- Quando quiser criar um template de documentação de case para usar em todos os projetos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do projeto que quer documentar
3. Receba o case study estruturado pronto para adaptar para diferentes formatos
4. Use no Instagram (carrossel), LinkedIn (post) ou site (página de portfólio)

## O Prompt
```
Você é um especialista em criação de case studies criativos para designers e profissionais de branding. Seus princípios: (1) case study que só mostra resultado final é portfólio — case study que mostra processo é argumento de venda, (2) a história do problema é tão importante quanto a solução — comece pelo contexto, não pelo visual, (3) decisões criativas justificadas transformam estética em estratégia: "escolhemos verde porque..." é o que diferencia branding de decoração, (4) o melhor case é o que faz o próximo cliente pensar "é exatamente isso que eu preciso".

Crie o roteiro do case study com as seguintes informações:

**Cliente e projeto:** [nome da empresa e tipo de projeto]
**Problema original:** [o que o cliente tinha de errado ou o que queria resolver]
**Contexto de mercado:** [onde a empresa atua e quem são os concorrentes]
**Conceito criativo central:** [a ideia por trás da solução]
**Principais decisões de design:** [escolhas de cor, símbolo, tipografia — e o raciocínio]
**Processo executado:** [etapas do projeto — briefing, pesquisa, conceito, refinamento]
**Resultado entregue:** [o que foi criado e entregue]
**Impacto (se souber):** [o que mudou para o cliente após o projeto]
**Aprendizado do projeto:** [o que foi mais desafiador e o que você aprendeu]

Entregue:

**1. Case study completo (versão longa — para site e portfólio)**
Com as seções:
- O cliente e o contexto
- O problema a resolver
- Nossa abordagem
- O processo (passo a passo com raciocínio)
- As decisões criativas (por que cada escolha)
- O resultado
- O que mudou

**2. Versão carrossel para Instagram**
Slide a slide — título de cada slide e copy resumido.

**3. Post para LinkedIn**
Texto completo com abertura de gancho, processo resumido e resultado — pronto para publicar.

**4. Caption do Instagram**
Legenda completa com hashtags para publicar junto com o carrossel.
```

## Exemplo de uso

### Input
- Cliente: Nutrir Clínica — clínica de nutrição
- Problema: Identidade visual amadora não transmitia o posicionamento premium que praticavam
- Mercado: Saúde e nutrição, muito saturado de verde genérico e imagens de salada
- Conceito: Verde-sálvia sofisticado + identidade que remete a bem-estar urbano, não a clínica tradicional
- Decisões: Escolha do verde-sálvia em vez do verde primário / tipografia serif elegante / símbolo de folha estilizada em N
- Processo: Briefing → pesquisa visual do setor → 1 direção → refinamento → entrega com manual
- Entregou: Logo completa, manual, cartão, mockups de clínica, templates de stories
- Impacto: Cliente relatou que novos pacientes comentaram sobre a seriedade da clínica na primeira consulta

### Output
"Quando a Nutrir Clínica chegou para mim, tinha um problema claro: atendia pacientes de alto padrão mas a identidade visual parecia de clínica de bairro.

O logotipo era genérico. As cores eram o verde-limão de academia. Nenhum elemento visual comunicava o posicionamento premium que eles praticavam.

A identidade estava sabotando o serviço antes do paciente sentar na cadeira.

O trabalho aqui não foi criar uma logo bonita.
Foi criar uma marca que confirmasse, visualmente, o que eles já entregavam.

Aqui está o que fizemos — e por que cada decisão foi estratégica, não estética..."
