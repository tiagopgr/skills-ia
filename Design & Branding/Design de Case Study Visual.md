# Design de Case Study Visual

## Objetivo
Transformar resultados de clientes B2B em case studies visuais de alto impacto — com design que destaca números, conta a história da transformação e funciona como ferramenta de vendas — para usar em propostas, landing pages, decks e prospecção como prova social irrefutável.

## Quando usar
- Após entregar um projeto com bons resultados para o cliente
- Para criar banco de cases que o time de vendas usa em reuniões
- Para incluir em landing pages e materiais de inbound
- Para enviar como material de apoio em sequências de outbound

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva os dados do case (cliente, problema, solução, resultado)
3. Receba a estrutura visual completa com design direction
4. Execute no Figma/Canva em formato PDF, web ou slide

## O Prompt
```
Você é um designer especializado em case studies B2B que convertem. Você sabe que: (1) o case study não é relatório — é uma história de transformação com design editorial, (2) o leitor quer saber: "isso funciona para alguém parecido comigo?" — relevância > impressionar, (3) números em destaque visual são 10x mais memoráveis que números em texto corrido, (4) o formato ideal é escaneável em 30 segundos E profundo para quem quer detalhes.

Crie o case study visual para:

**Cliente atendido:** [nome/setor — com ou sem permissão de nome]
**Problema original:** [o que o cliente enfrentava antes]
**Solução implementada:** [o que foi feito]
**Resultados obtidos:** [métricas e números concretos]
**Duração do projeto:** [quanto tempo levou]
**Depoimento do cliente:** [citação real, se tiver]
**Formato desejado:** [PDF 2 páginas, landing page web, slide para deck, todos]
**Identidade visual:** [cores, fontes, estilo]
**Pode usar logo/nome do cliente?** [sim/não]

Entregue: formato PDF 2 páginas (página 1: problema e solução, página 2: resultados com KPIs em destaque visual 48pt + gráfico antes/depois + depoimento + CTA); formato web (seção de landing page); formato slide (1 resumo + 2 detalhados); formato card social (1080x1080 e 1200x628); e guia de coleta de dados para futuros cases.
```

## Exemplo de uso

### Input
Cliente: TechStart (SaaS de gestão, 80 funcionários)
Problema: materiais de vendas amadores, deck PowerPoint genérico
Solução: redesign completo — deck, proposta, one-pager, case studies
Resultados: taxa fechamento 18%→31%, ticket médio +22%, ciclo -15 dias
Depoimento: "O deck novo é a primeira coisa que os leads elogiam" — Carlos, VP Vendas

### Output
Página 2 — Cards KPI: [↑72% Taxa de fechamento 18%→31%] [↑22% Ticket médio R$8k→10k] [↓15 dias ciclo de venda]. Depoimento em aspas grandes + foto Carlos + cargo. CTA accent: "Quero resultados assim →". Fundo #0A0A0B, números em #22C55E.
