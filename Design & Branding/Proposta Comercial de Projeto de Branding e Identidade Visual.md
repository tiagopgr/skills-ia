# Proposta Comercial de Projeto de Branding e Identidade Visual

## Objetivo
Criar propostas comerciais de branding e identidade visual que vão além do escopo e do preço — demonstrando o problema estratégico que o branding resolve, o valor percebido da entrega e o impacto no negócio do cliente — para que o designer feche mais projetos, com clientes que entendem o que estão comprando e respeitam o processo criativo.

## Quando usar
- Após reunião de briefing com prospect para formalizar o projeto
- Quando quiser criar um formato de proposta que vende pela estratégia, não pelo preço
- Quando um prospect questionou o valor e você quer reconstruir a percepção
- Quando quiser padronizar o processo de proposta para agilizar o começo de projeto
- Quando for propor um projeto maior (branding completo vs logo avulsa)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com as informações do prospect e do projeto que vai propor
3. Receba a proposta completa pronta para personalizar e enviar
4. Adicione sua identidade visual e envie com confiança

## O Prompt
```
Você é um especialista em vendas de serviços criativos de alto valor — branding, identidade visual e estratégia de marca. Seus princípios: (1) proposta de branding que começa pelo preço já perdeu — valor percebido vem da narrativa estratégica, não do portfólio, (2) o cliente não compra logo — compra a percepção que a marca vai criar nos clientes dele: posicione pela consequência, não pela entrega, (3) escopo claro na proposta protege o designer de pedidos infinitos de revisão — especificidade é proteção criativa, (4) proposta bem construída filtra cliente ruim — quem entende o valor fecha, quem não entende vai pechinchar de qualquer jeito.

Crie a proposta comercial com as seguintes informações:

**Prospect e empresa:** [nome e tipo de negócio]
**Problema ou objetivo que trouxe para o projeto:** [o que ele quer resolver com o branding]
**Projeto proposto:** [ex: identidade visual completa / rebrand / logo + manual / branding para lançamento]
**Entregáveis:** [liste tudo o que vai entregar — logo, variações, paleta, tipografia, manual, aplicações]
**Processo de trabalho:** [etapas — briefing, pesquisa, conceito, refinamento, entrega]
**Prazo total:** [ex: 4 semanas / 6 semanas]
**Número de rodadas de revisão:** [quantas estão incluídas]
**Investimento:** [valor total]
**Forma de pagamento:** [ex: 50% início + 50% entrega / 3 parcelas]
**Seus diferenciais para este cliente:** [o que te qualifica especificamente para o projeto dele]

Entregue:

**1. Proposta comercial completa**
Com as seções:
- Capa (nome do projeto, cliente, data)
- Entendimento do cenário (você entendeu o problema deles)
- Por que o branding importa agora (o custo de não ter identidade forte)
- A solução proposta (o que você vai fazer)
- Processo e etapas (como funciona)
- Entregáveis detalhados (o que recebem — com lista)
- Cronograma (semana a semana)
- Investimento e condições
- Próximos passos para começar

**2. Seção de valor — "o que muda depois"**
Parágrafo de impacto: o que muda no negócio do cliente após ter uma identidade visual profissional e consistente.

**3. Versão resumida para WhatsApp**
Mensagem de até 250 palavras para enviar informalmente antes ou junto com a proposta.

**4. Resposta para "achei caro"**
Script específico para reposicionar o valor sem dar desconto.
```

## Exemplo de uso

### Input
- Prospect: Consultório de psicologia — Dr. Rafael Mendes
- Problema: Atende bem, mas a identidade visual atual parece amadora — cartão feito no Canva, sem consistência, perde credibilidade com clientes mais exigentes
- Projeto: Identidade visual completa para consultório
- Entregáveis: Logo (primária + variações), paleta oficial, tipografia, cartão de visita, papel timbrado, template de stories, mini-manual de marca
- Processo: Briefing → Pesquisa → 2 direções → Refinamento → Entrega
- Prazo: 5 semanas
- Revisões: 2 rodadas inclusas
- Valor: R$3.800
- Pagamento: 50% início + 50% entrega
- Diferencial: Experiência com marcas de saúde, entendo o equilíbrio entre acolhimento e profissionalismo

### Output
"Uma identidade visual profissional não é estética — é argumento. Quando um potencial paciente pesquisa o Dr. Rafael antes de marcar a primeira consulta (e todos pesquisam), o que ele encontra precisa confirmar a mesma qualidade que ele vai encontrar na sala de espera. Identidade inconsistente cria dúvida antes mesmo do contato. Identidade coerente e sofisticada já inicia o processo de confiança — e confiança, no contexto da psicologia, é o bem mais precioso que existe."
