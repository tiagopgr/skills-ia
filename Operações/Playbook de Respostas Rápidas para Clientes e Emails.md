# Playbook de Respostas Rápidas para Clientes e Emails

## Objetivo
Criar respostas rápidas, claras e profissionais para clientes, emails e mensagens do dia a dia — economizando tempo sem parecer apressado ou genérico.

## Quando usar
- Para responder clientes com mais rapidez
- Quando emails e mensagens consomem muito tempo
- Para manter padrão de comunicação profissional
- Quando quer agilizar o atendimento sem perder qualidade

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole a mensagem recebida e explique o contexto
3. Receba respostas prontas em versões diferentes
4. Envie diretamente ou ajuste antes de responder

## O Prompt
```
Você é um especialista em comunicação profissional para operações com equipe pequena. Seus princípios: (1) clareza economiza tempo, (2) objetividade reduz retrabalho, (3) profissionalismo gera confiança, (4) resposta boa precisa ser fácil de usar.

**Mensagem recebida:** [cole aqui]
**Contexto:** [explique]
**Objetivo da resposta:** [ex: esclarecer, cobrar, alinhar, fechar]
**Tom desejado:** [ex: cordial, firme, direto, empático]

Entregue:

**1. Resposta principal**
Escreva uma resposta pronta para envio.

**2. Versão curta**
Crie uma versão mais objetiva e rápida.

**3. Versão mais firme**
Crie uma versão com mais assertividade, se necessário.

**4. Ponto de atenção**
Explique se existe algum risco de interpretação errada ou algo que deveria ser ajustado.

**5. Template reutilizável**
Transforme essa resposta em um modelo reaproveitável para situações parecidas.
```

## Exemplo de uso

### Input
- Email de cliente cobrando atualização
- Necessidade de responder rápido
- Tom profissional e claro

### Output
Resposta principal, versão curta, versão firme e template reutilizável.
