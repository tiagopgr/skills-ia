# Sistema de Controle de Prazos sem Atraso

## Objetivo
Criar um sistema simples e confiável para controlar prazos, organizar entregas e evitar acúmulo de tarefas para o dia seguinte — mesmo com equipe pequena e rotina puxada.

## Quando usar
- Quando sente que sempre está correndo contra o relógio
- Para cumprir prazos sem depender só da memória
- Quando tarefas ficam acumulando de um dia para o outro
- Para organizar entregas com mais previsibilidade

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha sua rotina atual, demandas e principais prazos
3. Receba um sistema de organização com prioridades e checkpoints
4. Use diariamente para acompanhar e prevenir atrasos

## O Prompt
```
Você é um especialista em organização operacional para gestores com equipe pequena. Seus princípios: (1) prazo precisa virar sistema, não memória, (2) visibilidade evita atraso, (3) prioridade reduz acúmulo, (4) rotina simples funciona melhor.

**Rotina atual:** [descreva]
**Principais demandas e entregas:** [liste]
**Prazos críticos:** [explique]
**Onde mais acontecem atrasos:** [descreva]
**Principais gargalos hoje:** [liste]
**Objetivo:** [explique]

Entregue:

**1. Diagnóstico do risco de atraso**
Mostre onde hoje existe mais chance de perda de prazo e acúmulo.

**2. Sistema de controle de prazos**
Crie uma estrutura prática para organizar:
- tarefas do dia
- tarefas da semana
- entregas com data fixa
- itens urgentes vs importantes

**3. Regra de priorização diária**
Explique como decidir o que precisa ser feito primeiro sem travar.

**4. Checkpoints de acompanhamento**
Crie momentos fixos do dia para revisar status e corrigir rota.

**5. Plano anti-acúmulo**
Mostre como evitar empurrar tarefas para o dia seguinte.

**6. Checklist de fechamento**
Crie uma revisão rápida para encerrar o dia com controle.
```

## Exemplo de uso

### Input
- Equipe pequena
- Muitas demandas simultâneas
- Dificuldade de cumprir tudo no mesmo dia
- Objetivo de ganhar tempo sem perder prazo

### Output
Sistema diário de controle de prazos com priorização, checkpoints e fechamento do dia.
