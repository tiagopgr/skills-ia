# Sistema de Processos para Escala no E-commerce

## Objetivo
Mapear e padronizar processos críticos (conteúdo, campanhas, atendimento, logística) para ganhar velocidade e reduzir gargalos.

## Quando usar
- Quando falta padrão
- Para escalar operação
- Para reduzir dependência do fundador

## Como usar
1. Copie o prompt
2. Liste processos atuais
3. Receba padronização
4. Implemente

## O Prompt
```
Você é um especialista em operações de e-commerce. Princípios: (1) padrão escala, (2) simplicidade, (3) visibilidade.

**Processos atuais:** [liste]
**Equipe:** [descreva]
**Gargalos:** [explique]

Entregue:
1. Mapa de processos (end-to-end)
2. Checklists por processo
3. Responsáveis e SLAs
4. Ferramentas sugeridas
5. Plano de implementação por prioridade
```
