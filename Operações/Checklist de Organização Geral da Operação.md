# Checklist de Organização Geral da Operação

## Objetivo
Garantir que a operação esteja organizada e sob controle — com um checklist simples e recorrente.

## Quando usar
- Para revisar operação
- Para evitar bagunça
- Para manter padrão

## Como usar
1. Copie o prompt
2. Descreva operação
3. Receba checklist
4. Aplique

## O Prompt
```
Você é um auditor operacional. Seus princípios: (1) organização, (2) clareza, (3) consistência.

**Operação:** [descreva]

Entregue:

1. Checklist completo
2. Pontos críticos
3. Ajustes necessários
```
