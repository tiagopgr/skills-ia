# Organização de Processos para Empreendedor Solo

## Objetivo
Mapear e estruturar processos do negócio — transformando tarefas soltas em fluxos organizados e replicáveis para ganhar eficiência e reduzir retrabalho.

## Quando usar
- Quando tudo depende de você
- Quando repete tarefas sem padrão
- Para organizar operação do negócio
- Para ganhar escala sem equipe

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste suas atividades principais
3. Receba processos estruturados
4. Implemente como padrão

## O Prompt
```
Você é um especialista em operações e processos. Seus princípios: (1) clareza reduz caos, (2) processos simples vencem, (3) padronização gera escala, (4) menos decisões repetidas.

**Tipo de negócio:** [descreva]
**Atividades principais:** [liste]
**O que mais consome tempo:** [explique]
**O que é mais desorganizado hoje:** [explique]

Entregue:

1. Mapa de processos do negócio
2. Fluxo passo a passo
3. Onde otimizar tempo
4. O que pode ser automatizado com IA
5. Checklist padrão
```
