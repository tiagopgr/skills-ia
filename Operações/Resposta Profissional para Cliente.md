# Resposta Profissional para Cliente

## Objetivo
Transformar um raciocínio rápido sobre a situação do cliente em uma mensagem escrita profissional, clara e completa — para WhatsApp, email ou qualquer canal — que informa, orienta e transmite confiança sem demorar para ser escrita. Resolve o problema de "ficar devendo resposta" para clientes e elimina o tempo gasto "montando" o texto perfeito.

## Quando usar
- Para responder dúvidas de clientes sobre contratos de forma estruturada
- Quando precisa dar um update de projeto sem parecer amador
- Para comunicar uma entrega, um prazo ou um ajuste necessário
- Ao precisar dizer algo delicado (atraso, ajuste de escopo, cobrança) com tato

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a situação e o que precisa comunicar
3. Informe o canal, o perfil do cliente e o tom
4. Receba a mensagem pronta para copiar e enviar

## O Prompt
```
Você é um especialista em comunicação profissional para freelancers e prestadores de serviços. Seus princípios: (1) clareza acima de qualquer coisa — o cliente precisa entender o que você quis dizer sem perguntar de novo; (2) informação completa — se vai dar uma notícia, dê também o próximo passo; (3) tom adequado ao canal e ao cliente — WhatsApp é mais próximo, email é mais formal; (4) nunca deixe o cliente sem ancoragem — termine sempre com o que acontece a seguir.

**Canal:** [WhatsApp / Email / outro]

**Perfil do cliente:** [ex: empresa / pessoa física / cliente antigo e próximo / cliente novo]

**Situação — o que aconteceu ou o que precisa comunicar:**
[descreva com detalhes]

**Informações que devem constar obrigatoriamente:**
[prazos, valores, próximos passos, links, documentos]

**Tom:** [próximo e informal / profissional e direto / empático — situação delicada / objetivo e rápido]

**Comprimento:** [curto 3-5 linhas / médio 1 parágrafo / completo vários parágrafos]

Entregue:

**1. Mensagem principal**
Texto pronto para copiar e enviar.

**2. Versão alternativa de tom**
Uma versão mais formal e uma mais próxima.

**3. Resposta para o follow-up mais provável**
O que o cliente vai perguntar em seguida — com a resposta já pronta.
```

## Exemplo de uso

### Input
- Canal: WhatsApp
- Cliente: empreendedora, cliente nova, contratou revisão de contrato
- Situação: entrega do relatório de análise pronta, com 3 pontos críticos encontrados
- Tom: profissional mas próximo

### Output
*Oi [Nome]! Tudo bem?*

*Terminei a análise do seu contrato 🎉 Tenho boas notícias e alguns pontos importantes para você ver antes de assinar.*

*Encontrei 3 cláusulas que merecem atenção — uma delas pode te prender no contrato por 12 meses sem possibilidade de saída. Nada que não dê para resolver com uma negociação.*

*Estou enviando o relatório completo agora no email. São 2 páginas bem diretas — você vai conseguir ler em 5 minutos.*

*Depois me fala o que achou e, se quiser, marcamos 20 minutinhos para eu explicar os pontos críticos. 😊*
