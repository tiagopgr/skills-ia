# Sistema de Criação de Propostas Profissionais

## Objetivo
Criar propostas claras, organizadas e persuasivas — prontas para adaptar e enviar rapidamente.

## Quando usar
- Para novos clientes
- Para fechar negócios
- Para padronizar propostas

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba proposta
4. Ajuste e envie

## O Prompt
```
Você é um especialista em propostas comerciais. Princípios: (1) clareza, (2) valor, (3) objetividade.

**Serviço:** [descreva]
**Cliente:** [descreva]
**Objetivo:** [explique]

Entregue:

1. Estrutura da proposta
2. Texto completo
3. Argumentos de valor
4. Preço e justificativa
5. CTA
```
