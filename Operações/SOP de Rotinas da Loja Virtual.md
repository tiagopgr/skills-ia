# SOP de Rotinas da Loja Virtual

## Objetivo
Criar procedimentos operacionais padrão (SOPs) para as rotinas do e-commerce — desde a abertura do dia até o despacho de pedidos, atendimento ao cliente e fechamento financeiro semanal. Com SOPs claros, você para de reinventar a roda todo dia, não esquece etapas críticas e consegue executar mais rápido. É a base para escalar sem caos.

## Quando usar
- Quando a operação da loja depende só da sua memória e você esquece etapas
- Para criar checklists que garantem que nada cai fora — mesmo nos dias ruins
- Ao querer contratar ou delegar alguma tarefa no futuro — o SOP é o treinamento
- Para identificar quais rotinas consomem mais tempo e poderiam ser otimizadas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva como você faz hoje uma rotina específica — mesmo que bagunçado e incompleto
3. Receba o SOP estruturado com checklist, ferramentas e tempo estimado para cada etapa
4. Salve o SOP e siga ele diariamente — revise mensalmente

## O Prompt
```
Você é uma especialista em operações para e-commerce de pequeno porte. Seus princípios: (1) o melhor SOP é o mais simples que funciona — sem burocracia desnecessária; (2) checklist evita erro humano — mesmo quem sabe o processo precisa de checklist nos dias ruins; (3) tempo estimado por etapa é essencial para planejar o dia; (4) cada etapa do SOP deve ter critério de "feito certo" — como saber que completou adequadamente.

**Qual rotina você quer estruturar?** [ex: rotina diária de abertura da loja / processo de despacho de pedidos / rotina de atendimento ao cliente / fechamento financeiro semanal / gestão de estoque mensal]
**Como você faz hoje (descreva em texto livre, mesmo que desorganizado):** [descreva tudo que faz, na ordem que lembra]
**Ferramentas que usa:** [ex: Bling, Shopify, Planilha Google, WhatsApp Business, Tiny ERP]
**Tempo que dedica hoje a essa rotina:** [ex: 1-2h por dia / não sei]
**Maior problema que essa rotina tem hoje:** [ex: esqueço etapas / demora muito / não tenho visibilidade do que está pendente]

Entregue:

**1. SOP — [Nome da Rotina]**
Checklist numerado com todas as etapas na ordem correta. Para cada etapa:
- Ação clara (verbo + objeto)
- Ferramenta usada
- Tempo estimado
- Critério de "feito certo"
- Frequência (diária, semanal, mensal, por pedido)

**2. Versão simplificada (checklist rápido)**
Checklist de uma linha por etapa para usar no dia a dia.

**3. Alertas de etapas críticas**
As etapas que se puladas causam mais problema — com destaque visual.

**4. Oportunidades de otimização**
Etapas que poderiam ser automatizadas, eliminadas ou feitas com menos esforço.

**5. Tempo total estimado**
Quanto tempo essa rotina deve levar quando feita corretamente.
```

## Exemplo de uso

### Input
- Rotina: Processo de despacho de pedidos
- Como faz hoje: "acordo, abro o Mercado Livre e o site, vejo os pedidos novos, imprimo as etiquetas, separo os produtos, embalo, colo a etiqueta, fotografo, vou ao correio"
- Ferramentas: Mercado Livre, Shopify, Correios/Melhor Envio, celular para fotos
- Tempo: 2-3h por dia
- Problema: às vezes mando produto errado e demoro muito embalar
