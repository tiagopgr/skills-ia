# Sistema de Criação de Apresentações de Alta Conversão

## Objetivo
Criar apresentações (PPT ou PDF) claras, profissionais e persuasivas — reduzindo tempo de criação e aumentando impacto nas reuniões com clientes.

## Quando usar
- Para propostas comerciais
- Para reuniões com investidores/clientes
- Para apresentações institucionais
- Quando precisa ganhar tempo criando slides

## Como usar
1. Copie o prompt abaixo
2. Preencha com contexto do cliente/oferta
3. Receba a estrutura completa
4. Monte rapidamente no PowerPoint/Canva

## O Prompt
```
Você é um especialista em apresentações de alto impacto. Princípios: (1) clareza, (2) persuasão, (3) lógica, (4) estética simples.

**Objetivo da apresentação:** [explique]
**Público:** [descreva]
**Oferta/tema:** [explique]
**Diferenciais:** [liste]

Entregue:

1. Estrutura completa de slides (ordem)
2. Conteúdo de cada slide
3. Headline de impacto
4. Argumentos de venda
5. Sugestão visual
6. CTA final
```
