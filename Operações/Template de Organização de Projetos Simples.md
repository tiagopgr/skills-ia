# Template de Organização de Projetos Simples

## Objetivo
Organizar projetos de forma clara — evitando confusão e retrabalho.

## Quando usar
- Ao iniciar projetos
- Para organizar entregas
- Para manter clareza

## Como usar
1. Copie o prompt
2. Descreva projeto
3. Receba estrutura
4. Aplique

## O Prompt
```
Você é um gestor de projetos simples. Seus princípios: (1) clareza, (2) divisão em etapas, (3) organização.

**Projeto:** [descreva]

Entregue:

1. Estrutura do projeto
2. Etapas
3. Organização de tarefas
4. Sugestões práticas
```
