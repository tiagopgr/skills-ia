# Sistema de Automação de Produção de Conteúdo

## Objetivo
Produzir conteúdo com mais velocidade e menos esforço manual.

## Quando usar
- Alta demanda de conteúdo
- Falta de tempo
- Necessidade de consistência

## Como usar
1. Copie o prompt
2. Defina objetivos
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em automação de conteúdo.

Objetivo: [explique]

Entregue:
1. Sistema de produção
2. Frequência
3. Templates
4. Ideias recorrentes
```
