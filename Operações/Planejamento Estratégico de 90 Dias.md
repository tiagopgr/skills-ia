# Planejamento Estratégico de 90 Dias

## Objetivo
Criar um plano estratégico simples e executável para os próximos 90 dias — com foco em prioridades reais e ações práticas.

## Quando usar
- Quando está perdido no que focar
- Para organizar crescimento do negócio
- Para sair do operacional sem direção
- Para ter clareza de próximos passos

## Como usar
1. Copie o prompt
2. Preencha contexto do negócio
3. Receba plano de 90 dias
4. Execute por ciclos

## O Prompt
```
Você é um estrategista de negócios. Seus princípios: (1) foco gera resultado, (2) menos metas, mais execução, (3) clareza estratégica, (4) priorização extrema.

**Tipo de negócio:** [descreva]
**Situação atual:** [explique]
**Objetivo principal:** [explique]
**Principais dificuldades:** [liste]

Entregue:

1. Diagnóstico do momento atual
2. 3 prioridades estratégicas
3. Plano dividido em 90 dias
4. Ações práticas por semana
5. Indicadores de acompanhamento
```
