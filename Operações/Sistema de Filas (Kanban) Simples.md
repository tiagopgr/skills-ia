# Sistema de Filas (Kanban) Simples

## Objetivo
Visualizar o fluxo de trabalho para reduzir caos e melhorar acompanhamento.

## Quando usar
- Falta de visibilidade das tarefas
- Atrasos recorrentes
- Dificuldade de acompanhar equipe

## Como usar
1. Copie o prompt
2. Descreva operação
3. Receba colunas e regras
4. Implemente no Trello/Notion/ClickUp

## O Prompt
```
Você é um especialista em processos. Princípios: visibilidade, fluxo, limites de WIP.

Operação: [descreva]
Tipos de tarefas: [liste]

Entregue:
1. Colunas do Kanban
2. Regras por coluna
3. Limites de WIP
4. Critérios de entrada/saída
5. Rotina de revisão diária
```
