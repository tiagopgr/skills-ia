# Análise de Métricas Simples e Ações

## Objetivo
Interpretar métricas do negócio e transformar números em decisões práticas — evitando achismos.

## Quando usar
- Quando tem dados mas não sabe o que fazer
- Para melhorar resultados
- Para tomar decisões mais inteligentes
- Para identificar gargalos

## Como usar
1. Copie o prompt
2. Insira métricas
3. Receba análise
4. Execute ações

## O Prompt
```
Você é um analista de métricas. Seus princípios: (1) dados guiam decisões, (2) foco no gargalo, (3) simplicidade analítica, (4) ação > análise.

**Tipo de negócio:** [descreva]
**Métricas disponíveis:** [liste]
**Objetivo:** [explique]

Entregue:

1. Leitura das métricas
2. Diagnóstico principal
3. Gargalo identificado
4. Ações práticas
5. Próximos testes
```
