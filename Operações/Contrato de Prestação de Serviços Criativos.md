# Contrato de Prestação de Serviços Criativos

## Objetivo
Criar contratos de prestação de serviços para projetos de design e branding — com escopo definido, política de revisão, propriedade intelectual, condições de pagamento e rescisão — para que o designer proteja seu trabalho, evite revisão infinita, receba corretamente e tenha respaldo legal quando o cliente tentar extrapolar o que foi combinado.

## Quando usar
- Para todo novo projeto de branding antes de começar qualquer trabalho
- Quando quiser criar um contrato padrão para reutilizar com adaptações
- Quando um projeto anterior teve problema e quer ter proteção agora
- Quando o cliente questionar algo que você considera óbvio mas não estava escrito
- Quando quiser profissionalizar a operação e mostrar seriedade para clientes maiores

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do projeto e do cliente
3. Receba o contrato completo pronto para assinar digitalmente ou por e-mail
4. Revise com seu advogado se necessário para projetos de maior valor

## O Prompt
```
Você é um especialista em contratos para profissionais criativos — designers, brandistas e consultores de marca. Seus princípios: (1) contrato de criativo sem definição de revisão é convite para trabalho infinito — quantas revisões, o que conta como revisão e o que conta como novo pedido precisa estar escrito, (2) propriedade intelectual de projeto criativo deve ser transferida somente após pagamento integral — isso protege o designer sem ser agressivo, (3) linguagem do contrato deve ser acessível ao cliente — juridiquês excessivo gera insegurança e questionamento, (4) contrato bem feito é ferramenta de relação, não de guerra — ele facilita o projeto porque todos sabem o que está combinado.

Crie o contrato de prestação de serviços para o seguinte projeto:

**Prestador (você):** [seu nome, CPF ou CNPJ, endereço]
**Cliente:** [nome ou empresa, CPF ou CNPJ, endereço]
**Projeto:** [o que será feito — ex: identidade visual completa]
**Entregáveis:** [lista do que será entregue]
**Prazo total:** [data de início e data de entrega]
**Valor total:** [R$ do projeto]
**Forma de pagamento:** [ex: 50% na assinatura + 50% na entrega]
**Rodadas de revisão incluídas:** [quantas e o que conta como revisão]
**Foro:** [cidade/estado]

Entregue:

**1. Contrato completo**
Com cláusulas de:
- Identificação das partes
- Objeto do contrato (escopo detalhado)
- Prazo e cronograma
- Remuneração e forma de pagamento
- Política de revisão (o que inclui e o que é cobrado à parte)
- Propriedade intelectual (quando transfere)
- Confidencialidade
- Rescisão (condições e consequências para ambas as partes)
- Disposições gerais
- Foro e assinatura

**2. Adendo de escopo extra**
Documento simples para usar quando o cliente pedir algo fora do escopo — formalizando o novo pedido e o valor adicional.

**3. E-mail de envio do contrato**
Mensagem profissional para enviar o contrato ao cliente — que explica o que é cada parte sem parecer desconfiado.
```

## Exemplo de uso

### Input
- Prestador: Thiago Tavares — CPF 123.456.789-00 — São Paulo/SP
- Cliente: Empresa de moda sustentável — Terra Viva Ltda — CNPJ 00.111.222/0001-33
- Projeto: Identidade visual completa (logo, paleta, tipografia, manual, 5 aplicações)
- Prazo: 5 semanas
- Valor: R$5.500
- Pagamento: R$2.750 na assinatura + R$2.750 na entrega final
- Revisões: 2 rodadas inclusas
- Foro: São Paulo/SP

### Output
"Cláusula 5ª — Das Revisões
O presente contrato prevê até 2 (duas) rodadas de revisão ao longo do projeto, distribuídas conforme o cronograma aprovado.
Entende-se por revisão: ajustes de proporção, refinamento de elementos já aprovados, alterações de tonalidade dentro da paleta acordada e ajustes tipográficos menores.
Não constituem revisão, e serão orçados separadamente: nova direção criativa após aprovação de conceito, alteração de cor principal após aprovação de paleta, inclusão de elementos não previstos no briefing original ou refação completa de qualquer elemento após sua aprovação formal pelo CONTRATANTE.
Revisões adicionais além das 2 inclusas serão cobradas no valor de R$ [valor por revisão] cada, mediante orçamento aprovado antes do início dos ajustes."
