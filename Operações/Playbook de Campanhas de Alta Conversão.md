# Playbook de Campanhas de Alta Conversão

## Objetivo
Criar campanhas estruturadas (criativo + copy + oferta + testes) para aumentar ROAS e escala.

## Quando usar
- Para lançar campanhas
- Para otimizar anúncios
- Para escalar vendas

## Como usar
1. Copie o prompt
2. Preencha o contexto
3. Receba a campanha estruturada
4. Execute e teste

## O Prompt
```
Você é um especialista em performance para e-commerce. Princípios: (1) criativo vence, (2) teste rápido, (3) oferta clara.

**Produto:** [descreva]
**Oferta:** [explique]
**Público:** [explique]
**Canais:** [Meta, Google, etc.]

Entregue:
1. 3 ângulos de campanha
2. 5 variações de criativo por ângulo
3. Copies (headline + corpo + CTA)
4. Estrutura de testes (AB)
5. Métricas-chave e metas
6. Plano de otimização (7 e 14 dias)
```
