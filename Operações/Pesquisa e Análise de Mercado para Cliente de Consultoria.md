# Pesquisa e Análise de Mercado para Cliente de Consultoria

## Objetivo
Conduzir uma pesquisa e análise de mercado profunda para um cliente de consultoria de marketing — mapeando o cenário competitivo, o comportamento do público, as tendências do nicho e as oportunidades não aproveitadas — para entregar um entregável estratégico que o cliente não conseguiria produzir sozinho e que fundamenta todas as decisões de marketing e posicionamento.

## Quando usar
- Como entregável do onboarding de um novo cliente de consultoria
- Para fundamentar uma proposta estratégica com análise de mercado real
- Quando um cliente quer entrar em um novo nicho e precisa entender o terreno
- Para identificar oportunidades de posicionamento que os concorrentes não estão aproveitando
- Quando precisar de uma análise profunda rápida para embasar recomendação estratégica

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o nicho do cliente, os concorrentes conhecidos e o objetivo da pesquisa
3. Receba o framework de pesquisa completo com o que coletar, onde buscar e como analisar
4. Execute a pesquisa usando o framework e use o prompt de síntese para montar o entregável final

## O Prompt
```
Você é uma analista de marketing estratégico especializada em pesquisa de mercado aplicada para consultoria. Seus princípios: (1) pesquisa de mercado para consultora não precisa de survey formal — dados públicos bem analisados valem mais do que pesquisa mal interpretada, (2) o que mais importa não é o que o mercado está fazendo, mas o que está DEIXANDO DE FAZER — o gap é a oportunidade, (3) análise boa termina com decisão clara, não com "depende" — entregue recomendação, não apenas mapa, (4) o cliente paga pela interpretação, não pela coleta de dados.

Faça a pesquisa e análise de mercado com os seguintes dados:

**Nicho/mercado a pesquisar:** [ex: coaches de carreira para executivos / consultores de RH / agências de marketing para PMEs]
**Objetivo da pesquisa:** [entrar no mercado / reposicionar / identificar oportunidades / embasar proposta]
**O que o cliente faz:** [produto/serviço e modelo de entrega]
**Concorrentes a analisar (liste 3-5):** [nomes ou tipos de concorrentes]
**Público final do cliente:** [quem o cliente do cliente é]
**Onde o cliente está presente hoje:** [canais, volume de audiência]

Entregue:

**1. Análise do mercado**
Tamanho estimado, maturidade, nível de saturação e tendências de comportamento do público no nicho — com fontes consultáveis (pesquise com dados reais, não estimativas)

**2. Mapa da concorrência**
Para cada concorrente listado:
- Posicionamento (o que promete, para quem)
- Pontos fortes de comunicação
- Gaps e pontos fracos identificáveis
- Preço médio praticado (se público)
- Tom e canal de comunicação predominante

**3. Análise do público**
O que o público deste nicho busca, o que teme, o que já tentou, quais são suas frustrações recorrentes — com base em padrões observáveis (comentários, avaliações, perguntas frequentes, grupos)

**4. Gaps de mercado identificados**
O que os concorrentes não estão fazendo, comunicando ou entregando — espaços disponíveis para diferenciação

**5. Oportunidades de posicionamento**
3 ângulos de posicionamento possíveis para o cliente, com justificativa estratégica para cada um

**6. Recomendação estratégica**
Com base em toda a análise: qual oportunidade tem maior potencial de retorno e menor barreira de entrada — e por quê

**7. Fontes e método**
O que pesquisar e onde — lista de fontes gratuitas para coletar os dados (LinkedIn, YouTube, comentários, grupos, Reclame Aqui, G2, SimilarWeb, etc.)
```

## Exemplo de uso

### Input
- Nicho: Coaches de carreira para profissionais em transição
- Objetivo: Identificar gaps de posicionamento e oportunidades
- O que o cliente faz: Coaching individual de 3 meses para executivos mudando de área ou buscando promoção
- Concorrentes: 4 coaches com audiência média de 10-50k no Instagram
- Público final: Executivos entre 30-45 anos, cargo gerencial, salário entre R$8k-R$20k
- Presença: Instagram 3k seguidores, sem LinkedIn

### Output
1. Nenhum concorrente fala especificamente para executivos em transição involuntária (demissão) — todos falam sobre "crescimento" e "evolução", não sobre o momento de crise e urgência
2. Ausência de conteúdo específico para a fase de "limbo" — depois da demissão, antes da nova posição — que é o momento de maior dor e maior abertura para comprar
3. Nenhum concorrente usa LinkedIn como canal principal, mesmo sendo onde o público está mais ativo profissionalmente — oportunidade de construir autoridade numa plataforma com menos ruído para esse nicho

Recomendação: Posicionar o cliente especificamente para executivos em transição involuntária — o coach que ajuda quem foi desligado a se reposicionar com velocidade e clareza. Canal prioritário: LinkedIn.
