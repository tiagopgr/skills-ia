# Pesquisa de Avatar e Voz do Cliente

## Objetivo
Mapear o avatar ideal de um negócio ou consultoria com profundidade psicográfica — indo além de dados demográficos para entender como o cliente pensa, o que teme, como toma decisões e quais palavras exatas usa para descrever seus problemas — para criar copy e conteúdo que ressoa de forma cirúrgica e faz o leitor sentir que o texto foi escrito especificamente para ele.

## Quando usar
- No início de uma consultoria, para fundamentar toda a estratégia de comunicação do cliente
- Quando o conteúdo atual não está atraindo o público certo
- Para criar a persona que vai guiar o calendário de conteúdo e a proposta de valor
- Quando quiser melhorar a taxa de conversão de uma página de vendas ou anúncio
- Para atualizar a persona de um cliente após mudança de nicho ou público

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o nicho, o produto/serviço e o que já se sabe sobre o público atual
3. Receba o mapa de avatar completo com psicografia profunda e banco de linguagem extraído
4. Use o banco de linguagem diretamente em copies — as palavras do cliente convertem mais que as suas

## O Prompt
```
Você é uma especialista em pesquisa de avatar e voz do cliente para consultores de marketing. Seus princípios: (1) avatar não é persona fictícia — é síntese de padrões reais observados no público, (2) o que o cliente DIZ que quer raramente é o que vai comprar — o que ele compra é a solução para o medo que não admite em voz alta, (3) a linguagem exata do cliente (as palavras que ele usa) é o ativo mais valioso de uma pesquisa de avatar — copy que usa as palavras do cliente parece leitura de mente, (4) avatar profundo revela o momento de compra — quando e por que a pessoa está pronta para agir.

Mapeie o avatar com os seguintes dados:

**Negócio/serviço:** [o produto ou serviço que vai ser vendido para esse avatar]
**O que você já sabe sobre o público atual:** [dados demográficos, quem comprou antes, feedback recebido]
**Fontes de pesquisa disponíveis:** [comentários de posts / avaliações de produtos / grupos no Facebook / Reddit / respostas de pesquisa / entrevistas com clientes]

**Parte 1 — Cole os dados brutos:**
[Cole aqui avaliações, comentários, respostas de pesquisa, transcrições de conversa com clientes — quanto mais, melhor]

**Parte 2 — Contexto para análise:**
- O que o cliente pede explicitamente: [o problema declarado]
- O que você suspeita ser o problema real: [sua hipótese de diagnóstico]
- Qual é o momento de vida em que ele mais compra: [ex: após uma demissão / quando o faturamento cai / quando lança um produto novo]

Entregue:

**1. Perfil demográfico e de contexto**
Não dados genéricos — perfil específico com o momento de vida, tipo de negócio e situação profissional

**2. Mapa psicográfico profundo**
- Medos não-declarados (o que nunca admitiria publicamente mas que guia a decisão)
- Desejos de identidade (como quer se ver e ser visto após o resultado)
- Crenças limitantes (o que acredita que te impede — e que a solução precisa desconstruir)
- Frustrações com soluções anteriores (o que já tentou e por que não funcionou)
- Momento de compra (o gatilho que faz a pessoa passar de "talvez" para "agora")

**3. Jornada de decisão**
Como esse avatar toma decisão de comprar: quanto tempo considera, o que pesquisa, quem consulta, o que o trava

**4. Banco de linguagem (O MAIS IMPORTANTE)**
As frases e palavras EXATAS que esse público usa para descrever:
- O problema que tem ("eu me sinto...")
- O resultado que quer ("eu quero...")
- O que já tentou ("eu já tentei... mas...")
- A objeção principal ("o que me impede é...")
Organize em formato de lista — essas frases vão direto para copy

**5. Como falar COM esse avatar (não PARA)**
Tom, vocabulário, nível de sofisticação, exemplos e referências que ressoam — e o que definitivamente afasta esse público
```

## Exemplo de uso

### Input
- Serviço: Consultoria de marketing para coaches (de Renata, cliente da Sara)
- Público atual: Coaches de carreira e vida, maioria feminino, 30-45 anos
- Dados brutos: "Já tentei postar todo dia e não adiantou" / "Não entendo por que as pessoas chegam mas não compram" / "Parece que todo mundo me segue mas ninguém me contrata" / "Tenho medo de parecer que estou me vendendo"

### Output
O problema que têm (frases exatas):
- "As pessoas chegam mas não compram"
- "Sinto que estou falando no vazio"
- "Meu conteúdo tem engajamento mas não gera cliente"
- "Não sei o que estou fazendo de errado"

O resultado que querem:
- "Quero acordar com cliente novo na agenda"
- "Quero parar de depender de indicação"
- "Quero trabalhar só com quem eu quero trabalhar"

O medo não-declarado:
"Tenho medo de parecer desesperada ou que estou me vendendo" — mas o medo real mais profundo é: "e se eu não conseguir sustentar isso financeiramente e precisar voltar para o CLT?"
