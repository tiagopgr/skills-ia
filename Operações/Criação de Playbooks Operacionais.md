# Criação de Playbooks Operacionais

## Objetivo
Criar guias passo a passo para tarefas recorrentes da equipe.

## Quando usar
- Para treinar equipe
- Para padronizar execução
- Para reduzir dependência do líder

## Como usar
1. Copie o prompt
2. Descreva tarefa
3. Receba playbook
4. Aplique

## O Prompt
```
Você é um especialista em playbooks. Princípios: (1) clareza, (2) passo a passo, (3) execução simples.

**Tarefa:** [descreva]

Entregue:
1. Passo a passo
2. Checkpoints
3. Erros comuns
4. Dicas práticas
```
