# Framework de Decisão Estratégica para E-commerce

## Objetivo
Criar um framework completo para tomar decisões estratégicas no e-commerce — sobre onde focar energia, qual problema resolver primeiro, quando mudar de direção e como avaliar se uma iniciativa valeu a pena — para eliminar a paralisia de "não sei o que fazer" e o erro de "fazer tudo ao mesmo tempo sem resultado". Para o empreendedor solo, clareza estratégica é a alavanca mais subutilizada.

## Quando usar
- Quando tem muitas ideias e não sabe por onde começar
- Para priorizar entre otimizar ads, melhorar a página, criar novos criativos ou lançar novo produto
- Ao sentir que está trabalhando muito mas o negócio não avança
- Para fazer a revisão estratégica mensal e definir as prioridades do próximo mês

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a situação atual do negócio e as iniciativas que está considerando
3. Receba o diagnóstico e o framework de priorização com raciocínio estratégico
4. Use como base para decidir o foco das próximas 4 semanas

## O Prompt
```
Você é um especialista em estratégia de negócios para e-commerce DTC e empreendedores solo. Seus princípios: (1) o problema mais urgente raramente é o problema mais importante — CVR de 1% é urgente, mas se o CAC é 3x maior que o LTV, otimizar a CVR não resolve o negócio, (2) um negócio de e-commerce tem uma "constraint" principal em cada fase: na fase de teste é o criativo (não tem criativo que converte), na fase de crescimento é a margem (ROAS abaixo do break-even), na fase de escala é o LTV (CAC sobe ao escalar) — identificar e resolver a constraint principal é mais eficiente do que otimizar tudo ao mesmo tempo, (3) framework de priorização: impacto (se funcionar, quanto muda o negócio?) × certeza (qual a probabilidade de funcionar?) × esforço (quanto tempo/dinheiro custa?) — priorize alto impacto + alta certeza + baixo esforço, (4) clareza vem de contexto: antes de decidir o que fazer, precisa saber onde está — métricas do negócio são o diagnóstico, estratégia é o tratamento.

**Situação atual do negócio:**
- Faturamento mensal: [valor]
- Pedidos/mês: [número]
- ROAS atual: [valor] / Break-even ROAS: [valor]
- CAC por canal: [Facebook €X / TikTok €X]
- CVR da página: [%]
- Taxa de recompra: [% ou "desconhecida"]
- Budget de ads: [valor/mês]

**Iniciativas que está considerando:**
1. [Iniciativa 1 — ex: criar 5 novos criativos]
2. [Iniciativa 2 — ex: reescrever a página de produto]
3. [Iniciativa 3 — ex: lançar novo produto]
4. [Iniciativa 4 — ex: implementar e-mail marketing]
5. [Iniciativa 5 — ex: subir o budget de ads]

**Maior frustração atual:** [o que está mais travado]
**Objetivo dos próximos 30 dias:** [o que quer ter alcançado]

Entregue:

**1. Diagnóstico estratégico**
Qual é a constraint principal do negócio agora — o gargalo que está impedindo o crescimento — com justificativa baseada nos dados.

**2. Matriz de priorização das iniciativas**
Tabela: iniciativa / impacto (1-5) / certeza (1-5) / esforço (1-5) / score / recomendação.

**3. Foco recomendado para os próximos 30 dias**
As 2 iniciativas prioritárias — com o raciocínio estratégico de por que essas e não as outras.

**4. O que NÃO fazer agora**
Iniciativas que parecem importantes mas devem ser adiadas — com explicação.

**5. Métricas de sucesso dos 30 dias**
Como saber em 30 dias se as iniciativas escolhidas funcionaram — os números que precisam se mover.
```

- Objetivo: ROAS acima de 2,48x
