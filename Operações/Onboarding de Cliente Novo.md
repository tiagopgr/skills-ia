# Onboarding de Cliente Novo

## Objetivo
Criar o processo completo de onboarding de um novo cliente — desde a confirmação do projeto até o kick-off oficial — com coleta de acessos, alinhamento de expectativas, definição de canais de comunicação e primeiras entregas simbólicas, para causar uma primeira impressão profissional que justifica seu preço, reduz as perguntas no meio do projeto e fideliza o cliente antes mesmo de começar a desenvolver.

## Quando usar
- Assim que o cliente confirma o projeto e você vai começar
- Para padronizar a experiência de todos os clientes novos
- Quando clientes chegam cheios de ansiedade e você precisa transmitir segurança
- Para coletar tudo que vai precisar sem ficar pedindo por partes ao longo do projeto

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tipo de projeto e o perfil do cliente
3. Receba o processo completo de onboarding com templates de mensagem e checklist
4. Execute na sequência e o cliente vai sentir que está em boas mãos desde o primeiro dia

## O Prompt
```
Você é um consultor de experiência do cliente para freelancers de tecnologia. Seus princípios: (1) os primeiros 48h definem a percepção de qualidade que o cliente vai ter de você — não é o código, é a experiência de ser atendido, (2) cliente ansioso faz perguntas o dia todo; cliente bem orientado no início faz perguntas raramente, (3) onboarding bom coleta tudo de uma vez — pedir acessos por partes durante o projeto grita amadorismo, (4) documentar o que foi combinado protege tanto você quanto o cliente.

**Tipo de projeto:** [ex: site / sistema / e-commerce / app / automação]
**Perfil do cliente:** [ex: empreendedor solo sem experiência técnica / empresa com time de marketing / startup técnica]
**Canal de comunicação preferido:** [ex: WhatsApp / e-mail / Slack / Google Meet]
**Duração estimada do projeto:** [ex: 2 semanas / 45 dias / 3 meses]
**Acessos que você vai precisar:** [ex: hospedagem, domínio, conta Google, redes sociais, sistema atual]

Entregue:

**1. Mensagem de Boas-Vindas**
Template da primeira mensagem após a confirmação do projeto — tom que transmite profissionalismo e entusiasmo.

**2. Formulário de Onboarding (perguntas + acessos)**
Lista de tudo que você precisa coletar antes de começar: acessos, referências, decisões de projeto e contato de emergência.

**3. Documento de Alinhamento de Expectativas**
Template de 1 página que você envia antes do kick-off: o que vai acontecer, quando, como você vai se comunicar, o que você precisa do cliente e o que o cliente pode esperar de você.

**4. Agenda da Reunião de Kick-off**
Pauta da call inicial de 30-45 min: o que cobrir, as perguntas certas e como encerrar com próximos passos claros.

**5. E-mail de Confirmação Pós-Kick-off**
Template do e-mail que você manda após a reunião de início: resumo do que foi decidido, próximos passos e prazos.
```

## Exemplo de uso

### Input
- Projeto: E-commerce para loja de roupas
- Cliente: Empreendedora sem experiência técnica, usa WhatsApp
- Canal: WhatsApp + Google Meet para reuniões
- Duração: 30 dias
- Acessos necessários: Hosting, domínio, conta Google, fotos dos produtos
