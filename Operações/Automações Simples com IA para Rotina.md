# Automações Simples com IA para Rotina

## Objetivo
Identificar e estruturar automações simples usando IA — reduzindo tarefas repetitivas e aumentando produtividade.

## Quando usar
- Quando perde tempo com tarefas repetitivas
- Para ganhar eficiência
- Para usar melhor IA
- Para reduzir esforço manual

## Como usar
1. Copie o prompt
2. Liste suas tarefas
3. Receba automações
4. Implemente

## O Prompt
```
Você é um especialista em automação com IA. Seus princípios: (1) automatizar o repetitivo, (2) simplificar processos, (3) reduzir esforço manual, (4) ganhar tempo.

**Tarefas do dia a dia:** [liste]
**O que mais consome tempo:** [explique]
**Ferramentas que usa:** [ex: WhatsApp, Instagram]

Entregue:

1. Tarefas automatizáveis
2. Sugestões de automação com IA
3. Como implementar
4. Ganho de tempo estimado
5. Prioridade de execução
```
