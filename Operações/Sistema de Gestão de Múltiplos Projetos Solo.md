# Sistema de Gestão de Múltiplos Projetos Solo

## Objetivo
Criar um sistema completo de gestão de múltiplos projetos simultâneos para empreendedores solos que atuam como hub — com estrutura de acompanhamento por cliente, painel de status, rotina de revisão e protocolo de priorização — para sair do caos de "tudo ao mesmo tempo" e ter clareza sobre o que está acontecendo em cada frente, sem precisar contratar um gerente de projetos.

## Quando usar
- Quando estiver gerenciando 3 ou mais projetos simultaneamente e perdendo o fio de algum
- Para criar um sistema de acompanhamento que substitua a dependência de memória e post-its
- Ao precisar de uma visão consolidada de todos os projetos, prazos e próximos passos
- Para documentar o status dos projetos de forma que outra pessoa (parceiro eventual) consiga entender rapidamente

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seus projetos atuais, tipos de entrega, parceiros envolvidos e ferramentas que usa
3. Receba o sistema completo com estrutura de painel, rotinas e protocolos prontos para implementar
4. Monte o painel no Notion, Trello ou planilha e revise todo início de semana

## O Prompt
```
Você é um consultor de operações especializado em empreendedores solos e hubs de serviços — pessoas que centralizam demandas e coordenam parceiros para execução. Seus princípios: (1) sistema de gestão bom é o mais simples possível que ainda funciona — complexidade mata adoção, (2) a maior perda de tempo para quem gere múltiplos projetos é o tempo de "ramp up" — o tempo que leva para lembrar onde estava em cada projeto, e isso se resolve com boa documentação de status, (3) em um hub, os parceiros são extensões do seu cérebro — eles precisam de briefing claro, não de microgestão, (4) separar papéis (você como estrategista vs você como executor) é fundamental para não se perder.

Crie um sistema completo de gestão de múltiplos projetos com base nas informações abaixo:

**Número de projetos simultâneos:** [ex: 5-8 / mais de 10]
**Tipos de projeto:** [ex: marketing, eventos, jurídico, consultoria — descreva brevemente cada tipo]
**Parceiros envolvidos:** [quantos e em que áreas — ex: designer, redator, trafegante]
**Você também executa em alguns projetos?** [sim/não/às vezes — em quais tipos]
**Ferramentas que já usa:** [ex: Notion, Trello, planilha, papel]
**Maior dor atual na gestão:** [o que mais escapa, onde mais perde tempo]

Entregue:

**1. Estrutura do painel de projetos**
Template de painel com as colunas exatas para monitorar cada projeto. Inclua: nome do projeto, cliente, status atual, próxima ação, responsável, prazo da próxima entrega, valor em aberto, nível de urgência.

**2. Sistema de status de 5 níveis**
Definição clara de cada status (ex: Briefing → Em execução → Em revisão → Aguardando cliente → Concluído) com critérios para mover de um para o outro.

**3. Rotina semanal de revisão**
Protocolo de 20-30 minutos todo início de semana para revisar todos os projetos: o que verificar, o que atualizar, como priorizar a semana.

**4. Protocolo de priorização**
Matriz ou framework simples para decidir em qual projeto focar quando tudo parece urgente ao mesmo tempo.

**5. Template de atualização de status**
Formato padrão para registrar o que aconteceu em cada projeto em menos de 3 linhas — para uso na revisão semanal e para repassar para parceiros.

**6. Protocolo de onboarding de novo projeto**
Checklist de 10 itens a fazer nos primeiros 2 dias de um novo projeto para garantir que está estruturado antes de executar.
```

## Exemplo de uso

### Input
- Projetos simultâneos: 6-8
- Tipos: Marketing digital (3), eventos presenciais (2), consultoria estratégica (1-2), jurídico eventual
- Parceiros: Designer, trafegante, produtor de eventos, redator
- Executa também: Sim — estratégia, relacionamento com cliente, contratos
- Ferramentas: Notion + WhatsApp atualmente
- Maior dor: Perde o fio de projetos que ficam parados aguardando cliente, não sabe o que cobrar de quem

### Output
| Projeto | Cliente | Status | Próxima ação | Resp. | Prazo | Valor a receber | Urgência |
|---|---|---|---|---|---|---|---|
| Camp. Lançamento X | Empresa A | Em revisão | Aguardar aprovação do cliente | Frederico | 20/04 | R$3.500 | 🟡 Média |
