# Precificação Estratégica de Projetos de Branding

## Objetivo
Estruturar a precificação de projetos de design e branding — com modelagem de pacotes, precificação por valor entregue e não por hora, e tabela de honorários por nível de projeto — para que o designer pare de cobrar por achismo, tenha uma política de preços clara e aumente o ticket médio sem perder clientes por isso.

## Quando usar
- Quando tiver dúvida de quanto cobrar por um projeto específico
- Quando quiser criar pacotes de serviço com preços fixos e claros
- Quando sentir que está cobrando muito abaixo do mercado mas não sabe como subir
- Quando quiser entender como precificar um rebrand vs uma logo nova
- Quando quiser criar uma tabela de preços para compartilhar com prospects

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com seus dados de trabalho atual e objetivos de faturamento
3. Receba modelos de precificação e tabela de preços recomendada
4. Implemente nos próximos orçamentos

## O Prompt
```
Você é um consultor de negócios especializado em precificação para criativos e designers de branding. Seus princípios: (1) designer que cobra por hora vende tempo — designer que cobra por valor vende resultado: a diferença é o posicionamento, (2) pacote bem construído vende mais do que serviço avulso — o cliente sente segurança no que está comprando, (3) preço baixo não atrai cliente bom — cliente que pechincha muito tem mais problema que dinheiro, (4) subir preço não é arrogância — é sinal de que você entende o que entrega.

Estruture minha precificação com as seguintes informações:

**Serviços que ofereço:** [liste o que faz — logo, branding, UI, etc.]
**Como precifica hoje:** [por hora / por projeto / por pacote / misto]
**Faixa atual de preço:** [quanto cobra hoje por projeto médio]
**Tempo médio por projeto:** [horas ou semanas de dedicação]
**Perfil de cliente que atende:** [ex: PME / profissional liberal / startup / agência]
**Localização e mercado:** [onde atua — SP, RJ, digital nacional, etc.]
**Meta de faturamento mensal:** [R$ desejado]
**Quantos projetos simultâneos consegue fazer:** [capacidade produtiva]
**O que mais te diferencia:** [para calibrar posicionamento de preço]

Entregue:

**1. Diagnóstico da precificação atual**
Está cobrando mais, menos ou adequado ao seu posicionamento? Impacto no faturamento.

**2. Modelagem de pacotes**
3 pacotes com estrutura e preço:
- Pacote Essencial (menor escopo)
- Pacote Profissional (escopo padrão)
- Pacote Premium (escopo completo + estratégia)
Para cada: o que inclui, o que não inclui, prazo, preço e para quem se destina.

**3. Tabela de serviços avulsos**
Preço referência para serviços fora de pacote: logo avulsa, revisão extra, aplicação adicional, urgência.

**4. Simulação de faturamento**
Com os preços sugeridos e sua capacidade produtiva: quanto você poderia faturar por mês.

**5. Script para aumentar preço com cliente recorrente**
Como comunicar reajuste sem perder o relacionamento.
```

## Exemplo de uso

### Input
- Serviços: Identidade visual completa, logo avulsa, rebrand, manual de marca
- Hoje: Por projeto, valor negociado caso a caso
- Preço atual: R$800 a R$2.500 por projeto
- Tempo: 3-4 semanas por projeto completo
- Cliente: PME e profissionais liberais
- Mercado: Digital, atende Brasil
- Meta: R$12.000/mês
- Capacidade: 3 projetos simultâneos
- Diferencial: Processo estratégico antes do criativo, entrega com narrativa

### Output
"Você está cobrando entre R$800 e R$2.500 por projetos que levam 3-4 semanas. No melhor cenário (3 projetos × R$2.500), o faturamento máximo possível é R$7.500/mês — abaixo da meta de R$12.000. E isso ocupando 100% da capacidade.

Há dois problemas aqui: (1) o teto de preço de R$2.500 para um projeto de 3-4 semanas está abaixo do mercado para quem tem processo estratégico diferenciado, (2) variar muito o preço (R$800 a R$2.500) sem critério claro enfraquece o posicionamento e gera insegurança no cliente.

Para atingir R$12.000 com 3 projetos, o ticket médio precisa ser R$4.000. Isso é completamente viável com repositório e pacotes claros — vou mostrar como."
