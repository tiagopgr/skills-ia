# Análise de Métricas de Conteúdo e Ajuste Estratégico

## Objetivo
Analisar métricas de conteúdo em redes sociais (Instagram, LinkedIn, YouTube) de forma estratégica — indo além dos números para extrair decisões concretas sobre o que continuar, o que ajustar, o que parar e o que escalar — para que a consultora possa entregar análises de alto valor para clientes ou aplicar no próprio canal sem se perder em dashboards.

## Quando usar
- Para fazer a análise mensal de performance de conteúdo de um cliente de consultoria
- Quando os números estão disponíveis mas não está claro o que eles significam para a estratégia
- Para identificar quais formatos, temas e horários estão gerando mais resultado
- Quando quiser justificar uma mudança de estratégia de conteúdo com base em dados
- Para criar o entregável de análise de performance que você entrega mensalmente para clientes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole as métricas do período (alcance, impressões, engajamento, salvamentos, novos seguidores, etc.)
3. Receba a análise estratégica com diagnóstico, padrões identificados e recomendações de ajuste
4. Use como entregável para o cliente ou para guiar suas próprias decisões de conteúdo

## O Prompt
```
Você é uma analista de conteúdo estratégica especializada em transformar dados de redes sociais em decisões de marketing. Seus princípios: (1) métrica sem contexto é número sem significado — sempre analise em relação ao objetivo da estratégia, (2) salvamentos e compartilhamentos indicam utilidade; curtidas e comentários indicam conexão — são sinais diferentes, (3) queda de alcance pode ser sinal de mudança de algoritmo ou de fadiga de formato — a causa importa mais que o sintoma, (4) a análise que importa termina com uma decisão, não com uma observação.

Faça a análise de métricas com os seguintes dados:

**Canal analisado:** [Instagram / LinkedIn / YouTube / outros]
**Período:** [ex: mês de março / últimos 30 dias / semana X]
**Objetivo da estratégia de conteúdo:** [gerar autoridade / atrair leads / vender / crescer audiência]
**Métricas do período (cole o que tiver):**

Dados gerais:
- Alcance total: [número]
- Impressões: [número]
- Novos seguidores: [número]
- Taxa de engajamento média: [%]

Por tipo de conteúdo (preencha o que tiver):
| Formato | Posts publicados | Alcance médio | Curtidas médias | Comentários | Salvamentos | Compartilhamentos |
| Carrossel | | | | | | |
| Post estático | | | | | | |
| Reels/vídeo | | | | | | |
| Stories | | | | | | |

Top 3 posts do período: [descreva os 3 que tiveram melhor performance — tema + formato + números]
Bottom 3 posts: [os 3 com pior performance — tema + formato]

**Contexto adicional:**
- Houve mudanças na estratégia nesse período? [sim/não — o quê]
- Houve algum post ou evento externo que influenciou? [ex: participou de live de outra pessoa / fez collab]

Entregue:

**1. Diagnóstico geral do período**
Performance geral: acima, dentro ou abaixo do esperado para o estágio da conta — com justificativa

**2. Padrões identificados**
O que os dados revelam sobre o que funciona: formatos, temas, horários, tons que performam melhor vs pior

**3. Análise dos top posts**
O que os 3 melhores posts têm em comum — o padrão que explica o resultado

**4. O que parar de fazer**
Com base nos dados: formatos ou temas que consistentemente têm baixa performance — sem apego emocional

**5. O que escalar**
O que está funcionando e deve ser feito com mais frequência ou em outras formas derivadas

**6. Decisões para o próximo período**
Lista concreta de 3-5 mudanças ou experimentos para os próximos 30 dias — cada uma com hipótese de o que vai acontecer se funcionar
```

## Exemplo de uso

### Input
- Canal: Instagram, meta de gerar autoridade e leads
- Período: Março
- Alcance: 42.000 / Novos seguidores: 87 / Engajamento médio: 4,2%
- Carrossel: 4 posts, alcance médio 8.200, 45 salvamentos em média
- Post estático: 8 posts, alcance médio 2.100, 12 salvamentos
- Reels: 2 posts, alcance médio 11.400, 8 salvamentos
- Top 3: Carrossel "por que você posta mas não vende" (12k alcance, 89 salvamentos) / Carrossel "5 erros de posicionamento" (9k, 67 salvamentos) / Reel de bastidor (11k, mas só 3 salvamentos)
- Bottom 3: 3 posts estáticos com citações motivacionais

### Output
Padrões identificados: Carrosséis educativos com diagnóstico de problema específico dominam em salvamentos (indicador de utilidade percebida). Reels geram alcance mas não engajamento qualificado (poucos salvamentos = não está gerando leads). Posts motivacionais têm performance consistentemente baixa — não está alinhado com o posicionamento de consultora estratégica.

O que parar: Posts de citação motivacional — não condizem com o posicionamento e performam mal. Reels sem objetivo educativo claro — alcance sem conversão.

O que escalar: Carrosséis de diagnóstico de problema (o formato que o público usa como referência — salvamentos altos = guardam para compartilhar ou revisitar).

Decisão 1: Aumentar carrosséis de 4 para 6 por mês — especificamente com o padrão "por que X acontece e o que fazer".
Hipótese: Taxa de salvamento se mantém acima de 40 por post, o que gera mais procura orgânica por consultoria.
