# Funil de Conteúdo para Advocacia

## Objetivo
Atrair, educar e converter clientes por meio de conteúdo estratégico.

## Quando usar
- Para crescer no Instagram/LinkedIn
- Para atrair leads qualificados
- Para estruturar marketing

## Como usar
1. Copie o prompt
2. Preencha seu contexto
3. Receba o funil
4. Execute

## O Prompt
```
Você é um estrategista de marketing jurídico.

Área: [descreva]
Público: [descreva]

Entregue:
1. Estrutura do funil
2. Conteúdo para topo, meio e fundo
3. CTAs
4. Estratégia de conversão
```
