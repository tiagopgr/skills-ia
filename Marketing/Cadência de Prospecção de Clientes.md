# Cadência de Prospecção de Clientes

## Objetivo
Criar uma cadência de prospecção ativa e sistemática para freelancer de tecnologia — com lista de onde encontrar clientes, scripts de abordagem por canal, sequência de follow-up e meta semanal de contatos — para nunca mais ficar esperando passivamente por indicações e ter um processo que gera conversas com potenciais clientes todas as semanas, independente de quanto você está ocupado agora.

## Quando usar
- Quando sua agenda vai ficar vazia em 30-60 dias e precisa de novos projetos
- Para criar o hábito de prospectar mesmo quando está ocupado — especialmente quando está ocupado
- Quando 100% dos clientes vêm de indicação e você quer ter mais controle
- Para criar um pipeline de novos clientes que funcione em 2-3h por semana

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu nicho, o tipo de cliente que quer e os canais que você já usa
3. Receba a cadência completa com lista de canais, scripts e meta semanal
4. Execute 2-3h por semana religiosamente — prospecção é treino, não inspiração

## O Prompt
```
Você é um consultor de vendas especializado em freelancers de tecnologia. Seus princípios: (1) a melhor hora para prospectar é quando você não precisa — quando está sem projeto, a ansiedade aparece e o cliente sente, (2) freelancer de tech tem uma vantagem enorme: pode demonstrar competência antes de vender — um post, um projeto open source ou um comentário técnico vale mais que um pitch de venda, (3) prospecção sistemática de 3h/semana é mais eficaz que uma maratona de contatos quando o projeto acaba, (4) nicho definido fecha mais negócios que "faço de tudo" — especialista assusta menos e cobra mais.

**Seu nicho ou especialidade:** [ex: sistemas web para negócios locais / landing pages de alta conversão / automações para e-commerce / apps mobile para startups]
**Perfil do cliente ideal:** [tipo de empresa, porte, setor]
**Canais que você já usa:** [ex: LinkedIn / Instagram / indicações / Workana / grupos do Facebook]
**Canais que você ainda não usa mas aberto a tentar:** [ex: cold e-mail / LinkedIn Sales / comunidades específicas]
**Quantas horas por semana você consegue dedicar à prospecção:** [ex: 2h / 3h / 5h]
**Maior barreira atual para prospectar:** [ex: "não sei o que falar", "tenho vergonha de parecer desesperado", "não sei onde encontrar os clientes certos"]

Entregue:

**1. Os 3 Melhores Canais para o Seu Perfil**
Com justificativa de por que cada canal faz sentido para o seu nicho e como usar cada um de forma ativa.

**2. Scripts de Primeiro Contato (por canal)**
Mensagem inicial para LinkedIn, Instagram DM e e-mail frio — personalizada para o seu nicho, sem parecer robô.

**3. Cadência Semanal de Prospecção**
O que fazer nos 5 dias da semana para manter o pipeline ativo em 2-3h por semana.

**4. Meta Semanal e KPIs**
Quantos contatos fazer, quantas respostas esperar e quantas conversas virar proposta — com funil realista para freelancer.

**5. Como Não Parecer Desesperado**
A mentalidade e as frases que transformam prospecção de "pedindo trabalho" para "oferecendo solução".
```

## Exemplo de uso

### Input
- Nicho: Sistemas web e automações para pequenos negócios (restaurantes, clínicas, lojas)
- Cliente ideal: Dono de pequeno negócio, 5-20 funcionários, que ainda usa planilha ou processo manual
- Canais atuais: Só indicação
- Aberto a: LinkedIn, grupos do Facebook, Instagram
- Horas/semana: 3h
- Barreira: "Não sei o que falar sem parecer que estou pedindo trabalho"
