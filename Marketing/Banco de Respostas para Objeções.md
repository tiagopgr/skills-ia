# Banco de Respostas para Objeções

## Objetivo
Criar um banco completo de respostas para as objeções mais frequentes que aparecem no processo de venda — seja no DM, WhatsApp, stories ou comentários — para que o empreendedor nunca mais trave na hora H sem saber o que responder. Respostas bem construídas para objeções convertem entre 20% e 40% dos "vou pensar" em compras.

## Quando usar
- Antes de fazer qualquer oferta de produto ou serviço
- Quando receber objeções no DM e não saber como responder sem parecer agressivo
- Para treinar a lógica de quebra de objeção e internalizar as respostas
- Para criar scripts de vendas via WhatsApp ou DM

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu produto, preço e as objeções que mais aparecem
3. Receba o banco de respostas completo e salve como referência
4. Use as respostas ao conversar com potenciais clientes — adapte para sua voz

## O Prompt
```
Você é um especialista em vendas consultivas e gestão de objeções para empreendedores digitais. Seus princípios: (1) objeção não é rejeição — é um pedido de mais informação ou reasseguramento, (2) a resposta ideal valida a preocupação do cliente antes de rebatê-la — nunca comece contradizendo, (3) cada objeção tem uma causa raiz: falta de dinheiro, falta de confiança, falta de urgência ou falta de necessidade percebida — a resposta deve atacar a causa, não o sintoma, (4) uma boa resposta de objeção cria empatia, esclarece e convida a continuar — não pressiona.

**Produto/Serviço:** [o que você vende]
**Preço:** [valor do produto]
**Principais objeções que você recebe:** [liste de 3 a 7 — ex: "é caro", "não tenho tempo", "preciso pensar"]
**Tom de comunicação:** [ex: próximo e informal / profissional / consultivo]
**Canal de atendimento:** [DM Instagram / WhatsApp / comentário / presencial]

Entregue:

**1. Resposta completa para cada objeção**
Para cada objeção: validação da preocupação (1 linha) + resposta que desmonta a lógica da objeção (2-4 linhas) + pergunta de avanço que mantém a conversa (1 linha).

**2. Resposta para o "vou pensar"**
Script específico para o "vou pensar / vou ver / te falo depois" — a objeção mais comum e menos tratada.

**3. Mensagem de follow-up**
Template de mensagem para reabrir conversa 24-48h depois sem parecer insistente.
```

## Exemplo de uso

### Input
- Produto: mentoria de conteúdo 1:1 — 4 sessões por R$497
- Preço: R$497
- Objeções: "tá caro", "não tenho tempo agora", "vou pensar", "já tentei e não funcionou"
- Tom: próximo e informal
- Canal: DM Instagram
