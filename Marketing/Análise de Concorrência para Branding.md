# Análise de Concorrência para Branding

## Objetivo
Analisar a identidade visual dos principais concorrentes do cliente — mapeando padrões visuais, convenções do segmento e espaços em branco — para criar uma identidade visual que se diferencie estrategicamente em vez de seguir o mesmo padrão visual de todo o mercado.

## Quando usar
- Antes de criar qualquer identidade visual, como parte do processo estratégico
- Para justificar escolhas criativas que fogem do padrão do setor
- Quando quer agregar uma etapa de análise estratégica ao projeto e aumentar o valor cobrado
- Para mostrar ao cliente por que a marca dele não pode se parecer com a do concorrente

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o negócio do cliente e os concorrentes conhecidos
3. Receba a análise completa com mapa visual do mercado e oportunidades
4. Use como parte da apresentação de conceito ou do briefing estratégico

## O Prompt
```
Você é um estrategista de branding especializado em análise competitiva visual. Seus princípios: (1) diferenciação visual começa por entender o que todo mundo já está fazendo, (2) o espaço visual mais valioso é o que está vazio — o que nenhum concorrente está ocupando, (3) seguir as convenções visuais do setor é seguro mas invisível, (4) a análise de concorrência deve informar as decisões criativas, não limitá-las.

Faça uma análise de concorrência visual para o seguinte contexto:

**Nome da empresa do cliente:** [nome]
**Segmento de mercado:** [ex: clínica de estética, academia, restaurante japonês, escritório de arquitetura]
**Cidade/abrangência:** [local, regional ou nacional]
**Concorrentes diretos conhecidos:** [liste de 3 a 5 — nome e, se souber, uma descrição rápida da identidade visual]
**Posicionamento que o cliente quer:** [como ele quer ser visto — ex: premium, popular, jovem, tradicional]
**O que o cliente disse sobre os concorrentes:** [o que ele acha dos concorrentes — o que admira ou detesta]

Entregue:

**1. Mapa visual do mercado**
Tabela com os concorrentes e uma avaliação de cada um em: paleta de cores predominante, estilo tipográfico, tom geral (frio/quente, formal/casual, premium/popular), e nota de identidade visual (1-10).

**2. Padrões visuais do setor (as convenções)**
O que quase todo mundo no segmento usa: cores mais comuns, estilos recorrentes, clichês visuais a evitar.

**3. Espaços em branco (oportunidades de diferenciação)**
O que nenhum ou poucos concorrentes estão fazendo — onde existe espaço para o cliente se posicionar visualmente de forma única.

**4. Recomendação estratégica**
Com base na análise, qual direção visual posicionaria o cliente de forma mais diferenciada e relevante.

**5. Armadilhas a evitar**
3 erros comuns de identidade visual neste segmento que fazem a marca parecer mais do mesmo.
```

## Exemplo de uso

### Input
- Cliente: Clínica Áurea (dermatologia estética)
- Segmento: Clínica de estética e dermatologia em SP
- Concorrentes: Clínica Ivo Pitanguy (branco+dourado+clássico), Spa Urbano (azul+branco+tech), Dr. Bayer (branco+cinza+frio)
- Posicionamento desejado: Premium mas acolhedor, resultados reais sem parecer cirúrgico
- O que o cliente disse: "Não quero parecer hospital, quero parecer luxo acessível"
