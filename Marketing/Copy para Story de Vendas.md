# Copy para Story de Vendas

## Objetivo
Criar uma sequência completa de stories de venda — com abertura que prende, construção de desejo, quebra de objeções e CTA irresistível — para qualquer produto ou serviço. Stories são o canal mais direto para conversão no Instagram, mas a maioria das pessoas não sabe como estruturá-los de forma que venda sem parecer invasivo.

## Quando usar
- Ao lançar ou relançar um produto ou serviço
- Para fazer uma oferta rápida sem precisar criar um post elaborado
- Quando quiser converter seguidores em clientes usando apenas os stories
- Para testar uma oferta antes de criar uma campanha maior

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o que você está vendendo, para quem e qual o gatilho de urgência
3. Receba sequência completa de 8 a 12 stories prontos para publicar
4. Adapte com sua voz, publique e acompanhe os cliques no link

## O Prompt
```
Você é um especialista em vendas via Instagram Stories para empreendedores e criadores de conteúdo. Seus princípios: (1) stories de venda precisam primeiro criar conexão antes de fazer oferta — pular direto para o preço é o maior erro, (2) a sequência deve ter ritmo: curiosidade → problema → solução → prova → oferta → urgência → CTA, (3) linguagem casual e próxima converte mais do que copy formal em stories, (4) cada story deve ser curto o suficiente para ser lido em 3 segundos — textos longos perdem o swipe.

**Produto/Serviço:** [o que você está vendendo]
**Preço:** [valor — influencia o nível de aquecimento necessário]
**Público:** [quem você quer atingir com esta sequência]
**Principal dor que resolve:** [o maior problema que seu produto resolve]
**Principal resultado entregue:** [o que a pessoa conquista ao comprar]
**Gatilho de urgência:** [ex: vagas limitadas, desconto até sexta, bônus por tempo]
**CTA final:** [ex: link na bio, responder story, DM com palavra-chave]

Entregue:

**1. Sequência de stories (8 a 12 stories)**
Para cada story: número, texto completo, sugestão de formato (texto, vídeo falando, enquete, caixa de perguntas, contagem regressiva), e intenção estratégica do story.

**2. Story de abertura alternativo**
Uma versão diferente do story 1 para teste A/B.

**3. Checklist de publicação**
Lista de 5 itens para verificar antes de publicar a sequência (link funcionando, urgência real, CTA claro, etc.).
```

## Exemplo de uso

### Input
- Produto: consultoria de posicionamento digital — 1h por R$297
- Preço: R$297
- Público: empreendedores que não sabem como se posicionar nas redes
- Dor: aparecem muito mas vendem pouco
- Resultado: clareza de nicho, bio e estratégia de conteúdo definidos
- Urgência: apenas 3 vagas essa semana
- CTA: DM com a palavra "POSICIONAMENTO"
