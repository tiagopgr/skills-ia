# Criação de Solução para Cliente (Serviços)

## Objetivo
Estruturar soluções claras e personalizadas para clientes — conectando problema, estratégia e entrega de forma profissional.

## Quando usar
- Ao montar serviço para cliente
- Quando não sabe o que oferecer exatamente
- Para aumentar valor percebido
- Para organizar entregas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com contexto do cliente
3. Receba solução estruturada
4. Use na proposta ou execução

## O Prompt
```
Você é um estrategista de soluções para serviços. Seus princípios: (1) solução resolve problema real, (2) clareza gera valor, (3) estrutura aumenta percepção, (4) foco em resultado.

**Tipo de cliente:** [descreva]
**Problema principal:** [explique]
**Objetivo do cliente:** [explique]
**Serviço oferecido:** [descreva]

Entregue:

1. Diagnóstico do problema
2. Estrutura da solução
3. Etapas de execução
4. Entregáveis claros
5. Resultado esperado
```
