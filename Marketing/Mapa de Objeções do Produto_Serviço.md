# Mapa de Objeções do Produto/Serviço

## Objetivo
Criar um mapa completo de todas as objeções que o público-alvo pode ter antes de comprar — classificadas por tipo, gravidade e fase do funil — com respostas prontas para cada uma, usáveis em vendas, copy, FAQ e treinamento.

## Quando usar
- Ao treinar equipe de vendas
- Para criar copy de página de vendas e FAQs
- Em campanhas de remarketing e nutrição de leads
- Antes de lançar um produto (antecipar resistências)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o produto, preço e público
3. Receba o mapa completo de objeções com respostas
4. Use como documento de referência para toda a equipe

## O Prompt
```
Você é um estrategista de vendas que sabe que toda venda perdida é uma objeção não tratada. A maioria dos negócios conhece 3-4 objeções óbvias, mas ignora as 15-20 objeções silenciosas que fazem o lead simplesmente sumir.

Crie o mapa de objeções completo para:

**Produto/Serviço:** [descreva]
**Preço:** [valor]
**Público-alvo:** [quem compra]
**Principais concorrentes:** [se souber]
**Canal de venda:** [site, WhatsApp, call, presencial]

Mapeie as objeções em 6 categorias:
1. Objeções de preço (4-5)
2. Objeções de confiança (4-5)
3. Objeções de timing (3-4)
4. Objeções de adequação (3-4)
5. Objeções de autoridade (2-3)
6. Objeções silenciosas (3-5) — as que o lead PENSA mas não diz

Para CADA objeção:
- Objeção literal (como o cliente fala)
- Objeção real (o que ele realmente quer dizer)
- Resposta para vendas (script)
- Resposta para copy (página de vendas)
- Prova/Evidência que destrói a objeção
```

## Exemplo de uso

### Input
Produto: Software de gestão financeira para PMEs (R$197/mês)
Público: Donos de empresa que usam planilha Excel
Concorrentes: ContaAzul, Nibo, Bling

### Output
Objeção silenciosa (o lead não diz nada — simplesmente não compra):
Objeção real: "Tenho medo de migrar e perder meus dados / não conseguir usar."

Resposta para vendas: "Uma dúvida que muita gente tem mas não fala: 'e se eu não conseguir usar?' Por isso fazemos a migração dos seus dados e treinamos sua equipe nos primeiros 7 dias."

Resposta para copy: Seção "Migração sem dor": "Nós importamos seus dados da planilha, configuramos tudo e treinamos sua equipe."

Gravidade: ALTA — essa objeção é a #1 responsável por leads que pedem demo e somem.
