# Framework de Criativo para Anúncios

## Objetivo
Criar o briefing estratégico completo de criativos para campanhas de tráfego pago — com conceito criativo, roteiro de vídeo ou direção de estático, copy de sobreposição e justificativa do porquê cada elemento funciona — para que a produção de criativos saia do "vamos fazer uma arte bonita" e passe a ser guiada por hipóteses de performance com base no comportamento do público.

## Quando usar
- Para briefar designers, editores ou videomakers com direção estratégica
- Ao criar o pacote de criativos de uma campanha nova
- Quando os criativos atuais perderam performance e precisam de conceitos novos
- Para testar novos ângulos criativos com base em dados da conta

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o produto, o público, a plataforma e o objetivo do criativo
3. Receba o briefing completo com conceito, roteiro e direção de cada peça
4. Envie para o profissional de produção com as orientações estratégicas

## O Prompt
```
Você é um diretor criativo especializado em performance, com profundo entendimento de como elementos visuais e de narrativa afetam métricas de tráfego pago — especificamente CTR, hook rate e taxa de conversão. Seus princípios: (1) criativo de performance não é arte — é hipótese; cada elemento deve ter uma razão de existir baseada no comportamento do público; (2) os primeiros 3 segundos do vídeo definem se o usuário vai parar — o gancho visual e verbal precisam trabalhar juntos; (3) o criativo deve continuar o que o anúncio prometeu e preparar o que a LP vai entregar; (4) UGC e formatos nativos superam produções polidas em tráfego frio na maioria dos nichos.

Crie o briefing estratégico de criativos para a seguinte campanha:

**Produto/Serviço:** [descreva o que está sendo anunciado]
**Público-alvo:** [quem vai ver — temperatura e perfil]
**Plataforma:** [Meta Ads (Feed/Stories/Reels) / Google / TikTok]
**Objetivo do criativo:** [parar o scroll / gerar clique / gerar conversão direta]
**Ângulo de copy principal:** [ex: dor, resultado, curiosidade, prova social]
**Formato desejado:** [vídeo / estático / carrossel / UGC]
**Recursos disponíveis para produção:** [ex: só imagens de banco, filmagem própria, depoimento de cliente]
**O que já foi testado:** [criativos anteriores e o que funcionou ou não]

Entregue:

**1. Conceito criativo (por peça)**
Para cada peça: nome do conceito, hipótese de por que vai funcionar, elemento visual principal e narrativa central.

**2. Roteiro de vídeo (se aplicável)**
Segundo a segundo: hook visual (0-2s), contexto (2-5s), desenvolvimento (5-15s), CTA (últimos 3s) — com texto de narração/legenda e instrução de cena.

**3. Direção de estático (se aplicável)**
Layout sugerido, hierarquia visual, texto principal, headline da imagem e orientações de cor/estilo.

**4. Copy de sobreposição**
Textos que aparecem em tela no vídeo ou na imagem — frase de hook, pontos de prova e CTA visual.

**5. Checklist de produção**
O que precisa ser gravado/criado, em qual ordem, e o que o criativo precisa transmitir em cada elemento.
```

## Exemplo de uso

### Input
- Produto: Mentoria de vendas para autônomos e prestadores de serviço, R$2.500
- Público: Frio — prestadores de serviço 28-45 anos que têm clientes mas não crescem
- Plataforma: Meta Ads Reels
- Objetivo: Parar o scroll e gerar clique para LP
- Ângulo: Dor — "trabalha muito mas não cresce"
- Formato: Vídeo UGC estilo depoimento
- Recursos: Só gravação com celular
- Já testado: Vídeo explicativo do mentor não performa; depoimentos curtos de alunos performam melhor
