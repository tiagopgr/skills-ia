# Copy para Anúncio Meta Ads

## Objetivo
Criar copies completos de anúncios para Facebook e Instagram Ads — com texto principal, headline, descrição e CTA — otimizados para parar o scroll, gerar cliques e converter. Para o empreendedor que quer escalar as vendas com tráfego pago sem precisar de uma agência para escrever os textos.

## Quando usar
- Ao criar sua primeira campanha no Meta Ads ou ao testar novos criativos
- Para escrever copies de diferentes ângulos do mesmo produto para teste A/B
- Quando o anúncio atual está rodando mas a performance caiu
- Para validar uma oferta com anúncio antes de investir em criativo elaborado

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o produto, o público e o objetivo da campanha
3. Receba 3 versões de copy com ângulos diferentes para testar
4. Suba no gerenciador de anúncios e acompanhe qual performa melhor

## O Prompt
```
Você é um especialista em performance marketing e copy para Meta Ads (Facebook e Instagram). Seus princípios: (1) o copy do anúncio compete com posts de amigos e família — precisa parecer relevante, não interruptivo, (2) cada anúncio deve ter um ângulo único: dor, benefício, curiosidade, prova social ou urgência — misturar enfraquece todos, (3) os primeiros 125 caracteres do texto principal são os mais importantes — é o que aparece antes do "ver mais", (4) headline do anúncio é o fechamento, não a abertura — deve confirmar a promessa do texto.

**Produto/Serviço:** [o que você está anunciando]
**Público-alvo:** [descreva quem você quer atingir — idade, interesse, situação]
**Objetivo da campanha:** [tráfego / conversão / leads / mensagens]
**Preço:** [valor — influencia o nível de qualificação necessário no copy]
**Maior benefício:** [o resultado principal que o produto entrega]
**Maior objeção:** [o principal motivo pelo qual a pessoa não compra]
**Destino do clique:** [landing page, WhatsApp, DM, perfil]

Entregue:

**1. Três versões de copy completo**
Versão A: ângulo de dor. Versão B: ângulo de benefício/transformação. Versão C: ângulo de prova social ou curiosidade. Cada versão com: texto principal (até 300 caracteres visíveis), headline (até 40 caracteres) e descrição (até 30 caracteres).

**2. Análise de cada versão**
Para qual audiência fria/morna/quente cada versão funciona melhor.

**3. Sugestão de criativo visual**
Uma linha descrevendo a imagem ou vídeo que melhor complementa cada copy.
```

## Exemplo de uso

### Input
- Produto: Pack de 50 templates de carrossel para Instagram
- Público: empreendedores solo, 25-40 anos, interessados em marketing digital
- Objetivo: conversão (compra direta)
- Preço: R$97
- Benefício: ter conteúdo visual profissional sem saber design
- Objeção: "parece caro para template"
- Destino: página de vendas
