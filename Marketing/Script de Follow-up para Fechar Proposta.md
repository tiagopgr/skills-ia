# Script de Follow-up para Fechar Proposta

## Objetivo
Gerar mensagens de follow-up estratégicas e não invasivas para clientes que receberam sua proposta de branding mas não responderam — para parar de ficar esperando passivamente e começar a converter mais propostas enviadas em projetos fechados, sem parecer desesperado.

## Quando usar
- Quando enviou a proposta e o cliente sumiu por 2 ou mais dias
- Para ter uma sequência de follow-up pronta sem precisar inventar na hora
- Quando tem medo de parecer insistente e acaba não fazendo follow-up
- Para recuperar propostas antigas que ficaram esquecidas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe os dados da proposta e do cliente
3. Receba a sequência completa de 3-4 mensagens com timing sugerido
4. Envie pelo WhatsApp ou e-mail conforme o canal usado com o cliente

## O Prompt
```
Você é um consultor de vendas especializado em freelancers criativos. Seus princípios: (1) follow-up não é chato — é profissional quando bem feito, (2) cada mensagem deve entregar valor ou nova informação, nunca ser só "oi, viu minha proposta?", (3) a sequência deve criar urgência real sem inventar pressão artificial, (4) o objetivo é retomar a conversa, não forçar uma resposta imediata.

Crie uma sequência de follow-up para a seguinte situação:

**Serviço proposto:** [ex: identidade visual completa, rebranding]
**Valor da proposta:** [valor em reais]
**Quando a proposta foi enviada:** [ex: há 3 dias / há 1 semana]
**Meio de comunicação com o cliente:** [WhatsApp / e-mail / Instagram]
**Algo que o cliente mencionou como urgência ou preocupação no briefing:** [ex: "preciso disso antes do lançamento", "estou perdendo clientes por causa da marca"]
**Tom da conversa até agora:** [formal / informal / descontraído]

Entregue:

**1. Mensagem de follow-up 1 (D+2 após envio)**
Curta, leve, agrega valor. Não pergunta só "viu?".

**2. Mensagem de follow-up 2 (D+5)**
Traz um dado, insight ou mini case que reforça o valor do projeto. Gera um pouco mais de urgência.

**3. Mensagem de follow-up 3 (D+9 — "último contato")**
Tom mais direto, deixa a porta aberta mas sinaliza que a proposta tem prazo de validade.

**4. Mensagem de reengajamento (D+20 — se o cliente sumir)**
Para usar semanas depois caso o cliente ressurja ou você queira tentar uma última vez.

**5. O que fazer enquanto espera a resposta**
2-3 ações práticas para não depender de uma única proposta aberta.
```

## Exemplo de uso

### Input
- Serviço: Identidade visual completa
- Valor: R$ 2.500
- Enviada há: 4 dias
- Canal: WhatsApp
- Urgência do cliente: "Preciso lançar o negócio em 2 meses"
- Tom: Informal, já se conhecem um pouco
