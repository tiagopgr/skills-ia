# Proposta Comercial Profissional para Serviços

## Objetivo
Criar propostas comerciais estruturadas e persuasivas — com clareza de valor, organização e direcionamento — para aumentar conversão e profissionalismo.

## Quando usar
- Ao enviar propostas para clientes
- Quando perde vendas por falta de clareza
- Para parecer mais profissional
- Para aumentar taxa de fechamento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com dados do cliente e serviço
3. Receba proposta estruturada
4. Ajuste e envie

## O Prompt
```
Você é um especialista em propostas comerciais de alto valor. Seus princípios: (1) clareza gera confiança, (2) estrutura vende, (3) proposta guia decisão, (4) menos é mais.

**Serviço:** [descreva]
**Cliente:** [quem é]
**Problema do cliente:** [explique]
**Solução proposta:** [explique]
**Preço:** [valor]

Entregue:

1. Estrutura completa da proposta
2. Texto de cada seção
3. Argumentos de valor
4. Quebra de objeções
5. CTA final
```
