# Banco de Respostas a Objeções de Vendas

## Objetivo
Criar um banco completo e estratégico de respostas para as principais objeções que aparecem em negociações de serviços de marketing digital — com argumentos que validam a objeção, reposicionam o valor e encaminham para o fechamento — para nunca mais travar quando o cliente diz "é caro" ou "vou pensar".

## Quando usar
- Durante reuniões de vendas ou negociações para ter respostas prontas
- Para treinar como lidar com objeções antes de uma reunião importante
- Quando perde contratos porque não sabe o que dizer quando o cliente resiste
- Para criar um playbook de vendas que funciona em qualquer situação

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe seu serviço e as objeções mais comuns que você enfrenta
3. Receba o banco completo com resposta para cada objeção
4. Salve e revise antes de reuniões de vendas

## O Prompt
```
Você é um especialista em vendas consultivas com foco em serviços de marketing digital. Seus princípios: (1) objeção é pedido de mais informação, não rejeição — quem não tem interesse não objeta, desaparece, (2) nunca discuta a objeção — valide, reposicione e encaminhe para o próximo passo, (3) a melhor resposta a uma objeção é uma pergunta que faz o cliente pensar, (4) humor leve e confiança transmitem mais credibilidade que argumentos defensivos.

Crie o banco completo de respostas a objeções para este contexto:

**Meu serviço:** [ex: gestão de tráfego pago, consultoria de marketing, social media]
**Ticket médio:** [valor aproximado — para calibrar as respostas de preço]
**Tipo de cliente que mais atendo:** [ex: pequenas empresas locais, e-commerces, profissionais liberais]

Crie respostas completas para estas objeções (e qualquer outra relevante para o seu contexto):

1. "Está caro / não tenho esse orçamento agora"
2. "Vou pensar e te retorno"
3. "Já tenho alguém fazendo isso" (ou "já tentei e não funcionou")
4. "Não sei se marketing digital funciona para o meu negócio"
5. "Posso fazer eu mesmo / tem tutorial no YouTube"
6. "Precisa de quanto tempo para ver resultado?"
7. "Não tenho tempo para te passar as informações que precisa"
8. "Me manda o contrato para eu analisar com calma"
9. "Meu sócio/marido/financeiro precisa aprovar"
10. "Prefiro esperar um momento melhor para investir"

Para cada objeção:
- **Validação:** como reconhecer a objeção sem concordar com ela
- **Reposicionamento:** o argumento que muda a perspectiva
- **Pergunta de retorno:** a pergunta que devolve o controle para você
- **Variação agressiva:** para quando o cliente está prestes a desistir
- **O que nunca dizer:** os erros mais comuns nesta objeção específica
```

### Input
- Serviço: Gestão de tráfego pago para pequenas empresas
- Ticket: R$ 1.800-2.500/mês
- Clientes: Pequenos negócios locais (academia, clínica, restaurante)
