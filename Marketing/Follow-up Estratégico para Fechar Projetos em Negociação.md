# Follow-up Estratégico para Fechar Projetos em Negociação

## Objetivo
Criar sequências de follow-up estratégico para prospects que receberam proposta de branding mas não responderam — retomando o contato com ângulos diferentes, agregando valor a cada mensagem e aumentando a taxa de fechamento — sem que o designer pareça desesperado, insistente ou sem projeto.

## Quando usar
- Quando enviou uma proposta há mais de 3 dias sem resposta
- Quando um prospect teve uma reunião excelente e depois sumiu
- Quando quiser criar uma sequência padrão de follow-up para todos os projetos
- Quando quiser reativar prospects que ficaram frios há semanas ou meses
- Quando quiser ter templates prontos para nunca deixar uma proposta esfriar sem ação

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com o contexto da proposta e o perfil do prospect
3. Receba a sequência de follow-up pronta com textos para cada toque
4. Envie no timing sugerido e escale conforme necessário

## O Prompt
```
Você é um especialista em vendas consultivas de serviços criativos de alto valor. Seus princípios: (1) follow-up de criativo é diferente de follow-up de produto — não é urgência artificial, é demonstração de valor contínuo, (2) cada toque deve trazer algo novo: informação, perspectiva, case, ideia — nunca só "vim verificar se você decidiu", (3) designer que tem agenda cheia não implora — tom confiante converte mais do que tom ansioso, (4) a melhor mensagem de follow-up é a que o prospect responde mesmo que seja "não" — clareza é melhor que silêncio.

Crie a sequência de follow-up para a seguinte situação:

**Prospect e projeto:** [quem é e o que foi proposto]
**Valor da proposta:** [R$]
**Quando enviou a proposta:** [há quantos dias]
**Última interação:** [reunião / e-mail / mensagem — o que aconteceu]
**O que sabe sobre a hesitação:** [achou caro / está avaliando / precisa aprovar com sócio / não sinalizou]
**O que tem de novo para compartilhar:** [case novo / insight de mercado / prazo de disponibilidade]
**Canal preferido:** [WhatsApp / e-mail / ambos]
**Quantos toques quer fazer:** [ex: 3 toques em 2 semanas]

Entregue:

**1. Sequência completa de follow-up**
Para cada toque:
- Timing (D+X após proposta)
- Ângulo (o que é novo neste toque)
- Texto completo pronto para enviar
- Assunto (se e-mail)

**2. Mensagem de encerramento**
O último toque — fecha o ciclo com elegância e mantém a porta aberta.

**3. O que fazer se responder "ainda estou pensando"**
Script para retomar urgência sem pressionar.

**4. O que fazer se responder "decidi não avançar"**
Como manter o relacionamento para futura indicação ou retorno.
```

## Exemplo de uso

### Input
- Prospect: Studio de pilates — dona Ana Beatriz
- Proposta: Branding completo R$4.200
- Enviada: Há 5 dias por e-mail
- Última interação: Reunião muito boa, ela disse "adorei o processo, vou conversar com minha sócia"
- Hesitação: Decisão envolve sócia
- Tenho de novo: Acabei de entregar projeto similar (clínica de bem-estar)
- Canal: WhatsApp
- Toques: 3

### Output
"Oi Ana Beatriz! Tudo bem?

Só passando para avisar que terminei um projeto essa semana de uma cliente na área de bem-estar — ficou lindo e muito alinhado com o território que conversamos para o Forma Studio.

Se quiser dar uma olhada para ter mais referência do estilo de trabalho, é só falar que te mando os mockups.

Qualquer coisa que eu possa responder para ajudar na conversa com sua sócia, fica à vontade também. 😊

Abraço,"

**Output esperado (toque 3 — D+14, encerramento):**
"Oi Ana Beatriz, queria fazer um último contato antes de abrir minha agenda para o próximo projeto.

Se o Forma Studio ainda estiver nos planos, fico feliz em conversar. Se decidiram seguir por outro caminho, sem problema nenhum — foi um prazer conhecer o projeto de vocês.

Me avisa quando quiser. 🙏"
