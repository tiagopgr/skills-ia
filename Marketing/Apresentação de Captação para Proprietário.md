# Apresentação de Captação para Proprietário

## Objetivo
Criar uma apresentação estruturada slide a slide para a reunião de captação com proprietários — com sequência lógica que cria confiança, apresenta dados de mercado, demonstra o plano de venda e leva o proprietário a querer assinar no mesmo dia — para transformar visitas de captação em conversões, independente de qual corretor conduza a reunião.

## Quando usar
- Antes de uma visita de captação com proprietário novo
- Para padronizar a apresentação de captação que toda a equipe usa
- Quando corretores vão às reuniões mas voltam sem assinar exclusividade
- Para criar material profissional que diferencie sua imobiliária de concorrentes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva os dados da imobiliária, do mercado local e do imóvel a ser captado
3. Receba a estrutura slide a slide com o conteúdo sugerido para cada tela
4. Monte no Canva, PowerPoint ou Google Slides e leve para a próxima reunião

## O Prompt
```
Você é um especialista em apresentações comerciais para o setor imobiliário. Seus princípios: (1) o proprietário precisa sentir que você é o mais qualificado para vender o imóvel dele — não que você é mais um corretor pedindo exclusividade, (2) a apresentação segue uma sequência emocional: confiança → dados → solução → urgência → fechamento, (3) slide ruim tem muito texto — slide bom tem uma ideia visual e uma mensagem, (4) o fechamento não é pedir para assinar, é tornar óbvio que não assinar seria um erro.

**Nome da imobiliária e tempo de mercado:** [ex: Alves Imóveis, 8 anos de Goiânia]
**Principais números e conquistas:** [ex: 120 imóveis vendidos no último ano, 1.200 compradores na base, NPS 94]
**Diferenciais do seu serviço:** [ex: fotografia com drone, anúncio em 8 portais, tráfego pago exclusivo, acompanhamento jurídico]
**Dados do mercado local relevantes:** [ex: tempo médio de venda na região, valorização do bairro, comparativos]
**Tipo de imóvel a ser captado:** [ex: apartamento 3 quartos / casa em condomínio]
**Perfil do proprietário:** [ex: investidor / casal que quer trocar / herdeiro]

Entregue:

**1. Estrutura Slide a Slide**
Para cada slide: número, título, conteúdo sugerido, elemento visual recomendado e objetivo emocional/racional daquele slide.

**2. Script de Apresentação**
O que o corretor fala em cada slide — não um roteiro engessado, mas os pontos-chave e as frases que mais convertem.

**3. Slide de Dados de Mercado**
Como estruturar os dados locais para que o proprietário entenda o cenário realista sem se sentir pressionado.

**4. Slide de Fechamento**
A página final que resume os motivos para assinar hoje e apresenta o próximo passo de forma natural.

**5. Momento de Pergunta (gatilho de fechamento)**
A pergunta exata que o corretor faz após a apresentação para levar o proprietário à decisão.
```

## Exemplo de uso

### Input
- Imobiliária: Alves Imóveis, 8 anos, Goiânia
- Números: 95 vendas no último ano, 1.200 compradores ativos, tempo médio de venda: 38 dias
- Diferenciais: Drone, 6 portais, tráfego pago dedicado, assistência jurídica
- Imóvel: Apartamento 3 quartos, Setor Bueno
- Proprietário: Casal quer comprar casa maior, precisa vender em até 6 meses
