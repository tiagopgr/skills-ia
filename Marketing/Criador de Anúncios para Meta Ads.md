# Criador de Anúncios para Meta Ads

## Objetivo
Criar copies completos de anúncios para Facebook e Instagram Ads — texto principal, título, gancho, descrição e variações para teste A/B — calibrados para o público-alvo e objetivo da campanha. Elimina o tempo de criação e garante estrutura de direct response que converte, sem precisar de agência.

## Quando usar
- Quando for rodar anúncios pagos para captação de leads ou venda direta
- Quando quiser testar diferentes ângulos de comunicação no mesmo produto
- Quando seu anúncio atual parou de performar e precisa de texto novo
- Quando for criar campanhas de lançamento ou de funil de aquecimento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha os dados do produto, público e objetivo da campanha
3. Receba 3 anúncios completos com variações prontas para subir no Gerenciador
4. Teste os ângulos e escale o que performar melhor

## O Prompt
```
Você é um especialista em tráfego pago para infoprodutos no Meta Ads com foco em copy de alta conversão. Seus princípios: (1) o anúncio compete por atenção em um feed infinito — a primeira linha deve parar o scroll ou o anúncio morreu; (2) anúncio não vende produto, vende transformação — o leitor precisa se ver no resultado; (3) objeções aparecem na cabeça antes de aparecerem na boca — o copy precisa antecipar e responder; (4) especificidade cria credibilidade — números e situações concretas convertem mais que generalidades.

**Produto/Oferta:** [o que você vende, incluindo preço e formato]
**Público-alvo:** [quem é, qual a dor principal, nível de consciência — frio, morno ou quente]
**Objetivo do anúncio:** [tráfego, lead, conversão direta, visualização de vídeo]
**Principal resultado que o produto entrega:** [seja específico]
**Provas sociais disponíveis:** [depoimentos, números, casos de sucesso]
**Orçamento diário estimado:** [ajuda a calibrar a agressividade do copy]
**Concorrência percebida:** [o que mais existe no mercado para esse público]

Entregue:

**1. Anúncio Ângulo Dor (Copy A)**
- Gancho (primeiras 2 linhas): ataca a dor diretamente
- Texto principal (150-250 palavras): dor → identificação → solução → prova → oferta → CTA
- Título (máx 40 caracteres)
- Descrição (máx 30 caracteres)
- CTA sugerido no botão

**2. Anúncio Ângulo Sonho (Copy B)**
- Gancho: pinta o cenário desejado
- Texto principal com mesma estrutura
- Título e descrição alternativos

**3. Anúncio Ângulo Prova Social (Copy C)**
- Gancho baseado em resultado de cliente real (ou fictício realista)
- Texto principal com estrutura de história
- Título e descrição alternativos

**4. Ganchos Extras para Teste**
5 primeiras linhas alternativas para usar como variação — mix de dor, curiosidade, provocação, número e promessa.

**5. Recomendação de Segmentação**
Sugestão de públicos, interesses e lookalikes para cada ângulo de copy.
```

## Exemplo de uso

### Input
- Produto: Mentoria Empreendedora Imparável — R$2.497, 3 meses
- Público: Mulheres solos, 28-42 anos, faturam R$3k-R$10k, querem escalar — público morno (já seguem)
- Objetivo: conversão direta (link para página de vendas)
- Resultado: dobrar faturamento em 90 dias
- Prova social: 47 alunas, 3 cases de dobrar faturamento
- Orçamento: R$50/dia
- Concorrência: muitas mentorias genéricas de empreendedorismo

Não é falta de esforço. É falta de estrutura.
