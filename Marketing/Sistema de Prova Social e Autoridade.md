# Sistema de Prova Social e Autoridade

## Objetivo
Aumentar confiança do cliente com provas e construção de autoridade.

## Quando usar
- Para melhorar conversão
- Para páginas de produto
- Para campanhas

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em persuasão. Princípios: prova, confiança, clareza.

**Produto:** [descreva]

Entregue:
1. Tipos de prova social
2. Como coletar
3. Como aplicar
4. Exemplos
```
