# Script de Primeira Reunião com Cliente

## Objetivo
Gerar um roteiro estruturado para a primeira reunião com um cliente de branding — com as perguntas certas, a ordem certa e as transições certas — para conduzir a conversa com confiança, entender o que o cliente realmente precisa e já criar autoridade antes de enviar a proposta.

## Quando usar
- Antes de qualquer reunião de apresentação ou descoberta com um novo cliente
- Quando trava na hora de conduzir a conversa e acaba deixando o cliente falar solto
- Para substituir reuniões sem foco que terminam sem próximo passo claro
- Para preparar reuniões online (Meet, Zoom) ou presenciais

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto do cliente e do projeto
3. Receba o script completo com falas, perguntas e transições
4. Imprima ou deixe aberto no celular durante a reunião

## O Prompt
```
Você é um consultor de branding experiente que também treina freelancers criativos. Seus princípios: (1) quem conduz a reunião controla a percepção de valor, (2) perguntas inteligentes demonstram mais expertise que respostas prontas, (3) a primeira reunião deve terminar com o cliente animado, não com ele pedindo orçamento, (4) silêncio e escuta ativa valem mais que falar muito.

Crie um script completo para a primeira reunião com este cliente:

**Tipo de negócio do cliente:** [ex: nutricionista autônoma, loja de roupas, startup de tecnologia]
**O que o cliente mencionou que precisa:** [o que ele disse no primeiro contato]
**Formato da reunião:** [online ou presencial — duração prevista]
**Serviço que você oferece:** [ex: identidade visual, branding completo, rebranding]
**Seu nível de experiência:** [iniciante / intermediário]

Entregue:

**1. Abertura (2-3 minutos)**
Fala de boas-vindas que cria rapport e define a estrutura da reunião.

**2. Fase de descoberta (10-15 minutos)**
5-7 perguntas estratégicas em ordem lógica, com a transição entre cada uma. Inclua por que cada pergunta importa (em itálico).

**3. Fase de apresentação da sua abordagem (5 minutos)**
Fala curta explicando como você trabalha — sem parecer pitch de venda, mas já posicionando seu método.

**4. Apresentação de cases ou exemplos (3-5 minutos)**
Instrução de como mostrar trabalhos anteriores (mesmo que poucos) de forma estratégica.

**5. Próximos passos e fechamento (2-3 minutos)**
Fala para encerrar a reunião com um compromisso claro: quando você envia a proposta, o que o cliente deve fazer, qual o prazo.

**6. Follow-up pós-reunião**
Mensagem de WhatsApp ou e-mail para enviar em até 2 horas após a reunião.
```

## Exemplo de uso

### Input
- Cliente: Personal trainer autônomo
- O que disse: "Preciso de um logo, minha marca está bagunçada"
- Formato: Online, 30 minutos
- Serviço: Identidade visual completa
- Experiência: Iniciante
