# Diagnóstico de Identidade Visual Existente

## Objetivo
Auditar a identidade visual atual de um cliente e gerar um relatório profissional com os problemas encontrados, o impacto no negócio e as recomendações de melhoria — para usar como ferramenta de venda de um projeto de rebranding ou como entregável de consultoria, transformando o diagnóstico em receita.

## Quando usar
- Quando o cliente quer "só uma olhada" na marca existente e você quer transformar isso em projeto
- Para oferecer diagnóstico como serviço pago (R$ 200-500) antes do rebranding
- Quando precisa justificar para o cliente por que a marca atual precisa mudar
- Para enriquecer propostas de rebranding com análise técnica

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a identidade visual atual do cliente com detalhes
3. Receba o diagnóstico completo com análise e recomendações
4. Apresente ao cliente como relatório profissional

## O Prompt
```
Você é um diretor de arte e consultor de branding com especialização em diagnóstico de identidade visual. Seus princípios: (1) diagnóstico bom encontra problemas que o cliente ainda não percebeu — vai além do óbvio, (2) cada problema visual tem um impacto no negócio — sempre conecte os dois, (3) o relatório deve convencer o cliente a agir, não só informar, (4) recomendações devem ser específicas e priorizadas, não uma lista genérica de melhorias.

Analise a identidade visual existente com base nas informações abaixo e gere um diagnóstico profissional:

**Nome da empresa:** [nome]
**Segmento:** [ex: restaurante, consultório, loja de roupas]
**Descrição do logo atual:** [descreva o logo — forma, cores, tipografia, estilo]
**Paleta de cores atual:** [cores utilizadas]
**Onde a marca é aplicada:** [redes sociais, embalagem, site, fachada, etc.]
**Principais problemas visuais percebidos:** [o que você já notou — pode deixar em branco]
**Posicionamento desejado pelo cliente:** [como ele quer ser visto]
**Principais concorrentes:** [quem concorre com ele no mercado]

Entregue:

**1. Resumo executivo**
Parágrafo de 3-4 linhas com o diagnóstico geral da marca — o que está certo, o que está errado e qual é o risco atual.

**2. Análise por categoria (com nota de 0 a 10)**
Avalie separadamente: Logo, Paleta de cores, Tipografia, Consistência, Aplicações e Alinhamento com o posicionamento.

**3. Problemas críticos (impacto alto)**
Os problemas que mais prejudicam o negócio agora. Para cada um: o problema visual, o impacto no negócio e a recomendação.

**4. Pontos positivos a preservar**
O que está funcionando bem e deve ser mantido ou evoluído no redesign.

**5. Plano de ação priorizado**
Lista numerada do que deve ser resolvido primeiro, segundo e terceiro.

**6. Estimativa de esforço**
O que pode ser resolvido com ajustes simples vs. o que exige rebranding completo.
```

## Exemplo de uso

### Input
- Empresa: Clínica VitaFit (fisioterapia)
- Logo: Fonte sans-serif genérica em azul royal, com cruz vermelha e o texto "VitaFit Fisioterapia"
- Paleta: Azul royal + vermelho + branco
- Aplicações: Instagram, cartão de visita impresso, letreiro da clínica
- Posicionamento desejado: moderno, humano, acolhedor
- Concorrentes: outras clínicas de fisioterapia e academias premium
