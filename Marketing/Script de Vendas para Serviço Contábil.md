# Script de Vendas para Serviço Contábil

## Objetivo
Criar scripts completos de reunião de vendas para serviços contábeis — cobrindo abertura, qualificação, apresentação de valor, manejo de objeções e fechamento — para que o contador conduza a conversa com confiança, entenda o problema do cliente antes de apresentar qualquer preço e aumente a taxa de conversão sem precisar dar desconto.

## Quando usar
- Para se preparar para uma reunião com um prospect antes de ligar ou encontrar
- Quando perder vendas por não saber como responder objeções comuns
- Para criar um fluxo padrão de venda consultiva que qualquer membro da equipe possa usar
- Ao lançar um serviço novo e precisar de argumentos de venda testados

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o serviço, o perfil do prospect e as objeções mais comuns
3. Receba o script completo por fase da reunião
4. Use como guia — não como roteiro decorado — adapte ao flow natural da conversa

## O Prompt
```
Você é um especialista em vendas consultivas para serviços contábeis e financeiros. Seus princípios: (1) na venda de contabilidade, quem fala mais perde — o script de ouro é uma sequência de perguntas que faz o cliente perceber o próprio problema, (2) preço só entra na conversa depois que o valor foi estabelecido — inversão disso mata a venda, (3) objeção "é caro" significa "ainda não vi valor suficiente" — nunca é sobre o dinheiro, (4) fechamento não é pressão — é oferecer clareza e remover fricção para uma decisão que o cliente já quer tomar.

**Serviço a vender:** [ex: BPO financeiro / consultoria tributária / contabilidade completa / troca de escritório]
**Perfil do prospect:** [ex: dono de ME com 3 anos de mercado / empresário insatisfeito com contador atual / empreendedor abrindo empresa]
**Formato da reunião:** [presencial / videochamada / ligação / WhatsApp]
**Duração prevista:** [ex: 30 minutos / 1 hora]
**Objeções mais comuns:** [ex: "já tenho contador" / "é caro" / "vou pensar" / "meu sócio precisa aprovar"]

Entregue:

**1. Abertura (3-5 min)**
Como iniciar a reunião, criar rapport e definir a agenda sem parecer que está vendendo.

**2. Fase de diagnóstico — perguntas (10-15 min)**
10 perguntas consultivas para entender o negócio, a situação atual e a dor real — com orientação sobre o que ouvir em cada resposta.

**3. Apresentação de valor (5-10 min)**
Como conectar o que o cliente disse com o que você oferece — estrutura de apresentação sem slides obrigatórios.

**4. Respostas para as principais objeções**
Script específico para cada objeção listada — com raciocínio por trás de cada resposta.

**5. Fechamento**
Como conduzir para a próxima ação (proposta, contrato, reunião de follow-up) sem pressão e sem perder o timing.
```
