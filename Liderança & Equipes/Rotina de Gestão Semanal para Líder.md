# Rotina de Gestão Semanal para Líder

## Objetivo
Criar uma rotina semanal estruturada — garantindo controle do negócio, equipe e prioridades.

## Quando usar
- Para organizar semana
- Para acompanhar equipe
- Para manter clareza
- Para melhorar gestão

## Como usar
1. Copie o prompt
2. Descreva rotina atual
3. Receba rotina
4. Aplique semanalmente

## O Prompt
```
Você é um gestor experiente. Seus princípios: (1) rotina gera controle, (2) clareza melhora decisão, (3) consistência.

**Rotina atual:** [descreva]

Entregue:

1. Estrutura semanal
2. Blocos de gestão
3. Momentos de revisão
4. Prioridades
```
