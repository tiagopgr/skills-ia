# Organização de Fluxo de Trabalho da Equipe

## Objetivo
Definir como o trabalho flui dentro da equipe.

## Quando usar
- Quando há confusão de tarefas
- Para melhorar organização
- Para evitar retrabalho

## Como usar
1. Copie o prompt
2. Descreva equipe
3. Receba fluxo
4. Aplique

## O Prompt
```
Você é um gestor de processos. Princípios: (1) clareza, (2) fluxo definido, (3) eficiência.

**Equipe:** [descreva]

Entregue:
1. Fluxo de trabalho
2. Etapas
3. Responsáveis
4. Melhorias
```
