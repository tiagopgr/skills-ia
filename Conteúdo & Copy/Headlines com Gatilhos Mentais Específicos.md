# Headlines com Gatilhos Mentais Específicos

## Objetivo
Criar headlines sob demanda usando um gatilho mental específico escolhido pelo usuário (escassez, autoridade, reciprocidade, prova social, urgência, antecipação, exclusividade, etc.).

## Quando usar
- Quando você sabe qual gatilho mental quer usar mas precisa de boas frases
- Para campanhas com estratégia emocional definida
- Para variar os tipos de headline nas suas comunicações
- Em testes A/B comparando diferentes gatilhos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Escolha o gatilho mental desejado
3. Preencha o contexto da oferta
4. Receba headlines calibradas para aquele gatilho específico

## O Prompt
```
Você é um copywriter especializado em psicologia da persuasão e neuromarketing. Você domina os principais gatilhos mentais e sabe como aplicá-los em headlines de forma ética e eficaz.

Crie 8 headlines usando EXCLUSIVAMENTE o seguinte gatilho mental:

**Gatilho escolhido:** [escolha um: escassez | urgência | prova social | autoridade | reciprocidade | antecipação | exclusividade | curiosidade | medo de perder (FOMO) | pertencimento | novidade | simplicidade]

**Contexto:**
- Produto/Serviço: [descreva]
- Público-alvo: [descreva]
- Canal de uso: [landing page, email, anúncio, post, etc.]
- Tom: [ex: agressivo, sutil, elegante, casual]

Para cada headline:

1. **A headline** (clara, direta, impactante)
2. **Onde usar** (melhor canal/contexto para essa headline)
3. **Por que funciona** (explicação breve de como o gatilho opera nessa frase)
4. **Nível de intensidade** (leve, moderado, forte) — para evitar ser apelativo demais
5. **Variação mais sutil** (versão da mesma headline com intensidade menor, para públicos mais sensíveis)

No final, explique:
- Quando esse gatilho funciona melhor
- Quando NÃO usar esse gatilho (contextos em que pode ser negativo)
- Com qual outro gatilho ele combina bem para potencializar o efeito
```

## Exemplo de uso

### Input
Gatilho: Prova Social
Produto: Software de gestão de projetos
Público: Gerentes de equipes remotas
Canal: Landing page
Tom: Profissional e confiável

### Output
- "12.500 equipes remotas já migraram para o [Produto]. Sua equipe é a próxima?"
  - Onde: Hero da landing page
  - Por que funciona: Número grande + identificação com "equipes remotas" + convite implícito
  - Intensidade: Moderada
  - Versão sutil: "Usado por milhares de equipes para organizar projetos remotos"
