# Sistema de Modelos Reutilizáveis de Copy

## Objetivo
Reduzir tempo de escrita criando templates reutilizáveis para e-mails, respostas e posts.

## Quando usar
- Muito tempo escrevendo do zero
- Necessidade de padrão de comunicação
- Alto volume de mensagens

## Como usar
1. Copie o prompt
2. Informe tipos de mensagem
3. Receba templates
4. Salve e reutilize

## O Prompt
```
Você é um copywriter operacional. Princípios: clareza, objetividade, reutilização.

Tipos de mensagem: [liste]
Tom de voz: [defina]

Entregue:
1. 10 templates prontos
2. Variáveis para personalizar
3. Versões curta e longa
4. CTAs sugeridos
```
