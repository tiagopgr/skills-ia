# Script de Stories com CTA

## Objetivo
Criar sequências de Stories para Instagram com roteiro estratégico que conduzem o espectador por uma narrativa curta e terminam com uma chamada para ação clara e eficaz.

## Quando usar
- Para direcionar tráfego ao link da bio, WhatsApp ou página de vendas
- Para aquecer a audiência antes de um lançamento
- Para promover conteúdo, produto ou oferta nos Stories
- Para criar sequências de Stories que não parecem propaganda

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o objetivo e contexto da sequência de Stories
3. Receba o roteiro story por story
4. Grave e publique seguindo a sequência

## O Prompt
```
Você é um estrategista de Instagram Stories que entende como usar Stories para gerar ação (cliques, respostas, compras) sem parecer forçado. Você sabe que Stories são consumidos rapidamente (1.7 segundos em média por story), então cada frame precisa ser simples, visual e direto.

Crie uma sequência de Stories com CTA para:

**Objetivo:** [vender produto | direcionar para link | gerar DMs | promover post | aquecer para lançamento | coletar respostas]
**O que está promovendo:** [descreva]
**Público:** [quem assiste seus Stories]
**Tom:** [casual, profissional, empolgado, íntimo, urgente]
**Número de Stories na sequência:** [5 | 7 | 10]
**CTA final desejado:** [link na bio | responder stories | clicar no link | mandar DM]

Entregue a sequência, story por story:

Para CADA story:
- **Formato:** [texto na tela | selfie video | foto + texto | enquete | quiz | caixinha | contagem regressiva | link]
- **Texto na tela:** (máximo 3 linhas, fonte grande)
- **Texto falado (se vídeo):** roteiro do que falar (máximo 15 segundos)
- **Elemento interativo:** [enquete, quiz, emoji slider, caixinha — se aplicável]
- **Sticker/recurso:** [música, GIF, localização, menção — se aplicável]
- **Objetivo desse story específico:** [curiosidade, engajamento, informação, CTA]

Estrutura geral da sequência:
- **Stories 1-2:** Hook — capturar atenção, gerar curiosidade
- **Stories 3-5:** Valor — informação, contexto, prova
- **Stories 6-7:** CTA — direcionamento claro para ação

Regras:
- Cada story deve funcionar em menos de 5 segundos de leitura
- Textos curtos e fonte grande (legível sem pausar)
- Variar formatos (não fazer 7 stories de texto puro)
- O CTA deve parecer consequência natural, não interrupção
```

## Exemplo de uso

### Input
Objetivo: Direcionar para inscrição de masterclass gratuita
Promovendo: Masterclass "Como criar seu primeiro produto digital"
Público: Aspirantes a empreendedores digitais
Tom: Empolgado e acessível
Número de Stories: 7
CTA: Link na bio

### Output
**Story 1:** [Selfie video] "Eu ia compartilhar isso só com meus alunos, mas decidi abrir pra todo mundo."

**Story 2:** [Texto na tela] "Na quarta-feira vai rolar uma masterclass GRATUITA" + contagem regressiva

**Story 3:** [Enquete] "Você já pensou em criar um produto digital?" Sim / Ainda não

**Story 4:** [Selfie video] "Nessa masterclass eu vou mostrar os 3 passos pra criar seu primeiro produto do zero."

**Story 5:** [Texto na tela] "Ao vivo. Gratuita. Quarta às 20h. Vagas limitadas."

**Story 6:** [Foto + texto] "Quem já se inscreveu no último evento disse isso:" + print de depoimento

**Story 7:** [Link] "Inscreva-se agora — link aqui" + sticker de link + seta apontando
