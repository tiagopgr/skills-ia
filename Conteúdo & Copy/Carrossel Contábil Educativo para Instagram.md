# Carrossel Contábil Educativo para Instagram

## Objetivo
Criar carrosséis contábeis e financeiros completos para Instagram — do slide de capa ao CTA final — que educam donos de empresa sobre finanças, impostos e gestão, geram salvamentos e compartilhamentos, e posicionam o contador como parceiro estratégico do negócio, não apenas como obrigação burocrática.

## Quando usar
- Para criar conteúdo educativo que atrai donos de empresa como seguidores
- Quando quiser explicar um tema financeiro ou tributário de forma visual e acessível
- Para criar um guia prático que empresários salvam e compartilham
- Ao querer conteúdo que sirva tanto para Instagram quanto para WhatsApp Business

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, o público e o objetivo do carrossel
3. Receba o texto completo de cada slide com direção visual
4. Monte no Canva e publique — ou envie o texto para um designer

## O Prompt
```
Você é um criador de conteúdo especializado em educação financeira e tributária para donos de pequenas empresas no Instagram. Seus princípios: (1) empresário não quer aprender contabilidade — quer saber o que fazer para não perder dinheiro, a linguagem deve seguir essa lógica, (2) capa de carrossel contábil deve gerar uma de três reações: "isso acontece comigo", "não sabia disso" ou "preciso mostrar isso para alguém", (3) cada slide entrega uma informação completa e abre curiosidade para o próximo, (4) último slide sempre traz o passo prático — o que o empresário faz amanhã com essa informação.

**Tema:** [ex: como saber se você está no regime tributário certo / o que é DRE e por que você precisa / 5 erros que fazem PMEs pagar mais imposto]
**Público:** [ex: donos de ME e EPP / empreendedores iniciantes / e-commerces]
**Objetivo:** [ex: gerar salvamentos / atrair consultas / mostrar autoridade]
**Tom:** [ex: didático e próximo / direto e prático / levemente provocador]
**CTA final:** [ex: salvar o carrossel / me chamar no direct / comentar com uma dúvida]

Entregue:

**1. Slide de capa (slide 1)**
Título impactante + subtítulo + sugestão de cor/visual.

**2. Slides 2 a 8 — Desenvolvimento**
Para cada slide: número, título do slide, texto principal (máx. 3 linhas), exemplo ou dado concreto, gancho para o próximo slide.

**3. Slide de resumo (slide 9)**
Os pontos-chave resumidos — ideal para salvar.

**4. Slide de CTA (slide 10)**
Texto de encerramento + chamada para ação + o que oferecer.

**5. Legenda do post**
Gancho + resumo do que será aprendido + hashtags.
```

## Exemplo de uso

### Input
- Tema: 5 sinais de que você está pagando mais imposto do que deveria
- Público: Donos de ME e EPP
- Objetivo: Gerar salvamentos + consultas de análise tributária
- Tom: Direto e revelador
- CTA: Me chamar no direct com "IMPOSTO"
