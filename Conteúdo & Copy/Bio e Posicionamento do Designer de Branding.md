# Bio e Posicionamento do Designer de Branding

## Objetivo
Criar textos de bio e posicionamento profissional para o designer de branding — para Instagram, LinkedIn, site e propostas — que comunicam valor estratégico antes de estética, deixam claro quem é o cliente ideal e diferenciam o profissional de outros designers que só mostram portfólio — para que o perfil do designer trabalhe como captação passiva 24 horas por dia.

## Quando usar
- Quando quiser atualizar o Instagram ou LinkedIn com um perfil mais estratégico
- Quando precisar de uma bio para o site ou portfólio
- Quando sentir que atrai cliente errado ou cliente que pechincha
- Quando quiser reposicionar para um nicho específico
- Quando for criar ou reformular a apresentação profissional para propostas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com suas informações profissionais e o que te diferencia
3. Receba versões de bio para diferentes canais
4. Personalize e publique

## O Prompt
```
Você é um especialista em personal branding para designers criativos e profissionais de branding. Seus princípios: (1) bio de designer que só fala de estética atrai cliente que só quer estética — se quer cliente que paga por estratégia, posicione por estratégia, (2) "designer gráfico" é invisível — especialidade específica cria percepção de expertise e justifica preço premium, (3) resultado concreto na bio converte mais que qualquer prêmio — "criei a marca de 40+ negócios que cresceram depois do rebrand" é mais forte que "premiado pela ABD", (4) tom da bio define o cliente que vai chegar — sofisticado atrai sofisticado, genérico atrai qualquer um.

Crie minha bio e posicionamento com as seguintes informações:

**Meu nome:** [nome]
**Minha especialidade:** [ex: branding estratégico / identidade visual / rebrand / design para profissionais liberais]
**Nicho de cliente que quero atrair:** [ex: empreendedores premium / clínicas de saúde / consultores / startups]
**Principal resultado que entrego:** [o que muda para quem contrata]
**Minha diferença:** [o que faço diferente de outros designers — processo, nicho, metodologia]
**Números ou resultados concretos:** [ex: projetos feitos, anos de experiência, tipo de cliente atendido]
**Tom desejado:** [ex: sofisticado e estratégico / acessível e empático / provocador e direto]
**Onde vai ser usada:** [Instagram bio / LinkedIn / site / proposta / todos]

Entregue:

**1. Bio do Instagram (até 150 caracteres)**
Específica, com especialidade e para quem — sem usar palavras genéricas.

**2. Sobre mim para LinkedIn (200-280 palavras)**
Em primeira pessoa — especialidade, para quem, como funciona o processo e o que muda.

**3. Headline de LinkedIn**
Até 220 caracteres — com especialidade, nicho e resultado.

**4. Tagline criativa**
Uma frase de posicionamento para usar em qualquer lugar.

**5. Texto "sobre mim" para proposta (100 palavras)**
Focado em credibilidade e relevância para o cliente específico.

**6. Elevator pitch (30 segundos)**
Como responder "o que você faz?" de forma que o outro queira saber mais.
```

## Exemplo de uso

### Input
- Nome: Thiago Tavares
- Especialidade: Branding estratégico — especializado em identidade visual para profissionais e negócios em crescimento
- Nicho: Consultores, coaches, profissionais liberais e negócios de serviços premium
- Resultado: Marca que combina com o nível de serviço entregue e atrai o cliente certo
- Diferença: Processo estratégico antes do criativo — pesquisa de mercado, briefing profundo, narrativa de apresentação
- Números: 50+ projetos, diversos nichos de serviços
- Tom: Sofisticado, estratégico, direto

### Output
"Branding estratégico para quem cobra caro e precisa que a marca confirme isso 🎯
Logo não. Identidade."

**Output esperado (tagline):**
"Sua marca precisa parecer tão boa quanto o que você entrega."

**Output esperado (headline LinkedIn):**
"Designer de Branding Estratégico | Identidade visual para profissionais premium | Processo que vai além do logo | 50+ projetos"
