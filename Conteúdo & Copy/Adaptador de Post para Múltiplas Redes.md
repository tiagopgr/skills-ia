# Adaptador de Post para Múltiplas Redes

## Objetivo
Pegar um post que funciona bem em uma rede social e adaptá-lo para outras plataformas, respeitando formato, tom, limites e comportamento do público de cada uma.

## Quando usar
- Para reaproveitar conteúdo bom em múltiplas redes
- Quando quer presença multi-canal sem criar tudo do zero
- Para maximizar o alcance de um conteúdo que performou bem
- Ao publicar simultaneamente em várias plataformas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o post original e indique a rede de origem
3. Selecione para quais redes quer adaptar
4. Receba versões prontas para cada plataforma

## O Prompt
```
Você é um estrategista de conteúdo multi-plataforma que entende as diferenças fundamentais de cada rede social: formato, limite de caracteres, comportamento do público, tom esperado e o que performa melhor em cada uma.

Adapte o seguinte post para múltiplas redes sociais:

**Post original:**
"""
[cole o post completo aqui — legenda, texto, conteúdo]
"""

**Rede de origem:** [Instagram, LinkedIn, Twitter, TikTok, YouTube, etc.]
**Performance:** [como performou — bom/ótimo/mediano]

Adapte para todas as redes abaixo:

**1. Instagram (Feed)**
- Formato: Legenda longa OK, visual é essencial
- Tom: Pessoal, visual, aspiracional
- Limite: 2.200 caracteres
- Entregue: Legenda + sugestão visual + hashtags (15-20)

**2. LinkedIn**
- Formato: Texto puro ou texto + imagem
- Tom: Profissional mas humano, insights de negócios
- Limite: 3.000 caracteres (engajamento cai após 1.300)
- Entregue: Post completo + hook forte na primeira linha

**3. Twitter/X**
- Formato: Tweet único (280 chars) ou thread
- Tom: Conciso, espirituoso, direto
- Limite: 280 caracteres por tweet
- Entregue: Versão tweet único + versão thread (se o conteúdo pedir)

**4. TikTok/Reels**
- Formato: Roteiro de vídeo curto (15-60s)
- Tom: Dinâmico, autêntico, nativo da plataforma
- Entregue: Script com marcação de tempo + texto na tela + hook

**5. Newsletter/Email**
- Formato: Texto para email pessoal
- Tom: Íntimo, como se falasse para 1 pessoa
- Entregue: Assunto + corpo do email

Para cada adaptação:
- O conteúdo adaptado (pronto para usar)
- O que mudou em relação ao original e por quê
- Melhor horário para publicar nessa rede
- Uma dica específica da plataforma
```

## Exemplo de uso

### Input
Post original (Instagram carrossel que performou bem):
"5 erros que freelancers cometem na precificação:
1. Cobrar por hora em vez de por valor
2. Não calcular custos fixos
3. Dar desconto por insegurança
4. Comparar preço com iniciantes
5. Não reajustar anualmente
Salve esse post e compartilhe com um freelancer que precisa ver isso."

### Output
**Twitter:** "Freelancers: parem de cobrar por hora. Cobrem por valor. Seu cliente não paga pelo tempo, paga pelo resultado. (e mais 4 erros de precificação no thread abaixo)"

**LinkedIn:** "Nos últimos 5 anos trabalhando com freelancers, vi os mesmos 5 erros de precificação se repetirem. O mais comum? Cobrar por hora. O mercado mudou..."

**TikTok:** "[0-3s] 'Se você é freelancer e cobra por hora, me desculpa...' [3-15s] explica os 5 erros rápido..."
