# Seção de Benefícios com Bullet Points

## Objetivo
Criar a seção de benefícios de uma landing page ou página de vendas com bullet points persuasivos que traduzem funcionalidades em resultados concretos para o cliente.

## Quando usar
- Na seção de benefícios de qualquer landing page
- Em páginas de venda (seção 'O que você recebe')
- Em emails de venda para listar benefícios
- Em propostas comerciais

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste as funcionalidades ou características do produto
3. Receba bullet points transformados em benefícios
4. Insira na landing page ou material de venda

## O Prompt
```
Você é um copywriter que domina a técnica de transformar funcionalidades em benefícios. Você sabe que o cliente não compra funcionalidades — compra resultados. "Tem 50GB de armazenamento" é funcionalidade. "Guarde todas as fotos da sua vida sem se preocupar com espaço" é benefício.

Crie bullet points de benefícios para:

**Produto/Serviço:** [descreva]
**Público-alvo:** [quem vai ler]
**Funcionalidades/Características para transformar em benefícios:**
1. [funcionalidade 1]
2. [funcionalidade 2]
3. [funcionalidade 3]
(liste de 4 a 10)

**Tom:** [ex: empolgado, sóbrio, profissional, casual]
**Onde será usado:** [landing page, email, proposta, anúncio]

Para CADA funcionalidade, entregue 3 versões de bullet point:

**Versão A — Resultado puro**
Foco no que o cliente ganha. Sem mencionar a funcionalidade.

**Versão B — Funcionalidade > Benefício**
Menciona a funcionalidade e traduz no resultado.

**Versão C — Dor eliminada**
Foco no problema que desaparece.

Para cada bullet, inclua:
- Sugestão de ícone/emoji
- Versão curta (para cards ou banners — máx. 8 palavras)

Ao final, entregue:
- Os 6 melhores bullets selecionados
- Ordem de apresentação recomendada
- Headline da seção
```

## Exemplo de uso

### Input
Produto: App de controle financeiro pessoal
Público: Jovens adultos
Funcionalidades: 1) Conexão automática com banco 2) Categorização automática de gastos 3) Alertas de limite de gastos
Tom: Casual e motivador

### Output
**Funcionalidade 2 — Categorização:**
- Versão A: "Saiba exatamente pra onde vai cada real do seu salário — sem digitar nada"
- Versão B: "Categorização automática de gastos para que você entenda seus hábitos sem precisar de planilha"
- Versão C: "Chega de anotar cada café no bloquinho. O app categoriza tudo sozinho e mostra onde você mais gasta."
- Ícone: 📊
- Versão curta: "Saiba pra onde vai seu dinheiro"
