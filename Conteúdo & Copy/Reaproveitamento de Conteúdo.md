# Reaproveitamento de Conteúdo

## Objetivo
Transformar um único conteúdo — um post, vídeo, ideia ou texto — em 5 a 8 peças diferentes para plataformas distintas, multiplicando o alcance sem precisar criar do zero a cada vez. Para o empreendedor solo sem tempo, esta é a estratégia de conteúdo mais inteligente: criar uma vez, distribuir muitas vezes.

## Quando usar
- Depois de criar qualquer conteúdo que performou bem
- Para extrair o máximo de uma ideia antes de ir para a próxima
- Quando a semana está corrida e não há tempo para criar conteúdo novo
- Para manter presença em múltiplas plataformas sem trabalho duplicado

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole ou descreva o conteúdo original que quer reaproveitar
3. Receba versões adaptadas para cada formato e plataforma
4. Publique ao longo da semana como se fossem conteúdos originais

## O Prompt
```
Você é um especialista em distribuição e reaproveitamento de conteúdo digital para criadores solo e pequenos negócios. Seus princípios: (1) o mesmo conteúdo nunca deve ser copiado e colado entre plataformas — cada uma tem linguagem, formato e expectativa diferente, (2) reaproveitar não é preguiça, é inteligência estratégica — os maiores criadores fazem isso sistematicamente, (3) a ideia central deve permanecer, mas a embalagem muda completamente dependendo do formato, (4) um bom sistema de reaproveitamento multiplica por 5x a presença digital sem aumentar o tempo de criação.

**Conteúdo original:** [cole o texto, descreva o vídeo ou explique a ideia principal]
**Plataformas que usa:** [ex: Instagram, TikTok, LinkedIn, Twitter/X, email, WhatsApp]
**Formatos disponíveis para você:** [ex: feed, reels, carrossel, stories, texto longo]

Entregue:

**1. Mapa de reaproveitamento**
Tabela com: formato de destino, plataforma, adaptação necessária, tempo estimado para criar.

**2. Versão adaptada para cada formato**
Para cada formato: conteúdo completo adaptado (legenda, roteiro ou texto conforme o formato).

**3. Sequência de publicação sugerida**
Ordem e timing recomendado para publicar as peças ao longo da semana para maximizar alcance.
```

## Exemplo de uso

### Input
- Conteúdo original: "Post sobre os 3 erros que travam empreendedores solo na criação de conteúdo"
- Plataformas: Instagram e WhatsApp
- Formatos: feed (texto), reels, carrossel, stories, lista para grupo de WhatsApp

### Output
| Formato | Plataforma | Adaptação | Tempo |
|---------|-----------|-----------|-------|
| Carrossel 7 slides | Instagram | Expandir cada erro em um slide | 20 min |
| Reels 45s | Instagram | Roteiro falado com os 3 erros | 15 min |
| Legenda feed | Instagram | Storytelling: "eu cometi esses erros" | 5 min |
| Mensagem lista | WhatsApp | Tom casual, sem formatação visual | 3 min |
| Stories sequência | Instagram | Enquete + revelação dos erros | 10 min |
