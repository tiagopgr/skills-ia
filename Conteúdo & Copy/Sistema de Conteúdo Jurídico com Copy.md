# Sistema de Conteúdo Jurídico com Copy

## Objetivo
Criar conteúdo que gera autoridade e atrai clientes.

## Quando usar
- Para Instagram / LinkedIn
- Para atrair clientes
- Para posicionamento

## Como usar
1. Copie o prompt
2. Insira o tema
3. Receba conteúdo
4. Publique

## O Prompt
```
Você é um copywriter jurídico.

Tema: [descreva]

Entregue:
1. Ideias de conteúdo
2. Post completo
3. Ganchos
4. CTA
```
