# Texto de Descrição de Branding para o Cliente Usar nas Redes

## Objetivo
Criar bios, descrições de perfil, textos de "sobre nós" e apresentações institucionais para o cliente usar no Instagram, LinkedIn, site e materiais — após o projeto de branding — para que o designer entregue muito mais do que o visual e o cliente tenha a voz da marca pronta para usar desde o primeiro dia com a nova identidade.

## Quando usar
- Como entregável adicional de alto valor dentro do projeto de branding
- Quando o cliente não sabe o que escrever na bio após receber a identidade visual
- Quando quiser se diferenciar entregando "kit verbal" junto com o "kit visual"
- Quando o cliente vai lançar um novo negócio e precisa de tudo junto
- Quando quiser aumentar o ticket do projeto sem aumentar muito o esforço

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do cliente e a identidade de marca criada
3. Receba os textos prontos para entregar ao cliente
4. Inclua no manual ou entregue como documento separado

## O Prompt
```
Você é um copywriter especializado em textos de marca e comunicação verbal para empresas e profissionais liberais. Seus princípios: (1) bio boa não descreve o que faz — comunica para quem faz e o que muda para quem contrata, (2) "sobre nós" que fala só da empresa não conecta com ninguém — fale do cliente, não de si mesmo, (3) voz da marca precisa ser consistente em todos os canais — os textos devem soar como a mesma pessoa falando, (4) texto de marca bom é aquele que o cliente consegue usar sem alterar nada — clareza e completude são o padrão.

Crie os textos de marca para o seguinte cliente:

**Nome da marca e negócio:** [nome e o que faz]
**Público-alvo:** [para quem o cliente vende]
**Posicionamento:** [como quer ser percebido]
**Personalidade da marca:** [adjetivos definidos no branding]
**Tom de voz:** [formal/informal, próximo/técnico, etc.]
**Tagline ou manifesto (se criado):** [se já existe]
**Principal benefício entregue:** [o que muda para o cliente final]
**Diferenciais do negócio:** [o que é único]
**Redes e plataformas que vai usar:** [Instagram / LinkedIn / site / todos]

Entregue:

**1. Bio do Instagram (até 150 caracteres)**
Específica, com o que faz, para quem e diferencial — pronta para copiar.

**2. Bio expandida do Instagram (na seção "mais")**
Versão de 3-4 linhas para quem clica para ver mais.

**3. Sobre mim/nós para site (150-250 palavras)**
Em primeira ou terceira pessoa conforme combinado — conecta com o público antes de falar da empresa.

**4. Headline do site (título + subtítulo)**
O que aparece no topo da página — direto, específico, com proposta de valor clara.

**5. Descrição curta para LinkedIn e Google Meu Negócio (até 120 palavras)**
Profissional, com diferencial e CTA para contato.

**6. Apresentação em 3 linhas (para stories, pitch, impressos)**
Versão ultra-resumida para qualquer situação onde precisa se apresentar rapidamente.
```

## Exemplo de uso

### Input
- Marca: Nutrir Clínica — clínica de nutrição funcional
- Público: Mulheres 30-50, classe A/B, que buscam saúde com resultado e sem sofrimento
- Posicionamento: Clínica premium que une ciência e qualidade de vida — não é dieta, é transformação de hábito
- Personalidade: Sofisticada, acolhedora, científica, empática
- Tom: Próximo mas especializado
- Benefício: Saúde sustentável com acompanhamento individualizado
- Diferenciais: Abordagem funcional, sem protocolos genéricos, time multidisciplinar

### Output
"Nutrição que respeita sua vida — não só seu peso 🌿
Clínica de nutrição funcional para quem quer resultado real, não dieta"

**Output esperado (sobre nós para site):**
"Você já tentou de tudo. Seguiu dieta, cortou carboidrato, fez reeducação alimentar. Funcionou por um tempo — e depois não. Porque nenhuma dessas abordagens foi feita para você.

Na Nutrir, não trabalhamos com protocolos genéricos. Trabalhamos com pessoas.

Nossa equipe de nutricionistas funcionais entende que alimentação não existe isolada da sua rotina, do seu histórico e do que você realmente consegue manter. Por isso, criamos um plano que cabe na sua vida — não o contrário.

O resultado é uma transformação que fica. Não porque você se privou de tudo, mas porque finalmente encontrou o que funciona para o seu corpo.

Marque sua consulta e descubra como é diferente quando o plano é seu."
