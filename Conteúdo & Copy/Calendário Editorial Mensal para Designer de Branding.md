# Calendário Editorial Mensal para Designer de Branding

## Objetivo
Criar um calendário editorial mensal completo para o designer de branding — com temas estratégicos, formatos variados e copies prontos para cada post — para que o profissional publique com consistência, construa autoridade no Instagram e LinkedIn e gere leads qualificados sem passar horas pensando no que postar toda semana.

## Quando usar
- No início de cada mês para planejar todo o conteúdo de uma vez
- Quando quiser parar de improvisar posts e ter um calendário para seguir
- Quando precisar de pautas que atraiam o cliente certo, não qualquer seguidor
- Quando quiser manter consistência mesmo nos meses mais corridos de projeto
- Quando quiser criar um sistema de conteúdo que funcione no piloto automático

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com seus dados, frequência desejada e temas relevantes do mês
3. Receba o calendário completo com temas, formatos e copies de cada post
4. Execute semana a semana sem precisar pensar do zero

## O Prompt
```
Você é um estrategista de conteúdo especializado em criação de calendários editoriais para designers, criativos e profissionais de branding. Seus princípios: (1) conteúdo de designer que converte tem mix de: autoridade técnica, bastidor de processo, educação do cliente e prova social — não só portfólio, (2) consistência vence perfeição — 3 posts bons por semana todo mês valem mais que 10 posts incríveis num mês e zero no outro, (3) calendário bom tem variedade de formato e profundidade — não pode ser só carrossel ou só stories, (4) cada post deve servir a um objetivo estratégico: educar / atrair / converter / relacionar.

Crie o calendário editorial mensal com as seguintes informações:

**Mês de referência:** [ex: Maio 2025]
**Plataformas:** [Instagram / LinkedIn / ambos]
**Frequência desejada:** [ex: 3x por semana / 5x por semana]
**Minha especialidade:** [ex: branding para profissionais liberais]
**Projetos que posso mostrar este mês:** [cases ou processos que posso documentar]
**Tema ou objetivo do mês:** [ex: captar clientes / lançar novo serviço / aumentar autoridade]
**Formatos que domino:** [carrossel / reel / post estático / stories / vídeo]
**O que mais performa no meu perfil:** [se souber — tipo de conteúdo que engaja mais]
**O que quero evitar:** [ex: muito conteúdo técnico / muito focado em preço]

Entregue:

**1. Calendário completo semana a semana**
Para cada post:
- Data
- Plataforma
- Formato
- Tema/título
- Objetivo estratégico (atrair / educar / converter / relacionar)
- Copy resumida (o que vai no texto)

**2. 5 copies completas prontas**
Para os 5 posts mais estratégicos do mês — texto completo pronto para publicar.

**3. Ideias de pautas recorrentes**
3-5 formatos/temas que você pode usar todo mês sem esgotar (ex: "Bastidor de projeto" / "Mito do branding" / "Antes e depois com contexto").

**4. Sugestão de stories para a semana**
Uma sequência de stories por semana para manter engajamento entre os posts.
```

## Exemplo de uso

### Input
- Mês: Maio 2025
- Plataforma: Instagram
- Frequência: 3x semana
- Especialidade: Branding para consultores e profissionais liberais
- Projetos: Nutrir Clínica (pode mostrar) + Forma Studio (iniciando, bastidor)
- Objetivo: Aumentar captação — quer fechar 2 projetos novos
- Formatos: Carrossel e post estático (não faz reel)
- Performa: Carrossel com dica prática e storytelling de case

### Output
"SEMANA 1 — MAIO
Segunda (05/05): Carrossel educativo
Tema: '3 sinais de que sua marca está custando clientes (sem você saber)'
Objetivo: Atrair — fazer o cliente ideal se identificar com o problema
Copy: [copy completa com 7 slides]

Quarta (07/05): Post estático — Bastidor
Tema: Processo de briefing do Forma Studio (imagem do quadro de referências)
Objetivo: Relacionar + mostrar processo — curiosidade sobre o projeto
Copy: 'Todo projeto começa antes de abrir qualquer arquivo. Esse é o moodboard de uma marca nova que estou desenvolvendo. Quando você entende o território visual antes de criar, o resultado não é achismo — é estratégia. Acompanha o processo aqui.' + hashtags

Sexta (09/05): Carrossel — Case
Tema: Nutrir Clínica — o que mudou (antes e depois com contexto estratégico)
Objetivo: Converter — prova social com raciocínio criativo
Copy: [copy completa de 8 slides]"
