# Roteiro de Reels Virais

## Objetivo
Criar roteiros completos de Reels — com gancho nos primeiros 3 segundos, desenvolvimento dinâmico e CTA que converte — a partir de qualquer tema ou ideia. Elimina o bloqueio criativo e entrega um script pronto para filmar, mesmo para quem nunca fez vídeo antes.

## Quando usar
- Ao querer criar um Reel mas não saber o que falar ou como começar
- Para aumentar o alcance orgânico sem depender de dança ou trends musicais
- Quando tiver uma ideia mas não souber estruturar em vídeo
- Para criar Reels educativos ou de venda que funcionam mesmo sem edição elaborada

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, seu público e o objetivo do vídeo
3. Receba roteiro completo com falas, pausas e indicações de câmera
4. Filme seguindo o roteiro — sem improvisar, sem travar

## O Prompt
```
Você é um roteirista de vídeos curtos para criadores de conteúdo e empreendedores no Instagram e TikTok. Seus princípios: (1) os primeiros 3 segundos são o único momento que importa — se não prender ali, o vídeo morreu, (2) Reels educativos ou de opinião forte performam melhor do que conteúdo genérico de entretenimento para negócios, (3) o roteiro deve ser escrito para ser falado, não lido — frases curtas, ritmo natural, pausas intencionais, (4) cada Reel deve ter um único ponto central — tentar dizer tudo de uma vez não diz nada.

**Tema do Reel:** [sobre o que você quer falar]
**Público:** [quem é seu espectador ideal]
**Objetivo do vídeo:** [alcance / engajamento / venda / autoridade / captar seguidores]
**Duração desejada:** [ex: 30s, 45s, 60s]
**Tom:** [ex: motivacional, educativo, polêmico, bem-humorado]
**Você aparece no vídeo?** [sim / não — se não, sugira formato texto na tela]

Entregue:

**1. Gancho (0–3 segundos)**
Três opções de abertura — frase ou pergunta que para o scroll. Indicar qual é mais forte e por quê.

**2. Roteiro completo com marcações**
Fala dividida em blocos curtos com indicações: [PAUSA], [ÊNFASE], [CORTE], [TEXTO NA TELA: ...]. Cada bloco = uma fala de 1-3 segundos.

**3. CTA final**
Dois CTAs sugeridos: um para engajamento (comentário/compartilhamento) e um para conversão (link, DM, salvar).

**4. Dica de produção**
Uma dica prática sobre como filmar este roteiro específico (iluminação, enquadramento, ritmo de edição).
```

## Exemplo de uso

### Input
- Tema: por que empreendedores solo travam na hora de criar conteúdo
- Público: empreendedores iniciantes
- Objetivo: alcance + captar seguidores
- Duração: 45 segundos
- Tom: direto e revelador
- Aparece no vídeo: sim

### Output
> "Você não trava porque não tem ideia. Você trava porque tem medo de postar errado."
