# Criador de Thread para Twitter/X

## Objetivo
Criar threads estruturadas e virais para Twitter/X que constroem autoridade, geram engajamento e atraem seguidores no formato que mais performa na plataforma.

## Quando usar
- Para construir presença e autoridade no Twitter/X
- Quando tem um tema que precisa de mais que 280 caracteres
- Para transformar conhecimento denso em conteúdo consumível
- Para campanhas de posicionamento ou lançamento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o tema e o ângulo da thread
3. Receba a thread completa pronta para publicar
4. Publique usando a função de thread do Twitter/X ou agende

## O Prompt
```
Você é um ghostwriter especializado em Twitter/X que já escreveu threads com mais de 10 mil curtidas. Você domina as técnicas de threads virais: hook forte no primeiro tweet, valor denso em cada tweet, formatação escaneável e CTA estratégico no final.

Crie uma thread completa para Twitter/X:

**Tema da thread:** [o assunto principal]
**Ângulo/Abordagem:** [ex: tutorial, opinião, lista, história, case, framework]
**Público-alvo:** [quem segue você no Twitter]
**Objetivo:** [ganhar seguidores, vender algo, gerar discussão, educar]
**Tom:** [ex: provocativo, didático, storytelling, data-driven]
**Tamanho:** [curta: 5-7 tweets | média: 8-12 tweets | longa: 13-20 tweets]

Entregue a thread completa com:

**Tweet 1 (Hook)**
O tweet mais importante. Deve fazer a pessoa PRECISAR ler o resto.
Regras: sem link, sem hashtag, sem "thread:", máximo 240 caracteres.

**Tweets 2-N (Corpo)**
Cada tweet deve:
- Ter uma ideia completa (funcionar sozinho se compartilhado)
- Máximo 280 caracteres
- Usar line breaks para legibilidade
- Não depender de "como eu disse acima" (cada um precisa ser autônomo)

**Tweet final (CTA)**
Opções de CTA:
- "Se isso foi útil, me siga para mais conteúdo assim"
- "RT o primeiro tweet para ajudar mais pessoas"
- Link para algo relevante (se quiser vender ou captar)
- Pergunta aberta (para gerar replies)

**Extras:**
- Sugestão de imagem/print para o primeiro tweet (aumenta CTR)
- Melhor horário para publicar threads
- Sugestão de tweet solo para publicar 24h depois linkando a thread (para quem não viu)

Formato de entrega:
Tweet 1/N:
[texto]

Tweet 2/N:
[texto]

...
```

## Exemplo de uso

### Input
Tema: 7 lições que aprendi faturando meu primeiro milhão
Ângulo: Lista com história pessoal em cada ponto
Público: Empreendedores e aspirantes a empreendedor
Objetivo: Ganhar seguidores
Tom: Direto e vulnerável
Tamanho: Média (10 tweets)

### Output
Tweet 1/10:
"Faturei meu primeiro milhão aos 28 anos.

Não foi sorte. Não foi herança. Foi erro atrás de erro até acertar o básico.

7 lições que ninguém me contou (e que teriam me poupado 4 anos):"

Tweet 2/10:
"1. Receita não é lucro.

Eu faturava R$80 mil/mês e não conseguia pagar as contas.

Parecia sucesso por fora. Por dentro era caos.

A primeira lição: margem importa mais que faturamento. Sempre."

Tweet 10/10:
"Se isso te ajudou:

→ Segue para mais conteúdo assim
→ RT o primeiro tweet pra chegar em mais gente

Estou compartilhando tudo que aprendi construindo negócios nos últimos 6 anos. Sem filtro."
