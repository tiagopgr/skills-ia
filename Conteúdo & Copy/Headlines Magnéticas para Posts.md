# Headlines Magnéticas para Posts

## Objetivo
Criar 10 opções de headlines e ganchos de alto impacto para qualquer post de feed — usando os gatilhos psicológicos mais eficazes do marketing de conteúdo. O gancho é a única parte que determina se alguém vai ler o post ou passar direto, e este prompt garante que você nunca mais publique com um título fraco.

## Quando usar
- Antes de publicar qualquer post onde o texto de abertura é visível no feed
- Para criar capas de carrossel com títulos que param o scroll
- Ao querer aumentar as visualizações de posts sem mudar o conteúdo
- Para testar diferentes abordagens de entrada para o mesmo tema

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tema do post e o que você quer que a pessoa sinta ao ler o gancho
3. Receba 10 opções com análise de cada uma
4. Escolha a mais forte ou combine elementos de diferentes opções

## O Prompt
```
Você é um especialista em copywriting de conteúdo para redes sociais, com foco em ganchos e headlines de alto desempenho. Seus princípios: (1) um gancho forte ativa um dos 5 gatilhos: curiosidade, dor, benefício, choque ou pertencimento — ganchos fracos tentam ser neutros, (2) especificidade vence generalidade sempre — "3 erros que empreendedores solo cometem no Instagram" bate "Erros no Instagram" em CTR, (3) o leitor ideal deve se ver no gancho — ele precisa pensar "isso é sobre mim", (4) números ímpares performam melhor que pares em listas (3, 5, 7, 9).

**Tema do post:** [sobre o que é o conteúdo]
**Público-alvo:** [quem você quer que pare e leia]
**Gatilho principal desejado:** [curiosidade / dor / benefício / choque / pertencimento]
**Tom:** [ex: direto, provocativo, empático, motivacional]
**Formato do post:** [feed texto / capa de carrossel / legenda de Reel]

Entregue:

**1. 10 opções de headline/gancho**
Numeradas, variando os gatilhos e estruturas (pergunta, afirmação, lista, negação, comparação).

**2. Análise de cada headline**
Uma linha por headline: qual gatilho usa, por que funciona, para qual sub-público é mais forte.

**3. Top 3 recomendadas**
As 3 com maior potencial explicadas em detalhe — e qual usar em qual situação.

**4. Versão A/B para a #1**
Uma variação da headline mais forte para teste de performance.
```

## Exemplo de uso

### Input
- Tema: por que postar todos os dias não garante crescimento
- Público: empreendedores solo que postam mas não crescem
- Gatilho: choque + curiosidade
- Tom: provocativo e direto
- Formato: capa de carrossel

### Output
> 1. "Você pode postar todos os dias e não crescer nada. (e aqui está o motivo)"
> 2. "Consistência sem estratégia é só barulho."
> 3. "O Instagram não recompensa quem posta mais. Recompensa quem posta certo."
