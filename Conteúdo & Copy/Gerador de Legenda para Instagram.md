# Gerador de Legenda para Instagram

## Objetivo
Criar legendas de Instagram completas — com gancho irresistível, corpo que prende atenção e CTA que converte — a partir de um simples briefing. Ideal para empreendedores solo que precisam postar com consistência sem gastar horas pensando no que escrever.

## Quando usar
- Quando precisar criar uma legenda para um post de feed mas travar na escrita
- Para postar com mais frequência sem perder qualidade
- Quando quiser testar diferentes estilos de legenda para o mesmo tema
- Ao lançar um produto, serviço ou oferta e precisar de copy rápido

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha o tema do post, o público e o objetivo (engajar, vender, educar)
3. Receba 3 versões de legenda com estilos diferentes
4. Escolha a que melhor se encaixa e publique

## O Prompt
```
Você é um copywriter especialista em Instagram para pequenos negócios e empreendedores solo. Seus princípios: (1) o gancho é tudo — os primeiros 2 segundos decidem se a pessoa para ou passa, (2) legendas que geram ação têm estrutura: dor ou desejo → solução ou história → CTA claro, (3) linguagem próxima bate linguagem formal em perfis de nicho, (4) um CTA fraco anula uma ótima legenda.

**Tema do post:** [sobre o que é este post — ex: procrastinação, vender sem parecer vendedor]
**Público:** [quem você quer atingir — ex: empreendedores iniciantes, mães que trabalham de casa]
**Objetivo da legenda:** [engajar / educar / vender / captar leads]
**Tom de voz:** [ex: direto, inspirador, bem-humorado, provocativo]
**Produto ou serviço (se for venda):** [descreva brevemente]

Entregue:

**1. Três versões de legenda completa**
Cada versão com: gancho (1-2 linhas), corpo (3-6 linhas) e CTA final. Versão A: emocional/storytelling. Versão B: direta/racional. Versão C: provocativa/questionamento.

**2. Análise rápida de cada versão**
Uma linha por versão explicando para qual situação ela funciona melhor.

**3. Hashtag sugeridas**
Bloco com 10-15 hashtags segmentadas para o nicho (mix de grandes, médias e pequenas).
```

## Exemplo de uso

### Input
- Tema: procrastinação
- Público: empreendedores solo sobrecarregados
- Objetivo: engajar
- Tom de voz: direto e provocativo
- Produto/serviço: não se aplica

### Output
> Você não está procrastinando por preguiça.
>
> Está procrastinando porque a tarefa parece grande demais, urgente demais, ou simplesmente você não sabe por onde começar.
>
> A solução não é força de vontade. É clareza.
>
> Pega uma tarefa hoje. Divide em 3 partes. Faz só a primeira.
>
> Me conta: qual tarefa você tá empurrando faz dias? 👇
