# Ideias de Conteúdo com Foco em Vendas

## Objetivo
Gerar ideias de conteúdo estratégicas — focadas em atrair, engajar e converter audiência em clientes.

## Quando usar
- Quando falta ideias de conteúdo
- Para gerar vendas com conteúdo
- Para crescer nas redes
- Para manter consistência

## Como usar
1. Copie o prompt
2. Preencha dados do negócio
3. Receba ideias
4. Execute

## O Prompt
```
Você é um estrategista de conteúdo com foco em vendas. Seus princípios: (1) conteúdo resolve problema, (2) consistência gera resultado, (3) autoridade vende, (4) simples funciona.

**Nicho:** [descreva]
**Público:** [quem é]
**Produto/serviço:** [descreva]
**Objetivo:** [ex: vender, crescer]

Entregue:

1. 10 ideias de conteúdo
2. Tipo de conteúdo
3. Estrutura de cada ideia
4. Gancho inicial
5. CTA sugerido
```
