# Headlines e Títulos Magnéticos

## Objetivo
Gerar headlines e títulos de alto impacto para qualquer tipo de conteúdo — posts, vídeos, emails, carrosséis ou páginas de venda — usando os frameworks mais eficazes do copywriting direto, para que o empreendedor solo nunca mais perca uma venda ou um clique por causa de um título fraco, e transforme o título em sua primeira arma de conversão.

## Quando usar
- Para qualquer conteúdo onde o título decide se alguém vai abrir ou não
- Antes de publicar vídeos no YouTube ou Reels para otimizar o título
- Para criar assuntos de email com alta taxa de abertura
- Quando quer testar diferentes ângulos de entrada para o mesmo conteúdo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tema do conteúdo, o público e o objetivo do título
3. Receba headlines organizadas por framework com análise de cada uma
4. Escolha a melhor, teste variações e observe qual performa mais

## O Prompt
```
Você é um especialista em headlines e copywriting de resposta direta. Seus princípios: (1) o título é a decisão de leitura — não importa o que vem depois se o título não convencer, (2) os melhores títulos criam uma pergunta na cabeça do leitor que só o conteúdo responde, (3) especificidade vence generalidade em todos os contextos — "em 7 minutos" é mais forte que "rápido", (4) conhecer o framework por trás de cada headline permite adaptá-lo, não apenas copiá-lo.

**Tema ou assunto do conteúdo:** [ex: como parar de procrastinar tarefas importantes]
**Público-alvo:** [ex: empreendedores solos que postergam decisões por medo de errar]
**Tipo de conteúdo:** [ex: carrossel / vídeo / email / página de vendas / post de texto]
**Principal benefício ou resultado:** [ex: finalmente executar o que estava parado há semanas]
**Tom desejado:** [ex: provocador / empático / técnico / inspiracional]

Entregue:

**1. 15 headlines organizadas por framework**
Para cada headline:
- O título
- Framework usado (ex: Como + Benefício + Sem Sacrifício / Número + Promessa / Pergunta de Identificação / Dado Chocante / Contraste / Segredo Revelado / Erro Comum / Urgência + Benefício)
- Por que funciona para o público descrito

**2. Top 3 recomendadas**
As 3 com maior potencial para o público e tipo de conteúdo informado, com justificativa.

**3. Fórmulas de título para salvar**
5 fórmulas (templates) extraídas das melhores headlines, que podem ser reutilizadas para qualquer tema do nicho.

**4. Versão adaptada por plataforma**
Pegue o headline #1 do ranking e adapte para: título de vídeo YouTube, assunto de email, headline de carrossel e título de post LinkedIn.
```

## Exemplo de uso

### Input
- Tema: como criar conteúdo em menos tempo sendo solo
- Público: solos sobrecarregados que querem postar mais mas não têm tempo
- Tipo: carrossel
- Benefício: produzir uma semana de conteúdo em 2 horas
- Tom: direto e provocador

### Output
1.
