# Script para Reels/TikTok (até 60s)

## Objetivo
Criar roteiros completos para vídeos curtos de Reels e TikTok (até 60 segundos) com estrutura otimizada para retenção, engajamento e viralização.

## Quando usar
- Para produção em lote de Reels e TikToks
- Quando quer gravar com roteiro e não 'na improviso'
- Para manter consistência e qualidade nos vídeos curtos
- Quando precisa de ideias e estrutura para vídeos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o tema e formato do vídeo
3. Receba o script completo com marcação de tempo
4. Grave seguindo o roteiro, adaptando ao seu estilo

## O Prompt
```
Você é um roteirista de vídeos curtos com experiência em conteúdo que viraliza no Instagram Reels e TikTok. Você domina as métricas de retenção e sabe que: os 3 primeiros segundos decidem se a pessoa fica, o meio precisa entregar valor rápido, e o final precisa ter um motivo para assistir de novo, salvar ou compartilhar.

Crie um script completo para vídeo curto:

**Tema do vídeo:** [sobre o que é]
**Nicho:** [sua área]
**Público:** [quem vai assistir]
**Formato:** [falando para câmera | texto na tela com narração | trend/áudio popular | tutorial visual | storytelling]
**Duração alvo:** [15s | 30s | 45s | 60s]
**Tom:** [educativo, engraçado, provocativo, inspiracional]
**Objetivo:** [engajamento, viralizar, vender, educar, atrair seguidores]

Entregue o script em formato timeline:

[0-3s] HOOK
Texto falado: "..."
Texto na tela: "..."
Visual: [descrição do que aparece]
Objetivo: Impedir o scroll

[3-10s] CONTEXTO
Texto falado: "..."
Texto na tela: "..."
Visual: [descrição]
Objetivo: Situar o espectador

[10-25s] CONTEÚDO PRINCIPAL
Texto falado: "..."
Texto na tela: "..."
Visual: [descrição]
Objetivo: Entregar valor

[25-28s] VIRADA/PUNCHLINE
Texto falado: "..."
Texto na tela: "..."
Visual: [descrição]
Objetivo: Momento memorável

[28-30s] CTA
Texto falado: "..."
Texto na tela: "..."
Visual: [descrição]
Objetivo: Gerar ação

Extras:
- **Sugestão de música/áudio** (tipo de trilha, trending audio se aplicável)
- **Sugestão de legenda** para a publicação
- **3 opções de hook alternativo** (para testar)
- **Hashtags** (5-10 relevantes)
- **Dica de edição** (cortes, zoom, transições)
```

## Exemplo de uso

### Input
Tema: 3 apps que uso todos os dias para produtividade
Nicho: Produtividade e tecnologia
Público: Profissionais que querem otimizar rotina
Formato: Falando para câmera com tela do celular
Duração: 30s
Tom: Direto e empolgado
Objetivo: Atrair seguidores

### Output
[0-3s] HOOK
Falado: "3 apps que mudaram minha produtividade e ninguém fala sobre."
Tela: "3 APPS SECRETOS"
Visual: Close no rosto, expressão de "vou te contar um segredo"

[3-12s] APP 1
Falado: "Primeiro: Notion Calendar. Conecta com tudo e mostra seu dia inteiro em uma tela."
Tela: Gravação da tela do app
Visual: Transição rápida para tela do celular

[12-21s] APP 2
Falado: "Segundo: Arc Browser. Substitui 5 extensões e organiza tudo em espaços."
Tela: Demo rápida
Visual: Corte seco para outra tela

[21-27s] APP 3
Falado: "Terceiro: Otter AI. Transcreve todas as minhas reuniões automaticamente."
Tela: App funcionando
Visual: Corte para tela

[27-30s] CTA
Falado: "Salva pra não esquecer e me segue pra mais dicas assim."
Tela: "SALVA + SEGUE"
Visual: Volta para câmera, gesto apontando para cima
