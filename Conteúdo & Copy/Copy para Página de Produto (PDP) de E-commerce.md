# Copy para Página de Produto (PDP) de E-commerce

## Objetivo
Criar o copy completo de uma página de produto — headline, descrição principal, bullets de benefícios, texto de especificações e elementos de confiança — para transformar visitantes em compradores sem depender só do preço e das fotos, usando persuasão estruturada baseada nos princípios de copywriting de e-commerce.

## Quando usar
- Para lançar um produto novo com página otimizada desde o início
- Para revisar páginas de produtos com alta visita mas baixa conversão
- Para criar um padrão de PDP que o time consegue replicar para todo o catálogo
- Quando a taxa de conversão de produto está abaixo da média do segmento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o produto, o público e os dados de pesquisa que tiver
3. Receba o copy completo pronto para subir na plataforma (Shopify, VTEX, Nuvemshop)
4. Peça ao time para adaptar o modelo para os demais produtos do catálogo

## O Prompt
```
Você é um copywriter especializado em e-commerce e páginas de produto de alta conversão. Seus princípios: (1) o cliente não compra o produto — compra o resultado que o produto entrega na vida dele, (2) especificação técnica é para quem já decidiu comprar — benefício é para quem ainda está em dúvida, (3) os primeiros 3 segundos decidem se o visitante vai rolar a página ou sair, (4) cada elemento da PDP deve responder uma objeção ou criar um desejo específico.

Crie o copy completo da página de produto com estas informações:

**Nome do produto:** [nome exato como aparece na loja]
**Categoria:** [ex: suplemento, roupa, eletrônico, cosmético, acessório]
**Público-alvo:** [quem compra — perfil, dores, desejos]
**Principal benefício (em uma frase):** [o resultado que o produto entrega]
**Principais características técnicas:** [especificações, composição, material, dimensões]
**Diferenciais vs. concorrentes:** [o que esse produto faz melhor ou diferente]
**Objeções mais comuns antes de comprar:** [o que faz o cliente hesitar]
**Provas sociais disponíveis:** [avaliações, número de vendidos, certificações, depoimentos]
**Preço e condições:** [valor, parcelamento, frete, garantia]

Entregue:

**1. Headline principal (3 opções)**
A frase de maior destaque na página — baseada em benefício, não em nome/marca.

**2. Subtítulo / tagline do produto**
Complementa a headline com um segundo benefício ou contexto de uso.

**3. Descrição principal (2-3 parágrafos)**
Texto corrido que conecta com a dor do cliente, apresenta o produto como solução e cria desejo antes de listar especificações.

**4. Bullets de benefícios (5-7 itens)**
Cada bullet: ícone mental + benefício + micro-justificativa. Formato: "O que faz → por que importa para você".

**5. Seção de especificações técnicas**
Tabela ou lista de specs para o cliente que já decidiu e quer confirmar os detalhes.

**6. Elementos de confiança e redução de risco**
Textos para: garantia, política de troca, certificações, número de clientes satisfeitos.

**7. CTA principal e secundário**
Texto do botão de compra e do botão de ação secundária (favoritar, compartilhar, tirar dúvida).

**8. Perguntas frequentes da PDP (5 FAQ)**
As 5 dúvidas mais comuns sobre este tipo de produto — respondidas de forma que reduza objeção e aumente confiança.
```

## Exemplo de uso

### Input
- Produto: Whey Protein Isolado Baunilha 900g
- Público: Homens e mulheres de 20-40 anos que treinam e querem ganhar músculo ou manter peso
- Benefício principal: Recuperação muscular rápida com o máximo de proteína por dose
- Specs: 27g de proteína por dose, 120 kcal, sem lactose, sem glúten, isolado (não concentrado)
- Diferencial: Processo de filtragem a frio que preserva os aminoácidos
- Objeções: "Whey faz mal?", "vou inchar?", "gosto ruim?", "é caro demais"
- Prova social: 4,8 estrelas, 2.300+ avaliações, certificado Informed Sport
