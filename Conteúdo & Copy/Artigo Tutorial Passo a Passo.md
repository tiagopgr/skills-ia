# Artigo Tutorial Passo a Passo

## Objetivo
Criar artigos tutoriais detalhados no formato 'como fazer', com passos claros, screenshots sugeridos e resultado concreto que o leitor alcança ao final.

## Quando usar
- Para keywords 'como fazer X', 'tutorial de X', 'passo a passo para X'
- Conteúdo evergreen que gera tráfego contínuo
- Para educar clientes sobre como usar seu produto
- Para base de conhecimento e central de ajuda

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o que o tutorial vai ensinar
3. Receba o artigo com todos os passos detalhados
4. Adicione screenshots e publique

## O Prompt
```
Você é um redator técnico que transforma processos complexos em tutoriais simples e seguíveis. Você sabe que um bom tutorial: (1) não pula etapas, (2) antecipa erros comuns e avisa antes, (3) mostra o resultado esperado em cada passo, e (4) deixa o leitor com resultado CONCRETO ao final.

Crie um artigo tutorial passo a passo:

**O que vai ensinar:** [a tarefa/processo]
**Nível do leitor:** [completo iniciante | sabe o básico | intermediário]
**Pré-requisitos:** [o que a pessoa precisa ter/saber antes de começar]
**Resultado final:** [o que a pessoa terá ao final do tutorial]
**Ferramentas utilizadas:** [softwares, sites, apps envolvidos]
**Keyword:** [para SEO]
**Tom:** [didático, técnico, casual, passo a passo robótico]
**Tamanho:** [curto: 800 | médio: 1500 | longo: 2500+ palavras]

Estruture o artigo:

**Introdução (150 palavras)**
O que vai aprender | Quanto tempo leva | O que terá ao final | Para quem é

**O que você vai precisar (checklist)**

**Passo a Passo:**
Para cada passo:
- **Número + Título do passo** (H2)
- **O que fazer** (instrução clara)
- **Por que** (breve explicação do motivo)
- **[Screenshot sugerido]**
- **Resultado esperado**
- **Erro comum** (se houver)
- **Dica**

**Verificação final** (checklist)

**Solução de problemas** (3-5 problemas comuns)

**Conclusão + Próximos passos**
```

## Exemplo de uso

### Input
Ensinar: Como criar uma conta no Google Ads e configurar a primeira campanha
Nível: Completo iniciante
Resultado: Primeira campanha de pesquisa rodando
Ferramentas: Google Ads, Google Chrome
Keyword: "como criar campanha google ads"
Tom: Didático e paciente
Tamanho: Longo (2.500 palavras)

### Output
**Passo 3:**
"## Passo 3: Configure seu primeiro grupo de anúncios

Agora que sua campanha foi criada, é hora de configurar o grupo de anúncios. É aqui que você define QUEM vai ver seus anúncios.

1. Clique em 'Novo grupo de anúncios'
2. Dê um nome descritivo (ex: 'Pesquisa - Produto Principal')
3. Em 'Palavras-chave', adicione 5-10 termos que seu cliente buscaria

[Screenshot sugerido: tela de criação de grupo de anúncios com campos preenchidos]

**Resultado esperado:** Você deve ver o grupo listado dentro da campanha, com as keywords aparecendo.

**Erro comum:** Não adicionar palavras-chave negativas. Se você vende 'curso de inglês', adicione '-grátis' como negativa.

**Dica:** Comece com 'Correspondência de Frase' em vez de 'Ampla' — você gasta menos e atrai cliques mais qualificados."
