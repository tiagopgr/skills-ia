# Carrossel de Autoridade

## Objetivo
Criar carrosséis educativos de alto valor — com profundidade de conteúdo, estrutura pedagógica e visual orientado para salvamento — para que o empreendedor solo construa autoridade no nicho de forma consistente, atraia seguidores qualificados e seja lembrado como referência antes mesmo de ter uma audiência grande.

## Quando usar
- Para criar o tipo de conteúdo que posiciona como especialista no nicho
- Quando quer aumentar salvamentos e compartilhamentos organicamente
- Para educar o público sobre um tema que conecta com seu produto ou serviço
- Quando quer criar conteúdo que continua gerando alcance semanas depois de publicado

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema educativo, seu público e quantos slides quer
3. Receba o roteiro completo do carrossel com copy por slide e orientações de design
4. Leve para o Canva e produza com seu template padrão

## O Prompt
```
Você é um estrategista de conteúdo educativo especializado em carrosséis para Instagram. Seus princípios: (1) autoridade se constrói com profundidade, não com volume — um carrossel com insight real vale mais que 10 superficiais, (2) o carrossel deve ensinar algo que o público não sabia antes de abrir — se é óbvio, não salva, (3) a progressão deve criar tensão crescente: problema → causa → solução → aplicação prática, (4) o slide final de CTA deve surgir naturalmente como extensão do aprendizado.

**Tema do carrossel:** [ex: por que empreendedores solos se sabotam sem perceber]
**Público:** [ex: empreendedores solos que trabalham muito mas produzem pouco]
**Número de slides:** [ex: 8 slides]
**Nível de profundidade:** [ex: introdutório / intermediário / avançado]
**Conexão com produto/serviço (opcional):** [ex: leva naturalmente para meu curso de sistemas]
**Tom:** [ex: direto e provocador / didático / inspiracional]

Entregue:

**1. Estrutura narrativa do carrossel**
Antes dos slides, defina: qual crença o leitor tem antes de abrir o carrossel e qual crença ele deve ter depois. Isso orienta todo o conteúdo.

**2. Roteiro slide a slide**
Para cada slide:
- Número + papel na narrativa (ex: Slide 3 — Causa raiz do problema)
- Headline do slide (texto principal em destaque — máx 8 palavras)
- Texto de apoio (max 3 linhas ou 3 bullets)
- Elemento visual sugerido (ícone, gráfico, comparação, antes/depois)
- Insight chave que esse slide entrega

**3. Slide de gancho (Slide 1) — 3 versões**
Três versões diferentes de Slide 1 para testar: uma pela dor, uma pelo dado e uma pela promessa.

**4. Slide de CTA (último slide) — otimizado**
CTA que conecta organicamente com o aprendizado do carrossel e convida para próximo passo sem parecer venda abrupta.
```

## Exemplo de uso

### Input
- Tema: os 3 erros que mantêm o empreendedor solo sempre sobrecarregado
- Público: solos que trabalham 10h/dia sem ver crescimento
- Slides: 8
- Profundidade: intermediário
- Conexão: curso de sistemas para solos
- Tom: direto e provocador
