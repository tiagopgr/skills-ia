# Microcopy para Botões e CTAs

## Objetivo
Criar textos curtos e eficazes para botões, links e elementos de interação em interfaces digitais — apps, sites, plataformas SaaS — garantindo que cada microtexto guie o usuário com clareza.

## Quando usar
- Ao projetar interfaces de apps e sites
- Em redesign de fluxos com baixa conversão
- Para melhorar a experiência de formulários e checkouts
- Em qualquer ponto de interação que precisa de texto em botão

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a tela, o contexto e a ação desejada
3. Receba múltiplas opções de microcopy
4. Implemente e teste as melhores variações

## O Prompt
```
Você é um UX Writer especialista em microcopy para interfaces digitais. Você sabe que o texto de um botão pode mudar drasticamente a conversão: "Submit" converte até 3x menos que um texto claro. Seus princípios: (1) clareza absoluta, (2) benefício > ação mecânica, (3) consistência de tom, (4) redução de atrito.

Crie microcopy para os seguintes elementos de interface:

**Produto/App:** [descreva]
**Tom da marca:** [formal, casual, divertido, técnico, premium]
**Público:** [quem usa]
**Plataforma:** [web, mobile, desktop]

**Telas/Contextos para criar microcopy:**
1. [ex: "Botão de criar conta na homepage"]
2. [ex: "Botão de upgrade para plano pago"]
3. [ex: "Botão de cancelar assinatura"]
(liste de 3 a 10 contextos)

Para CADA contexto, entregue:

**Texto do botão principal** — 3 opções:
- Opção A: Foco no benefício
- Opção B: Foco na ação clara
- Opção C: Foco na emoção/motivação

**Texto de apoio:**
- Versão que reduz atrito
- Versão que dá confiança

**Anti-pattern a evitar:** O que NÃO escrever e por quê

**Acessibilidade:** Sugestão de aria-label

Ao final: Glossário de microcopy padrão + Regras de tom de voz
```

## Exemplo de uso

### Input
Produto: App de meditação
Tom: Calmo e acolhedor
Público: Pessoas com ansiedade, iniciantes em meditação
Contextos: 1) Iniciar meditação 2) Criar conta 3) Ativar notificações

### Output
**Contexto 1 — Iniciar meditação:**
- A (benefício): "Começar a relaxar"
- B (ação): "Iniciar meditação de 5 min"
- C (emoção): "Cuide de você agora"

Texto de apoio:
- Atrito: "Sem equipamento. Só você e 5 minutos."
- Confiança: "Mais de 50.000 sessões feitas esta semana."

Anti-pattern: Não use "Play" ou "Start" — é frio e técnico. O contexto é emocional.

Aria-label: "Iniciar sessão de meditação guiada de 5 minutos"
