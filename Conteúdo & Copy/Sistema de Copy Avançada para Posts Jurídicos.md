# Sistema de Copy Avançada para Posts Jurídicos

## Objetivo
Criar conteúdos com maior impacto, retenção e conversão.

## Quando usar
- Posts no Instagram/LinkedIn
- Anúncios
- Conteúdo educativo

## Como usar
1. Copie o prompt
2. Defina o tema
3. Receba a copy
4. Publique

## O Prompt
```
Você é um copywriter jurídico especialista.

Tema: [descreva]

Entregue:
1. 5 ganchos fortes
2. Post completo
3. Estrutura persuasiva
4. CTA
```
