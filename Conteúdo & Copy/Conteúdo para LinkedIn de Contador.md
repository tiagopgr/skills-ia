# Conteúdo para LinkedIn de Contador

## Objetivo
Criar posts, artigos e sequências de conteúdo para LinkedIn de contadores — com linguagem que equilibra autoridade técnica e acessibilidade para o empresário leigo — para construir uma presença digital que gera conexões qualificadas, indicações e leads de clientes B2B sem precisar de anúncios pagos, apenas com consistência de conteúdo.

## Quando usar
- Para criar posts semanais no LinkedIn sem gastar horas escrevendo do zero
- Quando quiser posicionar o escritório como referência em um nicho específico
- Para criar conteúdo que atraia donos de empresa — não outros contadores
- Ao querer transformar uma mudança tributária ou fiscal em oportunidade de conteúdo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, o público e o objetivo do post
3. Receba 2-3 versões de post prontas para publicar ou ajustar
4. Publique com frequência mínima de 2x por semana para construir presença

## O Prompt
```
Você é um especialista em personal branding e conteúdo para contadores no LinkedIn. Seus princípios: (1) contador no LinkedIn que fala com contadores fica estagnado — o crescimento vem de falar COM empresários SOBRE os problemas deles, (2) post do LinkedIn não é artigo de revista contábil — abertura curta, linguagem simples, formatação arejada, (3) dado concreto + caso real (anonimizado) + lição aplicável = fórmula do post que gera saves e compartilhamentos, (4) o objetivo do post não é vender — é criar o pensamento "esse contador entende do meu problema".

**Tema do post:** [ex: reforma tributária / planejamento tributário para PMEs / erro comum de empresário no IR]
**Público-alvo:** [ex: donos de pequenas empresas / e-commerces / profissionais liberais]
**Tipo de post:** [ex: educativo / provocador / case anonimizado / opinião / dado surpreendente]
**Objetivo:** [ex: gerar conexões / mostrar autoridade / atrair leads / engajar atuais seguidores]
**Tom:** [ex: consultivo e próximo / técnico mas acessível / direto e objetivo]

Entregue:

**1. Post versão A — Abertura com dado/estatística**
Post completo com número ou dado de impacto na abertura. Máx. 1.300 caracteres.

**2. Post versão B — Abertura com situação do empresário**
Post que começa descrevendo uma situação que o dono de empresa reconhece imediatamente.

**3. Post versão C — Abertura com provocação ou opinião**
Post com ponto de vista claro que pode gerar debate — aumenta alcance orgânico.

Para cada versão: abertura, desenvolvimento (3-4 parágrafos curtos), fechamento + CTA suave.

**4. Sugestão de hashtags**
5-8 hashtags otimizadas para LinkedIn no nicho contábil/empresarial.
```

## Exemplo de uso

### Input
- Tema: Empresas pagando mais imposto do que deveriam por regime tributário errado
- Público: Donos de PMEs com faturamento entre R$ 500k e R$ 4,8M
- Tipo: Educativo com dado + caso anonimizado
- Objetivo: Atrair leads para consulta tributária
- Tom: Consultivo e próximo

### Output
"Uma empresa de serviços com R$ 1,2M de faturamento anual me procurou porque as parcelas do contador antigo chegavam todo mês — e nada mais.

Quando analisei o enquadramento tributário, descobri que eles estavam no Simples Nacional pagando uma alíquota efetiva de 14,7%.

Migramos para o Lucro Presumido. A alíquota caiu para 11,3%.

Economia: R$ 41.000 por ano. Sem fazer nada ilegal. Sem mudar o negócio. Só trocando de regime.

Planejamento tributário não é coisa de grande empresa. É o que separa o empresário que cresce do que trabalha para pagar imposto.

Quando foi a última vez que seu contador fez essa análise por você?"
