# Legenda Completa com CTA

## Objetivo
Criar legendas completas e persuasivas para posts no Instagram, LinkedIn ou Facebook — com hook que para o scroll, desenvolvimento que entrega valor e CTA que gera ação real — para que o empreendedor solo pare de usar legendas genéricas como "novo post no ar" e comece a ter legendas que trabalham por ele, amplificando o alcance orgânico e convertendo leitores em seguidores, leads e clientes.

## Quando usar
- Para qualquer post que precise de legenda com mais impacto do que o visual sozinho
- Quando quer aumentar comentários, salvamentos e compartilhamentos organicamente
- Para complementar carrosséis ou Reels com uma legenda que reforça a mensagem
- Quando vai impulsionar um post e quer que a legenda também converta

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o post, o tema e o que você quer que o leitor faça depois
3. Receba a legenda completa com variações de CTA
4. Revise o tom para sua voz e publique

## O Prompt
```
Você é um copywriter especializado em conteúdo orgânico para redes sociais. Seus princípios: (1) a legenda começa antes do "mais" — as primeiras 2 linhas são o gancho que faz o leitor expandir o texto, (2) a legenda deve agregar valor independente do visual — quem só lê a legenda deve sair com algo útil, (3) o CTA deve pedir uma coisa só — pedir comentário E seguir E compartilhar ao mesmo tempo dilui tudo, (4) hashtags são SEO, não decoração — escolhidas estrategicamente para alcance.

**Tema ou conteúdo do post:** [ex: 3 erros que empreendedores solos cometem na gestão do tempo]
**Tipo de post:** [ex: carrossel educativo / Reels / foto / post de texto]
**Objetivo da legenda:** [ex: gerar comentários / aumentar salvamentos / direcionar para link / aquecer para oferta]
**Público:** [ex: empreendedores solos que se sentem sobrecarregados]
**Tom de voz:** [ex: próximo e direto / técnico / inspiracional]
**Comprimento desejado:** [ex: curta (até 150 palavras) / média (150-300) / longa (+300)]

Entregue:

**1. Legenda completa**
Legenda formatada com:
- Linha 1-2: hook (antes do "mais")
- Desenvolvimento: 2-4 parágrafos curtos com o valor principal
- Linha de espaçamento antes do CTA
- CTA final: claro, direto, uma única ação

**2. Variações de CTA**
3 versões de CTA para o mesmo post com objetivos diferentes:
- CTA para comentários
- CTA para salvamento
- CTA para DM ou link

**3. Bloco de hashtags**
15-20 hashtags segmentadas em: nicho (5), tema específico (5), alcance amplo (5) + 2-3 de posicionamento. Com breve explicação da estratégia.

**4. Versão curta da legenda**
Versão condensada de até 100 palavras para quem prefere legendas diretas.
```
