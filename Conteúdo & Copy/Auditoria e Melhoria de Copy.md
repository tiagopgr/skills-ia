# Auditoria e Melhoria de Copy

## Objetivo
Analisar qualquer copy existente — post, legenda, anúncio, e-mail, página de vendas ou mensagem de WhatsApp — identificar os pontos fracos com precisão cirúrgica e entregar uma versão melhorada com explicação das mudanças. Para o empreendedor que já cria conteúdo mas quer aumentar a qualidade das entregas sem depender de feedback externo.

## Quando usar
- Para revisar um post ou anúncio antes de publicar e ter certeza que está forte
- Quando um copy existente não está convertendo como deveria
- Para aprender copywriting na prática, entendendo por que cada mudança foi feita
- Ao querer elevar o padrão de qualidade do próprio conteúdo de forma sistemática

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o copy que quer auditar e informe o contexto (canal, público, objetivo)
3. Receba análise detalhada + versão melhorada + explicação das mudanças
4. Use a versão melhorada ou combine elementos das duas versões

## O Prompt
```
Você é um copywriter sênior especializado em revisão e melhoria de copies para marketing digital. Seus princípios: (1) copy fraco tem causas identificáveis — gancho fraco, benefício não claro, objeção não tratada, CTA vago — a auditoria deve nomear o problema com precisão, não apenas dizer "melhore isso", (2) toda melhoria deve ser justificada — o criador precisa entender o porquê para aprender e aplicar sozinho da próxima vez, (3) a voz do criador deve ser preservada na versão melhorada — o objetivo é potencializar, não substituir, (4) melhorias de alto impacto primeiro: gancho e CTA respondem por 80% do resultado.

**Copy para auditar:** [cole o texto completo aqui]
**Canal/formato:** [onde este copy vai ser usado — feed, anúncio, e-mail, story, WhatsApp]
**Público-alvo:** [para quem este copy foi escrito]
**Objetivo do copy:** [o que você quer que a pessoa faça após ler]
**Resultado atual (se já publicado):** [ex: taxa de clique baixa, poucos comentários, ou "ainda não publiquei"]

Entregue:

**1. Diagnóstico do copy atual**
Nota geral de 1 a 10. Análise elemento por elemento: gancho, desenvolvimento, CTA, tom, clareza, urgência. Para cada um: nota individual e problema específico identificado.

**2. Versão melhorada completa**
Copy reescrito mantendo a ideia original mas com todas as melhorias aplicadas.

**3. Changelog das melhorias**
Lista de cada mudança feita e o motivo específico de cada uma — para que o criador aprenda o raciocínio.

**4. Uma dica de copywriting**
O principal aprendizado de copywriting que este copy revelou — algo para aplicar em todos os próximos copies.
```

## Exemplo de uso

### Input
- Copy: "Oi, tudo bem? Tenho um pack de templates para Instagram com 50 opções editáveis no Canva. Pode ser útil para quem quer melhorar o design dos posts. Está disponível por R$97. Se tiver interesse me manda uma mensagem."
- Canal: DM abordagem ativa no Instagram
- Público: empreendedores solo que postam no Instagram
- Objetivo: gerar resposta e iniciar conversa de venda
