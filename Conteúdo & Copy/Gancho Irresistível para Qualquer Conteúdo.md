# Gancho Irresistível para Qualquer Conteúdo

## Objetivo
Gerar 10 variações de ganchos de alta performance — para os primeiros 3 segundos de Reels, a primeira linha de legendas, o slide de capa de carrosséis ou a headline de anúncios — para qualquer tema e nicho. Ganchos fortes são a diferença entre conteúdo que para o scroll e conteúdo que é ignorado, e esta skill entrega variações testáveis com ângulos diferentes para o mesmo tema.

## Quando usar
- Antes de gravar qualquer Reels e precisar da frase de abertura ideal
- Para criar múltiplas capas de carrossel com ângulos diferentes para testar
- Ao escrever a primeira linha de uma legenda que precisa parar o scroll
- Para criar headlines de anúncios testáveis no Meta Ads

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema do conteúdo, o público e o objetivo do gancho
3. Receba 10 ganchos em ângulos diferentes com análise de qual tende a performar melhor
4. Escolha 2-3 para testar em formatos diferentes e observe qual gera mais resultado

## O Prompt
```
Você é um especialista em ganchos de conteúdo digital com foco em parar o scroll e gerar atenção imediata. Seus princípios: (1) gancho bom cria um loop aberto na mente do leitor — uma pergunta ou tensão que só será resolvida se ele continuar, (2) especificidade e números são os maiores aliados do gancho — "3 erros" converte mais do que "alguns erros", "perdi R$5.000" converte mais do que "perdi muito dinheiro", (3) os 6 ângulos que mais param o scroll são: dor identificável, número específico, contra-intuição, segredo revelado, prova de resultado e provocação direta, (4) o melhor gancho é o que o público sente que foi escrito especificamente para ele.

Gere 10 ganchos de alta performance com base nas informações abaixo:

**Tema do conteúdo:** [ex: como criar conteúdo sem aparecer / os erros mais comuns em anúncios]
**Público:** [quem vai ler ou assistir]
**Formato de destino:** [Reels / carrossel / legenda / anúncio / e-mail]
**Objetivo:** [engajamento / crescimento de seguidores / venda / captação de leads]
**Tom:** [direto / provocador / empático / curioso / chocante]

Gere 10 ganchos usando os seguintes ângulos (2 de cada para os 5 primeiros, depois variações livres):

1. **Dor identificável** — começa com uma situação que o público reconhece como própria
2. **Número específico** — usa dado concreto que gera credibilidade e curiosidade
3. **Contra-intuição** — contradiz uma crença comum no nicho
4. **Segredo revelado** — sugere que vai revelar algo que poucos sabem
5. **Provocação direta** — desafia o leitor de forma respeitosa

Para cada gancho:
- Texto exato do gancho
- Ângulo usado
- Por que funciona para este público e tema (1 linha)

Ao final, destaque os 3 com maior potencial de performance e sugira em qual formato cada um brilha mais.
```

## Exemplo de uso

### Input
- Tema: Como criar conteúdo para Instagram sem aparecer em vídeo
- Público: Empreendedores tímidos ou que preferem não aparecer
- Formato: Reels + carrossel
- Objetivo: Engajamento e crescimento de seguidores
- Tom: Empático e revelador

### Output
1. "Você não precisa aparecer em vídeo para crescer no Instagram — e vou te provar agora" *(contra-intuição — Reels com texto na tela)*
2. "Conheço 4 formatos de conteúdo que crescem perfis sem mostrar o rosto — o quarto surpreende todo mundo" *(número + segredo — capa de carrossel)*
3. "Eu não aparecia em nenhum vídeo e cresci 3.000 seguidores em 30 dias. Isso é o que eu usava." *(prova de resultado — legenda de Reels)*
