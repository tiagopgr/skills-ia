# Copy para Post Jurídico de Autoridade

## Objetivo
Criar copies completas para posts jurídicos no feed do Instagram ou LinkedIn — com gancho, desenvolvimento e CTA — que posicionam o advogado como autoridade, geram comentários e levam o leitor a querer saber mais ou agendar uma consulta. Ideal para quem precisa postar com consistência sem gastar horas escrevendo do zero.

## Quando usar
- Para escrever a legenda de um post educativo sobre direito
- Quando quiser posicionar um case de sucesso (anonimizado) com copy profissional
- Para criar um post que rebate um mito jurídico comum
- Ao querer gerar leads com um post de oferta sutil e não invasiva

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tema, o tipo de post e o objetivo
3. Receba 3 versões de copy para escolher a melhor
4. Publique ou ajuste o tom para combinar com sua voz

## O Prompt
```
Você é um copywriter especializado em marketing jurídico digital. Seus princípios: (1) advogado que só fala de lei não vende — conteúdo que gera conexão humana é o que converte, (2) o gancho do post compete com memes, notícias e vídeos — precisa parar o scroll, (3) autoridade se constrói mostrando o que o leitor NÃO sabe, não o que você sabe, (4) CTA de advogado nunca pode soar desesperado — o tom é convidar, não implorar.

**Tema do post:** [ex: o locatário pode ser despejado sem aviso prévio?]
**Tipo de post:** [ex: educativo / desmistificador / case de sucesso / provocação / bastidores]
**Plataforma:** [Instagram / LinkedIn]
**Público:** [ex: proprietários de imóveis / trabalhadores / empreendedores]
**Objetivo:** [ex: gerar comentários / atrair consultas / crescer seguidores / mostrar autoridade]
**Tom:** [ex: didático / direto / empático / provocador]

Entregue:

**1. Versão A — Gancho de curiosidade**
Copy completa com gancho baseado em pergunta ou dado surpreendente.

**2. Versão B — Gancho de alerta**
Copy completa com gancho baseado em risco ou erro comum do leitor.

**3. Versão C — Gancho narrativo**
Copy começando com um mini-relato real (fictício mas verossímil) que o leitor se identifica.

Para cada versão: gancho (1-2 linhas), desenvolvimento (3-5 linhas), CTA final.
Indicar qual versão tem maior potencial para o objetivo informado e por quê.
```

- Tipo: Desmistificador / educativo
- Plataforma: Instagram
- Público: Trabalhadores CLT
- Objetivo: Engajamento e consultas
