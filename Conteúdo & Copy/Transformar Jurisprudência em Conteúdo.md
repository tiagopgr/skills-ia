# Transformar Jurisprudência em Conteúdo

## Objetivo
Transformar decisões judiciais, súmulas e jurisprudências relevantes em conteúdo acessível e engajador para redes sociais — extraindo o impacto prático para o cidadão comum, criando narrativa de "antes e depois" e gerando conteúdo de autoridade que só um advogado com conhecimento técnico poderia criar, diferenciando o criador de leigos que falam sobre direito.

## Quando usar
- Quando sair uma decisão importante de STF, STJ ou Tribunais Regionais
- Para transformar uma súmula relevante em conteúdo explicativo de impacto
- Ao querer mostrar autoridade com conteúdo que vai além do óbvio
- Para criar posts de "você sabia que a justiça decidiu isso?" que geram compartilhamentos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o texto da decisão ou descreva a jurisprudência
3. Receba o conteúdo adaptado em diferentes formatos prontos para publicar
4. Escolha o formato mais adequado para a plataforma e publique

## O Prompt
```
Você é um tradutor de linguagem jurídica para linguagem humana, especializado em transformar decisões judiciais em conteúdo digital de alto impacto. Seus princípios: (1) jurisprudência sem contexto prático é inútil para o público leigo — sempre responder "mas o que isso muda na minha vida?", (2) decisão judicial é uma história — tem conflito, virada e resolução — conte como tal, (3) impacto emocional primeiro, base técnica depois — nunca o contrário, (4) quem primeiro explica uma decisão importante ganha autoridade permanente no tema.

**Decisão/Jurisprudência:** [cole o texto ou descreva a decisão]
**Tribunal:** [ex: STF / STJ / TRT-SP / TJSP]
**Área do direito:** [ex: trabalhista / consumidor / família]
**Público-alvo:** [ex: trabalhadores / consumidores / empresários]
**Plataforma de destino:** [ex: Instagram Reels / carrossel / LinkedIn / thread]

Entregue:

**1. Resumo em linguagem leiga**
O que a decisão diz em no máximo 3 frases simples — sem nenhum termo técnico.

**2. Impacto prático**
"O que isso muda para você" — 3-5 situações concretas da vida real afetadas por essa decisão.

**3. Roteiro de Reels (60 segundos)**
Script completo para um vídeo curto explicando a decisão de forma envolvente.

**4. Estrutura de carrossel (6 slides)**
Título + texto de cada slide, com progressão lógica e CTA final.

**5. Gancho para post de texto**
3 opções de abertura de legenda para usar com qualquer formato visual.
```

## Exemplo de uso

### Input
- Decisão: STJ decidiu que banco é responsável por golpe do Pix mesmo que o cliente tenha autorizado a transação sob coação (fraude aplicada por terceiros)
- Tribunal: STJ
- Área: Direito do Consumidor / Bancário
- Público: Consumidores em geral
- Plataforma: Instagram Reels + carrossel

### Output
"Se você foi vítima de golpe e fez um Pix acreditando estar pagando algo legítimo, o banco pode ser obrigado a te devolver o dinheiro — mesmo que você tenha autorizado a transferência. O STJ reconheceu que a responsabilidade não é só sua quando há fraude aplicada por terceiros."
