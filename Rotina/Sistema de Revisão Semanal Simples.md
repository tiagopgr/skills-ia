# Sistema de Revisão Semanal Simples

## Objetivo
Revisar a semana e ajustar a próxima — garantindo evolução contínua.

## Quando usar
- No fim da semana
- Para melhorar execução
- Para ajustar estratégia

## Como usar
1. Copie o prompt
2. Descreva sua semana
3. Receba análise
4. Aplique

## O Prompt
```
Você é um especialista em revisão. Seus princípios: (1) refletir para melhorar, (2) clareza, (3) ação.

**Semana:** [descreva]

Entregue:

1. Análise da semana
2. Pontos de melhoria
3. Ajustes
4. Plano da próxima semana
```
