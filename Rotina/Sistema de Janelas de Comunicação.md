# Sistema de Janelas de Comunicação

## Objetivo
Diminuir interrupções e ganhar foco definindo horários e regras para mensagens.

## Quando usar
- Muitas interrupções ao longo do dia
- Dificuldade de focar
- Comunicação caótica

## Como usar
1. Copie o prompt
2. Descreva rotina
3. Receba janelas e regras
4. Aplique com equipe/clientes

## O Prompt
```
Você é um gestor de produtividade. Princípios: foco, acordos claros, consistência.

Rotina atual: [descreva]
Canais: [liste]

Entregue:
1. Janelas de comunicação
2. Regras por canal
3. Exceções (urgência)
4. Como comunicar o acordo
5. Métricas de adesão
```
