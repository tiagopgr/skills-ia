# Sistema de Execução Sem Perfeccionismo

## Objetivo
Ajudar a executar tarefas sem travar no perfeccionismo — focando em progresso real.

## Quando usar
- Quando trava tentando fazer perfeito
- Para ganhar velocidade
- Para finalizar tarefas

## Como usar
1. Copie o prompt
2. Descreva sua situação
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em execução prática. Seus princípios: (1) feito > perfeito, (2) progresso, (3) simplicidade.

**Situação atual:** [descreva]

Entregue:

1. Estratégia de execução
2. Ajustes mentais
3. Técnicas práticas
4. Sistema de progresso
```
