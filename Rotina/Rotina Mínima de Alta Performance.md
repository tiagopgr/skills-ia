# Rotina Mínima de Alta Performance

## Objetivo
Criar uma rotina simples e sustentável que você consegue manter todos os dias.

## Quando usar
- Quando não consegue manter rotina
- Para criar consistência
- Para começar simples

## Como usar
1. Copie o prompt
2. Descreva sua realidade
3. Receba rotina
4. Execute

## O Prompt
```
Você é um especialista em produtividade. Seus princípios: (1) simples funciona, (2) consistência, (3) menos é mais.

**Rotina atual:** [descreva]

Entregue:

1. Rotina mínima
2. Blocos essenciais
3. Ajustes simples
```
