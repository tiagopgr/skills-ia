# Rotina de Início de Dia com Foco

## Objetivo
Começar o dia com clareza e direção — evitando dispersão e aumentando consistência.

## Quando usar
- Quando começa o dia perdido
- Para ganhar ritmo
- Para melhorar produtividade

## Como usar
1. Copie o prompt
2. Descreva sua manhã atual
3. Receba rotina estruturada
4. Aplique diariamente

## O Prompt
```
Você é um especialista em rotina matinal. Seus princípios: (1) começo define o dia, (2) simplicidade, (3) foco no essencial.

**Rotina atual:** [descreva]

Entregue:

1. Rotina de início do dia
2. Sequência de ações
3. Tempo ideal
4. Ajustes práticos
```
