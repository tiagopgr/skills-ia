# Sistema de Retrospectiva Diária e Semanal

## Objetivo
Aprender com o dia/semana para reduzir erros e melhorar consistência.

## Quando usar
- Repetição de erros
- Falta de melhoria contínua
- Sensação de caos recorrente

## Como usar
1. Copie o prompt
2. Descreva período
3. Receba análise
4. Aplique ajustes

## O Prompt
```
Você é um facilitador de retrospectivas. Princípios: simplicidade, ação, melhoria contínua.

Período: [descreva]
Resultados: [liste]

Entregue:
1. O que funcionou
2. O que não funcionou
3. Causas
4. 3 ações de melhoria
5. Plano para próxima semana
```
