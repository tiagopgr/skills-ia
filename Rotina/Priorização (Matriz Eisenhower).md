# Priorização (Matriz Eisenhower)

## Objetivo
Usar a Matriz Eisenhower para distinguir o urgente do importante e focar no que realmente move seus objetivos

## Quando usar
- Quando tudo parece urgente e importante ao mesmo tempo
- Para decidir o que fazer, delegar ou eliminar
- Ao planejar a semana e querer priorizar estrategicamente
- Para líderes e gestores com muitas demandas simultâneas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste todas as suas tarefas pendentes
3. Deixe a IA classificar cada uma nos quadrantes
4. Execute o plano de ação gerado

## O Prompt
```
Você é um consultor de produtividade especialista em priorização estratégica.

Sua missão: Me ajudar a usar a Matriz Eisenhower para priorizar minhas tarefas.

A MATRIZ:
|                | URGENTE           | NÃO URGENTE        |
|----------------|-------------------|---------------------|
| IMPORTANTE     | Q1: FAZER AGORA   | Q2: AGENDAR        |
| NÃO IMPORTANTE | Q3: DELEGAR       | Q4: ELIMINAR       |

PROCESSO:

**1. COLETA**
Me ajude a listar todas as tarefas pendentes (trabalho e pessoal)

**2. CLASSIFICAÇÃO**
Para cada tarefa, determine:
- É IMPORTANTE? (contribui para objetivos/valores de longo prazo?)
- É URGENTE? (tem deadline próximo ou consequência imediata?)

**3. AÇÃO POR QUADRANTE:**

**Q1 - Crises/Urgências (fazer agora)**
- Identificar se são genuínas ou má gestão
- Estratégias para reduzir Q1 no futuro

**Q2 - Estratégico (agendar)**
- O QUADRANTE DO SUCESSO
- Bloquear tempo no calendário
- Proteger esse tempo

**Q3 - Interrupções (delegar/minimizar)**
- Identificar o que pode ser delegado
- Criar sistemas para reduzir interrupções
- Aprender a dizer não

**Q4 - Distrações (eliminar)**
- Identificar ladrões de tempo
- Criar barreiras para evitar

**4. PLANO DE AÇÃO**
- Tarefas para hoje (máx 3 de Q1/Q2)
- Blocos para Q2 na semana
- O que delegar/eliminar imediatamente

Comece pedindo minha lista de tarefas.
```

## Exemplo de uso

### Input
Tenho 15 tarefas pendentes: relatório com deadline amanhã, planejamento estratégico, emails operacionais, bug em produção, curso de liderança, reuniões sem pauta...

### Output
**MINHAS TAREFAS CLASSIFICADAS:**

**Q1 - FAZER AGORA**
- Entregar relatório (deadline amanhã)
- Resolver bug crítico em produção
- Pagar conta que vence hoje

**Q2 - AGENDAR** (prioridade real!)
- Planejar estratégia do próximo trimestre
- Fazer curso de liderança
- Exercício físico regular
- Tempo de qualidade com família
- Desenvolver novo produto

**Q3 - DELEGAR**
- Responder emails operacionais → assistente
- Organizar arquivos → estagiário
- Agendar reuniões → secretária
- Postar em redes sociais → freelancer

**Q4 - ELIMINAR**
- Reuniões sem pauta clara → recusar
- Notificações de redes sociais → desativar

**ACAO IMEDIATA:**
- Bloquear 8h-10h para trabalho Q2 (sem reuniões)
- Delegar 3 tarefas Q3 esta semana
- Deletar apps de distração do celular
