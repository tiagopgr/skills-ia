# Rotina de Fechamento do Dia

## Objetivo
Encerrar o dia com organização — preparando o próximo dia e mantendo consistência.

## Quando usar
- No fim do dia
- Para evitar bagunça
- Para manter ritmo

## Como usar
1. Copie o prompt
2. Descreva seu fim de dia
3. Receba rotina
4. Aplique

## O Prompt
```
Você é um especialista em produtividade. Seus princípios: (1) organização, (2) fechamento consciente, (3) continuidade.

**Rotina atual:** [descreva]

Entregue:

1. Rotina de fechamento
2. Checklist
3. Preparação do dia seguinte
```
