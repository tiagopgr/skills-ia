# Sistema de Indicadores Operacionais Simples

## Objetivo
Medir o que realmente importa sem complicação.

## Quando usar
- Para acompanhar resultados
- Para melhorar operação
- Para tomar decisões

## Como usar
1. Copie o prompt
2. Descreva operação
3. Receba indicadores
4. Acompanhe

## O Prompt
```
Você é um analista operacional. Princípios: (1) medir o essencial, (2) clareza, (3) ação.

**Operação:** [descreva]

Entregue:
1. Indicadores principais
2. Como medir
3. Frequência
4. Interpretação
```
