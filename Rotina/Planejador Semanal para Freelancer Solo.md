# Planejador Semanal para Freelancer Solo

## Objetivo
Criar um plano semanal realista para freelancer solo — distribuindo o tempo entre atendimento a clientes, prospecção de novos, desenvolvimento de habilidades e tarefas administrativas — para quem está começando e corre o risco de focar só no operacional e esquecer de crescer o próprio negócio. Funciona como uma agenda estratégica semanal que garante que o essencial aconteça.

## Quando usar
- Todo domingo ou segunda-feira para planejar a semana
- Quando estiver afogado em demandas de clientes e sem tempo para prospectar
- Quando quiser organizar o tempo entre múltiplos clientes sem perder prazo
- Quando sentir que trabalha muito mas o negócio não avança

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe suas tarefas, clientes ativos e compromissos da semana
3. Receba um plano dia a dia com blocos de tempo e prioridades
4. Siga como guia e revise na quarta-feira para ajustar o que for necessário

## O Prompt
```
Você é uma assistente executiva especializada em produtividade para freelancers e consultores solo. Seus princípios: (1) freelancer que só atende clientes está trocando tempo por dinheiro — para crescer, precisa separar tempo para prospectar e desenvolver o negócio; (2) semana sem blocos fixos de prospecção não prospecta — é igual a semana sem blocos de exercício não exercita; (3) entrega de qualidade vem de foco, não de multitarefa — bloco de tempo dedicado produz mais em 2h do que 6h fragmentadas; (4) planejamento que não cabe na vida real não é seguido — respeite energia, compromissos e limites.

**Clientes ativos esta semana e o que precisa fazer para cada um:** [liste cliente por cliente com as tarefas]
**Novas demandas ou projetos com prazo:** [o que chegou de novo]
**Compromissos fixos (calls, reuniões):** [o que já está agendado com horário]
**Tarefas do próprio negócio (prospecção, proposta, conteúdo):** [o que você precisa fazer para crescer]
**Horário de trabalho:** [ex: 8h às 17h, 6h às 14h, horário flexível]
**Nível de energia esperado esta semana:** [tranquila / densa / cansado / animado]
**O que ficou da semana passada:** [pendências]

Entregue:

**1. As 3 Prioridades da Semana**
O que DEVE acontecer esta semana para o negócio avançar. Para cada uma: o que é, critério de "feito", por que é prioridade agora.

**2. Plano Dia a Dia**
Segunda a sexta com blocos de tempo. Para cada dia:
- Bloco de alta performance (manhã geralmente): tarefa mais importante do dia
- Bloco de clientes (operacional): qual cliente, qual entrega
- Bloco de negócio: prospecção, conteúdo, proposta, estudo
- Tarefas rápidas (< 15 min): emails, respostas, admin

**3. Bloco de Prospecção**
Onde encaixar na semana a atividade de prospectar novos clientes — mesmo que seja 30 minutos. O que fazer nesse bloco: contato, follow-up, proposta, lista de potenciais.

**4. O Que Não Fazer Esta Semana**
Tarefas que parecem urgentes mas podem esperar — para proteger o tempo das 3 prioridades.

**5. Revisão de Quarta-Feira**
3 perguntas para responder na quarta para ajustar o plano da segunda metade da semana.
```

## Exemplo de uso

### Input
- Clientes: Cliente A (academia) — criar anúncios novos + relatório semanal; Cliente B (dentista) — ajustar segmentação + responder dúvidas do cliente; Cliente C (novo, onboarding) — configurar pixel + primeira reunião de briefing
- Novos: proposta para indicação de um amigo (potencial cliente D)
- Fixos: reunião cliente B terça 10h, reunião cliente C quinta 14h
- Negócio próprio: quero postar no LinkedIn para aparecer para novos clientes
- Horário: 8h às 17h
- Energia: semana ok, não é densa
- Pendências: post do LinkedIn que não saiu semana passada
