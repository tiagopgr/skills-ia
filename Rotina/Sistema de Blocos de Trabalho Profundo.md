# Sistema de Blocos de Trabalho Profundo

## Objetivo
Organizar o dia em blocos de foco intenso — aumentando produtividade e qualidade das entregas.

## Quando usar
- Quando se distrai com facilidade
- Para tarefas importantes
- Para produzir com mais qualidade

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva sua rotina
3. Receba estrutura de blocos
4. Aplique no dia a dia

## O Prompt
```
Você é um especialista em foco profundo. Seus princípios: (1) blocos sem distração, (2) pausas estratégicas, (3) foco em uma tarefa.

**Rotina atual:** [descreva]
**Tarefas principais:** [liste]

Entregue:

1. Estrutura de blocos de trabalho
2. Duração ideal
3. Organização de pausas
4. Sugestões práticas
```
