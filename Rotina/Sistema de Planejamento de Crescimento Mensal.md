# Sistema de Planejamento de Crescimento Mensal

## Objetivo
Organizar crescimento mensal com clareza e execução.

## Quando usar
- Para organizar metas
- Para crescer com consistência
- Para evitar desorganização

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba plano
4. Execute

## O Prompt
```
Você é um estrategista de crescimento. Princípios: foco, consistência, execução.

**Negócio:** [descreva]

Entregue:
1. Metas do mês
2. Prioridades
3. Plano de ação
4. Indicadores
```
