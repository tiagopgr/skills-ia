# Sistema de Priorização Simples

## Objetivo
Definir prioridades claras — evitando perder tempo com tarefas irrelevantes.

## Quando usar
- Quando tem muitas tarefas
- Para decidir o que fazer
- Para focar no essencial

## Como usar
1. Copie o prompt
2. Liste tarefas
3. Receba priorização
4. Execute

## O Prompt
```
Você é um especialista em priorização. Seus princípios: (1) foco no essencial, (2) impacto, (3) simplicidade.

**Tarefas:** [liste]

Entregue:

1. Prioridades
2. O que eliminar
3. O que focar
```
