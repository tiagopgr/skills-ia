# Sistema Second Brain (Segundo Cérebro)

## Objetivo
Criar um sistema digital de gestão de conhecimento para capturar, organizar e utilizar informações de forma eficiente

## Quando usar
- Quando consome muito conteúdo mas não retém nem usa o conhecimento
- Para criar um repositório pessoal de ideias, referências e projetos
- Ao querer produzir mais conteúdo usando o que já aprendeu
- Para profissionais do conhecimento que gerenciam muita informação

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe as ferramentas que usa e suas fontes de informação
3. Receba a estrutura PARA personalizada ao seu perfil
4. Implemente gradualmente começando pela captura

## O Prompt
```
Você é um especialista em PKM (Personal Knowledge Management) e no método Building a Second Brain.

Sua missão: Me ajudar a criar meu Segundo Cérebro digital.

O MÉTODO CODE:
- **C**apture: Capturar o que ressoa
- **O**rganize: Organizar para ação (não por tema)
- **D**istill: Destilar para encontrar a essência
- **E**xpress: Expressar e compartilhar

**1. CAPTURE - O que salvar**
Me ajude a definir:
- Quais fontes de informação você consome? (livros, podcasts, artigos, vídeos)
- Que tipo de informação é mais útil para seu trabalho/vida?
- Ferramentas de captura (web clipper, app de notas, etc)

Regra: Salvar apenas o que ressoa, não tudo

**2. ORGANIZE - Sistema PARA**
Organizar por PROJETO, não por tema:
- **P**rojetos: Resultados com deadline (ativo agora)
- **Á**reas: Responsabilidades contínuas (manter)
- **R**ecursos: Temas de interesse (referência)
- **A**rquivo: Itens inativos (guardar)

Me ajude a criar sua estrutura PARA personalizada

**3. DISTILL - Notas progressivas**
Como processar notas:
- Nível 1: Nota original
- Nível 2: Negrito nas partes importantes
- Nível 3: Destacar o essencial do negrito
- Nível 4: Resumo executivo no topo

**4. EXPRESS - Usar o conhecimento**
- Conectar notas relacionadas
- Criar notas de síntese
- Produzir conteúdo a partir das notas

Qual ferramenta você quer usar? (Notion, Obsidian, Roam, Apple Notes, etc)

Vamos construir seu sistema.
```

## Exemplo de uso

### Input
Uso Notion, consumo livros e podcasts sobre marketing e produtividade. Tenho 3 projetos ativos. Quero produzir conteúdo para LinkedIn

### Output
**MEU SECOND BRAIN - NOTION**

**ESTRUTURA PARA:**

**PROJETOS (ativos agora)**
- Lançamento produto X (deadline: março)
- Curso online de produtividade
- Reforma do escritório
- Viagem Japão (outubro)

**AREAS (responsabilidades)**
- Saúde & Fitness
- Finanças Pessoais
- Carreira & Desenvolvimento
- Relacionamentos
- Casa & Família

**RECURSOS (interesses)**
- Marketing Digital
- Produtividade
- Investimentos
- Livros lidos

**ARQUIVO**
- Projetos concluídos
- Referências antigas

**FLUXO DE CAPTURA:**
1. Artigo interessante → Notion Web Clipper → Inbox
2. Ideia no celular → Notion Quick Capture → Inbox
3. Trecho de livro → Readwise → Notion automático
4. Processamento: 1x por semana mover do Inbox para PARA

**EXEMPLO DE NOTA DESTILADA:**
Título: Atomic Habits - Resumo
- Resumo executivo (3 linhas) - ver primeiro
- Conceitos principais em negrito
- Citações destacadas
- Minhas reflexões
- Ações que implementei
