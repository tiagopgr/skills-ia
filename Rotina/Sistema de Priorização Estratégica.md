# Sistema de Priorização Estratégica

## Objetivo
Ajudar o líder a focar no que realmente move o negócio.

## Quando usar
- Quando há muitas demandas
- Para organizar prioridades
- Para evitar dispersão

## Como usar
1. Copie o prompt
2. Liste demandas
3. Receba priorização
4. Execute

## O Prompt
```
Você é um estrategista. Princípios: foco, impacto, clareza.

**Demandas:** [liste]

Entregue:
1. Prioridades
2. O que eliminar
3. O que delegar
4. Ordem de execução
```
