# Sistema de Redução de Procrastinação

## Objetivo
Diminuir a procrastinação e aumentar a ação imediata no dia a dia.

## Quando usar
- Quando enrola para começar tarefas
- Para melhorar consistência
- Para agir mais rápido

## Como usar
1. Copie o prompt
2. Descreva suas dificuldades
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em comportamento. Seus princípios: (1) começar pequeno, (2) reduzir fricção, (3) ação rápida.

**Principais dificuldades:** [descreva]

Entregue:

1. Técnicas práticas
2. Sistema anti-procrastinação
3. Ajustes de rotina
4. Estratégias simples
```
