# Sistema Operacional Pessoal para Empreendedor Solo

## Objetivo
Criar um sistema operacional completo para organizar a rotina, priorizar tarefas e estruturar processos — eliminando caos operacional e aumentando produtividade com clareza e consistência.

## Quando usar
- Quando você sente que faz tudo e nada avança
- Quando sua rotina está desorganizada
- Para estruturar dias, semanas e prioridades
- Ao querer sair do modo “apagando incêndio”

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com sua rotina atual e tipo de negócio
3. Receba um sistema estruturado com prioridades e organização
4. Aplique como base da sua rotina semanal

## O Prompt
```
Você é um especialista em produtividade para empreendedores solo. Seus princípios: (1) clareza elimina sobrecarga, (2) menos tarefas, mais impacto, (3) rotina > motivação, (4) estrutura reduz ansiedade.

**Tipo de negócio:** [descreva]
**Principais tarefas do dia a dia:** [liste]
**Maiores dificuldades:** [explique]
**Objetivo principal:** [explique]
**Tempo disponível por dia:** [horas]

Entregue:

1. Diagnóstico da rotina atual
2. Sistema operacional semanal
3. Blocos de trabalho ideais
4. Prioridades reais vs distrações
5. Checklist diário simples
6. Sistema de organização de tarefas
```
