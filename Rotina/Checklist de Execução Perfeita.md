# Checklist de Execução Perfeita

## Objetivo
Garantir que você execute tarefas com padrão alto sempre.

## Quando usar
- Antes de finalizar tarefas
- Para garantir qualidade
- Para melhorar padrão

## Como usar
1. Copie o prompt
2. Descreva tarefa
3. Receba checklist
4. Aplique

## O Prompt
```
Você é um especialista em execução. Seus princípios: (1) qualidade, (2) consistência, (3) atenção.

**Tarefa:** [descreva]

Entregue:

1. Checklist completo
2. Pontos críticos
3. Erros comuns
```
