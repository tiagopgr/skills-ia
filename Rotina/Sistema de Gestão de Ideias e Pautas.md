# Sistema de Gestão de Ideias e Pautas

## Objetivo
Criar um sistema completo de captura, organização e priorização de ideias de conteúdo — para que nenhuma boa ideia se perca, o bank de pautas esteja sempre cheio e a decisão do que criar em seguida seja rápida e estratégica, eliminando o "não sei o que postar hoje" de vez.

## Quando usar
- Para criar o sistema de gestão de pautas que vai alimentar a produção de conteúdo de forma consistente
- Quando tem muitas ideias soltas que nunca viram conteúdo por falta de organização
- Para estruturar o fluxo de ideias desde a captura até a publicação
- Ao querer integrar o uso de Claude na gestão de pautas de forma sistemática

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe as ferramentas que usa e como é sua rotina atual de ideias
3. Receba o sistema completo com fluxo, estrutura de banco e critérios de priorização
4. Implemente no Notion, Obsidian ou qualquer ferramenta de gestão que preferir

## O Prompt
```
Você é um especialista em sistemas de gestão de conteúdo para creators solos. Seus princípios: (1) ideia não capturada imediatamente é ideia perdida — o sistema de captura precisa ter fricção zero; (2) banco de pautas é estoque estratégico — o que você produz na semana deveria ter sido decidido na semana anterior; (3) nem toda ideia boa vale ser desenvolvida agora — priorização é a habilidade mais subestimada em gestão de conteúdo; (4) IA pode fazer o trabalho pesado de triagem e desenvolvimento — o creator faz o julgamento final.

**Ferramentas disponíveis:** [ex: Notion, Obsidian, Apple Notes, Google Drive, Trello]
**Plataformas de conteúdo:** [onde você publica]
**Frequência de publicação atual:** [quantas vezes por semana]
**Como captura ideias hoje:** [o que já faz — mesmo que seja bagunçado]
**Principal problema com ideias:** [perde as ideias / não sabe o que priorizar / tem ideia mas não desenvolve]

Entregue:

**1. Sistema de Captura**
O método de captura de ideias com fricção zero — como registrar uma ideia em menos de 30 segundos, de qualquer lugar.

**2. Estrutura do Banco de Pautas**
Template do banco de pautas com os campos essenciais para cada ideia: o que registrar, como categorizar, como avaliar.

**3. Critérios de Priorização**
Como decidir qual ideia virar conteúdo primeiro — critérios objetivos de avaliação (relevância, tempo de produção, potencial de engajamento, alinhamento estratégico).

**4. Fluxo Semanal de Gestão**
Ritual semanal de: revisar ideias novas → avaliar e priorizar → mover para produção → arquivar o que não vai usar.

**5. Papel da IA no Processo**
Como usar Claude em cada etapa do fluxo — com prompts específicos para triagem, desenvolvimento e expansão de ideias.

**6. Template de Pauta Desenvolvida**
O modelo de documento que uma ideia vira quando é selecionada para produção — o que precisa estar definido antes de começar a criar.
```
