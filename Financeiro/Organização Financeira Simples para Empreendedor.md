# Organização Financeira Simples para Empreendedor

## Objetivo
Criar um sistema financeiro simples e funcional — para controlar entradas, saídas e lucro sem complicação.

## Quando usar
- Quando não tem controle financeiro
- Para organizar dinheiro do negócio
- Para entender lucro real
- Para tomar decisões melhores

## Como usar
1. Copie o prompt
2. Preencha dados financeiros
3. Receba estrutura
4. Implemente no dia a dia

## O Prompt
```
Você é um especialista em finanças para pequenos negócios. Seus princípios: (1) simplicidade, (2) clareza de números, (3) controle básico bem feito, (4) decisão baseada em dados.

**Faturamento médio:** [valor]
**Principais despesas:** [liste]
**Hoje você controla como:** [explique]
**Objetivo financeiro:** [explique]

Entregue:

1. Estrutura simples de controle financeiro
2. Categorias de entrada e saída
3. Modelo de organização mensal
4. Indicadores principais
5. Rotina financeira semanal
```
