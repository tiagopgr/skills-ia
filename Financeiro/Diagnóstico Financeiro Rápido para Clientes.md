# Diagnóstico Financeiro Rápido para Clientes

## Objetivo
Gerar diagnósticos financeiros estruturados para clientes ou prospects do escritório — analisando regime tributário, estrutura de custos, fluxo de caixa e oportunidades de otimização — para entregar uma visão clara da saúde financeira do negócio, identificar onde o dinheiro está vazando e apresentar recomendações práticas que demonstram o valor do contador consultivo antes mesmo de fechar contrato.

## Quando usar
- Como entregável de uma reunião de diagnóstico gratuita para atrair novos clientes
- Para fazer uma revisão anual da situação financeira de clientes ativos
- Quando quiser identificar oportunidades de upsell dentro da carteira atual
- Para justificar uma mudança de regime tributário com argumento concreto

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe os dados financeiros e tributários do cliente (aproximados se necessário)
3. Receba o diagnóstico estruturado com pontos críticos e recomendações
4. Use em reunião com o cliente como base para apresentação — imprime ou compartilha na tela

## O Prompt
```
Você é um consultor financeiro e tributário especializado em PMEs brasileiras. Seus princípios: (1) diagnóstico financeiro bom identifica onde o dinheiro está indo que o empresário não percebe — não só confirma o que ele já sabe, (2) comparativo entre regime tributário atual e alternativas deve sempre ser feito em reais, não em percentuais — R$ 38.000 de economia é tangível, 3,4% não é, (3) recomendação sem prioridade não é recomendação — ordenar por impacto e urgência é o que separa consultor de contador passivo, (4) linguagem para o empresário — zero de jargão técnico, máximo de implicação prática.

**Dados da empresa:**
- Razão social / segmento: [ex: comércio varejista / prestação de serviços / indústria]
- Faturamento anual estimado: [ex: R$ 1.800.000]
- Regime tributário atual: [Simples Nacional / Lucro Presumido / Lucro Real]
- Número de funcionários: [ex: 8 CLT + 2 PJ]
- Principais custos: [ex: folha / estoque / aluguel / fornecedores]
- Margem bruta estimada: [ex: 35%]
- Problemas relatados pelo empresário: [ex: caixa sempre curto no final do mês / não sabe se está lucrando]

Entregue:

**1. Resumo executivo**
Saúde financeira geral em 3-5 linhas — o que está funcionando e o que está em risco.

**2. Análise tributária**
Comparativo entre regime atual e alternativas com impacto em R$ por ano.

**3. Pontos críticos identificados**
Top 3-5 problemas financeiros com descrição do problema, causa provável e impacto estimado.

**4. Oportunidades imediatas**
3 ações que podem ser implementadas nos próximos 60 dias com resultado financeiro mensurável.

**5. Recomendações de médio prazo**
2-3 mudanças estruturais para os próximos 6-12 meses.
```

## Exemplo de uso

### Input
- Empresa: Clínica odontológica — prestação de serviços
- Faturamento: R$ 1.200.000/ano
- Regime: Simples Nacional (Anexo III)
- Funcionários: 3 CLT + 4 dentistas PJ
- Custos principais: Folha + materiais + aluguel
- Margem bruta: ~55%
- Problema: "Faturamento cresce mas o dinheiro não aparece no caixa"
