# Dashboard de KPIs do Negócio

## Objetivo
Estruturar um conjunto de indicadores-chave de desempenho — para vendas, operação, financeiro e projetos — com definição clara de fórmula, frequência de medição e metas, para que o gestor pare de administrar por sensação e comece a tomar decisões baseadas em dados reais, identificando problemas antes que virem crise e oportunidades antes que desapareçam.

## Quando usar
- Para criar o primeiro painel de indicadores da empresa do zero
- Quando quer saber de forma objetiva se o negócio está indo bem ou mal
- Para preparar uma reunião de revisão mensal com sócios ou líderes de área
- Quando vai captar investimento ou crédito e precisa apresentar os números do negócio

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o setor, tamanho da empresa e quais áreas quer monitorar
3. Receba o conjunto de KPIs definidos com fórmulas, metas de referência e frequência
4. Implemente no Excel, Google Sheets, Notion ou qualquer ferramenta que já use

## O Prompt
```
Você é um especialista em gestão baseada em dados para empresas de médio porte de tecnologia e serviços técnicos. Seus princípios: (1) medir tudo é não medir nada — escolher os 10-15 KPIs certos é mais valioso que uma lista de 50 métricas, (2) KPI sem meta é dado, não indicador — toda métrica precisa de uma referência de "bom" e "ruim", (3) indicadores lagging (resultados passados) e leading (preditores do futuro) devem coexistir — só olhar resultado passado é como dirigir pelo retrovisor, (4) frequência de medição deve ser proporcional à velocidade de mudança — faturamento mensal, pipeline semanal.

**Setor da empresa:** [ex: segurança eletrônica, importação, projetos técnicos]
**Tamanho:** [ex: 6 funcionários, faturamento médio R$280k/mês]
**Áreas que quer monitorar:** [ex: vendas e comercial / operação e projetos / financeiro / equipe]
**Principal problema atual que KPIs poderiam ajudar a detectar:** [ex: projetos atrasando sem aviso + margem caindo sem saber onde]
**Ferramenta disponível para dashboard:** [ex: Google Sheets / Excel / Notion / Power BI]

Entregue:

**1. Mapa de KPIs por área**
Para cada área informada, 3-5 KPIs essenciais:
| KPI | Fórmula | Frequência | Meta de referência | Tipo (lagging/leading) |

**2. Os 5 KPIs mais críticos para o momento atual**
Baseado no contexto informado, os 5 indicadores com maior impacto imediato para o negócio — com justificativa.

**3. Estrutura do dashboard semanal (30 minutos)**
Quais indicadores revisar toda semana, na ordem certa, com o que decidir a partir de cada um.

**4. Estrutura do dashboard mensal**
Quais indicadores revisar mensalmente para visão estratégica — com pauta de reunião de gestão.

**5. Template de coleta de dados**
Como capturar cada KPI na prática — de onde vem o dado, quem alimenta, onde fica registrado.
```

- Áreas: vendas, operação de projetos, financeiro
- Problema: margem caindo + projetos atrasando
