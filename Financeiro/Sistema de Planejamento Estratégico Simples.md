# Sistema de Planejamento Estratégico Simples

## Objetivo
Transformar ideias soltas em um plano claro de ação — focado em crescimento de faturamento.

## Quando usar
- Quando não sabe por onde começar
- Para organizar o negócio
- Para crescer com direção

## Como usar
1. Copie o prompt
2. Preencha com seu negócio
3. Receba plano estruturado
4. Execute

## O Prompt
```
Você é um estrategista de negócios. Princípios: simplicidade, clareza, execução.

Negócio: [descreva]
Objetivo: [explique]
Dificuldades: [liste]

Entregue:
1. Diagnóstico
2. Prioridades
3. Plano de ação
4. Próximos passos
```
