# Dashboard Financeiro para Renda Pulverizada

## Objetivo
Criar uma estrutura completa de controle financeiro para empreendedores com múltiplas fontes de renda simultâneas — projetos diferentes, clientes diferentes, parceiros a pagar — para ter clareza sobre quanto vai entrar, quanto vai sair, quanto já é lucro e qual projeto está valendo a pena. Substitui o "estou totalmente perdido financeiramente" por um painel simples que qualquer pessoa consegue manter atualizado em 15 minutos por semana.

## Quando usar
- Quando não souber ao certo quanto vai entrar no mês por conta de projetos simultâneos
- Para entender qual cliente ou projeto é realmente rentável e qual está consumindo mais do que paga
- Ao precisar planejar pagamentos a parceiros sem saber se vai ter caixa para isso
- Para criar uma visão financeira mensal que mostre lucro real, não só receita

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste seus projetos ativos, receitas previstas, custos de parceiros e despesas fixas
3. Receba a estrutura do dashboard financeiro com categorias, fórmulas e rotina de atualização
4. Monte em planilha (Google Sheets ou Excel) e atualize semanalmente

## O Prompt
```
Você é um consultor financeiro especializado em empreendedores solos e prestadores de serviços com múltiplas fontes de renda. Seus princípios: (1) fluxo de caixa é mais importante do que lucro para quem tem renda variável — saber quando o dinheiro vai entrar é mais urgente do que saber se vai ser lucro, (2) margem por projeto é a métrica mais importante para quem vende serviços — projeto com muito faturamento e margem baixa pode estar destruindo seu tempo, (3) separar conta pessoal de conta do negócio é o primeiro passo para ter clareza financeira — confundir as duas é confundir sua sobrevivência com o resultado do negócio, (4) o sistema financeiro ideal é o mais simples que você vai realmente usar.

Crie a estrutura completa do dashboard financeiro com base nas informações abaixo:

**Projetos/clientes ativos:** [liste cada um com: valor do contrato, duração, data de pagamento prevista]
**Parceiros que você paga:** [liste quem paga, quando e quanto — por projeto ou fixo]
**Despesas fixas do negócio:** [ex: ferramentas, escritório, contador, licenças]
**Despesas pessoais fixas (aproximadas):** [para calcular quanto o negócio precisa gerar]
**Forma de pagamento dos clientes:** [ex: boleto mensal, entrada + saldo, parcelado]
**Mês atual:** [para criar a projeção do mês corrente]

Entregue:

**1. Estrutura do dashboard — abas recomendadas**
Quais abas criar na planilha e o que cada uma controla.

**2. Aba: Projetos ativos**
Template com colunas: cliente | valor total | valor já recebido | a receber | data prevista | status do pagamento | margem estimada (após custo de parceiros).

**3. Aba: Fluxo de caixa mensal**
Template de projeção mês a mês: entradas previstas por data, saídas por data, saldo projetado semana a semana.

**4. Aba: Rentabilidade por projeto**
Cálculo de margem real por projeto: receita − custo de parceiros − custo do seu tempo − custos indiretos = lucro real.

**5. Rotina de atualização**
O que atualizar toda semana (15 min), o que revisar todo início de mês (30 min) e os 3 alertas que devem acender luz vermelha imediatamente.

**6. Indicadores mínimos para acompanhar**
Os 5 números que você precisa saber toda semana para ter controle financeiro real do negócio.
```

## Exemplo de uso

### Input
- Projetos: Cliente A (R$4.500/mês, todo dia 5), Cliente B (R$8.000 projeto, 50% entrada pago, 50% na entrega em 30 dias), Cliente C (R$2.200/mês, atrasado), 2 propostas em negociação
- Parceiros a pagar: Designer (R$1.500/projeto B), Trafegante (R$800/mês cliente A)
- Despesas fixas negócio: R$600/mês (ferramentas, contador)
- Despesas pessoais: R$5.000/mês
- Mês atual: Abril

### Output
1.
