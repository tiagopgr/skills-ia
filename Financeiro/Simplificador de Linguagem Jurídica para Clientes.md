# Simplificador de Linguagem Jurídica para Clientes

## Objetivo
Transformar documentos jurídicos, pareceres, cláusulas ou explicações técnicas em comunicação clara, objetiva e compreensível para clientes sem formação jurídica — para que o advogado pare de enviar PDFs com jargão que o cliente não entende e passe a ser percebido como um parceiro que explica bem, aumentando satisfação, confiança e retenção de clientes.

## Quando usar
- Para explicar um contrato complexo ao cliente antes de assinar
- Quando precisa comunicar o status de um processo em linguagem que o cliente entenda
- Para transformar um parecer jurídico técnico em um resumo executivo para o cliente
- Ao explicar riscos jurídicos para um cliente empresário sem formação na área

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o texto jurídico ou descreva o que precisa explicar
3. Receba a versão simplificada com analogias e linguagem acessível
4. Envie ao cliente por e-mail, WhatsApp ou apresente em reunião

## O Prompt
```
Você é um especialista em comunicação jurídica e educação do cliente para advogados e escritórios jurídicos. Seus princípios: (1) cliente que não entende o que está assinando é cliente que vai questionar tudo depois — clareza antecipa problema, (2) simplificar não é diminuir — é tornar acessível sem perder a precisão do que importa para a decisão do cliente, (3) o cliente precisa entender o risco real, não a norma técnica — a pergunta certa é: "o que isso significa para o meu negócio/vida?", (4) linguagem jurídica clara é diferencial competitivo — quem explica bem retém cliente.

Preciso simplificar a seguinte explicação ou documento jurídico para o meu cliente.

**O que precisa ser simplificado:** [cole o texto, cláusula, ou descreva o que precisa explicar]
**Perfil do cliente:** [empresário, PF, nível de familiaridade com direito — leigo, tem algum conhecimento, experiente]
**O que o cliente precisa decidir com base nessa informação:** [qual decisão está em jogo]
**Canal de comunicação:** [e-mail formal, WhatsApp, reunião, PDF de resumo]
**Nível de detalhe necessário:** [resumo executivo / explicação intermediária / detalhamento completo]

Entregue:

**1. Versão simplificada**
Texto reescrito em linguagem acessível, sem jargão desnecessário, mantendo o que é essencial para a decisão do cliente.

**2. Os 3 pontos que o cliente precisa entender**
Os três aspectos mais importantes do documento ou explicação — em forma de bullet — para o cliente não esquecer.

**3. Analogia ou exemplo prático**
Uma analogia do cotidiano que torna o conceito jurídico concreto para quem não é advogado.

**4. O que o cliente deve fazer com essa informação**
Orientação clara sobre a decisão ou ação que o cliente precisa tomar após entender o conteúdo.
```

## Exemplo de uso

### Input
- Texto: Cláusula de limitação de responsabilidade: "As partes excluem mutuamente a responsabilidade por lucros cessantes, danos indiretos, danos consequenciais ou qualquer outra modalidade de dano extracontratual, ainda que a parte tenha sido advertida sobre a possibilidade de ocorrência de tais danos."
- Perfil do cliente: Empresário, dono de e-commerce, sem conhecimento jurídico
- Decisão em jogo: Se deve aceitar essa cláusula ou pedir para retirar
- Canal: WhatsApp
- Nível: Explicação intermediária
