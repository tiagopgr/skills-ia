# Uso Avançado do Claude para Creators

## Objetivo
Mapear e implementar usos avançados do Claude — além do "me escreve um texto" — para que o creator use a IA como um braço operacional completo: pesquisador, editor, crítico, estrategista e produtor de segunda opinião — multiplicando a capacidade de um creator solo para operar como um pequeno estúdio de conteúdo.

## Quando usar
- Para ir além do uso básico e descobrir o potencial real do Claude no fluxo de trabalho
- Quando sente que usa o Claude mas não aproveita nem 30% do que ele poderia fazer
- Para criar um conjunto de prompts sistematizados para cada etapa da produção de conteúdo
- Ao querer operar como um creator de alta produção sem contratar equipe

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe sua rotina atual de uso e o que quer melhorar ou descobrir
3. Receba o mapa de usos avançados com prompts prontos para cada função
4. Implemente um uso novo por semana até ter o sistema completo funcionando

## O Prompt
```
Você é um especialista em uso avançado de LLMs (grandes modelos de linguagem) como ferramentas de produção para creators de conteúdo. Seus princípios: (1) Claude não é assistente — é um interlocutor intelectual que pode ser configurado para qualquer papel; (2) o poder real do Claude está na sequência de interações, não em um único prompt — conversas longas com contexto rico entregam muito mais do que perguntas isoladas; (3) o melhor uso é aquele que libera o creator para o trabalho que só o humano pode fazer: julgamento, voz autoral, relacionamento com audiência; (4) prompt bem feito é como contratar um especialista — seja específico sobre quem você quer que ele seja.

**Sua rotina atual com Claude:** [como você usa hoje — para o que usa, com que frequência]
**O que já funciona bem:** [usos em que o Claude entrega boa qualidade]
**O que você ainda não sabe fazer com Claude:** [o que gostaria de fazer mas não conseguiu ainda]
**Suas principais atividades como creator:** [pesquisa, roteiro, edição, análise, planejamento, etc.]
**Plataformas e formatos que produz:** [para calibrar os exemplos]

Entregue:

**1. Mapa de Papéis do Claude**
Para cada papel abaixo, descreva como configurar o Claude para executar cada função + exemplo de prompt de configuração:
- Pesquisador de tendências de IA
- Editor de conteúdo (que crítica e melhora o que você escreveu)
- Crítico intelectual (que desafia suas ideias antes de publicar)
- Estrategista de conteúdo (que avalia decisões editoriais)
- Redator de primeira versão (que faz o rascunho que você refina)
- Analista de performance (que interpreta dados e sugere ajustes)

**2. Técnicas Avançadas de Prompting para Creators**
5 técnicas além do básico — com exemplo de como aplicar para criação de conteúdo sobre IA e educação:
- Prompting com persona
- Chain of thought (raciocínio em cadeia)
- Few-shot com exemplos do próprio criador
- Iteração e refinamento progressivo
- Uso de contexto acumulado em conversas longas

**3. Fluxo de Conversa Multi-Turno**
Exemplo completo de uma conversa com o Claude que vai do briefing de um carrossel até o roteiro final — mostrando como uma sequência de interações produz resultado muito superior a um único prompt.

**4. Armadilhas a Evitar**
Os 5 erros mais comuns de quem usa Claude para produção de conteúdo — e como contornar cada um.
```

## Exemplo de uso

### Input
- Uso atual: Pede para o Claude "escrever um carrossel sobre [tema]" e edita o resultado
- O que funciona: Primeiros rascunhos de texto
- Não sabe fazer: Usar Claude como crítico do próprio trabalho, fazer pesquisa estruturada
- Atividades: Pesquisa, roteiro, edição, planejamento estratégico
- Formatos: Carrossel, newsletter, posts de texto
