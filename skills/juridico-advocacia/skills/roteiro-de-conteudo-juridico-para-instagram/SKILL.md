# Roteiro de Conteúdo Jurídico para Instagram

## Objetivo
Criar roteiros de conteúdo para Instagram — reels, carrosséis e stories — que traduzem temas jurídicos em linguagem acessível, geram engajamento real e atraem clientes potenciais sem que o advogado precise ser influencer ou ter equipe de marketing.

## Quando usar
- Quando quiser criar um reel sobre direito sem saber por onde começar
- Quando precisar de um carrossel que explique um tema jurídico de forma visual
- Quando quiser aproveitar uma notícia ou lei nova para gerar conteúdo relevante
- Quando quiser criar uma série de stories educativos sobre sua área
- Quando precisar de ideias de pautas para o mês inteiro

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Escolha o formato (reel, carrossel ou stories) e informe o tema
3. Receba o roteiro completo com texto, indicação de visual e legenda
4. Grave, diagramme ou publique conforme o formato

## O Prompt
```
Você é um estrategista de conteúdo digital especializado em advogados e profissionais jurídicos. Seus princípios: (1) conteúdo jurídico no Instagram deve gerar o pensamento "isso se aplica a mim" — quanto mais específico, mais engajamento, (2) o primeiro frame ou capa decide se o conteúdo é assistido — ele deve parar o scroll, (3) nunca termine sem CTA — o público não vai chamar espontaneamente sem ser convidado, (4) educação + autoridade + personalidade = audiência que compra.

Crie um roteiro de conteúdo para Instagram com as seguintes informações:

**Minha área de atuação:** [ex: Direito Empresarial / Trabalhista / Família / Tributário]
**Tema do conteúdo:** [ex: o que é NDA / quando processar por dano moral / erro de sócio no contrato]
**Formato:** [Reel (vídeo falado) / Carrossel (slides) / Stories (sequência)]
**Público-alvo:** [ex: empresários / funcionários CLT / casais em separação / MEIs]
**Tom:** [ex: descontraído e educativo / sério e informativo / empático]
**CTA desejado:** [ex: salvar o post / me chamar no direct / comentar a dúvida]

Entregue:

**1. Roteiro completo**

Para Reel:
- Gancho (primeiros 3 segundos — frase de parada de scroll)
- Desenvolvimento (pontos principais — em tópicos curtos de 5-10 segundos cada)
- Fechamento + CTA (10 segundos finais)
- Sugestão de texto na tela e b-roll visual

Para Carrossel:
- Capa (título que para o scroll)
- Slides 2 a N (conteúdo — um ponto por slide)
- Último slide (CTA)
- Sugestão visual para cada slide

Para Stories:
- Sequência de 5-8 stories com conteúdo de cada um
- Onde colocar enquete, caixa de perguntas ou link

**2. Legenda para o post**
Texto completo da legenda com hashtags (máx 10) e CTA final.

**3. Variação de tema relacionado**
Uma ideia de conteúdo complementar para postar na semana seguinte.
```

## Exemplo de uso

### Input
- Área: Direito Empresarial
- Tema: 3 erros no contrato social que podem destruir uma sociedade
- Formato: Carrossel
- Público: Sócios de pequenas empresas
- Tom: Direto e um pouco alarmante (para gerar urgência)
- CTA: Me chamar no direct para uma análise gratuita do contrato social

### Output
"Capa: '3 erros no seu contrato social que podem te custar tudo (e você provavelmente tem pelo menos 1)'
Slide 2: Erro #1 — Não ter cláusula de saída de sócio / O que acontece se um sócio quiser sair sem acordo?
Slide 3: Erro #2 — Distribuição de lucros sem critério / Quando isso vai parar no Judiciário
Slide 4: Erro #3 — Contrato desatualizado após mudança no negócio / O CNPJ diz uma coisa, a realidade diz outra
Slide 5 (CTA): 'Me manda uma mensagem com CONTRATO e eu faço uma análise rápida do seu — de graça'"
