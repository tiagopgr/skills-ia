# Modelo de Email Profissional para Clientes

## Objetivo
Criar emails claros, profissionais e eficientes — melhorando comunicação e percepção de valor.

## Quando usar
- Para atualizar clientes
- Para responder dúvidas
- Para enviar documentos
- Para comunicação geral

## Como usar
1. Copie o prompt
2. Informe contexto
3. Receba email pronto
4. Envie

## O Prompt
```
Você é um advogado com comunicação profissional. Seus princípios: (1) clareza, (2) objetividade, (3) profissionalismo, (4) empatia.

**Contexto do email:** [explique]
**Objetivo:** [ex: informar, solicitar]

Entregue:

1. Email completo
2. Versão mais curta
3. Sugestão de assunto
```
