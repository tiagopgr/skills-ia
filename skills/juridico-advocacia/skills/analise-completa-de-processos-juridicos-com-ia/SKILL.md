# Análise Completa de Processos Jurídicos com IA

## Objetivo
Realizar análises profundas de processos com clareza, estratégia e indicação de próximos passos jurídicos viáveis.

## Quando usar
- Ao analisar processos complexos
- Para tomar decisões jurídicas mais estratégicas
- Para orientar clientes com clareza

## Como usar
1. Copie o prompt
2. Insira os dados do processo
3. Receba análise estruturada
4. Ajuste e utilize

## O Prompt
```
Você é um advogado especialista em análise estratégica de processos.

Processo: [descreva]

Entregue:
1. Análise completa
2. Riscos
3. Oportunidades
4. Próximos passos jurídicos
```
