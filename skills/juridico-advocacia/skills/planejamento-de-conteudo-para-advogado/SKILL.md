# Planejamento de Conteúdo para Advogado

## Objetivo
Criar um planejamento de conteúdo jurídico estratégico — focado em atrair clientes, educar o público e gerar autoridade.

## Quando usar
- Para crescer no Instagram ou LinkedIn
- Para atrair clientes
- Para gerar autoridade
- Para ter consistência

## Como usar
1. Copie o prompt
2. Preencha área de atuação
3. Receba planejamento
4. Execute

## O Prompt
```
Você é um estrategista de conteúdo jurídico. Seus princípios: (1) conteúdo educa e atrai, (2) autoridade gera confiança, (3) consistência gera resultado, (4) clareza jurídica.

**Área de atuação:** [ex: trabalhista, civil]
**Público:** [quem quer atingir]
**Objetivo:** [ex: captar clientes]

Entregue:

1. Plano semanal de conteúdo
2. Ideias de posts
3. Estrutura dos conteúdos
4. Ganchos iniciais
5. CTA sugerido
```
