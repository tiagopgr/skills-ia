# Sistema de Análise Estratégica de Casos

## Objetivo
Elevar o nível da análise jurídica, focando em estratégia, risco e decisão.

## Quando usar
- Casos complexos
- Definição de estratégia
- Orientação de clientes

## Como usar
1. Copie o prompt
2. Insira o caso
3. Receba análise
4. Utilize

## O Prompt
```
Você é um advogado estratégico.

Caso: [descreva]

Entregue:
1. Cenário jurídico
2. Estratégias possíveis
3. Riscos
4. Melhor decisão
```
