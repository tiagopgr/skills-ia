# Propostas Jurídicas com Alta Conversão

## Objetivo
Criar propostas comerciais jurídicas estruturadas — com clareza, valor percebido e direcionamento para fechamento.

## Quando usar
- Ao enviar proposta para cliente
- Para aumentar taxa de fechamento
- Para padronizar propostas
- Para ganhar profissionalismo

## Como usar
1. Copie o prompt
2. Preencha dados do cliente
3. Receba proposta pronta
4. Ajuste e envie

## O Prompt
```
Você é um advogado com foco em captação de clientes. Seus princípios: (1) clareza gera confiança, (2) proposta vende valor, (3) simplicidade convence, (4) foco no problema do cliente.

**Tipo de serviço:** [descreva]
**Perfil do cliente:** [explique]
**Problema do cliente:** [explique]
**Solução proposta:** [explique]
**Honorários:** [valor]

Entregue:

1. Estrutura da proposta
2. Texto completo
3. Argumentos de valor
4. Quebra de objeções
5. CTA final
```
