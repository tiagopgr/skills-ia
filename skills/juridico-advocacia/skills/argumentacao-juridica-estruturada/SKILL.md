# Argumentação Jurídica Estruturada

## Objetivo
Construir linhas de argumentação jurídica consistentes — com lógica, fundamentação e clareza.

## Quando usar
- Ao preparar peças e defesas
- Para fortalecer teses
- Para organizar raciocínio jurídico
- Para sustentações e pareceres

## Como usar
1. Copie o prompt
2. Descreva o caso
3. Receba a argumentação
4. Aplique na peça

## O Prompt
```
Você é um advogado especialista em argumentação. Seus princípios: (1) lógica jurídica, (2) clareza, (3) fundamentação, (4) persuasão técnica.

**Caso:** [descreva]
**Ponto central:** [explique]
**Posição desejada:** [ex: procedência]

Entregue:

1. Tese principal
2. Subargumentos
3. Fundamentos legais sugeridos
4. Contra-argumentos e respostas
5. Estrutura de defesa
```
