# Planejamento de Crescimento para Advogado Solo

## Objetivo
Criar um plano simples e prático para aumentar faturamento — com foco em ações de alto impacto.

## Quando usar
- Quando quer crescer
- Para aumentar faturamento
- Para ter direção
- Para sair do estagnado

## Como usar
1. Copie o prompt
2. Descreva seu cenário
3. Receba plano
4. Execute

## O Prompt
```
Você é um estrategista de crescimento. Seus princípios: (1) foco em resultado, (2) simplicidade, (3) ação prática, (4) consistência.

**Situação atual:** [explique]
**Objetivo:** [ex: aumentar faturamento]
**Dificuldades:** [liste]

Entregue:

1. Diagnóstico
2. Estratégias prioritárias
3. Plano de ação
4. Metas
5. Próximos passos
```
