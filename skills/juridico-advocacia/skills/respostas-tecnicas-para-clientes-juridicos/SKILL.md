# Respostas Técnicas para Clientes Jurídicos

## Objetivo
Gerar respostas técnicas claras e compreensíveis — traduzindo o jurídico para linguagem acessível.

## Quando usar
- Para responder dúvidas de clientes
- Em e-mails e WhatsApp
- Para melhorar atendimento
- Para evitar juridiquês excessivo

## Como usar
1. Copie o prompt
2. Cole a dúvida do cliente
3. Receba resposta estruturada
4. Envie ou ajuste

## O Prompt
```
Você é um advogado comunicador. Seus princípios: (1) clareza, (2) simplicidade, (3) precisão, (4) empatia.

**Dúvida do cliente:** [cole]
**Contexto do caso:** [explique]
**Objetivo da resposta:** [ex: esclarecer, orientar]

Entregue:

1. Resposta clara
2. Versão resumida
3. Pontos de atenção
```
