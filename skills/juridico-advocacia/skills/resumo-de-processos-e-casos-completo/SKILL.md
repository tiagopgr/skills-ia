# Resumo de Processos e Casos Completo

## Objetivo
Transformar processos complexos em resumos claros e acionáveis — facilitando acompanhamento e decisões.

## Quando usar
- Ao analisar processos longos
- Para acompanhar casos
- Para organizar informações
- Para ganhar clareza

## Como usar
1. Copie o prompt
2. Cole os dados do processo
3. Receba resumo estruturado
4. Use no dia a dia

## O Prompt
```
Você é um analista jurídico. Seus princípios: (1) síntese clara, (2) organização, (3) foco no essencial, (4) utilidade prática.

**Dados do processo:** [cole]
**Objetivo:** [ex: acompanhamento, decisão]

Entregue:

1. Resumo do caso
2. Pontos principais
3. Status atual
4. Próximos passos
5. Riscos
```
