# Proposta Comercial Visual B2B

## Objetivo
Criar propostas comerciais B2B com design profissional de alto impacto — estrutura visual que guia o olhar, hierarquia de informações que vende, e layout que transmite autoridade — para impressionar decisores e acelerar o fechamento.

## Quando usar
- Ao montar uma proposta comercial para um cliente B2B
- Para padronizar o template de propostas da empresa ou agência
- Quando as propostas atuais não estão convertendo (parecem amadoras)
- Para elevar o nível de apresentação em negociações de ticket alto

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o serviço, cliente e contexto da proposta
3. Receba a estrutura completa com direção de design para cada página
4. Execute no Figma, Canva ou ferramenta de design

## O Prompt
```
Você é um designer sênior especializado em materiais comerciais B2B. Você cria propostas que vendem porque entende que: (1) a proposta é um produto — se o design é amador, o cliente assume que o serviço também é, (2) decisores B2B escaneiam antes de ler — a hierarquia visual define se a proposta será lida ou ignorada, (3) menos texto + mais estrutura visual = mais compreensão e mais confiança, (4) cada página tem uma função — não é decoração, é persuasão estruturada, (5) a proposta compete com outras 3-5 propostas na mesa do decisor — design é diferencial.

Crie a proposta comercial visual para:

**Empresa que está vendendo:** [nome, setor, o que faz]
**Serviço/Produto oferecido:** [o que está sendo proposto]
**Cliente-alvo:** [empresa, setor, tamanho, cargo do decisor]
**Ticket da proposta:** [valor estimado — influencia o nível de sofisticação]
**Problema do cliente:** [o que o cliente precisa resolver]
**Diferenciais competitivos:** [por que escolher você e não o concorrente]
**Identidade visual disponível:** [tem brand guide? cores? fontes? logo?]
**Ferramenta de design:** [Figma, Canva, PowerPoint, InDesign]
**Formato final:** [PDF, apresentação interativa, link]

Entregue a estrutura completa página por página (capa, contexto, solução, diferenciais, cases, cronograma, investimento, próximos passos, contracapa), direção de design completa (paleta, tipografia, grid, ícones), e checklist de qualidade antes de enviar.
```

## Exemplo de uso

### Input
Empresa: Agência de marketing digital
Serviço: Gestão de tráfego pago + CRO para e-commerce
Cliente: E-commerce de moda feminina, faturamento R$200k/mês
Ticket: R$8.500/mês
Diferencial: metodologia própria de ROAS, cases no setor de moda
Ferramenta: Figma

### Output
Página 8 — Investimento com 3 cards lado a lado, card central (recomendado) 10% maior com borda em destaque. Badge "Mais escolhido" no card central. Preço em tamanho grande, itens em lista com check icons verdes. Fundo escuro, cards com fundo #111.
