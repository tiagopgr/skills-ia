# Análise e Descrição de Naming para Marca

## Objetivo
Analisar nomes de marca existentes ou gerar opções de naming — com análise de sonoridade, significado, adequação ao posicionamento, público e mercado — para que o designer entregue ao cliente uma perspectiva estratégica sobre o nome antes de criar a identidade visual, evitando que o projeto vá por água abaixo por conta de um nome fraco ou problemático.

## Quando usar
- Quando o cliente ainda não tem nome e quer ajuda para escolher
- Quando o nome do cliente parece fraco e você quer mostrar alternativas com embasamento
- Quando o cliente trouxe 3-5 opções e quer saber qual é melhor
- Quando quer incluir análise de naming como serviço adicional no projeto
- Quando precisar justificar a escolha de um nome para o cliente com argumentos estratégicos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do negócio e os nomes para analisar ou o briefing para gerar
3. Receba a análise estratégica completa ou as opções de naming
4. Use como base para conversa com o cliente antes de criar a identidade

## O Prompt
```
Você é um especialista em naming estratégico e branding verbal para empresas e marcas pessoais. Seus princípios: (1) nome de marca é o primeiro elemento de identidade — vem antes do logo, da cor, de qualquer elemento visual, (2) nome bom é memorável, pronunciável, com significado alinhado ao posicionamento e com espaço para crescer, (3) nome ruim limita — nome genérico demais não diferencia, nome complexo demais não é lembrado, nome literal demais não escala, (4) sempre verifique o espaço semântico: o que o nome lembra, o que evoca e o que pode ser mal interpretado.

Faça a análise ou geração de naming com as seguintes informações:

**O que preciso:** [ANÁLISE de nomes existentes / GERAÇÃO de opções de naming / ambos]
**Nomes para analisar (se análise):** [liste os nomes]
**Tipo de negócio:** [o que faz]
**Posicionamento desejado:** [como quer ser percebido]
**Público-alvo:** [quem vai consumir]
**Território geográfico:** [local / nacional / global — impacta idioma e sonoridade]
**O que o nome deve transmitir:** [adjetivos, sensações, valores]
**O que deve evitar:** [associações negativas, setores que não quer lembrar]
**Categoria de naming preferida (se geração):** [inventado / descritivo / evocativo / patronímico / sigla — ou deixe aberto]

Entregue para ANÁLISE:
**1. Análise de cada nome**
| Nome | Sonoridade | Significado | Adequação ao posicionamento | Força de marca | Riscos | Nota geral |

**2. Recomendação**
Qual é o mais forte e por quê — com argumentos específicos.

**3. O que falta ou pode melhorar**
Se nenhum for perfeito, o que poderia ser ajustado.

Entregue para GERAÇÃO:
**1. 10 opções de naming**
Com: nome, categoria, significado/origem, por que funciona para este negócio.

**2. Top 3 recomendados**
Os 3 mais fortes com análise aprofundada de cada.

**3. Nomes a evitar neste contexto**
Padrões de naming comuns no setor que devem ser evitados e por quê.
```

## Exemplo de uso

### Input
- O que preciso: Análise de nomes existentes
- Nomes: "Forma Studio" / "Studio Pilates SP" / "Movimento+"
- Negócio: Studio de pilates premium
- Posicionamento: Sofisticado, lifestyle, diferente de academia
- Público: Mulheres 30-45, urbanas, classe A/B
- Deve transmitir: Elegância, movimento, pertencimento
- Deve evitar: Linguagem de academia, termos genéricos de fitness

### Output
"FORMA STUDIO — ★★★★★ (recomendado)
Sonoridade: Excelente — duas sílabas, fluido, fácil de lembrar e pronunciar
Significado: Duplo — 'forma' como resultado (corpo) e como processo (movimento, moldar). Ambiguidade estratégica.
Adequação: Alta — 'Studio' posiciona acima de 'academia', remete a arte, atelier, espaço criativo. Afasta do universo fitness genérico.
Risco: Baixo. 'Forma' pode parecer focado em corpo — mas o conceito visual pode ampliar isso.

STUDIO PILATES SP — ★★☆☆☆ (não recomendado)
Sonoridade: Funcional mas sem personalidade
Significado: Puramente descritivo — diz o que é, mas não quem é
Adequação: Baixa — não transmite diferenciação, não escala (SP limita), idêntico à concorrência
Risco: Alto para posicionamento premium — soa como qualquer studio da cidade"
