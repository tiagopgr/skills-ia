# Brand Guidelines / Identidade B2B

## Objetivo
Criar um guia de identidade visual (brand guidelines) completo para marcas B2B — documentando logo, cores, tipografia, ícones, grid, tom visual e regras de aplicação — para garantir consistência em todos os materiais e permitir que qualquer designer reproduza a marca corretamente.

## Quando usar
- Ao criar a identidade visual de uma empresa/marca do zero
- Para documentar uma identidade visual que existe mas não está formalizada
- Quando múltiplas pessoas criam materiais e a marca fica inconsistente
- Para entregar ao cliente um guia profissional junto com o projeto de branding

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a marca, público e materiais que precisam de consistência
3. Receba a estrutura completa do brand guide
4. Execute no Figma/InDesign e entregue como PDF ou Figma file

## O Prompt
```
Você é um brand designer que cria sistemas de identidade visual para empresas B2B. Você sabe que: (1) brand guidelines não são sobre restrições — são sobre consistência que constrói reconhecimento, (2) um guia que ninguém segue é inútil — precisa ser prático e acessível, (3) em B2B a marca transmite confiança — inconsistência visual = desconfiança, (4) o guia deve cobrir os 20% de regras que resolvem 80% dos casos.

Crie o brand guidelines para:

**Nome da marca:** [nome]
**Setor:** [tech, finança, saúde, etc.]
**O que a empresa faz:** [em 1 frase]
**Público B2B:** [quem são os clientes — cargo, setor]
**Personalidade da marca:** [3-5 adjetivos — moderna, confiável, inovadora]
**Logo existente?** [sim — descreva / não — precisa criar]
**Cores existentes?** [sim — quais / não — sugerir]
**Materiais que precisam de consistência:** [site, deck, proposta, redes sociais, email]
**Ferramenta de design do time:** [Figma, Canva, PowerPoint]
**Formato do guia:** [PDF, Figma file, Notion, site]

Entregue: página de abertura com índice visual; essência da marca (missão, visão, valores, personalidade, tom de voz com do/don't); regras de logo (versões, clear space, tamanho mínimo, o que não fazer); paleta completa (HEX + RGB + CMYK + proporção 60/25/15 + acessibilidade WCAG); tipografia (escala H1-H2-H3-body-small com line-height); iconografia (estilo, espessura, biblioteca); imagens e fotografia; grid e layout (8px base); aplicações reais (cartão, assinatura, deck, post, proposta, site); e checklist de consistência.
```

## Exemplo de uso

### Input
Marca: NovaTech Solutions — SaaS B2B gestão de projetos
Personalidade: moderna, confiável, clean, tech
Cores existentes: azul escuro #1E3A5F e branco
Materiais: site, deck de vendas, proposta, LinkedIn, email
Ferramenta: Figma

### Output
Paleta: Navy #1E3A5F (fundos, headers), White #FAFAFA (conteúdo, texto em fundo escuro), Electric Blue #3B82F6 (CTAs, links, hover). Neutros: Gray 100 #F3F4F6, Gray 300 #D1D5DB, Gray 500 #6B7280, Gray 900 #111827. Semânticas: Sucesso #22C55E, Alerta #F59E0B, Erro #EF4444. Proporção: Navy 50% + White 30% + Electric Blue 15% + Neutros 5%.
