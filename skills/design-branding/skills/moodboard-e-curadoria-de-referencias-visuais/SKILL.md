# Moodboard e Curadoria de Referências Visuais

## Objetivo
Montar moodboards profissionais a partir de referências curadas no Dribbble, Behance, Pinterest e Awwwards — para alinhar expectativas com clientes antes de começar o design e garantir que o resultado final seja exatamente o que foi aprovado.

## Quando usar
- No kickoff de qualquer projeto de design (antes de abrir o Figma)
- Para alinhar direção visual com cliente que 'não sabe explicar o que quer'
- Ao buscar inspiração para um estilo que não domina
- Para apresentar opções de direção criativa e pedir aprovação

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o projeto, público e preferências visuais
3. Receba o roteiro de busca e estrutura do moodboard
4. Execute a curadoria nas plataformas e monte no Figma/Milanote

## O Prompt
```
Você é um diretor de arte que usa curadoria de referências como etapa fundamental de todo projeto de design. Você sabe que: (1) começar a designar sem referências é inventar a roda, (2) o moodboard alinha expectativas — evita o "não era isso que eu imaginava", (3) referências não são para copiar — são para extrair princípios (cor, layout, tipografia, mood), (4) curadoria é filtrar, não acumular — 8-12 referências focadas superam 50 aleatórias, (5) Dribbble (UI/visual), Behance (projetos completos), Pinterest (estilo/mood), Awwwards (web real).

Crie o moodboard para o projeto:

**Tipo de projeto:** [logo, identidade visual, landing page, app, material de vendas, ebook]
**Setor/Nicho do cliente:** [tech, saúde, finanças, educação, moda]
**Público-alvo do design:** [quem vai ver o design final]
**Adjetivos desejados:** [moderno, premium, minimalista, ousado, corporativo]
**O que o cliente NÃO quer:** [o que evitar — cores, estilos]
**Referências que o cliente gosta:** [sites, marcas, designs que admira]
**Entregáveis do projeto:** [o que vai ser desenhado ao final]
**Ferramenta para o moodboard:** [Figma, Milanote, Miro, Notion]

Entregue: estratégia de busca por plataforma (Dribbble, Behance, Pinterest, Awwwards) com palavras-chave exatas e filtros; estrutura do moodboard (mood/atmosfera, paleta, tipografia, layout, elementos visuais, anti-referências); template do moodboard para Figma (dimensão, grid, labels); script de apresentação ao cliente; e guia para traduzir referências em decisões de design.
```

## Exemplo de uso

### Input
Projeto: redesign de landing page + materiais de vendas para SaaS B2B fintech
Adjetivos: moderno, confiável, tech, premium, dark
Não quer: colorido demais, infantil, genérico corporativo
Referências: Linear, Stripe, Vercel

### Output
Dribbble — termos: "fintech dashboard dark", "SaaS landing page dark theme", "B2B website minimal", "pricing page dark", "tech startup hero section", "dark UI card design". Filtros: cor preto/dark, tipo shot. Referências-âncora: Linear (clean dark UI), Stripe (gradientes sutis, hierarquia tipográfica), Vercel (grid, espaçamento, developer aesthetic).
