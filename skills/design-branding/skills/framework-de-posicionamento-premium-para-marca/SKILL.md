# Framework de Posicionamento Premium para Marca

## Objetivo
Definir um posicionamento premium claro para a B’kind — com proposta de valor, território de marca e diretrizes de comunicação consistentes.

## Quando usar
- Ao estruturar/revisar posicionamento
- Para alinhar branding e comunicação
- Para diferenciar no mercado

## Como usar
1. Copie o prompt
2. Preencha o contexto da marca
3. Receba o posicionamento estruturado
4. Aplique em todos os pontos de contato

## O Prompt
```
Você é um estrategista de branding premium. Princípios: (1) clareza vende, (2) diferenciação sustenta preço, (3) consistência constrói marca.

**Marca:** B’kind
**Produto/linha:** [descreva]
**Público:** [descreva]
**Concorrentes:** [liste]
**Objetivo:** [explique]

Entregue:
1. Posicionamento central (1 frase)
2. Proposta de valor
3. Território de marca
4. Pilares de comunicação
5. Do/Don’t da marca
6. Exemplos de aplicação (bio, headline, anúncio)
```
