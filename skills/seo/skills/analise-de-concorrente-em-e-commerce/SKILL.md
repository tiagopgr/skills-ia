# Análise de Concorrente em E-commerce

## Objetivo
Analisar os criativos, estratégias de anúncio e posicionamento dos concorrentes no Meta Ads Library e TikTok — identificando os ângulos que estão usando, os hooks que performam no nicho, as ofertas e os formatos mais frequentes — para extrair inteligência competitiva que acelera a criação de criativos vencedores sem precisar descobrir tudo por tentativa e erro. O que está rodando há semanas no concorrente é sinal de que está funcionando.

## Quando usar
- Antes de criar novos criativos — para entender o que já funciona no nicho
- Para identificar ângulos de comunicação que nenhum concorrente está usando (gap de mercado)
- Quando criativos próprios não performam e quer ver o que o mercado está fazendo
- Para benchmarking regular a cada 30-45 dias

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva os concorrentes que quer analisar e cole os dados que encontrou no Ads Library
3. Receba a análise estruturada com insights acionáveis para seus criativos
4. Use os insights como input para o próximo ciclo de criação

## O Prompt
```
Você é um especialista em inteligência competitiva para e-commerce e análise de criativos de performance no Meta Ads e TikTok. Seus princípios: (1) o objetivo da análise de concorrente não é copiar — é entender o que funciona no mercado para criar algo superior ou diferenciado, (2) criativo que o concorrente mantém rodando por 30+ dias é criativo vencedor — o algoritmo de ads não mantém criativos ruins rodando sem o anunciante perceber e pausar, (3) a análise deve cobrir: ângulos de comunicação (o que prometem), formatos (vídeo/imagem, duração), hooks (como abrem), ofertas (preço, frete, desconto) e posicionamento (para quem vendem), (4) o gap competitivo mais valioso é o ângulo que nenhum concorrente está usando — é aí que a diferenciação é mais fácil e a audiência está menos saturada.

**Nicho/produto:** [perfume de nicho, e-commerce europeu]
**Concorrentes para analisar:** [nomes das marcas — para buscar no Ads Library]
**Dados coletados do Ads Library / TikTok:**
- Concorrente 1: [descreva os anúncios que encontrou — quantos, formatos, hooks, textos]
- Concorrente 2: [idem]
- Concorrente 3: [idem]
**O que você já usa:** [seus ângulos e formatos atuais — para identificar diferenciação]
**Mercados analisados:** [França / UK / Europa]

Entregue:

**1. Mapa de ângulos de comunicação do mercado**
Tabela: concorrente → ângulos usados → formato principal → frequência/persistência dos anúncios.

**2. Padrões que aparecem em múltiplos concorrentes**
O que todo mundo está fazendo — indica que funciona no nicho.

**3. Gaps identificados**
Ângulos e abordagens que nenhum concorrente está usando — oportunidades de diferenciação.

**4. Top 3 insights acionáveis para seus criativos**
Os 3 aprendizados mais importantes da análise — com recomendação específica de como aplicar.

**5. Hooks mais comuns no nicho**
Lista dos tipos de abertura mais usados pelos concorrentes — útil como benchmark de CTR e referência de linguagem.
```

## Exemplo de uso

### Input
- Nicho: perfume de nicho masculino, e-commerce França
- Concorrentes analisados: Marca A (10 anúncios ativos), Marca B (6 anúncios), Marca C (14 anúncios)
- Marca A: vídeos 15-30s, hooks com "le parfum que tout le monde remarque", ênfase em reviews, oferta com frete grátis
- Marca B: imagens estáticas, preço em destaque, hook "Le niche à prix accessible", sem storytelling
- Marca C: vídeos longos 60s, lifestyle (homem bem vestido, cenários parisienses), sem preço visível, ênfase em exclusividade
- Você: vídeo 30s com hook de durabilidade, sem prova social nos criativos

### Output
| Gap | Oportunidade | Ação sugerida |
|-----|-------------|---------------|
| Nenhum concorrente usa unboxing como formato principal | Unboxing gera curiosidade e sensação de presente — mercado de presente masculino não está sendo trabalhado | Criar criativo de unboxing com ângulo "le cadeau parfait pour lui" |
| Prova social ausente na maioria | Reviews e UGC constroem confiança — o mercado está subutilizando | Criar criativo com texto de review real sobreposto ao vídeo do frasco |
| Ângulo de "sabendo escolher" inexplorado | Homem que escolhe perfume de nicho vs marca de massa é identidade — nenhum concorrente usa | Criar hook "Ce que les hommes qui savent choisir portent" |
