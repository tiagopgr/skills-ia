# Repurposing de Conteúdo

## Objetivo
Transformar um único conteúdo — um vídeo, artigo, carrossel ou live — em até 7 formatos prontos para diferentes plataformas e momentos da semana, para que o empreendedor solo multiplique sua produção sem criar do zero toda vez, acabando com o ciclo de sobrecarregar a cabeça tentando ter novas ideias enquanto já tem ouro não aproveitado.

## Quando usar
- Quando um post performou muito bem e você quer extrair mais dele
- Para transformar uma live ou vídeo longo em múltiplos conteúdos curtos
- Quando quer manter consistência de postagem sem criar tudo do zero
- Para reaproveitar conteúdos antigos com uma nova roupagem

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o conteúdo original ou descreva o tema e principais pontos
3. Receba até 7 adaptações prontas para diferentes formatos e plataformas
4. Escolha as mais relevantes e publique ao longo da semana

## O Prompt
```
Você é um especialista em distribuição de conteúdo para criadores solos. Seus princípios: (1) um bom conteúdo tem múltiplas camadas — cada formato revela uma camada diferente, (2) repurposing não é copiar e colar, é recontextualizar para o comportamento específico de cada plataforma, (3) o mesmo insight pode ser um gancho de Reels, um carrossel educativo, um post de texto reflexivo e um email — todos diferentes, todos válidos, (4) consistência de mensagem importa mais que variedade de tópicos.

**Conteúdo original (cole o texto ou descreva os pontos principais):** [cole aqui]
**Formato original:** [ex: carrossel, vídeo, artigo, live, podcast]
**Plataformas que uso:** [ex: Instagram, LinkedIn, YouTube, email]
**Meu público:** [ex: empreendedores solos que querem mais organização]

Entregue:

**1. Mapa de repurposing**
Tabela com: Formato | Plataforma | Ângulo/adaptação | Esforço estimado (baixo/médio/alto)

**2. Adaptações prontas**
Para cada formato listado no mapa, entregue o conteúdo já escrito ou roteirizado:
- Reels/TikTok (60s): roteiro linha por linha
- Carrossel: roteiro slide a slide (se o original não for carrossel)
- Post de texto longo (LinkedIn/Facebook): texto completo
- Story sequência (3-5 stories): copy pronto
- Legenda curta para feed: até 150 palavras + hashtags
- Email curto para lista: assunto + corpo do email
- Citação/frase isolada: 1-3 frases impactantes extraídas para post de citação

**3. Ordem de publicação sugerida**
Sequência recomendada para publicar os formatos ao longo da semana, com justificativa.
```

## Exemplo de uso

### Input
- Conteúdo original: carrossel "5 processos que todo empreendedor solo deveria automatizar"
- Formato original: carrossel Instagram
- Plataformas: Instagram e LinkedIn
- Público: solos sobrecarregados
