# Copy para Onboarding de App/SaaS

## Objetivo
Criar toda a copy do fluxo de onboarding de um app ou SaaS — desde a tela de boas-vindas até o momento 'aha!' — garantindo que o usuário entenda o valor do produto rapidamente.

## Quando usar
- Ao lançar um app ou SaaS novo
- Quando a taxa de ativação está baixa (muitos signups, poucos engajados)
- No redesign do fluxo de primeiro uso
- Para reduzir churn nos primeiros 7 dias

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu produto e o fluxo de onboarding
3. Receba a copy para cada tela do fluxo
4. Implemente e meça a taxa de completude

## O Prompt
```
Você é um UX Writer especialista em onboarding de produtos digitais. 75% dos usuários abandonam um app na primeira semana. Um bom onboarding: (1) mostra valor antes de pedir esforço, (2) divide em passos pequenos, (3) celebra cada progresso, (4) leva ao momento "aha!" rápido.

Crie a copy completa do fluxo de onboarding para:

**Produto:** [descreva o app/SaaS]
**O que o produto faz:** [proposta de valor em 1 frase]
**Momento "aha!":** [quando o usuário experimenta o valor]
**Público:** [quem vai usar]
**Tom:** [formal, casual, motivador, técnico]
**Número de telas:** [3-7 telas]

**Passos do onboarding:**
1. [ex: Criar conta]
2. [ex: Preencher perfil]
3. [ex: Conectar integração]
4. [ex: Criar primeiro projeto]
(liste os passos reais)

Para CADA tela, entregue:

**Headline:** Curta, motivadora, orientada ao resultado
**Subtexto de apoio:** 1-2 frases
**Texto dos campos/inputs** (se houver)
**Texto do botão (CTA)**
**Indicador de progresso**
**Opção de pular** (quando aplicável)

Extras:
- Copy para email de boas-vindas
- Copy para notificação "complete seu perfil"
- Copy para tela de onboarding completo
```

## Exemplo de uso

### Input
Produto: Ferramenta de gestão de tarefas para equipes
Proposta de valor: "Organize tarefas, prazos e responsáveis em um só lugar"
Momento "aha!": Quando cria a primeira tarefa e atribui a alguém
Passos: 1) Criar conta 2) Nomear workspace 3) Criar primeiro projeto 4) Adicionar tarefa 5) Convidar colega

### Output
**Tela 3 — Criar primeiro projeto:**

Headline: "Dê um nome ao que sua equipe vai conquistar"

Subtexto: "Pode ser um projeto real ou só um teste. Você pode renomear, duplicar ou excluir depois — sem estresse."

Campo: Placeholder → "Ex: Lançamento do site, Sprint de março"

CTA: "Criar meu projeto"

Progresso: "Passo 3 de 5 — você está voando!"

Pular: "Criar depois — quero explorar primeiro"
