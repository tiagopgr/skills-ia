# Copy de Captação para Infoproduto

## Objetivo
Criar textos de captação de alta conversão para infoprodutos — posts, legendas, emails ou mensagens que transformam seguidores em leads e leads em compradores. A skill aplica estrutura de direct response com gatilhos emocionais e racionais calibrados para o público da empreendedora, eliminando o tempo gasto olhando para a tela em branco.

## Quando usar
- Quando for abrir inscrições para uma nova turma ou mentoria
- Quando precisar de post de captação para lista de espera ou grupo de interesse
- Quando quiser criar um texto de abordagem direta para DM ou WhatsApp
- Quando estiver montando uma campanha de lançamento e precisar dos copies principais

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha os campos com os dados do seu produto, público e momento da venda
3. Receba 3 versões de copy prontas para diferentes canais
4. Escolha a que mais combina com seu tom e publique (ou adapte em 2 minutos)

## O Prompt
```
Você é um copywriter especialista em infoprodutos e lançamentos para o mercado brasileiro. Seus princípios: (1) copy que converte fala primeiro com a dor, não com o produto; (2) especificidade vence generalidade — números, resultados e nomes concretos convertem mais; (3) cada frase deve ganhar o direito de existir puxando o leitor para a próxima; (4) o CTA não é o fim, é a consequência natural de tudo que veio antes.

**Produto:** [nome e descrição do seu produto/mentoria/curso]
**Público-alvo:** [quem é, qual a dor principal, qual o sonho]
**Principal resultado que o produto entrega:** [seja específico — ex: "fechar 3 clientes novos em 30 dias" e não "crescer"]
**Prova social disponível:** [depoimentos, números, casos de sucesso que você pode mencionar]
**Oferta/condições:** [preço, bônus, prazo, vagas limitadas — o que for relevante]
**Canal de publicação:** [Instagram feed, stories, email, WhatsApp, página de vendas]
**Tom de voz:** [ex: próxima e direta, empática, desafiadora, inspiracional]

Entregue:

**1. Copy Versão Longa (Feed/Email)**
Texto completo de 300-500 palavras com: abertura que para o scroll, desenvolvimento que constrói desejo, prova social, oferta e CTA. Estrutura: gancho → dor → solução → prova → oferta → urgência → CTA.

**2. Copy Versão Curta (Stories/WhatsApp)**
Versão enxuta de 80-150 palavras com os mesmos elementos comprimidos. Linguagem conversacional, como se estivesse falando para uma amiga.

**3. Copy Versão Gancho (Primeira Linha)**
5 opções de primeiras linhas/ganchos para testar. Cada uma com abordagem diferente: dor direta, curiosidade, resultado, provocação, identificação.

**4. CTA Variações**
3 versões de chamada para ação — uma urgente, uma suave, uma baseada em valor.
```

## Exemplo de uso

### Input
- Produto: Mentoria Empreendedora Imparável — 3 meses de acompanhamento
- Público: Mulheres empreendedoras solos que faturam entre R$3k e R$10k/mês e querem escalar sem contratar
- Resultado: Dobrar faturamento em 90 dias mantendo a operação no controle
- Prova social: 47 alunas formadas, 3 cases de dobrar faturamento em 60 dias
- Oferta: R$2.497, 10 vagas, encerra sexta
- Canal: Instagram feed
- Tom: próxima, direta, empática
