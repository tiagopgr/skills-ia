# Sistema de Reaproveitamento de Conteúdo

## Objetivo
Transformar um único conteúdo — uma ideia, um vídeo, um carrossel, uma newsletter — em múltiplas versões adaptadas para diferentes plataformas e formatos, multiplicando o alcance do mesmo esforço de criação sem precisar reescrever tudo do zero, para que o creator solo publique mais com o mesmo tempo disponível.

## Quando usar
- Após criar um conteúdo bom que merece mais alcance em outras plataformas
- Para transformar um vídeo longo em posts, carrosséis e threads automaticamente
- Quando quer estar presente em mais canais sem criar conteúdo diferente para cada um
- Para extrair o máximo de uma ideia ou pesquisa que levou tempo para desenvolver

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o conteúdo original (texto, roteiro, transcrição ou resumo)
3. Informe para quais formatos e plataformas quer adaptar
4. Receba todas as versões adaptadas prontas para revisão e publicação

## O Prompt
```
Você é um especialista em estratégia de conteúdo omnichannel para creators solos. Seus princípios: (1) reaproveitamento inteligente não é copiar e colar — é identificar o núcleo da ideia e reembalá-la no formato nativo de cada plataforma; (2) cada plataforma tem uma gramática própria — o que funciona como carrossel não funciona como thread sem adaptação real; (3) o mesmo insight pode ser o hook de um Reel, a abertura de uma newsletter e o slide 3 de um carrossel — o trabalho é identificar onde cada parte brilha mais; (4) um bom conteúdo original gera pelo menos 5 derivados sem perder qualidade.

**Conteúdo original (cole aqui):** [texto completo, roteiro, transcrição ou resumo detalhado do conteúdo que quer reaproveitar]
**Formato de origem:** [ex: vídeo YouTube / carrossel Instagram / newsletter / post LinkedIn / rascunho de ideias]
**Plataformas de destino:** [ex: LinkedIn, Instagram, Twitter/X, newsletter, YouTube Shorts]
**Formatos desejados:** [ex: thread, carrossel, Reels roteiro, post de texto, subject de email]

Entregue para cada formato solicitado:

**[Nome da Plataforma + Formato]**
- Adaptação completa do conteúdo no formato nativo da plataforma
- Ajustes de tom, tamanho e estrutura específicos para esse canal
- Indicação do que foi mantido do original e o que foi transformado

Ao final:

**Mapa de Reaproveitamento**
Tabela mostrando: Conteúdo original → Derivado → Plataforma → Status (pronto para publicar / precisa de ajuste)

**O que ficou de fora**
Partes do original que não foram usadas em nenhum derivado — com sugestão de como poderiam virar um conteúdo futuro.
```

## Exemplo de uso

### Input
- Conteúdo original: Vídeo de 8 minutos sobre "O que é IA agêntica e como ela vai mudar a educação" — com explicação do conceito, 3 exemplos práticos na educação e reflexão final sobre o papel do professor
- Formato de origem: Vídeo YouTube
- Plataformas: Instagram, LinkedIn, newsletter, Twitter/X
- Formatos: Carrossel (Instagram), post longo (LinkedIn), abertura de newsletter, thread (Twitter/X)
