# Post de Autoridade com Dados e Estatísticas

## Objetivo
Criar posts que usam dados, estatísticas, pesquisas e números concretos para construir autoridade, educar o público e gerar compartilhamentos por serem referência no assunto.

## Quando usar
- Para se posicionar como especialista no nicho
- Quando quer criar conteúdo altamente compartilhável
- Para embasar argumentos com credibilidade
- Em datas relevantes ou quando sai uma pesquisa nova do setor

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema e os dados que tem (ou peça para pesquisar)
3. Receba o post estruturado com dados apresentados de forma acessível
4. Publique com design que destaque os números

## O Prompt
```
Você é um criador de conteúdo data-driven que transforma dados e estatísticas em posts de redes sociais acessíveis, impactantes e compartilháveis. Você sabe que posts com dados concretos geram 2x mais compartilhamentos que posts opinativos, desde que os dados sejam apresentados de forma simples e visual.

Crie um post de autoridade baseado em dados para:

**Tema:** [sobre o que é o post]
**Nicho:** [sua área de atuação]
**Público:** [quem vai consumir]
**Dados disponíveis:** [se tiver dados específicos, cole aqui. Se não tiver, pesquise e use dados públicos relevantes]
**Formato:** [carrossel | imagem única | thread | infográfico]
**Objetivo:** [educar, chocar, gerar debate, posicionar autoridade]

Entregue:

**1. Hook com dado impactante**
Abrir com o número mais surpreendente para capturar atenção.

**2. Contextualização**
Explicar o que o dado significa na prática (traduzir número em impacto real).

**3. Análise / Opinião expert**
Sua interpretação profissional sobre o dado — o que poucos percebem.

**4. Implicação prática**
O que o público deveria FAZER com essa informação.

**5. CTA**
Chamada para ação (compartilhar, comentar opinião, salvar).

Para o post completo:
- **Legenda** (200-350 palavras, acessível, sem jargão acadêmico)
- **Fonte dos dados** (citação da pesquisa/estudo)
- **Sugestão visual** (como apresentar o número no design — gráfico, destaque, comparação)
- **3 frases de destaque** (para usar como texto na imagem ou stories)
- **Hashtags** (10-15 relevantes)

Regras:
- Nunca inventar dados. Se não tiver fonte, indicar claramente
- Traduzir porcentagens em exemplos concretos ("7 em cada 10 pessoas...")
- Comparar com algo tangível quando possível ("equivale a X estádios de futebol")
```

## Exemplo de uso

### Input
Tema: Crescimento do mercado de infoprodutos no Brasil
Nicho: Marketing digital / negócios online
Público: Empreendedores digitais e aspirantes
Dados: Tenho alguns dados da Hotmart e ABED
Formato: Carrossel

### Output
**Hook:** "O mercado de infoprodutos no Brasil cresceu 46% em 2025. E 73% dos criadores ainda faturam menos de R$5 mil/mês. O que isso significa pra você:"

**Slide 2:** "R$10 bilhões movimentados em cursos online no Brasil (Fonte: ABED 2025)" + gráfico de crescimento

**Slide 3:** "Mas 73% dos criadores faturam menos de R$5k/mês. O mercado cresce, mas a maioria fica para trás."

**Slide 4:** "O que separa os 27% que faturam bem? Dados mostram 3 fatores..."
