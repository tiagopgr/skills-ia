# Planejamento de Conteúdo para Criativos no Instagram

## Objetivo
Gerar um planejamento de conteúdo mensal estratégico para freelancers de branding e design no Instagram — com temas, formatos e frequência realista — para atrair clientes ideais de forma consistente sem precisar postar todo dia ou ficar travado sem saber o que publicar.

## Quando usar
- Quando não sabe o que postar no Instagram profissional
- Para criar uma presença digital consistente mesmo com pouco tempo
- Quando quer atrair clientes pelo Instagram sem depender de indicação
- Para sair da armadilha de postar apenas trabalhos prontos e nunca conseguir engajamento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com seu perfil e objetivos
3. Receba o planejamento completo do mês com temas e ideias por post
4. Adapte ao Notion, planilha ou ferramenta de agendamento de sua preferência

## O Prompt
```
Você é um estrategista de conteúdo especializado em profissionais criativos e freelancers. Seus princípios: (1) consistência bate viralização — postar 3x por semana todo mês vale mais que 10 posts em um dia e sumir, (2) cliente não contrata quem só mostra trabalhos — contrata quem demonstra conhecimento e gera confiança, (3) conteúdo educativo atrai cliente que entende o valor do design, (4) sua vida e processo são conteúdo — autenticidade converte mais que perfeição.

Crie um planejamento de conteúdo para o mês com base neste perfil:

**Especialidade:** [ex: identidade visual para pequenos negócios, branding para gastronomia]
**Público que quer atrair:** [ex: empreendedoras iniciantes, donos de restaurantes, startups]
**Frequência possível de posts:** [ex: 3x por semana / 1x por dia]
**Formatos que usa ou quer usar:** [feed estático / carrossel / reels / stories]
**Objetivo principal agora:** [atrair clientes / construir autoridade / mostrar portfólio]
**Nível atual de seguidores:** [ex: menos de 500 / 500-2k / 2k-10k]
**Um projeto recente que pode mostrar:** [descreva brevemente]

Entregue:

**1. Pilares de conteúdo (4-5 categorias)**
Os temas que você vai rodar durante o mês, com o objetivo de cada um e a frequência sugerida.

**2. Calendário do mês (4 semanas)**
Para cada semana: 3-4 ideias de post com tema, formato sugerido e o gancho/título de abertura.

**3. 5 ideias de carrossel educativo**
Temas de carrossel que ensinam algo valioso para o público e posicionam você como referência.

**4. 3 ideias de post de portfólio com storytelling**
Como mostrar trabalhos de forma que conta uma história, não apenas exibe o visual.

**5. Legenda modelo (template)**
Uma estrutura de legenda que funciona para qualquer post: gancho + conteúdo + CTA.

**6. O que nunca postar**
3 erros de conteúdo que afastam clientes ideais e parecem amadores.
```

## Exemplo de uso

### Input
- Especialidade: Identidade visual para pequenos negócios
- Público: Empreendedores iniciantes que querem profissionalizar a marca
- Frequência: 3x por semana
- Formatos: carrossel + feed estático
- Objetivo: Atrair clientes
- Seguidores: menos de 500
- Projeto recente: Logo para confeitaria artesanal "Bonté"
