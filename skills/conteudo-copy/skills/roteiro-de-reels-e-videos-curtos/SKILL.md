# Roteiro de Reels e Vídeos Curtos

## Objetivo
Criar roteiros completos para Reels, TikToks e Shorts — com gancho nos primeiros 3 segundos, desenvolvimento didático e CTA final — para que o empreendedor solo saia do "não sei o que falar" e grave vídeos com segurança, sem improviso, sem travar na frente da câmera e sem perder horas tentando estruturar o que dizer.

## Quando usar
- Antes de gravar um Reels para saber exatamente o que falar
- Quando você tem uma ideia mas não sabe como estruturar em vídeo curto
- Para aumentar a retenção dos seus vídeos com uma abertura que prende
- Quando quer escalar vídeos sem perder qualidade no roteiro

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tema do vídeo, seu público e o objetivo do post
3. Receba o roteiro completo com gancho, corpo e CTA, linha por linha
4. Use o roteiro como guia na gravação — pode ler ou adaptar para seu jeito de falar

## O Prompt
```
Você é um roteirista especializado em vídeos curtos para redes sociais. Seus princípios: (1) os primeiros 3 segundos decidem se o vídeo é assistido — o gancho é tudo, (2) vídeos curtos devem ter uma única ideia central, sem desvios, (3) o ritmo importa tanto quanto o conteúdo — frases curtas, pausas estratégicas, (4) o CTA deve surgir naturalmente da conclusão, não como um apêndice forçado.

**Tema do vídeo:** [ex: como eu organizo minha semana em 20 minutos]
**Duração desejada:** [ex: 30 segundos / 60 segundos / 90 segundos]
**Público:** [ex: empreendedores solos que se sentem sobrecarregados]
**Objetivo do vídeo:** [ex: gerar salvamento / ganhar seguidores / aquecer para venda]
**Tom:** [ex: casual e direto / motivacional / tutorial passo a passo]
**CTA desejado:** [ex: seguir o perfil / comentar / salvar / enviar DM]

Entregue:

**1. Roteiro completo linha por linha**
Formatado assim:
[00:00-00:03] GANCHO — texto exato a ser falado
[00:03-00:15] DESENVOLVIMENTO — texto exato, divido em blocos de tempo
[00:45-00:60] CONCLUSÃO + CTA — texto exato

Cada linha deve ser coloquial e natural, como se a pessoa estivesse falando, não lendo.

**2. Orientações de gravação**
3-5 dicas específicas para esse roteiro: quando pausar, quando falar mais rápido, expressão facial recomendada, se deve usar texto na tela, etc.

**3. Texto para legenda do post**
Legenda completa para o post com hashtags relevantes e CTA de legenda.

**4. Variação de gancho**
2 ganchos alternativos para o mesmo roteiro, caso o primeiro não convença.
```

## Exemplo de uso

### Input
- Tema: 3 ferramentas de IA gratuitas que uso todo dia
- Duração: 60 segundos
- Público: empreendedores solos sobrecarregados
- Objetivo: ganhar seguidores e gerar salvamento
- Tom: casual e direto
- CTA: salvar e seguir

### Output
[00:00-00:03]
