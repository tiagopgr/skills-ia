# Gerador de Outline de Artigo com Pesquisa

## Objetivo
Criar outlines (estruturas) completas e estratégicas para artigos de blog, incluindo pesquisa de concorrentes, ângulo diferenciado, estrutura SEO e talking points para cada seção — tudo antes de escrever uma palavra do artigo.

## Quando usar
- Antes de escrever qualquer artigo (planejar antes de executar)
- Para briefar redatores freelancers ou equipe de conteúdo
- Quando quer garantir que o artigo será melhor que os concorrentes
- Para estratégia de conteúdo em escala (criar outlines em lote)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o tema e keyword do artigo
3. Receba o outline completo pronto para ser escrito
4. Use como guia de redação ou briefing para redator

## O Prompt
```
Você é um estrategista de conteúdo que cria outlines de artigos profissionais antes da redação. Você sabe que 80% da qualidade de um artigo é definida na estrutura — um outline ruim gera um artigo ruim.

Crie um outline detalhado para o seguinte artigo:

**Keyword principal:** [termo alvo]
**Keywords secundárias:** [termos relacionados]
**Tema do artigo:** [sobre o que será]
**Público-alvo:** [quem vai ler]
**Nível do público:** [iniciante, intermediário, avançado]
**Tipo de artigo:** [tutorial, listicle, guia, comparação, opinião, case]
**Tamanho alvo:** [1.000 | 1.500 | 2.000 | 3.000 palavras]
**Blog/Site:** [onde será publicado]
**Tom:** [casual, profissional, técnico, didático]
**Objetivo:** [tráfego SEO, leads, autoridade, vendas indiretas]
**Concorrentes conhecidos:** [artigos que já rankeiam para essa keyword — se conhecer]

Entregue o outline em 5 partes:

**PARTE 1 — PESQUISA COMPETITIVA**
O que os concorrentes cobrem | O que NÃO cobrem | Ângulo diferenciado sugerido | Formato sugerido

**PARTE 2 — SEO PLANNING**
Title tag | Meta description | URL slug | Intenção de busca

**PARTE 3 — OUTLINE DETALHADO**
Para cada seção: H2 com keyword | Palavras estimadas | Objetivo da seção | Talking points | Exemplo/Dado a incluir | [Link interno sugerido]

**PARTE 4 — ELEMENTOS ESPECIAIS**
FAQ | Tabelas/Infográficos sugeridos | CTAs distribuídos | Imagens sugeridas

**PARTE 5 — BRIEFING PARA REDATOR**
Resumo executivo de 1 parágrafo
```

## Exemplo de uso

### Input
Keyword: "como começar a investir"
Público: Jovens adultos 20-30 que nunca investiram
Tipo: Guia tutorial
Tamanho: 2.500 palavras
Tom: Didático e acessível

### Output
**PARTE 1 — Pesquisa:**
"Concorrentes provavelmente cobrem: tipos de investimento, onde abrir conta, quanto investir.
O que NÃO cobrem: o lado emocional (medo, paralisia), erros psicológicos, como automatizar.
Ângulo diferenciado: 'O guia para quem tem medo de investir' — cobrir o emocional E o prático.
Formato: Guia passo a passo com tom empático, não financeirês."

**PARTE 3 (seção exemplo):**
## H2: Passo 2 — Abra uma conta em uma corretora
- Keyword: "abrir conta corretora"
- Palavras: 250
- Objetivo: Eliminar o medo de abrir conta
- Talking points:
  1. É grátis e leva 5 minutos
  2. Comparação de 3 corretoras fáceis
  3. O que acontece depois de abrir
- Dado: "Em 2025, X milhões de brasileiros abriram conta em corretora"
