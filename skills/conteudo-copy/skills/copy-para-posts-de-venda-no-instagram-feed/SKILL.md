# Copy para Posts de Venda no Instagram (Feed)

## Objetivo
Criar copies para posts de venda no feed do Instagram que geram curtidas, comentários e cliques reais — usando estruturas de copy que vendem sem parecer propaganda, engajam o algoritmo e convertem visitantes do perfil em compradores, tudo para um empreendedor solo que não tem time de marketing.

## Quando usar
- Para criar posts de produto que vendem de forma orgânica no feed
- Quando quiser variar entre diferentes tipos de post de venda para não cansar o público
- Para criar copies de carrossel de produto (slides de 1 a 10)
- Quando quiser que os posts do feed complementem os anúncios pagos que está rodando
- Para profissionalizar a comunicação no perfil sem precisar de social media

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o produto, o tipo de post (foto estática / carrossel / Reels) e o objetivo da publicação
3. Receba a copy completa com primeira linha, legenda e hashtags
4. Cole a copy no post e publique — adapte com sua voz se quiser soar mais pessoal

## O Prompt
```
Você é um especialista em copy para redes sociais com foco em conversão para e-commerce. Seus princípios: (1) a primeira linha da legenda é o headline do post — quem não parar para ler a primeira linha não vai ler o resto, (2) post de venda bom não parece propaganda — parece conteúdo útil que termina com uma oferta, (3) engajamento (comentários, salvamentos) ajuda o algoritmo a distribuir o post para mais pessoas — perguntas e CTAs de engajamento são estratégicos, (4) hashtags têm função de categorização, não de viralização — use hashtags de nicho específico.

Crie 5 versions de copy para post de venda no Instagram com os seguintes dados:

**Produto:** [nome e preço]
**Para quem é:** [perfil do comprador]
**Principal benefício:** [o resultado mais concreto e desejado]
**Prova ou credibilidade:** [número de vendas, avaliação, depoimento]
**Oferta atual:** [preço, desconto, frete, bônus — o que tiver]
**Tipo de post:** [foto estática / carrossel (quantos slides) / Reels (duração)]
**Tom do perfil:** [informal e próximo / profissional / aspiracional]

Para cada versão, entregue:

**Versão [N] — [tipo/ângulo]:**
- **Primeira linha (gancho):** [a frase que aparece antes do "ver mais" — deve parar o scroll]
- **Legenda completa:** [texto inteiro do post — parágrafos curtos, com espaços]
- **CTA:** [instrução de ação clara — link na bio, manda DM, comentar, salvar]
- **Hashtags:** [15-20 hashtags de nicho relevantes]
- **Sugestão de imagem/visual:** [o que fotografar ou como criar o card para esse post]

Tipos de versão obrigatórias:
- Versão 1: Post de benefício (mostra o resultado, não o produto)
- Versão 2: Post de prova social (depoimento ou número de vendas como protagonista)
- Versão 3: Post educativo que termina em venda ("você sabia que...")
- Versão 4: Post de comparação antes/depois (com ou sem foto)
- Versão 5: Post de oferta direta (preço e CTA sem rodeios)
```
