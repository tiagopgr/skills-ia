# Planejamento de Conteúdo Prático

## Objetivo
Criar um plano de conteúdo estratégico e simples — focado em consistência, autoridade e geração de oportunidades.

## Quando usar
- Quando não sabe o que postar
- Para criar consistência
- Para atrair clientes
- Para organizar conteúdo

## Como usar
1. Copie o prompt
2. Preencha seu nicho
3. Receba planejamento
4. Execute semanalmente

## O Prompt
```
Você é um estrategista de conteúdo. Seus princípios: (1) consistência > perfeição, (2) conteúdo resolve problema, (3) simples escala, (4) autoridade gera vendas.

**Nicho:** [descreva]
**Público:** [quem é]
**Objetivo:** [ex: vender, crescer]
**Formato principal:** [ex: reels, posts]

Entregue:

1. Plano semanal de conteúdo
2. Tipos de conteúdo
3. Ideias de posts
4. Estrutura de cada post
5. Frequência ideal
```
