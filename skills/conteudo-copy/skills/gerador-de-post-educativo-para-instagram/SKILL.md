# Gerador de Post Educativo para Instagram

## Objetivo
Criar posts educativos completos para Instagram (carrossel, imagem única ou texto longo) que geram valor, posicionam autoridade e aumentam salvamentos e compartilhamentos.

## Quando usar
- Para manter constância de postagem com conteúdo de valor
- Quando quer aumentar salvamentos e compartilhamentos (sinais fortes para o algoritmo)
- Para construir autoridade no nicho
- Quando precisa de ideias de conteúdo educativo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina o tema e formato do post
3. Receba o conteúdo completo pronto para publicar
4. Adapte o visual no Canva ou ferramenta de design

## O Prompt
```
Você é um estrategista de conteúdo para Instagram especializado em posts educativos de alta performance. Você sabe que posts educativos são os que mais geram salvamentos e compartilhamentos, e que esses dois sinais são os mais valorizados pelo algoritmo do Instagram para distribuição orgânica.

Crie um post educativo completo para Instagram:

**Tema do post:** [sobre o que você quer ensinar]
**Nicho:** [sua área de atuação]
**Público-alvo:** [quem segue seu perfil]
**Nível do público:** [iniciante, intermediário, avançado]
**Formato desejado:** [carrossel (quantos slides?) | imagem única com legenda longa | Reels com texto na tela]
**Tom:** [ex: didático, informal, profissional, divertido]
**Objetivo:** [educar, gerar salvamentos, atrair seguidores, vender indiretamente]

Entregue:

### Para formato CARROSSEL:
- **Slide 1 (Capa):** Texto de impacto + sugestão visual
- **Slides 2-8 (Conteúdo):** Texto de cada slide (máximo 40 palavras por slide) + sugestão visual
- **Slide 9 (CTA):** Chamada para ação (salvar, compartilhar, comentar, seguir)
- **Legenda:** Texto complementar para a legenda (150-300 palavras)
- **Hashtags:** 15-20 hashtags relevantes organizadas por tamanho (pequena, média, grande)

### Para formato IMAGEM ÚNICA:
- **Texto da imagem:** Frase de impacto para a arte
- **Legenda completa:** Texto de 200-400 palavras com estrutura de gancho > valor > CTA
- **Hashtags:** 15-20 relevantes

### Para ambos:
- **Melhor horário sugerido** para publicar esse tipo de conteúdo
- **Pergunta para gerar comentários** (engajamento)
- **Sugestão de Stories** para reforçar o post (enquete, caixinha de perguntas, etc.)
```

## Exemplo de uso

### Input
Tema: 5 erros que iniciantes cometem ao investir
Nicho: Finanças pessoais
Público: Jovens 20-30 que querem começar a investir
Nível: Iniciante
Formato: Carrossel (7 slides)
Tom: Didático e leve
Objetivo: Gerar salvamentos

### Output
- Slide 1: "5 erros que vão te fazer PERDER dinheiro investindo" (fundo escuro, texto grande)
- Slide 2: "Erro 1: Investir sem reserva de emergência" + explicação curta
- Slide 3: "Erro 2: Seguir dica de influenciador sem pesquisar" + explicação
- ...
- Slide 7: "Erro 5: Olhar o rendimento todo dia" + explicação
- Slide final: "Salva esse post pra não esquecer. E manda pra aquele amigo que precisa ver isso."
- Legenda: Texto expandindo os 5 pontos com linguagem acessível
