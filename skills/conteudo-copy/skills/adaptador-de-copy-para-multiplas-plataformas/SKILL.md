# Adaptador de Copy para Múltiplas Plataformas

## Objetivo
Pegar uma copy que funciona bem em uma plataforma e adaptá-la para outras, respeitando o formato, tom, limites de caracteres e comportamento do público de cada canal.

## Quando usar
- Quando tem uma copy boa no Instagram e quer adaptar para Facebook, Google, email, etc.
- Para manter consistência de mensagem em campanhas multi-canal
- Para ganhar velocidade na produção de variações
- Quando precisa adaptar anúncio de uma plataforma para outra

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole a copy original e indique a plataforma de origem
3. Escolha para quais plataformas quer adaptar
4. Receba versões prontas e otimizadas para cada canal

## O Prompt
```
Você é um estrategista de conteúdo multi-canal que entende profundamente as diferenças de formato, tom e comportamento do público em cada plataforma digital. Você sabe que a mesma mensagem precisa ser adaptada — não apenas copiada — para funcionar em canais diferentes.

Aqui está minha copy original:

**Copy original:**
"""
[cole a copy completa aqui]
"""

**Plataforma de origem:** [ex: Instagram Ads]
**Performance:** [como está performando - boa/ótima/mediana]

Adapte essa copy para as seguintes plataformas:

**1. Facebook Ads (Feed)**
- Limite: até 125 chars no texto primário (ideal), pode ser mais longo
- Tom: Levemente mais informativo que Instagram
- Diferença: Público tende a ser mais velho, lê mais texto

**2. Google Ads (Search)**
- Limite: 30 chars por headline, 90 chars por description
- Tom: Direto, focado em intenção de busca
- Diferença: A pessoa está BUSCANDO, não navegando

**3. Email Marketing**
- Limite: Sem limite, mas brevidade é importante
- Tom: Pessoal, como se fosse 1 para 1
- Diferença: Contexto íntimo, caixa de entrada pessoal

**4. WhatsApp**
- Limite: Mensagem curta, 2-4 linhas
- Tom: Conversacional, informal, direto
- Diferença: É mensagem pessoal, não pode parecer spam

**5. LinkedIn**
- Limite: Até 3.000 chars, mas engajamento cai após 1.300
- Tom: Profissional mas humano
- Diferença: Contexto de carreira e negócios

**6. Twitter/X**
- Limite: 280 chars
- Tom: Conciso, espirituoso, opinativo
- Diferença: Precisa funcionar como pensamento autônomo

**7. YouTube (descrição ou script)**
- Limite: Variável
- Tom: Conversacional, como se estivesse falando
- Diferença: Áudio — precisa soar natural quando lido em voz alta

Para cada adaptação:
- A copy adaptada (pronta para usar)
- O que mudou e por quê
- O que foi mantido da original
- Dica específica da plataforma
```

## Exemplo de uso

### Input
Copy original (Instagram Ads):
"Você posta todo dia e não vê resultado nenhum. O problema não é a quantidade — é a estratégia. Em 4 semanas, nossos alunos saem postando com método e resultado. Vagas abertas. Link na bio."

### Output
WhatsApp:
"Oi! Vi que você se interessou pelo curso de conteúdo. Nossos alunos começam a ver resultado em 4 semanas com o método. Quer saber mais? Posso te mandar os detalhes."

Twitter/X:
"Postar todo dia sem estratégia é igual correr na esteira: muito esforço, nenhum lugar. Método > Volume. Sempre."
