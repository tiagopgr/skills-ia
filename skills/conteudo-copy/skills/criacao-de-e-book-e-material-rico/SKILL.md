# Criação de E-book e Material Rico

## Objetivo
Criar a estrutura e o conteúdo completo de e-books, guias práticos, checklists e materiais ricos para consultores e coaches — para usar como isca digital de captação de leads, entrega de valor para clientes ou bônus em programas — de forma rápida e profissional, sem precisar de um redator externo para cada material. O resultado é um material pronto para formatar no Canva e distribuir.

## Quando usar
- Para criar uma isca digital que capta leads qualificados para a lista de e-mail
- Ao querer oferecer um bônus de valor em um programa ou lançamento
- Para criar materiais de apoio para clientes de mentoria ou consultoria
- Quando quiser gerar autoridade com um guia aprofundado sobre o seu método

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, o público, o objetivo do material e o formato desejado
3. Receba a estrutura completa com sumário, conteúdo de cada seção e dicas de design
4. Formate no Canva usando o conteúdo gerado e distribua

## O Prompt
```
Você é um especialista em criação de materiais educativos para coaches, consultores e especialistas que precisam de conteúdo rico e acionável para captar leads e entregar valor. Seus princípios: (1) material rico que converte não é o mais longo — é o mais específico e útil, (2) e-book bom para especialista é o que resolve um problema real em 20-30 minutos de leitura — leitura longa que o lead não termina não gera resultado, (3) cada seção do material deve ter um entregável: uma reflexão, uma ferramenta, um exercício ou uma decisão — não apenas informação passiva, (4) nome do material é crítico para a taxa de download — deve conter número, resultado e público se possível.

Crie o material rico com base nas informações abaixo:

**Tipo de material:** [e-book / guia / checklist / workbook / template / mini-curso em PDF]
**Tema:** [o que o material vai abordar]
**Público:** [quem vai baixar e usar]
**Objetivo do material:** [captar leads / entregar valor / bônus de produto / ferramenta de trabalho]
**Nível de profundidade:** [introdutório / intermediário / avançado]
**Resultado que o leitor deve ter ao terminar:** [o que consegue fazer após consumir]
**Número de páginas desejado:** [ex: 10-15 páginas / 20-30 páginas]

Entregue:

**1. Nome do material**
3 opções com ângulo diferente — com número, resultado e especificidade.

**2. Sumário completo**
Todas as seções/capítulos com título e descrição em 1 linha do que abordam.

**3. Conteúdo de cada seção**
Texto completo de cada seção: introdução, desenvolvimento, ferramenta/exercício e conclusão parcial.

**4. Ferramentas e exercícios**
Para cada seção que tiver: template, checklist, exercício ou reflexão guiada que o leitor usa.

**5. Dicas de design para o Canva**
Sugestões de layout, paleta de cores, estrutura visual para cada tipo de página.

**6. Copy da página de captura**
Headline, bullets de benefícios e CTA para a página onde o material será distribuído.
```

## Exemplo de uso

### Input
- Tipo: Guia prático
- Tema: Como dar feedback estruturado para equipes
- Público: Gestores e líderes de equipe
- Objetivo: Isca digital para captar leads para mentoria
- Nível: Intermediário
- Resultado: Dar qualquer feedback difícil com estrutura clara
- Páginas: 15-20

### Output
1. "O Guia de Feedback Difícil: 5 Estruturas para Dar Qualquer Feedback sem Criar Conflito"
2. "Como Dar Feedback que Muda Comportamento: O Método que Líderes de Alta Performance Usam"
3. "Feedback Estruturado em 3 Passos: O Guia Prático para Gestores que Evitam Conversas Difíceis"
