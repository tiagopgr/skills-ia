# Criador de Analogias e Metáforas para Produto

## Objetivo
Gerar analogias, metáforas e comparações poderosas que tornam produtos ou serviços complexos facilmente compreensíveis para qualquer público, acelerando a compreensão e o desejo de compra.

## Quando usar
- Quando seu produto é difícil de explicar em poucas palavras
- Para simplificar conceitos técnicos para público leigo
- Em pitch, vendas, copy e apresentações
- Quando a audiência não entende o que você faz

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu produto e o que ele faz
3. Receba analogias prontas para múltiplos contextos
4. Use a que melhor se encaixa na situação

## O Prompt
```
Você é um comunicador que domina a arte de explicar coisas complexas de forma simples usando analogias e metáforas. Você sabe que quando alguém diz "ah, entendi!" é porque encontrou uma referência mental que conecta o novo a algo que já conhece.

Exemplos clássicos:
- "Uber para X" — modelo de marketplace
- "Netflix dos livros" — assinatura com acesso ilimitado
- "Um personal trainer, mas para suas finanças" — acompanhamento personalizado

Crie analogias e metáforas para explicar o seguinte produto:

**Produto/Serviço:** [descreva em detalhes o que faz]
**Público que precisa entender:** [leigo, técnico, investidor, cliente final]
**O que é mais difícil de explicar:** [a parte que as pessoas não entendem]
**Benefício principal:** [o que o produto faz pela pessoa]
**Setor/Indústria:** [onde atua]

Gere 10 analogias/metáforas organizadas por tipo:

**Tipo 1 — "É como X para Y" (3 analogias)**
Formato: Referência conhecida + contexto do produto.

**Tipo 2 — Comparação do dia a dia (3 analogias)**
Usando situações cotidianas que todo mundo entende.

**Tipo 3 — Metáfora visual (2 analogias)**
Imagens mentais que ficam na memória.

**Tipo 4 — Antes/Depois comparativo (2 analogias)**
Contraste que mostra o impacto do produto.

Para cada analogia:
- A analogia completa (pronta para usar)
- Melhor contexto de uso
- Para qual público funciona melhor
- Nível de formalidade
- Cuidado: quando essa analogia pode NÃO funcionar
```

## Exemplo de uso

### Input
Produto: Plataforma que conecta empresas a freelancers pré-vetados em 24h
Público: Gestores de empresa que precisam contratar rápido
Difícil de explicar: O processo de vetting e match automático
Benefício: Contratar freelancer qualificado em 24h sem dor de cabeça
Setor: RH / Tecnologia

### Output
**"É como X para Y":** "É como um Tinder para contratação, mas onde todos os matches já foram aprovados pelo seu RH."

**Dia a dia:** "Contratar freelancer hoje é como pescar no oceano aberto — muito esforço, pouca certeza. Nossa plataforma é como pescar num aquário onde só tem peixe bom."

**Visual:** "Imagine uma prateleira onde cada freelancer já vem com selo de qualidade, nota de outros clientes e disponibilidade em tempo real. É isso."

**Antes/Depois:** "Antes: 2 semanas procurando, 15 entrevistas, 3 decepções. Depois: 1 briefing, 24 horas, freelancer qualificado trabalhando."
