# Copy para YouTube Ads (Pre-roll)

## Objetivo
Criar roteiros de anúncios em vídeo para YouTube Ads (pre-roll skippable e non-skippable) com estrutura otimizada para reter nos primeiros 5 segundos e converter.

## Quando usar
- Ao criar campanhas de vídeo no YouTube/Google Ads
- Para anúncios in-stream puláveis (skippable) e não puláveis
- Para anúncios bumper (6 segundos)
- Quando quer escalar campanhas de vídeo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu produto e público
3. Receba roteiros prontos com marcação de tempo
4. Passe o roteiro para produção ou grave você mesmo

## O Prompt
```
Você é um roteirista de anúncios para YouTube com experiência em campanhas de performance. Você sabe que em anúncios skippable o espectador pode pular após 5 segundos, então o hook inicial é crítico. E sabe que a estrutura do script deve levar a uma ação clara.

Crie roteiros de anúncio em vídeo para YouTube Ads:

**Produto/Serviço:** [descreva]
**Público-alvo:** [detalhado]
**Objetivo:** [conversão, awareness, leads, app install]
**Oferta:** [se houver]
**Tom:** [ex: profissional, casual, enérgico, storytelling]
**Quem aparece no vídeo:** [apresentador, animação, tela com texto, b-roll]
**CTA final:** [o que a pessoa deve fazer - clicar, acessar, comprar]

Gere 3 roteiros em formatos diferentes:

**Roteiro 1 — Skippable (30 segundos)**
[0-5s] HOOK — O que prende ANTES do botão de pular
[5-15s] PROBLEMA — Conectar com a dor do público
[15-25s] SOLUÇÃO — Apresentar o produto/serviço
[25-30s] CTA — Chamada para ação clara

**Roteiro 2 — Skippable (60 segundos)**
[0-5s] HOOK — Gancho impossível de ignorar
[5-15s] IDENTIFICAÇÃO — "Se você é [público], isso é pra você"
[15-30s] PROBLEMA + AGITAÇÃO — Aprofundar a dor
[30-45s] SOLUÇÃO + PROVA — Apresentar com credibilidade
[45-55s] OFERTA — O que a pessoa ganha
[55-60s] CTA — Ação clara e urgente

**Roteiro 3 — Bumper (6 segundos, non-skippable)**
[0-2s] IMPACTO VISUAL + FRASE
[2-4s] PRODUTO/MARCA
[4-6s] CTA

Para CADA roteiro, entregue:
- **Texto narrado** (exatamente o que será falado, com marcação de tempo)
- **Direção visual** (o que aparece na tela em cada momento)
- **Texto na tela** (overlays, legendas, destaques visuais)
- **Música/Som sugerido** (tipo de trilha ou efeito sonoro)
- **CTA visual** (botão, link, QR code, etc.)
```

## Exemplo de uso

### Input
Produto: Plataforma de cursos online para criadores
Público: Criadores de conteúdo que querem vender cursos
Objetivo: Cadastro gratuito
Tom: Energético e aspiracional
Quem aparece: Apresentador jovem
CTA: Criar conta grátis

### Output
Roteiro 1 — 30s:
- [0-5s] "Você cria conteúdo de graça e alguém está lucrando com isso."
  Visual: Tela de celular com redes sociais, números subindo
- [5-15s] "Enquanto você posta, plataformas ganham milhões com o SEU público..."
  Visual: Apresentador falando direto para câmera
- [15-25s] "Com a [Plataforma], você transforma seu conhecimento em curso e vende direto pro seu público."
  Visual: Demo rápida da plataforma
- [25-30s] "Crie sua conta grátis. Link aqui embaixo."
  Visual: Tela com botão CTA + URL
