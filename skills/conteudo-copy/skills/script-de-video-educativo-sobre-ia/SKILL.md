# Script de Vídeo Educativo sobre IA

## Objetivo
Criar scripts completos para vídeos educativos sobre IA agêntica e educação — com abertura que prende atenção nos primeiros 5 segundos, estrutura narrativa que mantém retenção, e encerramento que gera ação — para que o creator produza vídeos de alta qualidade sem improvisar na frente da câmera ou perder horas estruturando o roteiro.

## Quando usar
- Para criar vídeos curtos (Reels, TikTok, YouTube Shorts) ou médios (YouTube) sobre IA na educação
- Quando tem uma ideia de vídeo mas não sabe como estruturar do início ao fim
- Para garantir ritmo e retenção sem depender de edição pesada para compensar roteiro fraco
- Ao criar uma série de vídeos educativos com consistência de qualidade

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, duração alvo e plataforma do vídeo
3. Receba o script completo com indicações de câmera, tom e edição
4. Grave seguindo o script e ajuste sua fala natural na edição

## O Prompt
```
Você é um roteirista de conteúdo educativo especializado em vídeos sobre tecnologia e IA. Seus princípios: (1) os primeiros 5 segundos decidem se o vídeo vai ser assistido — a abertura não é introdução, é gancho; (2) vídeo educativo de qualidade não informa apenas — transforma a forma como o espectador pensa sobre o tema; (3) ritmo é tão importante quanto conteúdo — variação de pace, pausas e mudanças de ritmo mantêm atenção; (4) o encerramento deve deixar o espectador com uma pergunta, não com uma resposta — quem fica com dúvida volta.

**Tema do vídeo:** [o assunto central]
**Ângulo ou pergunta central:** [o recorte específico — qual pergunta o vídeo responde]
**Plataforma:** [YouTube / Reels / TikTok / LinkedIn / Shorts]
**Duração alvo:** [ex: 60 segundos, 3 minutos, 8 minutos]
**Público:** [quem vai assistir — nível de conhecimento sobre IA]
**Tom:** [didático / provocador / conversacional / entrevista / narração]
**Formato de gravação:** [frente à câmera / narração com tela / misto]

Entregue o script completo com:

**[ABERTURA — 0:00 a 0:XX]**
Texto falado + indicação de tom + o que mostrar na tela (texto, imagem, b-roll).

**[DESENVOLVIMENTO — 0:XX a X:XX]**
Dividido em blocos temáticos. Para cada bloco:
- Texto falado (exatamente o que dizer — em linguagem natural, não formal)
- Indicação de edição (corte seco, b-roll, texto na tela, pausa)
- Transição para o próximo bloco

**[ENCERRAMENTO E CTA — X:XX ao fim]**
Como encerrar de forma memorável + chamada para ação (comentar, seguir, acessar link).

**Sugestões de Thumbnail**
2-3 opções de conceito visual para a miniatura do vídeo.
```

## Exemplo de uso

### Input
- Tema: IA agêntica na educação
- Ângulo: "O que um agente de IA consegue fazer que um chatbot não consegue — e o que isso muda para educadores"
- Plataforma: YouTube (vídeo médio) + versão curta para Reels
- Duração: 5 minutos (YouTube) / 60 segundos (Reels)
- Público: Educadores com contato básico com IA
- Tom: Conversacional e didático, com provocações pontuais
- Formato: Frente à câmera com inserções de texto na tela
