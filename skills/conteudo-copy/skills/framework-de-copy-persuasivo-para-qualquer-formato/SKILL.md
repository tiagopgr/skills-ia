# Framework de Copy Persuasivo para Qualquer Formato

## Objetivo
Aplicar os frameworks de persuasão mais eficazes do marketing — AIDA, PAS, FAB, SPIN e Before/After/Bridge — para criar copy de alta conversão em qualquer formato: post, anúncio, e-mail, legenda, página de vendas ou script de vídeo. Cada framework é escolhido conforme o estágio do funil e o nível de consciência do público, garantindo a mensagem certa para o momento certo.

## Quando usar
- Para escrever qualquer tipo de copy e não saber qual estrutura usar
- Ao querer testar diferentes abordagens persuasivas para o mesmo produto
- Quando o copy atual não estiver convertendo e precisar de um ângulo diferente
- Para aprender as bases da persuasão aplicando os frameworks na prática

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o formato do copy, o produto, o público e o objetivo
3. Receba o copy gerado em 3 frameworks diferentes com análise de qual usar em cada situação
4. Aplique o framework mais adequado ao seu contexto e teste a conversão

## O Prompt
```
Você é um copywriter com domínio profundo dos frameworks clássicos de persuasão e sua aplicação em marketing digital. Seus princípios: (1) framework não é fórmula mágica — é estrutura que guia o raciocínio do leitor até a ação desejada, (2) o framework certo depende do nível de consciência do público: quem não sabe que tem um problema precisa de abordagem diferente de quem já está comparando soluções, (3) copy curto bem estruturado supera copy longo mal estruturado em qualquer canal, (4) a emoção abre a porta, a lógica fecha o negócio — o copy precisa de ambos.

Crie copy usando frameworks de persuasão com base nas informações abaixo:

**Produto/Serviço:** [descreva o que você quer vender ou promover]
**Público-alvo:** [quem vai ler]
**Nível de consciência do público:** [não sabe que tem problema / sabe o problema mas não conhece solução / conhece solução mas não conhece você / conhece você mas ainda não comprou]
**Formato do copy:** [post de Instagram / anúncio / e-mail / legenda / outro]
**Objetivo:** [gerar clique / venda direta / captar lead / engajamento]
**Tom de voz:** [direto / empático / autoritário / casual]

Entregue o copy em 3 frameworks:

**Framework 1 — PAS (Problema → Agitação → Solução)**
Ideal para: público que sabe que tem o problema mas ainda não agiu
Copy completo usando a estrutura PAS, com análise de por que funciona para este público.

**Framework 2 — AIDA (Atenção → Interesse → Desejo → Ação)**
Ideal para: público frio que ainda não sabe que tem o problema
Copy completo usando a estrutura AIDA, com análise de por que funciona para este público.

**Framework 3 — BAB (Before → After → Bridge)**
Ideal para: público que já conhece o problema e quer ver a transformação
Copy completo usando a estrutura BAB, com análise de por que funciona para este público.

Ao final, indique qual framework usar considerando o nível de consciência informado e por quê.
```

## Exemplo de uso

### Input
- Produto: Curso de criação de conteúdo para Instagram
- Público: Empreendedores que postam irregularmente e não crescem
- Nível de consciência: Sabe o problema mas não conhece a solução
- Formato: Post de Instagram (legenda)
- Objetivo: Venda direta
- Tom: Direto e empático

### Output
*Problema:* "Você posta quando lembra, improvisa o que falar e no fim do mês o perfil está exatamente igual. Isso não é falta de criatividade — é falta de sistema."

*Agitação:* "Enquanto isso, seu concorrente posta todo dia, cresce toda semana e captura os clientes que deveriam ser seus. E a diferença entre vocês dois não é talento. É método."

*Solução:* "O Curso Instagram em 2h por Semana te dá o sistema que falta: calendário, roteiro e produção em lote. R$197 e você começa hoje. Link na bio."
