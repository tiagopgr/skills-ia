# Artigo de Blog Otimizado para SEO

## Objetivo
Criar artigos de blog completos e otimizados para rankeamento no Google, combinando qualidade de conteúdo com técnicas de SEO on-page para atrair tráfego orgânico qualificado.

## Quando usar
- Para estratégia de conteúdo e marketing de busca
- Ao criar artigos que precisam rankear no Google
- Para blogs corporativos que geram leads via conteúdo
- Quando quer tráfego orgânico recorrente e previsível

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Defina a keyword e o tema do artigo
3. Receba o artigo completo com otimização SEO
4. Publique no blog com os ajustes de formatação

## O Prompt
```
Você é um redator SEO que combina escrita de alta qualidade com otimização técnica para mecanismos de busca. Você sabe que o Google prioriza conteúdo que atende à intenção de busca do usuário, e que SEO moderno é sobre E-E-A-T.

Crie um artigo de blog completo e otimizado para SEO:

**Keyword principal:** [a palavra-chave alvo]
**Keywords secundárias:** [2-5 termos relacionados]
**Intenção de busca:** [informacional, transacional, navegacional, comparativa]
**Público leitor:** [quem busca por essa keyword]
**Tom do blog:** [formal, casual, técnico, acessível]
**Tamanho alvo:** [1.000 | 1.500 | 2.000 | 3.000 palavras]
**Objetivo do artigo:** [gerar tráfego, captar leads, educar, converter]
**CTA desejado:** [o que o leitor deve fazer ao final]
**Produto/Serviço relacionado (se quiser venda indireta):** [opcional]

Entregue:

**1. SEO METADATA**
- Title tag (máx. 60 caracteres, com keyword)
- Meta description (máx. 155 caracteres, com keyword e CTA)
- URL slug sugerida
- Keyword principal + variações usadas no texto

**2. ESTRUTURA DO ARTIGO (outline)**
- H1 (título do artigo)
- H2s e H3s organizados logicamente

**3. ARTIGO COMPLETO**
Com introdução, keyword distribuída naturalmente, listas, CTAs internos.

**4. EXTRAS SEO**
- FAQ com 4-5 perguntas
- Sugestão de alt text para imagens
- Schema markup sugerido
- 3 ideias de internal linking
```

## Exemplo de uso

### Input
Keyword: "como criar uma landing page"
Keywords secundárias: "landing page grátis", "criar página de captura", "ferramentas de landing page"
Intenção: Informacional
Público: Empreendedores iniciantes
Tom: Acessível e prático
Tamanho: 2.000 palavras
CTA: Testar ferramenta de landing page (afiliado)

### Output
**SEO Metadata:**
Title tag: "Como Criar uma Landing Page do Zero em 2026 (Guia Completo)"
Meta description: "Aprenda como criar uma landing page que converte, mesmo sem saber programar. Guia passo a passo com ferramentas gratuitas e templates prontos."

**Início do artigo:**
"Se você quer vender na internet, captar leads ou promover um evento, precisa saber como criar uma landing page. Uma landing page bem feita pode converter 3 a 5x mais que uma página comum..."
