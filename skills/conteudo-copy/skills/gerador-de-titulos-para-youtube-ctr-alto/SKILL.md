# Gerador de Títulos para YouTube (CTR Alto)

## Objetivo
Criar títulos para vídeos do YouTube otimizados para alta taxa de cliques (CTR), combinando técnicas de curiosidade, clareza e apelo algorítmico.

## Quando usar
- Ao publicar novos vídeos no YouTube
- Quando seus vídeos têm impressões mas poucos cliques
- Para otimizar títulos de vídeos antigos que performam abaixo do esperado
- Ao planejar a pauta de conteúdo do canal

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o vídeo que vai publicar
3. Escolha os melhores títulos entre as opções
4. Combine com uma thumbnail estratégica

## O Prompt
```
Você é um consultor de YouTube especializado em CTR (Click-Through Rate) e crescimento orgânico de canais. Você sabe que o título e a thumbnail são responsáveis por 90% da decisão de clicar, e que o algoritmo do YouTube prioriza vídeos com CTR alto nos primeiros 60 minutos de publicação.

Crie 15 opções de título para o seguinte vídeo:

**Tema do vídeo:** [sobre o que é o vídeo]
**Nicho do canal:** [área do canal]
**Público do canal:** [quem assiste]
**Formato do vídeo:** [tutorial, vlog, review, lista, storytelling, etc.]
**Duração estimada:** [curto <10min, médio 10-20min, longo >20min]
**Palavra-chave principal:** [se tiver, para SEO]

Organize em 5 estilos (3 títulos cada):

**Estilo 1 — Curiosidade Gap**
Títulos que criam uma lacuna de informação que só se fecha assistindo.

**Estilo 2 — Resultado/Transformação**
Títulos que prometem um resultado claro e tangível.

**Estilo 3 — Número + Especificidade**
Títulos com listas, dados ou prazos concretos.

**Estilo 4 — Negativo/Erro/Alerta**
Títulos que alertam sobre erros, problemas ou coisas a evitar.

**Estilo 5 — Tendência/Atualidade**
Títulos que conectam com algo atual, novo ou trending.

Para cada título, inclua:
- O título (máximo 60 caracteres quando possível)
- Sugestão de thumbnail (descrição visual em 1 frase)
- CTR estimado relativo (baixo / médio / alto / muito alto)
- Dica de otimização (o que faz esse título funcionar)

Ao final, indique:
- Os 3 melhores títulos e por quê
- Palavras que devem ir na descrição para reforçar SEO
```

## Exemplo de uso

### Input
Tema: Como começar a investir com pouco dinheiro
Nicho: Finanças pessoais
Público: Jovens 18-30 que nunca investiram
Formato: Tutorial educativo
Duração: Médio (15 min)
Palavra-chave: "como começar a investir"

### Output
- Curiosidade: "Comecei a investir com R$50 e isso aconteceu"
- Resultado: "Como investir com pouco dinheiro (plano completo)"
- Número: "5 investimentos para quem tem menos de R$100"
- Negativo: "NÃO comece a investir antes de ver isso"
- Tendência: "O investimento que está bombando em 2026 (e custa R$1)"
