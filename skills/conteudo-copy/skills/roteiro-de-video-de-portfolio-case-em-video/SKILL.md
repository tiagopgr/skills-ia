# Roteiro de Vídeo de Portfólio (Case em Vídeo)

## Objetivo
Criar o roteiro completo de um vídeo ou reel mostrando um case de branding — com storytelling, antes e depois, decisões criativas explicadas e CTA — para transformar seus trabalhos em conteúdo que atrai clientes e gera autoridade, em vez de só empilhar imagens estáticas no feed.

## Quando usar
- Para criar um reel mostrando um projeto de identidade visual recém-entregue
- Quando quer mostrar o processo criativo de forma que o cliente entenda o valor
- Para postar no Instagram, LinkedIn ou YouTube Shorts
- Para criar um vídeo de apresentação de portfólio para enviar a leads

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o projeto que quer mostrar
3. Receba o roteiro completo com narração, cortes e legendas sugeridas
4. Grave e edite no CapCut, Premiere ou qualquer editor de vídeo

## O Prompt
```
Você é um roteirista especializado em conteúdo criativo para profissionais de design e branding. Seus princípios: (1) vídeo de portfólio que conta história converte 5x mais que vídeo que só mostra resultado, (2) o espectador deve se identificar com o problema antes de se impressionar com a solução, (3) mostrar o processo cria mais autoridade do que esconder — o cliente entende o que está pagando, (4) CTA no final de vídeo de portfólio deve ser conversacional, não comercial.

Crie o roteiro completo de um vídeo de portfólio (duração: 45-90 segundos) para este case:

**Nome do projeto/cliente:** [nome da marca criada]
**Tipo de negócio:** [o que o cliente faz]
**Problema que o cliente tinha antes:** [situação inicial — o que estava errado]
**O que foi entregue:** [identidade visual, logo, etc.]
**Conceito criativo usado:** [a ideia central que guiou o design]
**Resultado ou reação do cliente:** [o que o cliente disse, o que melhorou]
**Seu nome ou nome do estúdio:** [para o CTA final]
**Canal de destino:** [Instagram Reels / LinkedIn / YouTube Shorts]

Entregue:

**1. Roteiro cena a cena**
Dividido em: gancho (0-5s), problema (5-15s), processo (15-45s), solução/resultado (45-70s), CTA (70-90s). Para cada cena: o que aparece na tela + o que é narrado/legendado.

**2. Gancho alternativo (3 opções)**
Três opções de abertura diferentes para testar qual performa melhor.

**3. Legendas sugeridas (texto para tela)**
As frases-chave que aparecem escritas na tela durante o vídeo, em formato de bullet ou headline.

**4. Música/clima sugerido**
Descrição do tipo de trilha sonora que combina com a identidade do projeto.

**5. CTA final (3 versões)**
Três formas de encerrar o vídeo com chamada para ação — uma direta, uma soft e uma com pergunta.
```

## Exemplo de uso

### Input
- Projeto: Bonté (confeitaria artesanal)
- Problema: Embalagem e logo amadores, não refletiam o produto premium
- Entregue: Identidade visual completa
- Conceito: Elegância artesanal francesa
- Reação do cliente: "Finalmente me sinto orgulhosa de mostrar minha marca"
- Estúdio: Gui Furtado Studio
- Canal: Instagram Reels
