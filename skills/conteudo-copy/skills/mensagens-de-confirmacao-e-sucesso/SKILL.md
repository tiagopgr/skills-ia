# Mensagens de Confirmação e Sucesso

## Objetivo
Criar mensagens de confirmação e sucesso para interfaces digitais que celebram a ação do usuário, reforçam o que aconteceu, e guiam para o próximo passo.

## Quando usar
- Após ações importantes (compra, cadastro, envio de formulário)
- Em confirmações de operações (salvar, deletar, atualizar)
- Em marcos de progresso (completou onboarding, atingiu meta)
- Para feedback positivo após ações do usuário

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste os momentos de sucesso do seu produto
3. Receba mensagens prontas para cada cenário
4. Implemente e observe o impacto na experiência

## O Prompt
```
Você é um UX Writer que sabe que mensagens de sucesso são oportunidades desperdiçadas na maioria dos produtos. Um "Salvo com sucesso" genérico perde a chance de: (1) celebrar o progresso, (2) guiar para o próximo passo, (3) reforçar o valor do produto, (4) criar um micro-momento positivo.

Crie mensagens de confirmação e sucesso para:

**Produto/App:** [descreva]
**Tom da marca:** [formal, casual, celebratório, minimalista]
**Público:** [quem vai ver as mensagens]

**Cenários de sucesso:**
1. [ex: "Conta criada com sucesso"]
2. [ex: "Primeiro projeto criado"]
3. [ex: "Pagamento confirmado"]
4. [ex: "Perfil atualizado"]
(liste de 5 a 12 cenários)

Para CADA cenário, entregue:

**Mensagem principal** (3 opções):
- Versão funcional
- Versão celebratória
- Versão com próximo passo

**Texto do CTA/próximo passo**
**Detalhe de apoio** (se aplicável)
**Formato recomendado** (toast, modal, tela dedicada, inline)
**Tempo de exibição sugerido**

Ao final: Hierarquia de celebração + Guia de tom + Template padrão
```

## Exemplo de uso

### Input
Produto: App de finanças pessoais
Tom: Motivador e amigável
Cenários: 1) Primeira despesa registrada 2) Meta de economia criada 3) Mês fechado sem estourar orçamento 4) Exportou relatório

### Output
**Cenário 3 — Mês fechado no orçamento:**

Mensagens:
- Funcional: "Orçamento de março fechado dentro do limite. Parabéns!"
- Celebratória: "Você mandou bem! Março fechou no verde. Isso é disciplina de verdade."
- Com próximo passo: "Março no azul — consistência que dá orgulho. Que tal definir uma meta de economia para abril?"

CTA: "Criar meta para abril" | "Ver relatório do mês"

Detalhe: "Você economizou R$342 a mais que o planejado."

Formato: Tela dedicada com animação de celebração (é um marco mensal).
