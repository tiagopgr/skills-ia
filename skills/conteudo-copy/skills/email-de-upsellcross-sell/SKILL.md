# Email de Upsell/Cross-sell

## Objetivo
Criar emails pós-compra que oferecem produtos complementares (cross-sell) ou upgrades (upsell) de forma natural e relevante, aumentando o ticket médio sem parecer oportunista.

## Quando usar
- Após uma compra (automação pós-venda)
- Para clientes ativos que podem comprar mais
- Ao lançar um produto complementar ao que o cliente já tem
- Para aumentar LTV (Lifetime Value) da base de clientes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o produto original e o que quer oferecer
3. Receba emails prontos para cada estratégia
4. Configure na automação pós-compra

## O Prompt
```
Você é um especialista em email marketing de pós-venda com foco em maximização de receita por cliente. Você sabe que vender para quem já comprou é 5-7x mais barato do que adquirir um novo cliente, e que o timing e a relevância são tudo.

Crie emails de upsell/cross-sell para:

**Produto que o cliente já comprou:** [descreva]
**Preço que já pagou:** [valor]
**Tempo desde a compra:** [quando enviar — 3 dias, 7 dias, 14 dias]
**Produto que quer oferecer agora:** [descreva o upsell ou cross-sell]
**Preço do novo produto:** [valor]
**Relação entre os produtos:** [complementar, versão avançada, acessório, pacote]
**Condição especial para quem já é cliente:** [desconto, bônus, condição — se houver]
**Tom:** [gratidão, consultivo, direto, exclusivo]

Gere 3 versões de email:

**Versão 1 — Recomendação genuína**
Tom: "Quem comprou X geralmente ama Y. Achei que você ia querer saber."

**Versão 2 — Resultado amplificado**
Tom: "Você já deu o primeiro passo com X. Para potencializar seus resultados, Y é o próximo nível."

**Versão 3 — Oferta exclusiva de cliente**
Tom: "Como você já é cliente, preparei uma condição que ninguém mais tem."

Para cada versão:
- **3 opções de assunto**
- **Preview text**
- **Corpo completo** (200-300 palavras)
- **CTA**
- **Timing ideal de envio**
- **Gatilho principal**
```

## Exemplo de uso

### Input
Produto comprado: Curso básico de fotografia (R$197)
Tempo: 14 dias após compra
Upsell: Curso avançado de edição no Lightroom (R$297)
Relação: Complementar
Condição: 30% de desconto para alunos do básico
Tom: Consultivo e amigável

### Output
**Versão 2 — Resultado amplificado:**
- Assunto: "Suas fotos vão ficar 10x melhores depois disso"
- Preview: "O passo que separa amador de profissional"
- Corpo: "Oi [nome], faz 2 semanas que você começou o curso de fotografia. A essa altura, suas fotos já devem estar bem melhores — enquadramento, luz, composição. Mas vou ser honesto: a maioria das fotos incríveis que você vê por aí não saem assim da câmera. Elas passam pelo Lightroom. A edição é o que transforma uma foto boa em uma foto de tirar o fôlego. E como você já domina a base, esse é o momento perfeito para dar o próximo passo. Preparei algo especial para quem já é aluno: 30% OFF no Curso de Edição no Lightroom. De R$297 por R$207,90. [Botão: QUERO EDITAR COMO PROFISSIONAL]"
