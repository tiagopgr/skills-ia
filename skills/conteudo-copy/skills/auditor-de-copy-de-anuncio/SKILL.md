# Auditor de Copy de Anúncio

## Objetivo
Analisar e auditar uma copy de anúncio existente, identificando fraquezas, oportunidades de melhoria e entregando uma versão otimizada com justificativa para cada mudança.

## Quando usar
- Quando um anúncio não está performando bem
- Antes de subir um anúncio novo (revisão final)
- Para auditar copies da equipe ou de freelancers
- Para aprender a identificar problemas em copy

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole a copy do anúncio que quer auditar
3. Receba análise detalhada + versão melhorada
4. Aplique as correções e teste a nova versão

## Exemplo de uso

### Input
Copy: "Nosso software é o melhor do mercado. Com ele, sua empresa vai crescer. Clique e saiba mais."
Plataforma: Facebook Ads
Objetivo: Conversão
Público: Donos de pequenas empresas

### Output
- Hook: 2/10 — "Nosso software é o melhor" é genérico e egocêntrico, fala da empresa e não do cliente
- Clareza: 3/10 — "vai crescer" é vago, não diz como nem quanto
- CTA: 4/10 — "Saiba mais" é fraco, não dá motivo para clicar
- Nota geral: 25/100
- Versão otimizada: "Você perde quantas horas por semana com controles manuais? Nosso software automatiza a gestão da sua empresa em um só lugar. 2.400 PMEs já usam. Teste grátis por 14 dias — sem cartão."
