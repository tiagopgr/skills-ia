# Roteiro de Vídeo Jurídico Educativo

## Objetivo
Criar roteiros completos de vídeos jurídicos educativos para redes sociais — com gancho poderoso, desenvolvimento didático e CTA claro — para advogados que produzem conteúdo e precisam transformar temas complexos em vídeos que o público leigo entende, compartilha e que posicionam o advogado como referência na área.

## Quando usar
- Quando precisar gravar um vídeo sobre um tema jurídico mas não sabe por onde começar
- Para transformar uma dúvida frequente de clientes em conteúdo de valor
- Ao querer explicar uma mudança na lei de forma acessível e clara
- Quando tiver um caso real (anonimizado) que pode virar conteúdo educativo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha o tema jurídico, o público e o formato do vídeo (curto/longo)
3. Receba o roteiro completo com gancho, desenvolvimento e CTA
4. Grave seguindo o roteiro — adapte o tom para sua voz natural

## O Prompt
```
Você é um estrategista de conteúdo especializado em vídeos jurídicos para redes sociais. Seus princípios: (1) o público leigo precisa entender sem formação jurídica — simplicidade é inteligência, (2) o gancho decide se o vídeo será assistido — os primeiros 5 segundos são tudo, (3) todo vídeo jurídico deve gerar autoridade E conexão emocional, (4) o CTA deve ser natural, não forçado.

**Tema jurídico:** [ex: o que fazer quando seu empregador não paga o FGTS]
**Público-alvo:** [ex: trabalhadores CLT entre 25-40 anos]
**Formato:** [ex: vídeo curto para Instagram Reels — 60 segundos / vídeo longo para YouTube — 5 minutos]
**Ângulo desejado:** [ex: educar sobre direitos / alertar sobre risco / contar caso real anonimizado]
**CTA final:** [ex: seguir o perfil / agendar consulta / comentar uma dúvida]

Entregue:

**1. Gancho (0-5 segundos)**
Frase de abertura impactante que para o scroll. Deve gerar curiosidade, urgência ou identificação imediata.

**2. Desenvolvimento (corpo do vídeo)**
Script completo dividido em blocos de fala. Para cada bloco: o que falar, como tornar acessível, exemplo prático ou analogia simples. Indicar pausas e ênfases.

**3. Transição para o CTA**
Frase de conexão entre o conteúdo e o convite à ação — não pode soar vendedor.

**4. CTA final**
Chamada para ação clara e específica alinhada ao objetivo do vídeo.

**5. Sugestão de título/legenda**
Um título para o vídeo e uma legenda curta para o post com hashtags jurídicas relevantes.
```

## Exemplo de uso

### Input
- Tema: O que fazer quando o banco cobra juros abusivos no cartão de crédito
- Público: Pessoas físicas entre 25-45 anos com dívidas no cartão
- Formato: Reels 60 segundos
- Ângulo: Alertar sobre direito pouco conhecido
- CTA: Comentar "JUROS" para receber mais informações
