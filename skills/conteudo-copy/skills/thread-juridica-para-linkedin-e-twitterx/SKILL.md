# Thread Jurídica para LinkedIn e Twitter/X

## Objetivo
Criar threads jurídicas completas e bem estruturadas para LinkedIn e Twitter/X — com abertura que para o scroll, desenvolvimento progressivo que prende o leitor até o fim e encerramento que converte — para que o advogado publique conteúdo de alto alcance nessas plataformas sem precisar escrever do zero cada post da sequência.

## Quando usar
- Para explicar um processo jurídico complexo em etapas digestíveis
- Quando um tema tiver muitos desdobramentos que não cabem num único post
- Para criar conteúdo de autoridade no LinkedIn que gere conexões e indicações
- Ao querer viral orgânico no Twitter/X com um tema jurídico relevante do momento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema, a plataforma e o objetivo da thread
3. Receba a thread completa numerada, pronta para publicar
4. Copie cada item e publique em sequência na plataforma escolhida

## O Prompt
```
Você é um criador de conteúdo jurídico especializado em threads para LinkedIn e Twitter/X. Seus princípios: (1) o primeiro tweet/post decide tudo — se não parar o scroll, o restante da thread não existe, (2) cada item da thread deve entregar valor sozinho E criar curiosidade para o próximo, (3) thread jurídica que só cita lei é chata — misturar lei + caso real + consequência prática é o que engaja, (4) último item deve ter o CTA mais suave possível — nunca vender diretamente em thread de valor.

**Tema da thread:** [ex: como funciona o processo de inventário no Brasil]
**Plataforma:** [LinkedIn / Twitter/X]
**Número de itens:** [ex: 8 itens / 10 itens / 12 itens]
**Público:** [ex: pessoas que perderam um parente recentemente / empreendedores / leigo geral]
**Objetivo:** [ex: gerar DMs para consulta / crescer seguidores / ser compartilhado por outros advogados]
**Tom:** [ex: empático / direto / provocador / educativo]

Entregue:

**1. Item 1 — Abertura**
Gancho irresistível — máx. 280 caracteres para Twitter ou 3 linhas para LinkedIn. Deve gerar curiosidade ou urgência imediata.

**2. Itens 2 a N-1 — Desenvolvimento**
Cada item com: número, conteúdo principal (máx. 280 caracteres Twitter / 3 linhas LinkedIn), dado ou exemplo concreto, gancho para o próximo.

**3. Último item — Encerramento + CTA**
Síntese do que foi aprendido + convite natural à ação (comentar, seguir, compartilhar ou contato).

**4. Sugestão de imagem ou mídia**
Para qual item vale adicionar uma imagem, gráfico ou documento para aumentar o alcance.
```

## Exemplo de uso

### Input
- Tema: O que acontece com as dívidas de uma pessoa quando ela morre
- Plataforma: LinkedIn
- Número de itens: 8
- Público: Leigo geral, pessoas que têm pais idosos
- Objetivo: Engajamento e DMs para consulta
- Tom: Empático e direto

### Output
"Seu pai morreu e deixou dívidas. Você é obrigado a pagar?

A resposta correta vai contra o que a maioria das famílias acredita — e o desconhecimento pode custar muito caro.

Aqui está o que a lei diz: 🧵"
