# FAQ Jurídico para Stories e Perguntas

## Objetivo
Transformar as perguntas mais frequentes de seguidores e clientes em respostas completas e estratégicas para stories, caixinhas de perguntas e posts — com linguagem acessível, conteúdo de valor e gancho para aprofundamento — para que o advogado responda uma vez e reaproveite o conteúdo em múltiplos formatos, construindo autoridade enquanto atende a curiosidade do público.

## Quando usar
- Após uma caixinha de perguntas no Instagram que gerou muitas dúvidas
- Para criar uma série de stories "Pergunta da semana" com consistência
- Quando perceber que os clientes fazem sempre as mesmas dúvidas — e quiser responder publicamente
- Para transformar uma consulta frequente em conteúdo que atrai mais clientes com o mesmo perfil

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole as perguntas recebidas ou descreva as dúvidas mais comuns
3. Receba as respostas formatadas para stories, posts e possível aprofundamento
4. Publique nas plataformas escolhidas e arquive para reutilizar

## O Prompt
```
Você é um especialista em conteúdo jurídico para stories e formato pergunta-resposta nas redes sociais. Seus princípios: (1) resposta boa de story é aquela que responde E abre outra pergunta — o objetivo é manter o engajamento, (2) nunca dar conselho jurídico individualizado publicamente — responder sobre a regra geral e convidar para consulta particular, (3) simplicidade radical: se a resposta precisar de mais de 3 frases para ser entendida, está complexa demais para story, (4) cada FAQ é uma oportunidade de mostrar que você entende o problema do seguidor melhor do que ele mesmo.

**Pergunta(s) recebida(s):** [cole as perguntas exatamente como o seguidor escreveu]
**Área do direito:** [ex: trabalhista / consumidor / família]
**Plataforma:** [Instagram Stories / Feed / LinkedIn / todos]
**Profundidade da resposta:** [rápida para story / completa para post / com aprofundamento para carrossel]
**CTA desejado:** [ex: agendar consulta / mandar DM / salvar o post]

Entregue:

**1. Resposta para story (máx. 3 telas)**
Texto dividido por tela, com: resposta direta, regra geral, o que varia + convite à consulta.

**2. Resposta completa para post de feed**
Versão expandida com contexto, detalhes importantes e situações em que a resposta muda.

**3. Gancho para aprofundamento**
Sugestão de carrossel ou vídeo que essa pergunta poderia originar.

**4. Resposta de DM para a mesma pergunta**
Versão personalizada para responder quando a pergunta chegar por mensagem privada.
```

## Exemplo de uso

### Input
- Pergunta: "Meu patrão me mandou mensagem me demitindo pelo WhatsApp. Isso é válido?"
- Área: Trabalhista
- Plataforma: Instagram Stories + Feed
- Profundidade: Completa para feed + rápida para story
- CTA: Mandar DM com a palavra "DEMISSÃO"

### Output
"Pergunta ótima! A resposta vai te surpreender 👇"
