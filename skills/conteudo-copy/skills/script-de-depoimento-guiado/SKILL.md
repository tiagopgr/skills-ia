# Script de Depoimento Guiado

## Objetivo
Criar roteiros de perguntas para coletar depoimentos em vídeo de clientes de forma natural, autêntica e estrategicamente útil para vendas e marketing.

## Quando usar
- Quando vai gravar depoimentos de clientes
- Para enviar como guia ao cliente gravar por conta própria
- Para transformar depoimentos escritos em roteiro de vídeo
- Para padronizar a coleta de cases de sucesso

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto do depoimento
3. Receba o roteiro de perguntas + guia de gravação
4. Envie ao cliente ou use na hora da gravação

## O Prompt
```
Você é um produtor de conteúdo especializado em depoimentos de clientes para uso em marketing e vendas. Você sabe que um bom depoimento: (1) conta uma história de transformação, (2) é específico com resultados, (3) soa natural e não decorado, e (4) se conecta com o público que vai assistir.

Crie um roteiro de depoimento guiado para:

**Produto/Serviço:** [o que o cliente comprou/usou]
**Quem é o cliente:** [nome, cargo, empresa, contexto]
**Resultado obtido:** [que resultado o cliente teve — se souber]
**Para quê o depoimento será usado:** [página de vendas, anúncio, redes sociais, site]
**Público que vai assistir:** [quem precisa ser convencido por esse depoimento]
**Duração desejada do depoimento final:** [30s | 1 min | 2 min]

Entregue:

### Roteiro de Perguntas (na ordem de gravação)

**Bloco 1 — Contexto (quem é você)**
Perguntas para o cliente se apresentar naturalmente.
Objetivo: Gerar identificação com quem assiste.

**Bloco 2 — Antes (o problema)**
Perguntas para descrever a situação antes de usar o produto.
Objetivo: O espectador se ver na mesma situação.

**Bloco 3 — Decisão (por que escolheu)**
Perguntas sobre o que levou a comprar/contratar.
Objetivo: Quebrar objeções de quem está considerando.

**Bloco 4 — Depois (o resultado)**
Perguntas sobre resultados concretos e transformação.
Objetivo: Prova social com dados reais.

**Bloco 5 — Recomendação (fechamento)**
Perguntas para recomendar diretamente.
Objetivo: CTA implícito vindo do cliente.

Para cada pergunta:
- A pergunta exata (linguagem natural, não formal)
- O que esperar como resposta ideal
- Dica para o entrevistador (como conduzir se a resposta for curta ou vaga)

### Guia de Gravação (para enviar ao cliente)
- Onde gravar (iluminação, fundo, enquadramento)
- O que vestir
- Como falar (natural, sem decorar)
- Duração sugerida
- Erros comuns a evitar
```

## Exemplo de uso

### Input
Produto: Consultoria de gestão financeira
Cliente: Roberto, dono de restaurante
Resultado: Saiu de prejuízo para lucro de R$15k/mês em 4 meses
Uso: Página de vendas
Público: Donos de restaurantes
Duração: 1 minuto

### Output
**Bloco 2 — Antes (perguntas):**

1. "Roberto, como era a situação financeira do restaurante antes da consultoria?"
   - Resposta ideal: Detalhar o caos — não sabia o lucro, aperto todo mês, preocupação
   - Dica: Se responder "era difícil", aprofunde: "O que era mais difícil especificamente?"

2. "O que você já tinha tentado fazer pra resolver?"
   - Resposta ideal: Tentativas que não funcionaram (contador, planilha, cortar custos)
   - Dica: Isso mostra que a solução anterior não funcionava

3. "Como isso afetava você pessoalmente?"
   - Resposta ideal: Estresse, noites sem dormir, pensou em fechar
   - Dica: A emoção é o que conecta — deixe o cliente falar
