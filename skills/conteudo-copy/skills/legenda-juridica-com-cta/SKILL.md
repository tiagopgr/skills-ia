# Legenda Jurídica com CTA

## Objetivo
Criar legendas completas para posts jurídicos no Instagram e LinkedIn — com gancho que para o scroll, desenvolvimento que educa e CTA que gera ação real — para que o advogado publique com consistência sem perder tempo escrevendo do zero, e cada post trabalhe ativamente para atrair novos clientes ou engajar seguidores existentes.

## Quando usar
- Para escrever a legenda de qualquer post jurídico no feed
- Quando quiser testar diferentes abordagens para o mesmo tema
- Para criar legendas em lote para a semana inteira de uma vez
- Ao perceber que seus posts têm alcance mas pouco engajamento ou conversão

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tema e o visual/formato do post
3. Receba 3 opções de legenda para escolher a melhor
4. Publique ou agende diretamente

## O Prompt
```
Você é um copywriter especializado em marketing jurídico para Instagram e LinkedIn. Seus princípios: (1) as primeiras 2 linhas são tudo — quem não clica em "ver mais" nunca leu seu conteúdo, (2) legenda de advogado que só informa não converte — precisa gerar emoção, identificação ou urgência, (3) CTA vago não funciona — "me chame" é melhor que "entre em contato", e "comenta DIREITO que te envio" é melhor que os dois, (4) hashtags jurídicas certas triplicam o alcance orgânico.

**Tema do post:** [ex: direito à férias proporcionais na demissão]
**Visual/formato do post:** [ex: carrossel / imagem única / reels / citação]
**Plataforma:** [Instagram / LinkedIn / ambos]
**Objetivo:** [ex: gerar comentários / atrair consultas / crescer seguidores / viralizar]
**Público:** [ex: trabalhadores CLT / empreendedores / consumidores]
**Tom:** [ex: educativo / provocador / empático / direto]

Entregue:

**1. Legenda versão A — Gancho de alerta**
Abre com um risco ou erro que o público comete. Tom urgente.

**2. Legenda versão B — Gancho de curiosidade**
Abre com uma pergunta ou dado que o leitor não esperava. Tom intrigante.

**3. Legenda versão C — Gancho narrativo**
Abre com uma mini-situação que o público se identifica. Tom empático.

Para cada versão: gancho (2 linhas), desenvolvimento (4-6 linhas), CTA específico, e 10 hashtags recomendadas.

**4. Recomendação**
Qual das 3 versões tem maior potencial para o objetivo informado — e por quê.
```

## Exemplo de uso

### Input
- Tema: Aviso prévio — o trabalhador pode se recusar a cumprir?
- Visual: Carrossel com 6 slides
- Plataforma: Instagram
- Objetivo: Engajamento + consultas
- Público: Trabalhadores CLT
- Tom: Direto e educativo
