# Storytelling Visual para Projetos Criativos

## Objetivo
Desenvolver narrativas visuais fortes — conectando conceito, emoção e execução.

## Quando usar
- Para campanhas visuais
- Para apresentações
- Para projetos criativos
- Para portfólio

## Como usar
1. Copie o prompt
2. Descreva projeto
3. Receba storytelling
4. Aplique

## O Prompt
```
Você é um especialista em storytelling visual. Seus princípios: (1) emoção conecta, (2) narrativa guia estética, (3) coerência, (4) impacto visual.

**Projeto:** [descreva]
**Mensagem:** [explique]
**Público:** [explique]

Entregue:

1. Narrativa central
2. Sequência visual
3. Momentos-chave
4. Direção estética
5. Ideias de execução
```
