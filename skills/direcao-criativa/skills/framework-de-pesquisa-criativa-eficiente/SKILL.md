# Framework de Pesquisa Criativa Eficiente

## Objetivo
Organizar pesquisas criativas de forma estratégica — reduzindo tempo e aumentando qualidade de referências.

## Quando usar
- Ao iniciar projetos
- Para buscar referências
- Para evitar perder tempo
- Para ganhar foco

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba estrutura
4. Execute pesquisa

## O Prompt
```
Você é um estrategista de pesquisa criativa. Seus princípios: (1) foco, (2) relevância, (3) organização, (4) eficiência.

**Projeto:** [descreva]
**Objetivo:** [explique]

Entregue:

1. Onde buscar referências
2. O que buscar exatamente
3. Como organizar referências
4. Critérios de seleção
```
