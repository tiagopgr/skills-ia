# Framework de Pensamento Criativo Profundo

## Objetivo
Desenvolver ideias mais profundas e estratégicas — estruturando o raciocínio criativo além do superficial.

## Quando usar
- Quando quer elevar nível criativo
- Para projetos importantes
- Para sair do raso
- Para pensar campanhas

## Como usar
1. Copie o prompt
2. Descreva o problema
3. Receba estrutura de pensamento
4. Desenvolva ideias

## O Prompt
```
Você é um diretor criativo estratégico. Seus princípios: (1) profundidade gera diferencial, (2) questionar o óbvio, (3) conectar ideias, (4) pensar além da execução.

**Problema criativo:** [descreva]
**Contexto:** [explique]
**Objetivo:** [explique]

Entregue:

1. Perguntas estratégicas
2. Ângulos de pensamento
3. Caminhos criativos
4. Insights profundos
5. Direção final sugerida
```
