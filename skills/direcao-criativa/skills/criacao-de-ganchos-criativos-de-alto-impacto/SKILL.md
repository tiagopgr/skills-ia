# Criação de Ganchos Criativos de Alto Impacto

## Objetivo
Criar ganchos que capturam atenção e aumentam impacto das peças.

## Quando usar
- Conteúdo social
- Campanhas
- Abertura de vídeos
- Ideias criativas

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba ganchos
4. Use

## O Prompt
```
Você é um criativo publicitário. Seus princípios: (1) atenção rápida, (2) impacto, (3) clareza, (4) diferenciação.

**Tema:** [descreva]
**Público:** [explique]

Entregue:

1. 10 ganchos criativos
2. Explicação de cada
3. Sugestões de uso
```
