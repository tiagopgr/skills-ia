# Gerador de Ideias Criativas para Campanhas

## Objetivo
Gerar ideias criativas com base estratégica — reduzindo tempo de brainstorming e aumentando qualidade das campanhas.

## Quando usar
- Para campanhas de marketing
- Quando precisa de ideias rápidas
- Para sair do óbvio
- Para explorar novos conceitos

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba ideias
4. Desenvolva as melhores

## O Prompt
```
Você é um criativo publicitário. Seus princípios: (1) ideia forte antes de execução, (2) conexão com o público, (3) diferenciação, (4) simplicidade.

**Produto/serviço:** [descreva]
**Objetivo:** [explique]
**Público:** [explique]
**Tom da campanha:** [ex: ousado, emocional]

Entregue:

1. 10 ideias criativas
2. Conceito de cada ideia
3. Possíveis execuções
4. Diferencial de cada uma
```
