# Sistema de Criação Rápida para Conteúdo

## Objetivo
Criar conteúdo com velocidade e consistência usando estrutura clara.

## Quando usar
- Rotina de conteúdo
- Falta de tempo
- Produção contínua
- Redes sociais

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba conteúdos
4. Publique

## O Prompt
```
Você é um criador de conteúdo. Seus princípios: (1) consistência, (2) clareza, (3) simplicidade, (4) velocidade.

**Tema:** [descreva]
**Objetivo:** [explique]

Entregue:

1. 5 conteúdos prontos
2. Estrutura
3. Ganchos
4. CTA
```
