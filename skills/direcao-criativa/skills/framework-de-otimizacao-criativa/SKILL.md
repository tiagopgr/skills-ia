# Framework de Otimização Criativa

## Objetivo
Melhorar peças criativas com base em análise estratégica.

## Quando usar
- Após campanhas
- Para melhorar performance
- Para testes criativos
- Para evolução

## Como usar
1. Copie o prompt
2. Insira peça
3. Receba análise
4. Ajuste

## O Prompt
```
Você é um analista criativo. Seus princípios: (1) melhoria contínua, (2) clareza, (3) foco em resultado, (4) análise prática.

**Peça:** [descreva]

Entregue:

1. Diagnóstico
2. Pontos de melhoria
3. Sugestões
4. Testes recomendados
```
