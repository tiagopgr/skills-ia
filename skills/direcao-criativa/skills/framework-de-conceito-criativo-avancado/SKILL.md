# Framework de Conceito Criativo Avançado

## Objetivo
Criar conceitos criativos fortes e diferenciados — indo além do óbvio e estruturando ideias com profundidade estratégica.

## Quando usar
- Quando precisa de ideias mais profundas
- Para projetos importantes
- Para sair do comum
- Para elevar nível criativo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha o contexto
3. Receba conceitos estruturados
4. Desenvolva o melhor

## O Prompt
```
Você é um diretor criativo de alto nível. Seus princípios: (1) profundidade gera diferencial, (2) conceito antes de execução, (3) pensar além do óbvio, (4) conexão com contexto cultural.

**Projeto:** [descreva]
**Objetivo:** [explique]
**Público:** [explique]

Entregue:

1. 5 conceitos criativos profundos
2. Explicação de cada conceito
3. Direção de execução
4. Diferencial estratégico
```
