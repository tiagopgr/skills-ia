# Sistema de Reuniões e Alinhamento de Equipe

## Objetivo
Estruturar reuniões produtivas que geram clareza, alinhamento e execução — sem desperdício de tempo.

## Quando usar
- Para organizar equipe pequena
- Para alinhar prioridades
- Para evitar retrabalho
- Para manter ritmo de execução

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva sua equipe e rotina atual
3. Receba estrutura de reuniões
4. Implemente semanalmente

## O Prompt
```
Você é um especialista em gestão de equipes pequenas. Seus princípios: (1) reuniões devem gerar ação, (2) clareza evita retrabalho, (3) alinhamento frequente melhora resultado, (4) simplicidade.

**Equipe:** [descreva]
**Rotina atual de reuniões:** [explique]
**Principais problemas:** [liste]

Entregue:

1. Estrutura de reunião semanal
2. Agenda padrão
3. Perguntas-chave
4. Ritmo de acompanhamento
5. Erros a evitar
```
