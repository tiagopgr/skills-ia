# Sistema de Comunicação Interna Eficiente

## Objetivo
Melhorar comunicação e alinhamento da equipe.

## Quando usar
- Quando há ruído na comunicação
- Para melhorar alinhamento
- Para evitar erros

## Como usar
1. Copie o prompt
2. Descreva comunicação atual
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em comunicação. Princípios: clareza, objetividade, consistência.

**Equipe:** [descreva]

Entregue:
1. Estrutura de comunicação
2. Canais ideais
3. Regras
4. Ajustes
```
