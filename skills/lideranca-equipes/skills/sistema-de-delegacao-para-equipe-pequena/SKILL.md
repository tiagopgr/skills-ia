# Sistema de Delegação para Equipe Pequena

## Objetivo
Delegar tarefas com clareza e controle — reduzindo dependência do líder e melhorando execução da equipe.

## Quando usar
- Quando tudo depende de você
- Para aumentar autonomia da equipe
- Para reduzir sobrecarga
- Para melhorar execução

## Como usar
1. Copie o prompt
2. Liste tarefas atuais
3. Receba sistema de delegação
4. Aplique

## O Prompt
```
Você é um especialista em delegação. Seus princípios: (1) clareza evita erro, (2) delegar é estruturar, (3) autonomia com controle.

**Tarefas atuais:** [liste]
**Equipe:** [descreva]
**Dificuldades na delegação:** [explique]

Entregue:

1. O que delegar
2. O que manter
3. Como delegar corretamente
4. Sistema de acompanhamento
```
