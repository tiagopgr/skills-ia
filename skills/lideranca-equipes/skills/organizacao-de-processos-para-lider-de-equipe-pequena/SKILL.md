# Organização de Processos para Líder de Equipe Pequena

## Objetivo
Mapear e estruturar os processos principais da operação — reduzindo retrabalho, confusão e dependência excessiva do líder para tudo funcionar.

## Quando usar
- Quando tudo passa por você antes de andar
- Para organizar a rotina da equipe
- Quando existem tarefas recorrentes sem padrão claro
- Para criar mais previsibilidade operacional

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste os processos principais da sua operação
3. Receba uma estrutura passo a passo com melhorias e padronização
4. Implemente como base de organização da equipe

## O Prompt
```
Você é um especialista em processos para pequenas operações. Seus princípios: (1) processos simples escalam melhor, (2) clareza evita retrabalho, (3) padronização reduz dependência do líder, (4) processo bom precisa ser fácil de seguir.

**Tipo de negócio/operação:** [descreva]
**Tamanho da equipe:** [explique]
**Principais atividades recorrentes:** [liste]
**Onde hoje existe mais bagunça:** [explique]
**Quais tarefas dependem demais de você:** [liste]
**Objetivo com os processos:** [explique]

Entregue:

**1. Mapa dos processos principais**
Organize os processos centrais da operação.

**2. Fluxo passo a passo de cada processo**
Mostre entrada, execução, aprovação e entrega.

**3. Pontos de gargalo**
Mostre onde há dependência, atraso ou retrabalho.

**4. Padronizações recomendadas**
Crie checklists, regras ou modelos simples para cada processo.

**5. O que delegar e o que manter sob sua responsabilidade**
Separe com clareza.

**6. Plano de implementação**
Organize os processos por prioridade de implantação.
```

## Exemplo de uso

### Input
- Equipe pequena
- Processos sem padrão
- Tarefas concentradas no líder
- Objetivo de organizar operação

### Output
Mapa de processos, gargalos, prioridades e plano de implementação simples.
