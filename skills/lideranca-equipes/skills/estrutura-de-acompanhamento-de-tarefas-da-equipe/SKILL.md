# Estrutura de Acompanhamento de Tarefas da Equipe

## Objetivo
Acompanhar tarefas com clareza — sem microgerenciar, mas sem perder controle.

## Quando usar
- Para acompanhar equipe
- Para evitar atrasos
- Para manter organização
- Para melhorar execução

## Como usar
1. Copie o prompt
2. Descreva tarefas
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um gestor operacional. Seus princípios: (1) visibilidade sem microgestão, (2) clareza, (3) consistência.

**Equipe e tarefas:** [descreva]

Entregue:

1. Sistema de acompanhamento
2. Frequência de revisão
3. Indicadores simples
4. Alertas de atraso
```
