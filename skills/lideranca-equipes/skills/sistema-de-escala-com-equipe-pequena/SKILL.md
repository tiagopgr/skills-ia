# Sistema de Escala com Equipe Pequena

## Objetivo
Crescer sem perder controle e qualidade.

## Quando usar
- Para escalar operação
- Para crescer faturamento
- Para organizar time

## Como usar
1. Copie o prompt
2. Descreva operação
3. Receba plano
4. Execute

## O Prompt
```
Você é um estrategista de escala. Princípios: processos, clareza, foco.

**Negócio:** [descreva]

Entregue:
1. Plano de escala
2. Gargalos
3. Ajustes necessários
4. Prioridades
```
