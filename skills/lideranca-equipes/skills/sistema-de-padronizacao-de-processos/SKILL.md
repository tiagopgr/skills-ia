# Sistema de Padronização de Processos

## Objetivo
Criar padrões claros para processos recorrentes — reduzindo erros, retrabalho e dependência de memória.

## Quando usar
- Quando cada tarefa é feita de um jeito
- Para reduzir erros da equipe
- Para escalar operação

## Como usar
1. Copie o prompt
2. Liste processos
3. Receba padronização
4. Aplique

## O Prompt
```
Você é um especialista em padronização. Princípios: (1) padrão reduz erro, (2) simples funciona, (3) repetição gera eficiência.

**Processos:** [liste]

Entregue:
1. Padrões por processo
2. Checklists
3. Regras claras
4. Sugestões de melhoria
```
