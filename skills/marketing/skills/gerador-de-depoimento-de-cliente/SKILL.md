# Gerador de Depoimento de Cliente

## Objetivo
Criar o script para pedir depoimento ao cliente de forma que ele responda com as informações certas — e transformar a resposta dele em um depoimento formatado, poderoso e estratégico — para construir prova social que converte novos clientes mesmo com poucos trabalhos no portfólio.

## Quando usar
- Logo após a entrega de um projeto bem recebido
- Para transformar um "adorei!" no WhatsApp em prova social real
- Quando o cliente quer te indicar mas não sabe o que dizer
- Para criar depoimentos para site, Instagram, proposta e portfólio

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o projeto entregue e o que o cliente disse espontaneamente
3. Receba o script de pedido de depoimento e o depoimento formatado
4. Envie o script ao cliente e use o depoimento formatado nas suas aplicações

## O Prompt
```
Você é um especialista em marketing de prova social para freelancers. Seus princípios: (1) depoimento genérico não converte — "ótimo profissional" vale zero, (2) o depoimento mais poderoso conta: o problema antes, o que o profissional fez e o resultado depois, (3) o cliente raramente sabe como dar um bom depoimento — sua tarefa é guiá-lo com as perguntas certas, (4) depoimento curto e específico converte mais que depoimento longo e vago.

Crie o script de pedido e o depoimento formatado para este contexto:

**Nome do cliente:** [nome]
**Projeto entregue:** [o que foi feito]
**O que o cliente disse espontaneamente:** [copie as mensagens reais que ele mandou — ex: "ficou incrível, amei tudo!"]
**Qual era o problema/situação antes do projeto:** [o que você sabe que estava errado]
**Qual foi o resultado ou mudança:** [o que melhorou após a entrega]
**Onde o depoimento vai ser usado:** [Instagram / site / proposta / Behance]

Entregue:

**1. Mensagem para pedir o depoimento (WhatsApp)**
Texto natural e não constrangedor para pedir o depoimento, com 3 perguntas específicas embutidas que vão gerar um depoimento rico.

**2. Depoimento formatado (versão curta — 2-3 linhas)**
Para usar em cards de Instagram e proposta comercial.

**3. Depoimento formatado (versão longa — 1 parágrafo)**
Para usar em site, portfólio e Behance.

**4. Card de depoimento (estrutura visual)**
Descrição de como montar o card: o que colocar, em que ordem, quais elementos incluir (foto, nome, profissão, estrelas).

**5. Legenda para postar o depoimento no Instagram**
Texto que contextualiza o depoimento e termina com CTA para novos clientes.
```

## Exemplo de uso

### Input
- Cliente: Camila (personal trainer)
- Projeto: Identidade visual completa
- O que disse: "Gui, ficou PERFEITO. Esse logo tem a minha cara, não acredito como você entendeu o que eu queria sem eu saber explicar"
- Problema antes: Marca inconsistente, usava fontes aleatórias e cores diferentes em tudo
- Resultado: Marca nova, já atualizou todas as redes e recebeu elogios das alunas
- Uso: Instagram + proposta
