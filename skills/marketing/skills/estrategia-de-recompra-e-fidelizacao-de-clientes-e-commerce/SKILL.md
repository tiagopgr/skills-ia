# Estratégia de Recompra e Fidelização de Clientes (E-commerce)

## Objetivo
Criar uma estratégia completa de recompra e fidelização de clientes para e-commerce — com sequência pós-compra, ofertas para a segunda compra, comunicação de relacionamento e programa de indicação — para aumentar o valor vitalício de cada cliente sem precisar gastar mais em tráfego para adquirir novos.

## Quando usar
- Quando tiver base de clientes que compraram uma vez mas não voltaram
- Para criar uma sequência pós-compra que ativa o cliente logo após a primeira compra
- Quando quiser aumentar o faturamento sem aumentar o investimento em tráfego
- Para criar um programa simples de indicação que traz novos clientes via boca a boca
- Sempre que quiser transformar clientes pontuais em compradores recorrentes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe sua base de clientes, os produtos que vende e os canais de contato disponíveis
3. Receba a estratégia completa com sequência de mensagens e copy pronta
4. Execute a estratégia de reativação com clientes antigos primeiro, depois automatize para novos

## O Prompt
```
Você é um especialista em retenção e monetização de base de clientes para e-commerce. Seus princípios: (1) é 5-7x mais barato vender para quem já comprou do que para cliente novo, (2) a janela de recompra varia por produto — precisa saber quando o cliente está propenso a comprar de novo, (3) fidelização não é desconto — é relevância e experiência, (4) cliente que indica vale 3x mais que cliente que veio de anúncio.

Crie uma estratégia de recompra e fidelização com os seguintes dados:

**Produtos que você vende:** [lista com preços]
**Base de clientes:** [quantos clientes já compraram, quando foi a última compra da maioria]
**Tempo médio de recompra (se souber):** [ex: cliente geralmente volta em 2-3 meses / produto de compra única]
**Canais de contato com clientes:** [email / WhatsApp / Instagram / outros]
**Tem novos produtos para oferecer?** [ex: produtos complementares, novidades de estoque]
**Tem programa de indicação hoje?** [sim / não]

Entregue:

**1. Diagnóstico de oportunidade de recompra**
Com base nos produtos e no tempo médio, qual é o momento ideal para contatar o cliente

**2. Sequência pós-compra (para novos clientes a partir de agora)**
Mensagem 1 (no dia da compra): Confirmação + boas-vindas
Mensagem 2 (no dia da entrega): Recebeu? Pedido de avaliação
Mensagem 3 (7-14 dias após): Follow-up de experiência + produto complementar
Mensagem 4 (30-45 dias): Oferta da segunda compra

Para cada mensagem: canal recomendado + texto completo

**3. Campanha de reativação (para clientes inativos)**
Sequência de 3 contatos para trazer de volta clientes que compraram há 60+ dias e sumiram

**4. Programa de indicação simples**
Como estruturar um programa de "indique um amigo" sem precisar de sistema complexo — apenas com WhatsApp e cupom de desconto manual

**5. Oferta da segunda compra**
Como estruturar a oferta para o segundo pedido ser mais fácil do que o primeiro

**6. Segmentação da base**
Como dividir sua base de clientes em grupos para comunicação mais relevante
```

## Exemplo de uso

### Input
- Produtos: Suporte notebook R$89 / Mousepad R$39 / Luminária LED R$69
- Base: 180 clientes nos últimos 6 meses, maioria comprou 1 vez
- Última compra da maioria: há 2-4 meses
- Canais: WhatsApp e Instagram
- Produtos novos: Sim — suporte para monitor R$149 acaba de chegar
- Programa de indicação: Não tem

### Output
Canal: WhatsApp
Timing: 10 dias após a compra
Texto: "Oi [nome]! Já faz uns dias desde que você pegou o suporte ergonômico. Tá gostando?
Fico feliz quando o pessoal manda foto do setup novo 😄

Uma coisa que muita gente percebe: depois de resolver a postura do notebook, o problema vira o mouse em posição ruim. Sei que parece pequeno, mas faz diferença.

Tenho um mousepad gamer que complementa bem — antiderrapante, grande o suficiente para mouse + teclado, R$39 com frete grátis.

Se quiser, é só me falar que te mando o link."
