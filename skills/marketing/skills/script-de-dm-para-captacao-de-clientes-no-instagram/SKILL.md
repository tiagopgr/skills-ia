# Script de DM para Captação de Clientes no Instagram

## Objetivo
Criar scripts de mensagem direta (DM) para abordar potenciais clientes no Instagram — que chegaram via conteúdo, comentários ou enquetes — de forma natural, sem parecer vendedor, para converter audiência engajada em leads qualificados que agendam uma consulta ou conversa inicial.

## Quando usar
- Para responder alguém que comentou no post ou respondeu um story
- Para abordar quem respondeu uma enquete ou caixinha de perguntas sinalizando interesse
- Para reativar pessoas que interagiram mas nunca entraram em contato
- Para transformar DMs de "que legal!" em conversas com potencial de venda

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto do lead (de onde ele veio, o que fez)
3. Receba os scripts de DM prontos para cada situação
4. Personalize com o nome e um detalhe específico antes de enviar

## O Prompt
```
Você é um especialista em vendas conversacionais para profissionais de saúde no Instagram. Seus princípios: (1) DM é conversa, não pitch — começa com interesse genuíno, nunca com oferta, (2) a primeira mensagem deve ser tão pessoal que a pessoa sente que você a ouviu de verdade, (3) nunca envie link na primeira mensagem — link premature mata a conversa, (4) objetivo da DM é agendar uma conversa, não fechar a venda — a venda acontece na conversa.

Crie scripts de DM para os seguintes contextos:

**Minha especialidade:** [ex: nutricionista, coach de saúde, fisioterapeuta]
**Serviço principal:** [ex: consultas online, programa de 8 semanas]
**Tom da minha comunicação:** [próximo e empático / profissional / motivacional]

Crie scripts para estas situações:

**Situação 1:** Alguém que comentou no post algo como "que conteúdo incrível, precisava muito disso"
**Situação 2:** Alguém que respondeu a enquete dizendo que tem o problema que você abordou
**Situação 3:** Alguém que respondeu a caixinha de perguntas com uma dor específica
**Situação 4:** Alguém que salvou ou compartilhou o post (você viu nas métricas ou a pessoa avisou)
**Situação 5:** Alguém que interagiu muito nos últimos dias mas nunca foi ao DM
**Situação 6:** Lead antigo que sumiu — reativação após 30+ dias sem contato

Para cada script:
- Mensagem inicial (abertura)
- Como prosseguir se a pessoa responder positivamente
- Como prosseguir se a pessoa responder só com "haha", "obrigada" ou emoji
- Como fazer a transição natural para a oferta do serviço

**Entregue também:**
**Erros comuns de DM** — o que nunca fazer ao abordar um lead no Instagram
```
