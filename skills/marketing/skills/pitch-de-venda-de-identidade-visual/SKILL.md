# Pitch de Venda de Identidade Visual

## Objetivo
Criar um pitch de venda personalizado para abordar potenciais clientes e oferecer serviços de identidade visual — de forma natural, sem parecer vendedor chato — para quem está começando e ainda não tem um fluxo constante de indicações e precisa prospectar ativamente.

## Quando usar
- Para abordar um negócio local que claramente precisa de identidade visual
- Quando quer responder ao perfil de alguém que falou "precisaria de um logo"
- Para comentários ou DMs no Instagram com potencial de virar cliente
- Para criar a mensagem certa para indicações que ainda não te conhecem

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto do cliente que você quer abordar
3. Receba o pitch personalizado para cada canal
4. Ajuste o tom e envie pelo canal mais adequado

## O Prompt
```
Você é um consultor de vendas especializado em serviços criativos. Seus princípios: (1) pitch bom não parece pitch — parece uma conversa útil, (2) o gancho deve tocar na dor ou na oportunidade do cliente, não no seu serviço, (3) a primeira mensagem não fecha venda — abre uma conversa, (4) especificidade vende mais que generalidade — mostrar que você pesquisou o negócio do cliente vale mais que qualquer argumento genérico.

Crie pitches de venda personalizados para a seguinte situação:

**Tipo de cliente que quero abordar:** [ex: nutricionista autônoma, loja de roupas local, personal trainer]
**Onde vou fazer a abordagem:** [Instagram DM / WhatsApp / pessoalmente / comentário em post]
**O que observei na marca dele:** [algo específico que está faltando ou poderia melhorar — seja honesto]
**Serviço que quero oferecer:** [ex: identidade visual, logo, rebranding]
**Meu diferencial para este cliente:** [algo específico que faz sentido para o perfil dele]
**Tom desejado:** [formal / informal / descontraído / direto]

Entregue:

**1. Pitch curto para Instagram DM (3-4 linhas)**
Direto, pessoal, com gancho no negócio do cliente. Não começa com "Olá, sou designer".

**2. Pitch médio para WhatsApp (6-8 linhas)**
Um pouco mais de contexto, mostra que você pesquisou o negócio, termina com pergunta aberta.

**3. Pitch para comentário de post (1-2 linhas)**
Comentário que chama atenção sem parecer spam. Gera curiosidade.

**4. Script de abordagem pessoal (para negócios locais)**
O que dizer pessoalmente em uma conversa de 2-3 minutos quando você encontra o dono do negócio.

**5. Respostas para as 3 objeções mais comuns**
"Já tenho logo", "não tenho dinheiro agora", "vou pensar" — com resposta que mantém a conversa viva.
```

## Exemplo de uso

### Input
- Cliente: Barbearia local com logo pixelado e paleta inconsistente
- Canal: Instagram DM
- Observação: Perfil com bons cortes mas logo amador que parece feito no Word
- Serviço: Identidade visual completa
- Diferencial: Entrego rápido, em até 15 dias
- Tom: Descontraído
