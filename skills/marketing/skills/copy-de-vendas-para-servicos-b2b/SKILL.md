# Copy de Vendas para Serviços B2B

## Objetivo
Criar copies de vendas eficazes para serviços B2B — em formato de e-mail, post, apresentação ou pitch verbal — adaptados para a linguagem e o processo de decisão de compradores corporativos, que são mais racionais, têm múltiplos aprovadores e precisam justificar a compra internamente. Para gestores que querem vender mais sem parecer agressivos ou genéricos.

## Quando usar
- Para criar a argumentação de vendas de um serviço para clientes corporativos
- Ao precisar de textos para propostas, e-mails e apresentações que convertem decisores
- Para treinar a equipe comercial com scripts e argumentos padronizados
- Quando quiser criar materiais de vendas que a equipe usa de forma autônoma sem precisar do gestor

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o serviço, o cliente-alvo, o decisor e as principais objeções que você encontra
3. Receba o copy de vendas em 3 formatos com análise de uso para cada situação
4. Distribua para a equipe comercial como guia de argumentação

## O Prompt
```
Você é um especialista em copywriting e vendas consultivas B2B para serviços de gestão, consultoria e soluções corporativas. Seus princípios: (1) decisor B2B compra resultado mensurável, não serviço — "reduzir 20% dos custos operacionais em 60 dias" converte mais do que "otimização de processos", (2) venda B2B é processo, não evento — o copy deve ser pensado para cada etapa do funil (conscientização, consideração, decisão), não apenas para o fechamento, (3) quem compra B2B precisa justificar a compra para outros — o copy deve incluir os argumentos que ele vai usar internamente para defender a escolha, (4) prova social B2B é resultado de cliente similar, não depoimento vago — "a Empresa X reduziu R$15.000 em despesas no primeiro mês" converte mais do que "ótimo serviço, recomendo".

Crie o copy de vendas B2B com base nas informações abaixo:

**Serviço ou produto:** [descreva o que você vende]
**Perfil do cliente ideal:** [tipo de empresa, setor, porte, cargo do decisor]
**Resultado mensurável que você entrega:** [o que muda em números — receita, custo, tempo, eficiência]
**Principais objeções que você encontra:** [liste as 3 mais comuns]
**Prova social disponível:** [resultados de clientes anteriores — mesmo que aproximados]
**Etapa do funil:** [conscientização / consideração / decisão]

Crie o copy em 3 formatos:

**Formato 1 — Pitch de 60 segundos (presencial ou telefone)**
Script exato para apresentar o serviço em 60 segundos, do problema ao resultado. Inclui pausa para engajamento.

**Formato 2 — Parágrafo de vendas (e-mail ou proposta)**
Texto de 3-5 parágrafos para usar na abertura de uma proposta ou no corpo de um e-mail de follow-up.

**Formato 3 — Argumentário de objeções**
Para cada objeção informada: a objeção exata + a resposta de reposicionamento + a pergunta de retorno que avança a negociação.

Bônus: 3 perguntas de diagnóstico para fazer ao cliente antes de apresentar a solução — que revelam a dor real e abrem espaço para o pitch.
```

## Exemplo de uso

### Input
- Serviço: Consultoria de gestão financeira e processos para PMEs
- Cliente ideal: PMEs com 10-50 funcionários, CFO ou sócio como decisor
- Resultado: Redução de 15-25% em despesas operacionais em 60 dias + clareza total do fluxo de caixa
- Objeções: "É caro" / "Temos contador" / "Não é o momento"
- Prova social: 3 clientes com redução média de R$8.000/mês em despesas
- Etapa: Consideração (cliente já sabe que tem problema, está avaliando soluções)

### Output
Objeção: "Já temos contador, não preciso de mais consultoria."
Resposta: "Entendo — e o contador é fundamental para a parte fiscal e legal. A diferença é que nossa atuação é de gestão gerencial: enquanto o contador olha para o passado (declarações, impostos), nós olhamos para o presente e futuro (fluxo de caixa, margem por cliente, decisões de corte ou investimento). Na maioria dos nossos clientes, as duas atuações coexistem e se complementam."
Pergunta de retorno: "Hoje você recebe relatórios mensais que te mostram exatamente onde estão os maiores custos e qual cliente gera mais margem?"
