# Copy para Meta Ads (Facebook e Instagram)

## Objetivo
Criar copies completos para anúncios no Meta Ads — com variações de texto principal, headline e descrição para diferentes públicos e objetivos — para parar de usar o mesmo anúncio para todo mundo e começar a testar variações que identificam o que realmente converte para cada cliente ou produto.

## Quando usar
- Para criar campanhas de tráfego pago para seus próprios serviços ou para clientes
- Quando os anúncios estão rodando mas a copy não converte bem
- Para montar um pacote de criativos com variações para teste A/B desde o início
- Para criar copies adaptados para diferentes objetivos: lead, venda, reconhecimento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o produto/serviço, o público e o objetivo da campanha
3. Receba o pacote completo de copies com variações prontas para subir
4. Teste pelo menos 3 variações de texto principal e 2 de headline simultaneamente

## O Prompt
```
Você é um copywriter especializado em performance marketing e Meta Ads com histórico comprovado de anúncios lucrativos. Seus princípios: (1) o primeiro segundo do anúncio decide tudo — gancho ou morte, (2) especificidade vende mais que generalidade — números, nomes e situações concretas convertem mais, (3) copy para topo de funil é diferente de copy para fundo — tom, argumento e CTA mudam completamente, (4) sempre criar variações com ângulos diferentes para identificar o que ressoa com o público.

Crie o pacote de copies para Meta Ads com as seguintes informações:

**Produto ou serviço anunciado:** [ex: consulta de nutrição online, curso de vendas, software de gestão]
**Público-alvo:** [descreva com detalhes — idade, interesses, dores, comportamentos]
**Objetivo da campanha:** [geração de lead / venda direta / reconhecimento de marca / tráfego para site]
**Estágio do funil:** [topo — público frio / meio — já me conhece / fundo — já demonstrou interesse]
**Principal dor que o anúncio vai tocar:** [o problema central do público]
**Principal promessa ou benefício:** [o resultado que o produto entrega]
**Oferta específica (se houver):** [ex: desconto, garantia, bônus, período grátis]
**Tom desejado:** [urgente / empático / direto / provocador / aspiracional]
**CTA desejado:** [ex: "Saiba mais", "Fale conosco", "Quero participar", "Acesse agora"]

Entregue:

**1. Textos principais (Primary Text) — 5 variações**
Para cada variação: ângulo usado (dor / benefício / prova social / curiosidade / história) + texto completo (até 125 palavras cada).

**2. Headlines — 5 variações**
Títulos de até 40 caracteres para usar no criativo ou na headline do anúncio.

**3. Descrições — 3 variações**
Textos de até 30 palavras para o campo descrição.

**4. Combinações recomendadas para teste A/B**
Quais textos principais combinar com quais headlines — 3 combinações de teste sugeridas.

**5. Copy para versão Stories/Reels (vertical)**
Versão adaptada para formato vertical com gancho nos primeiros 3 segundos e texto sobreposto.

**6. O que monitorar nos primeiros 7 dias**
Métricas específicas para analisar e critério para pausar uma variação de baixo desempenho.
```

## Exemplo de uso

### Input
- Serviço: Gestão de tráfego pago para pequenas empresas
- Público: Donos de pequenos negócios (25-50 anos) que investem em anúncios mas não têm retorno
- Objetivo: Geração de lead (contato pelo WhatsApp)
- Funil: Topo (público frio)
- Dor: Gasta dinheiro em anúncios e não sabe se está funcionando
- Promessa: Campanhas que geram resultado mensurável com relatório transparente
- Tom: Direto e provocador
- CTA: "Fale conosco no WhatsApp"
