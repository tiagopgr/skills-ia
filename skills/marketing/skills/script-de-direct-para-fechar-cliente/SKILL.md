# Script de Direct para Fechar Cliente

## Objetivo
Criar scripts completos de conversa para direct do Instagram ou WhatsApp — do primeiro contato ao agendamento da consulta — para que o advogado transforme seguidores interessados em clientes pagantes sem improvisar na hora, sem soar desesperado e sem perder leads por falta de acompanhamento estruturado.

## Quando usar
- Quando um seguidor manda mensagem perguntando sobre um serviço e você não sabe exatamente o que responder
- Para criar um fluxo padrão de atendimento no direct que qualifique e converta
- Quando receber muito "quanto custa?" sem converter e quiser mudar a abordagem
- Para treinar atendentes ou estagiários no padrão de comunicação do escritório

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o tipo de serviço e a situação típica do lead
3. Receba o script completo com variações para diferentes objeções
4. Salve como template no WhatsApp Business ou no Instagram

## O Prompt
```
Você é um especialista em vendas consultivas para serviços jurídicos. Seus princípios: (1) venda de advocacia é venda de confiança — quem vende rápido demais perde o cliente, (2) perguntas qualificadoras constroem autoridade antes de qualquer proposta, (3) "quanto custa?" é uma objeção disfarçada de pergunta — nunca responder com número antes de entender o caso, (4) o objetivo do direct não é fechar o contrato — é agendar a consulta.

**Tipo de serviço:** [ex: consulta inicial trabalhista / assessoria empresarial / divórcio]
**Situação típica do lead:** [ex: foi demitido e quer saber seus direitos / está abrindo empresa e precisa de contratos]
**Canal:** [Instagram Direct / WhatsApp]
**Objeções mais comuns:** [ex: "é muito caro" / "vou pensar" / "já tentei resolver sozinho"]
**Oferta de entrada:** [ex: consulta gratuita de 30min / consulta paga de R$ X / análise de caso grátis]

Entregue:

**1. Script de primeiro contato (lead chegou sozinho)**
Sequência de 3-4 mensagens do primeiro "oi" até o agendamento — com perguntas qualificadoras naturais.

**2. Script de abordagem proativa (lead comentou no post)**
Como iniciar a conversa com alguém que interagiu com um post sem que pareça spam.

**3. Resposta para "quanto custa?"**
Script específico para redirecionar a pergunta de preço para uma conversa de valor.

**4. Script de follow-up (lead sumiu)**
Mensagem de retomada para leads que não responderam em 2-3 dias — sem soar desesperado.

**5. Frase de encerramento positivo**
Para leads que disseram "não agora" — mantendo o relacionamento para futuras oportunidades.
```

## Exemplo de uso

### Input
- Serviço: Consulta trabalhista para trabalhadores demitidos
- Situação: Lead foi demitido e perguntou sobre seus direitos
- Canal: WhatsApp
- Objeções: "é caro" e "vou resolver no sindicato"
- Oferta de entrada: Consulta inicial de 30 min por R$ 150

### Output
"Boa pergunta! Antes de te passar qualquer valor, preciso entender melhor a sua situação para saber qual é o caminho mais adequado pro seu caso.

Me conta: há quanto tempo você estava na empresa? Você foi demitido com ou sem justa causa?

Com essas informações consigo te dizer exatamente o que você tem direito a receber — aí o investimento faz mais sentido."
