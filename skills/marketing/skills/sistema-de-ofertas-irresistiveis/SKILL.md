# Sistema de Ofertas Irresistíveis

## Objetivo
Criar ofertas que aumentam desejo e conversão — sem depender só de desconto.

## Quando usar
- Para aumentar vendas
- Para campanhas
- Para novos produtos

## Como usar
1. Copie o prompt
2. Preencha dados
3. Receba ofertas
4. Aplique

## O Prompt
```
Você é um estrategista de ofertas. Princípios: valor percebido, clareza, urgência.

**Produto:** [descreva]
**Público:** [explique]

Entregue:
1. 5 ofertas diferentes
2. Estrutura de cada oferta
3. Gatilhos
4. Sugestões de melhoria
```
