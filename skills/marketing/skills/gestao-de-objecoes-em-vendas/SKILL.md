# Gestão de Objeções em Vendas

## Objetivo
Criar respostas estratégicas para objeções comuns — aumentando confiança e taxa de fechamento.

## Quando usar
- Quando cliente trava na decisão
- Para melhorar conversão
- Para estruturar argumentos
- Para lidar com objeções frequentes

## Como usar
1. Copie o prompt
2. Liste objeções
3. Receba respostas
4. Use nas vendas

## O Prompt
```
Você é um especialista em vendas. Seus princípios: (1) objeção é dúvida, (2) clareza reduz resistência, (3) empatia vende, (4) lógica + emoção.

**Produto/serviço:** [descreva]
**Público:** [quem é]
**Principais objeções:** [liste]

Entregue:

1. Lista de objeções
2. Resposta estratégica para cada uma
3. Versão curta
4. Argumentos de reforço
```
