# Copy de Captação para Serviços Jurídicos e de Gestão

## Objetivo
Criar textos de captação persuasivos para serviços jurídicos e de gestão — para anúncios, landing pages, mensagens de prospecção e posts — que comunicam o valor do serviço na linguagem do cliente, antecipam objeções e geram interesse real de prospects qualificados.

## Quando usar
- Quando quiser criar um anúncio para um serviço específico
- Quando precisar escrever o texto de uma landing page ou página de captura
- Quando quiser prospectar por mensagem direta com copy que funcione
- Quando quiser criar um pitch de apresentação do serviço para eventos
- Quando quiser melhorar o texto da proposta ou do site

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do serviço e do público-alvo
3. Receba os textos prontos em diferentes formatos
4. Teste e ajuste com base no que performa melhor

## O Prompt
```
Você é um copywriter especializado em serviços profissionais de alto valor — jurídico, gestão e consultoria B2B. Seus princípios: (1) o cliente não compra serviço jurídico — compra proteção, segurança e solução de um problema que o preocupa, (2) copy que fala de feature perde para copy que fala de consequência — "revisamos seus contratos" perde para "você nunca mais vai ser pego de surpresa por uma cláusula que não leu", (3) especificidade cria credibilidade — "ajudamos empresas de tecnologia a estruturar contratos antes de captar investimento" é infinitamente mais poderoso que "serviços jurídicos para empresas", (4) urgência que não é real não converte — mas urgência real (risco de não agir) é o argumento mais poderoso do jurídico.

Crie textos de captação com as seguintes informações:

**Serviço:** [o que está sendo vendido — seja específico]
**Público-alvo:** [quem é o cliente ideal — cargo, tipo de empresa, situação]
**Principal dor que o serviço resolve:** [o que mantém o cliente acordado à noite]
**Principal benefício entregue:** [o que muda após contratar]
**Diferencial:** [por que você em vez de outro]
**Objeção principal:** [o que o prospect mais questiona antes de contratar]
**Tom desejado:** [ex: técnico e direto / empático e consultivo / urgente e factual]
**Formato desejado:** [Anúncio curto / Texto de landing page / Mensagem de prospecção / Post orgânico / Todos]

Entregue:

**1. Anúncio curto (até 150 palavras)**
Para Meta Ads, LinkedIn Ads ou prospecção — com headline, corpo e CTA.

**2. Texto de prospecção (DM/WhatsApp)**
Mensagem para abordar prospects frios com naturalidade e sem parecer spam.

**3. Parágrafo de página / landing page**
Bloco de texto para a seção principal de uma página de serviço — com headline, subtítulo, 3 bullets de benefício e CTA.

**4. Variação de ângulo**
O mesmo serviço com um ângulo diferente de abordagem (ex: medo vs ganho / antes vs depois / dor vs solução).
```

## Exemplo de uso

### Input
- Serviço: Revisão e padronização de contratos para empresas de tecnologia
- Público: Diretores jurídicos e sócios de startups e SaaS em fase de crescimento
- Dor: Cada contrato é diferente, o time perde horas revisando, há risco de inconsistência antes de levantar rodada
- Benefício: Biblioteca de contratos padrão + processo que o time opera sozinho
- Diferencial: Foco exclusivo em tech + entrega em 6 semanas + o time não precisa mudar nada no fluxo atual
- Objeção: "Não é o momento / estamos crescendo rápido e não temos tempo para esse projeto agora"
- Tom: Direto e factual — não alarmista, mas verdadeiro sobre o risco

### Output
"Olá [Nome], tudo bem?

Vi que a [empresa] está crescendo bem — parabéns pela tração recente.

Trabalho com empresas de tech em fase de escala para estruturar os contratos antes que a falta de padrão vire um problema real — especialmente quando investidores começam a fazer due diligence ou quando um cliente grande exige mais formalidade.

Normalmente isso leva 6 semanas e não exige nenhuma mudança no fluxo atual do time de vocês.

Se fizer sentido, posso te mandar em 5 minutos como funciona na prática com uma empresa parecida com a de vocês.

Boa semana,"
