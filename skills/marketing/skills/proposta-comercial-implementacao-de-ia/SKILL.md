# Proposta Comercial — Implementação de IA

## Objetivo
Criar uma proposta comercial completa para vender implementação de IA para empresas — com diagnóstico do problema, solução proposta, ROI estimado, escopo técnico em linguagem acessível, investimento e próximos passos — que convence o empresário cétic o de que IA não é luxo, é necessidade. Transforma a conversa técnica em argumento de negócio.

## Quando usar
- Após a reunião de diagnóstico com um cliente potencial
- Para formalizar uma proposta que o cliente vai levar para os sócios aprovarem
- Quando precisa justificar o investimento em IA com números concretos
- Para padronizar as propostas e ganhar velocidade no processo comercial

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o que você vai implementar e o que descobriu na reunião com o cliente
3. Informe o valor que quer cobrar e os resultados esperados
4. Receba a proposta completa pronta para enviar

## O Prompt
```
Você é um consultor de implementação de IA com experiência em vendas B2B para PMEs. Seus princípios: (1) empresário compra resultado, não tecnologia — a proposta começa pela dor e pelo ganho, nunca pelo nome da ferramenta; (2) ROI em R$ e horas — torne o retorno tangível e calculável; (3) escopo técnico em linguagem de negócio — "automação de atendimento via WhatsApp" bate "integração com API da OpenAI via webhook"; (4) risco percebido baixo — mostre que você gerencia a implementação, o cliente só precisa aprovar.

**Cliente:**
Empresa: [nome e setor]
Porte: [funcionários / faturamento aproximado]
Principais dores identificadas: [o que o cliente relatou]

**O que você vai implementar:**
[descreva as automações ou sistemas de IA propostos]

**Ferramentas que serão usadas:**
[ex: n8n + OpenAI + WhatsApp + Google Sheets]

**ROI esperado:**
[ex: economia de X horas/mês / redução de Y% nas faltas / atendimento 24h sem custo adicional]

**Escopo (o que está e o que não está incluído):**
[o que você entrega e o que não faz parte]

**Prazo de implementação:** [ex: 2 semanas / 30 dias]

**Valor do projeto:** [R$ valor]
**Modelo:** [projeto único / mensalidade / projeto + manutenção mensal]

**Seu diferencial como consultor:**
[o que te diferencia de outro que faz automação]

Entregue:

**1. Abertura — o problema em palavras do cliente**
1-2 parágrafos que mostram que você entendeu exatamente o que está travando o negócio.

**2. A oportunidade — o que IA pode fazer aqui**
Descrição das soluções propostas em linguagem de negócio — sem jargão técnico.

**3. O que você vai construir**
Descrição de cada entregável com: o que é | como funciona no dia a dia do cliente | o que muda para ele.

**4. ROI esperado**
Cálculo visual do retorno: horas economizadas × custo hora da equipe = R$/mês. Payback do investimento em meses.

**5. Como vamos trabalhar — 4 etapas**
Processo simplificado de implementação: diagnóstico → construção → testes → entrega + treinamento.

**6. Investimento**
Valor com contexto de ROI. Condições de pagamento.

**7. Próximos passos**
O que o cliente faz para aprovar — 1 ação clara.
```

## Exemplo de uso

### Input
- Cliente: Clínica Odontológica Sorriso, 8 funcionários
- Implementar: confirmação automática de consultas + respostas automáticas de dúvidas frequentes no WhatsApp
- ROI: redução de 50% nas faltas (R$1.200/mês recuperados) + 3h/dia da secretária liberadas
- Valor: R$2.800 implementação + R$400/mês manutenção

### Output
| | Situação atual | Com IA |
|---|---|---|
| Taxa de faltas | 20% (16 faltas/mês) | ~10% (8 faltas/mês) |
| Receita recuperada | — | +R$1.200/mês |
| Horas secretária em confirmações | 3h/dia (66h/mês) | 0h (automático) |
| Custo dessa hora | R$25/h | — |
| Economia mensal | — | R$1.650/mês |
|
