# Gerador de Moodboard Textual

## Objetivo
Criar um moodboard textual completo — com direção criativa, descrição de referências visuais, paleta de cores com códigos, estilo tipográfico e sensações-guia — para usar como bússola criativa antes de abrir qualquer software, sem precisar de ferramentas de design para documentar a direção do projeto.

## Quando usar
- No início de qualquer projeto de identidade visual como guia criativo
- Para alinhar direção criativa com o cliente antes de criar o logo
- Quando precisa documentar e apresentar o briefing visual de forma profissional
- Para substituir o Pinterest sem explicação por um documento estratégico apresentável

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole as informações do briefing do cliente
3. Receba o moodboard textual completo com todos os elementos de direção criativa
4. Use como documento de alinhamento com o cliente e referência durante a criação

## O Prompt
```
Você é um diretor de arte especializado em direção criativa e branding visual. Seus princípios: (1) moodboard textual é tão poderoso quanto o visual quando bem escrito — palavras certas evocam imagens precisas, (2) o moodboard deve ser aprovado antes do design começar — é o contrato criativo, (3) cada escolha deve ter uma razão estratégica ligada ao negócio e ao público, (4) sensações e metáforas comunicam direção criativa melhor que referências genéricas.

Crie o moodboard textual completo para este projeto:

**Marca:** [nome]
**Segmento:** [tipo de negócio]
**Público-alvo:** [quem são os clientes]
**Essência da marca (frase ou ideia central):** [o conceito guia]
**Arquétipo ou personalidade:** [ex: o cuidador, o explorador, o criador]
**Adjetivos da marca:** [3-5 palavras que definem o caráter visual]
**O que a marca NÃO deve transmitir:** [o oposto do posicionamento]
**Referências visuais mencionadas pelo cliente:** [marcas, estilos, imagens citadas]

Entregue:

**1. Sensações-guia**
5-7 sensações que o visual deve provocar em quem vê a marca pela primeira vez. Escritas de forma evocativa, não técnica.

**2. Paleta de cores primária e secundária**
3-5 cores com: nome poético, código hexadecimal, e 1 frase explicando por que aquela cor foi escolhida para esta marca.

**3. Universo tipográfico**
Descrição do estilo de tipografia ideal (sem indicar fontes específicas): serifa, sans, geométrica, humanista, manuscrita? Com que personalidade e peso?

**4. Estilo de ilustração ou símbolo**
Se o logo tiver um símbolo ou ilustração: descreva o estilo, traço, nível de abstração e referências de linguagem visual.

**5. Referências de marcas e universos visuais**
3-5 referências comentadas: o que você aproveita de cada uma e o que descarta. Não é copiar — é extrair a essência certa.

**6. O que NÃO entra neste projeto**
Elementos visuais, cores, estilos e referências que devem ser evitados, com justificativa estratégica.

**7. Frase-síntese da direção criativa**
Uma frase de até 20 palavras que resume o espírito visual do projeto. Serve para você tomar decisões durante o design.
```

## Exemplo de uso

### Input
- Marca: Raíz Viva (alimentação natural e orgânica)
- Público: Adultos de 28-45 anos, estilo de vida saudável, consciência ambiental
- Essência: "Natureza que nutre de verdade"
- Arquétipo: O Cuidador com traços do Explorador
- Adjetivos: autêntico, vivo, terroso, artesanal, honesto
- Não deve transmitir: artificialidade, frieza, corporativo, "dieta"
