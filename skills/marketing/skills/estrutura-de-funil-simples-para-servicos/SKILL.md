# Estrutura de Funil Simples para Serviços

## Objetivo
Criar um funil simples e funcional — organizando aquisição, relacionamento e conversão de clientes.

## Quando usar
- Quando não tem funil definido
- Para organizar aquisição de clientes
- Para aumentar conversão
- Para estruturar vendas

## Como usar
1. Copie o prompt
2. Preencha dados do negócio
3. Receba funil estruturado
4. Implemente

## O Prompt
```
Você é um estrategista de funil. Seus princípios: (1) simplicidade escala, (2) clareza no caminho do cliente, (3) relacionamento gera venda, (4) estrutura reduz esforço.

**Tipo de negócio:** [descreva]
**Como atrai clientes hoje:** [explique]
**Como vende hoje:** [explique]
**Objetivo:** [explique]

Entregue:

1. Estrutura do funil
2. Etapas do cliente
3. Estratégias de aquisição
4. Estratégias de conversão
5. Melhorias práticas
```
