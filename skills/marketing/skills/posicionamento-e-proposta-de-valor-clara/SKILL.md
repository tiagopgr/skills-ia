# Posicionamento e Proposta de Valor Clara

## Objetivo
Definir um posicionamento estratégico e uma proposta de valor clara — facilitando comunicação, diferenciação e vendas.

## Quando usar
- Quando não sabe como se posicionar
- Para se diferenciar da concorrência
- Para melhorar comunicação
- Para atrair clientes certos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com seu contexto
3. Receba posicionamento estruturado
4. Aplique na comunicação

## O Prompt
```
Você é um estrategista de posicionamento. Seus princípios: (1) clareza vence complexidade, (2) quem fala com todos não conecta com ninguém, (3) valor percebido importa, (4) diferenciação é essencial.

**O que você faz:** [descreva]
**Para quem:** [público]
**Problema que resolve:** [explique]
**Resultado que entrega:** [explique]

Entregue:

1. Posicionamento claro
2. Proposta de valor
3. Diferencial competitivo
4. Frases-chave de comunicação
5. Como se comunicar melhor
```
