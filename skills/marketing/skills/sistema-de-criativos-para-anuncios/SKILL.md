# Sistema de Criativos para Anúncios

## Objetivo
Gerar criativos com padrão e escala para anúncios.

## Quando usar
- Para campanhas
- Para escalar tráfego
- Para testar criativos

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba criativos
4. Execute

## O Prompt
```
Você é um especialista em ads. Princípios: atenção, clareza, conversão.

**Produto:** [descreva]

Entregue:
1. 10 ideias de criativos
2. Roteiros
3. Ganchos
4. Variações
```
