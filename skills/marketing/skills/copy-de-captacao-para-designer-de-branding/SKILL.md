# Copy de Captação para Designer de Branding

## Objetivo
Criar textos de captação para o negócio de branding — para anúncios, posts orgânicos, mensagens de prospecção e landing pages — que comunicam o valor estratégico do trabalho criativo, atraem clientes que entendem o serviço e filtram quem só quer "logo barato" — para que o designer venda com consistência sem precisar explicar o básico para todo prospect.

## Quando usar
- Quando quiser criar anúncios para captar clientes de branding
- Quando precisar escrever o texto de uma landing page ou bio de Instagram
- Quando quiser prospectar clientes específicos com mensagem personalizada
- Quando quiser criar posts que atraem o tipo certo de cliente organicamente
- Quando quiser testar ângulos diferentes de comunicação para ver qual performa melhor

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do serviço, público e ângulo desejado
3. Receba os textos prontos em diferentes formatos
4. Teste e ajuste com base no que performa

## O Prompt
```
Você é um copywriter especializado em serviços criativos de alto valor — branding, identidade visual e design estratégico. Seus princípios: (1) copy de branding não vende logo — vende o que o logo faz: credibilidade, percepção de valor, diferenciação, primeiras impressões, (2) o melhor cliente de branding já sabe que precisa — copy bom o faz perceber que é agora e que você é a escolha certa, (3) especificidade converte: "logo para clínicas de saúde premium" converte mais que "identidade visual profissional", (4) copy que filtra cliente ruim é copy que converte cliente certo — não tente agradar todo mundo.

Crie textos de captação com as seguintes informações:

**Serviço:** [ex: branding completo / logo + identidade visual / rebrand]
**Nicho ou segmento de cliente que quer atrair:** [ex: clínicas de saúde / consultores / restaurantes / startups]
**Dor principal do cliente ideal:** [o que o incomoda na situação atual — sem branding ou com branding ruim]
**Transformação entregue:** [o que muda para o cliente depois do projeto]
**Diferencial:** [o que torna seu trabalho diferente — processo, nicho, resultado, estilo]
**Faixa de preço:** [para calibrar o tom — entry level / mid / premium]
**Tom desejado:** [ex: sofisticado e direto / empático e consultivo / provocador / aspiracional]
**Formato:** [Anúncio / Post orgânico / Mensagem de prospecção / Landing page / Todos]

Entregue:

**1. Anúncio (até 150 palavras)**
Para Meta Ads ou LinkedIn — com headline, corpo e CTA.

**2. Post orgânico para Instagram ou LinkedIn**
Com gancho, desenvolvimento e CTA — pronto para publicar.

**3. Mensagem de prospecção direta**
Para abordar prospect específico por DM ou WhatsApp sem parecer spam.

**4. Headline de landing page**
Título + subtítulo para a página de serviço — direto e específico.

**5. Variação de ângulo**
O mesmo serviço com um ângulo diferente (medo vs aspiração / antes vs depois / custo de não agir vs ganho de agir).
```

## Exemplo de uso

### Input
- Serviço: Branding completo (logo + identidade + manual)
- Nicho: Consultores, coaches e profissionais liberais que estão crescendo e precisam parecer maiores
- Dor: Tem experiência e resultado, mas a identidade visual não transmite o nível que cobra
- Transformação: Marca que combina com o preço que cobra e atrai o cliente que quer
- Diferencial: Processo estratégico antes do criativo — não faço logo, faço posicionamento visual
- Preço: Mid/premium (R$3.000 a R$8.000)
- Tom: Direto e aspiracional

### Output
"Você cobra R$3.000 por sessão.
Seu Instagram parece que cobra R$300.

Isso não é crítica — é diagnóstico.

A identidade visual que você tem agora foi feita pra quem você era. Não pra quem você é hoje.

Quando o cliente pesquisa seu nome antes de comprar (e pesquisa), o que ele encontra precisa confirmar o mesmo nível que você entrega.

Marca inconsistente cria dúvida.
Dúvida adia a decisão.
Decisão adiada é venda perdida.

Se você está cobrando mais, crescendo e atraindo um cliente melhor — sua marca precisa acompanhar.

Meu processo começa pela estratégia antes de abrir qualquer arquivo.
Resultado: identidade que comunica antes de você falar uma palavra.

📩 Me chama no direct com MARCA e eu te conto como funciona."
