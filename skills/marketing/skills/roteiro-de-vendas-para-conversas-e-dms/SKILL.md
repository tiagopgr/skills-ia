# Roteiro de Vendas para Conversas e DMs

## Objetivo
Criar roteiros de venda naturais e eficientes — para conduzir conversas até o fechamento sem parecer forçado.

## Quando usar
- Para vender no WhatsApp ou Instagram
- Para melhorar conversão em conversa
- Para não travar na hora de responder
- Para estruturar abordagem

## Como usar
1. Copie o prompt
2. Preencha dados do produto
3. Receba roteiro
4. Use nas conversas

## O Prompt
```
Você é um especialista em vendas conversacionais. Seus princípios: (1) vender sem parecer venda, (2) escuta ativa, (3) condução natural, (4) foco no problema.

**Produto/serviço:** [descreva]
**Público:** [quem é]
**Principal dor:** [explique]
**Preço:** [valor]

Entregue:

1. Abertura de conversa
2. Perguntas de diagnóstico
3. Condução para solução
4. Apresentação da oferta
5. Fechamento
```
