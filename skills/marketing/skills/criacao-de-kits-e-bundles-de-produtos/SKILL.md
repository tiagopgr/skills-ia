# Criação de Kits e Bundles de Produtos

## Objetivo
Criar kits e bundles de produtos estratégicos que aumentam o ticket médio por pedido, reduzem o custo de aquisição de cliente, diferenciam a loja da concorrência e fazem o cliente sentir que está fazendo um ótimo negócio — com copy pronta e estratégia de precificação incluída.

## Quando usar
- Quando quiser aumentar o valor médio por pedido sem precisar de mais clientes
- Para criar uma oferta diferenciada que o cliente não encontra no concorrente
- Quando tiver produtos complementares que fazem sentido juntos
- Para reduzir custo de frete por pedido agrupando itens
- Ao criar promoções de Black Friday, Natal ou datas especiais com kits temáticos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste os produtos que você tem disponíveis e o objetivo (aumentar ticket, liquidar estoque, diferenciar)
3. Receba sugestões de kits com lógica de composição, precificação e copy pronta
4. Monte os kits na sua loja ou marketplace e use o copy sugerido

## O Prompt
```
Você é um especialista em estratégia de produto e merchandising para e-commerce. Seus princípios: (1) bundle deve fazer sentido para o cliente — a combinação precisa resolver um problema completo, não parecer que você está forçando itens, (2) o desconto do kit precisa ser real mas preservar margem — nunca vender kit com margem menor que o produto individual, (3) o nome e a narrativa do kit vendem mais do que o desconto, (4) kits reduzem custo de aquisição porque aumentam o ticket da mesma venda.

Crie sugestões de kits e bundles com os seguintes dados:

**Produtos disponíveis:** [liste todos os produtos com preço individual de cada um]
**Custo de cada produto (se souber):** [para calcular margem do kit]
**Objetivo principal:** [aumentar ticket médio / liquidar estoque / criar oferta diferenciada / data especial]
**Ticket médio atual por pedido:** [valor médio de compra hoje]
**Ticket que gostaria de atingir:** [meta]
**Canal de venda:** [Mercado Livre / Shopee / loja própria / WhatsApp]
**Público:** [perfil do comprador]

Entregue:

**1. Análise de oportunidade de bundle**
Quais combinações de produto fazem mais sentido para o perfil do comprador e por quê

**2. Sugestões de kits (mínimo 3 opções)**
Para cada kit:
- Nome do kit (deve ser aspiracional, não genérico)
- Composição (quais produtos + quantidades)
- Preço de cada item separado vs preço do kit
- Percentual de desconto aparente
- Margem estimada
- Para quem é esse kit (perfil específico)

**3. Copy de apresentação de cada kit**
Texto pronto para usar na página do produto ou no anúncio — apresentando a combinação como solução completa

**4. Estratégia de precificação**
Como precificar os kits para maximizar conversão sem comprometer margem

**5. Como apresentar na plataforma**
Dicas práticas de como montar o kit no Mercado Livre, Shopee ou loja própria (limitações, fotos, títulos)

**6. Upsell na hora do checkout**
Script de mensagem para oferecer o kit para quem já está comprando só um item
```

## Exemplo de uso

### Input
- Produtos: Suporte notebook R$89 / Mousepad gamer R$39 / Suporte para monitor R$149 / Luminária LED de mesa R$69
- Objetivo: Aumentar ticket médio
- Ticket atual: R$89 (vende majoritariamente o suporte)
- Meta: R$150+ por pedido
- Canal: Loja própria + Mercado Livre
- Público: Freelancers e empreendedores home office

### Output
Kit 1 — "Setup Home Office Completo"
Composição: Suporte notebook + Mousepad gamer
Preço separado: R$89 + R$39 = R$128
Preço do kit: R$109 (15% de desconto)
Margem: preservada se custo combinado < R$65
Copy: "Monte seu home office com tudo que você precisa de uma vez. Suporte ergonômico + mousepad premium — R$109 com frete grátis. Separado sairia R$128."
