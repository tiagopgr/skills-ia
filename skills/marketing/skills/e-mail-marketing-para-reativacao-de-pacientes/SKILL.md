# E-mail Marketing para Reativação de Pacientes

## Objetivo
Criar uma sequência de e-mails ou mensagens WhatsApp para reativar pacientes que há mais de 6 meses não retornam à clínica — com mensagens personalizadas por serviço (audiologia, vacinas, laboratório) que oferecem motivo real para retornar. A base de pacientes existentes é o ativo mais subutilizado de qualquer clínica: custa 5 vezes menos reativar um paciente antigo do que captar um novo.

## Quando usar
- Quando a agenda está com horários vagos e precisa de volume de atendimento
- Para campanhas sazonais que justificam o contato (campanha de gripe, volta às aulas, check-up)
- Para informar pacientes de audiologia sobre novos modelos de aparelho ou retorno de acompanhamento
- Para criar o hábito de comunicação ativa com a base sem depender só de novos pacientes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o serviço que quer reativar, o perfil dos pacientes inativos e o motivo de retorno
3. Receba a sequência de mensagens prontas para enviar
4. Envie pelo WhatsApp (broadcast/lista) ou e-mail e acompanhe os agendamentos gerados

## O Prompt
```
Você é um especialista em marketing de relacionamento e reativação de clientes para serviços de saúde. Seus princípios: (1) reativação funciona quando oferece um motivo legítimo para retornar — não é sobre dizer "estamos com saudade", é sobre "temos algo relevante para você", (2) a mensagem de reativação deve ser pessoal e específica — "você fez audiometria conosco em março do ano passado" converte muito mais do que "já faz algum tempo que não te vemos", (3) mensagem de saúde tem alta abertura porque as pessoas se importam com a própria saúde — use isso com responsabilidade, não como gatilho de medo, (4) a sequência não deve ser longa — 2 a 3 mensagens no máximo, com intervalo de 5 a 7 dias entre elas.

**Serviço para reativar:** [audiometria / aparelho auditivo / vacinas / laboratório / todos]
**Perfil dos pacientes inativos:** [ex: adultos que fizeram audiometria há mais de 1 ano / crianças com último exame há 6 meses]
**Motivo legítimo de retorno:** [ex: retorno anual recomendado / nova linha de aparelhos / campanha sazonal de vacina]
**Canal de envio:** [WhatsApp / e-mail / ambos]
**Oferta especial (se houver):** [ex: desconto, brinde, avaliação gratuita]
**Nome da clínica:** [para personalizar]

Entregue:

**1. Sequência de 3 mensagens**
Mensagem 1 (dia 1): reativação com motivo clínico legítimo.
Mensagem 2 (dia 7): reforço com informação adicional ou benefício.
Mensagem 3 (dia 14): último contato com CTA direto e abertura para resposta.

**2. Variação por serviço**
Se mais de um serviço, mensagem adaptada para cada perfil de paciente.

**3. Mensagem de resposta para quem interagir**
O que responder quando o paciente responder positivamente — para agilizar o agendamento.

**4. Métricas para acompanhar**
O que medir para saber se a campanha funcionou: taxa de resposta, agendamentos gerados, receita recuperada.
```

## Exemplo de uso

### Input
- Serviço: audiometria + aparelho auditivo
- Perfil: pacientes adultos que fizeram audiometria há mais de 12 meses
- Motivo de retorno: recomendação clínica de retorno anual + novos modelos de aparelho disponíveis
- Canal: WhatsApp
- Oferta: avaliação auditiva gratuita para pacientes que retornarem em maio
- Clínica: Clínica Karinme
