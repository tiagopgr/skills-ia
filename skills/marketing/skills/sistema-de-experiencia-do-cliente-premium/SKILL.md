# Sistema de Experiência do Cliente Premium

## Objetivo
Elevar experiência do cliente para aumentar retenção e valor percebido.

## Quando usar
- Para melhorar marca
- Para fidelizar clientes
- Para se posicionar premium

## Como usar
1. Copie o prompt
2. Descreva experiência atual
3. Receba melhorias
4. Aplique

## O Prompt
```
Você é um especialista em experiência. Princípios: consistência, detalhe, valor.

**Experiência atual:** [descreva]

Entregue:
1. Jornada do cliente
2. Pontos de melhoria
3. Ideias premium
4. Diferenciais
```
