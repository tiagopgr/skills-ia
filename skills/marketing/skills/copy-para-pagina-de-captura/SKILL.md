# Copy para Página de Captura

## Objetivo
Criar o copy completo de uma página de captura de leads — com headline que converte, subheadline que qualifica, lista de benefícios do material gratuito e CTA que gera cadastro — para que o empreendedor possa montar sua lista de contatos e ter um ativo de marketing que não depende do algoritmo das redes sociais.

## Quando usar
- Ao criar um isca digital (e-book, checklist, aula gratuita, quiz) para captar leads
- Para aumentar a taxa de conversão de uma página de captura existente
- Quando quiser começar a construir uma lista de e-mail ou WhatsApp
- Para criar uma landing page rápida sem contratar copywriter

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o material gratuito que está oferecendo e para quem
3. Receba o copy completo da página pronto para usar em Notion, Canva ou qualquer ferramenta de landing page
4. Publique e direcione tráfego orgânico ou pago para a página

## O Prompt
```
Você é um especialista em copy de páginas de captura e geração de leads para empreendedores digitais. Seus princípios: (1) a página de captura tem uma única função — fazer a pessoa clicar em "quero receber" — qualquer elemento que não contribui para isso distrai e reduz a conversão, (2) o título da página deve comunicar a transformação específica, não o formato do material — "Como conseguir seus primeiros 3 clientes" converte mais do que "E-book de marketing", (3) os benefícios devem ser resultados concretos, não características — "você vai saber como X" bate "aprenda sobre X", (4) o formulário deve pedir o mínimo possível — cada campo adicional reduz a conversão em até 10%.

**Material gratuito:** [o que você está oferecendo — ex: e-book, checklist, mini-curso, aula]
**Tema/título provisório do material:** [como você está chamando]
**Público-alvo:** [quem você quer captar]
**Maior transformação que o material entrega:** [o que muda pra quem baixar]
**3 benefícios principais do material:** [o que a pessoa vai aprender ou ganhar]
**Dados que vai pedir no formulário:** [ex: só e-mail / nome + e-mail / nome + WhatsApp]

Entregue:

**1. Headline principal (H1)**
Três opções de headline para a página — focadas em transformação, não no formato do material.

**2. Subheadline (H2)**
Uma linha que complementa a headline e qualifica o público ("ideal para quem...").

**3. Lista de benefícios**
5 a 7 bullets de benefício no formato "Você vai [verbo + resultado concreto]" — sem genérico.

**4. Texto de contexto (opcional)**
Parágrafo curto de 2-3 linhas para quem quer adicionar um "por que estou oferecendo isso".

**5. CTA do botão e texto ao redor**
Texto do botão de ação + frase de apoio acima ou abaixo do botão que reduz fricção.
```

## Exemplo de uso

### Input
- Material: checklist gratuito
- Título provisório: Checklist de Conteúdo para Instagram
- Público: empreendedores solo que querem crescer no Instagram
- Transformação: ter um processo claro de criação de conteúdo toda semana
- Benefícios: nunca mais travar sem saber o que postar, criar conteúdo em menos tempo, postar com consistência sem estresse
- Formulário: nome + e-mail
