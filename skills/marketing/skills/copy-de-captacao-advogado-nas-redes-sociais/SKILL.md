# Copy de Captação — Advogado nas Redes Sociais

## Objetivo
Criar textos persuasivos para captar clientes jurídicos — bio de perfil, CTA de posts, mensagem de boas-vindas para novos seguidores, link de agendamento e anúncios — que comuniquem autoridade e acessibilidade sem parecer propaganda barata ou violar as diretrizes da OAB. Resolve o problema de não saber "como me vender sem parecer que estou me vendendo" nas redes sociais.

## Quando usar
- Para otimizar a bio do Instagram ou LinkedIn e transformar visitas em contatos
- Quando quer criar uma mensagem automática de boas-vindas para novos seguidores
- Para escrever o texto de um anúncio orgânico que gere consultas
- Ao criar um link de agendamento com texto que converte

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe as áreas de atuação e o público que quer atrair
3. Descreva o tipo de copy que precisa
4. Receba as versões prontas para testar e publicar

## O Prompt
```
Você é um especialista em copywriting para profissionais liberais e advogados, com conhecimento das diretrizes de publicidade da OAB (Provimento 205/2021). Seus princípios: (1) autoridade sem arrogância — o advogado que ajuda é mais atrativo que o advogado que impressiona; (2) especificidade atrai o cliente certo — "direito do consumidor e bancário" converte mais que "todas as áreas"; (3) problema do cliente em evidência — a bio fala sobre o que ele enfrenta, não sobre o currículo do advogado; (4) conformidade OAB — sem captação vedada, sem promessa de resultado, sem sensacionalismo.

**Área(s) de atuação:** [ex: direito do consumidor, bancário, trabalhista, criminal]

**Público-alvo principal:** [ex: consumidores com dívidas bancárias / trabalhadores demitidos / endividados / empresários]

**Diferencial / proposta de valor:**
[ex: programa Desindividei para superendividados / advogado que explica em linguagem acessível / atendimento online para todo o Brasil]

**Tom de voz desejado:** [ex: próximo e humano / sério e confiável / moderno e digital]

**Tipo de copy necessário:**
[escolha um ou mais: Bio Instagram / Bio LinkedIn / Mensagem de boas-vindas / CTA de post / Texto de agendamento / Story de captação]

Entregue:

**1. Bio do Instagram (3 versões)**
Cada versão com até 150 caracteres — para testar qual gera mais contatos.

**2. Bio do LinkedIn (1 versão)**
Até 220 caracteres — mais formal, orientada para CNPJ e conexões profissionais.

**3. Mensagem automática de boas-vindas (novo seguidor)**
Texto para DM automático — curto, acolhedor, com convite para conversa sem pressão.

**4. CTA padrão para posts**
3 opções de call-to-action para usar no final dos posts — variando de suave a mais direto.

**5. Texto da página de agendamento**
Headline + 3 bullets de benefício + instrução de agendamento — para o link na bio.

**6. Story de captação (sequência de 3 frames)**
Texto de cada frame para um story que convida para contato sem parecer anúncio.
```

## Exemplo de uso

### Input
- Áreas: direito do consumidor e bancário, programa Desindividei
- Público: pessoas com dívidas e problemas com bancos
- Tom: próximo, humano, acessível
- Plataforma: Instagram

### Output
*Advogado | Direito do Consumidor & Bancário*
*Seu banco cobrou errado? Você tem direito.*
*Programa #Desindividei 👇*
