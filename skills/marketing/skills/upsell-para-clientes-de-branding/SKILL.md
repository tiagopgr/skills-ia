# Upsell para Clientes de Branding

## Objetivo
Criar ofertas de upsell e cross-sell estratégicas para clientes que já contrataram um projeto de identidade visual — identificando o momento certo, os serviços mais relevantes e a abordagem natural — para aumentar o faturamento por cliente sem precisar buscar novos o tempo todo.

## Quando usar
- Durante ou após a entrega de um projeto de identidade visual
- Quando percebe que o cliente vai precisar de materiais que não estão no escopo
- Para oferecer serviços complementares sem parecer que está "empurrando"
- Quando quer dobrar o ticket médio sem dobrar a quantidade de clientes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o projeto atual e o perfil do cliente
3. Receba as sugestões de upsell com abordagem e roteiro de oferta
4. Escolha as mais relevantes e aplique no momento certo

## O Prompt
```
Você é um consultor de crescimento de negócios para freelancers criativos. Seus princípios: (1) o melhor momento para vender mais é quando o cliente já confia em você — logo após uma entrega bem recebida, (2) upsell não é empurrar — é identificar a próxima necessidade real do cliente e se posicionar para resolvê-la, (3) oferta natural é mais eficaz que oferta estruturada — parece cuidado, não venda, (4) cliente satisfeito com a marca vai precisar de materiais — a questão é com quem ele vai contratar.

Crie uma estratégia de upsell e cross-sell para este contexto:

**Projeto entregue:** [ex: identidade visual completa — logo, paleta, tipografia, manual]
**Como foi a recepção do cliente:** [ex: adorou, pediu 1 revisão pequena, ficou muito satisfeito]
**Tipo de negócio do cliente:** [ex: nutricionista autônoma, loja física, serviço local]
**O que o cliente vai precisar a seguir:** [ex: redes sociais, cardápio, uniforme, site, embalagem]
**Seu momento no projeto:** [entregando agora / já entregou há X dias / cliente voltou com dúvida]

Entregue:

**1. Mapa de próximas necessidades**
Lista do que este tipo de cliente vai inevitavelmente precisar depois da identidade visual, organizada por urgência.

**2. Top 3 upsells recomendados para este perfil**
Os 3 serviços complementares com maior potencial de venda, com: o serviço, o valor sugerido, e o momento certo de oferecer.

**3. Script de oferta natural (para cada upsell)**
Como mencionar o serviço de forma que pareça uma dica, não uma venda. Frases reais para usar no WhatsApp ou na reunião de entrega.

**4. Pacote combinado sugerido**
Uma sugestão de como empacotar identidade visual + serviços extras com desconto progressivo para fechar tudo de uma vez.

**5. Mensagem de acompanhamento (D+30 após entrega)**
Mensagem para enviar 30 dias depois perguntando como está indo — que abre espaço para novas vendas de forma natural.
```

## Exemplo de uso

### Input
- Entregue: Identidade visual completa
- Recepção: Cliente amou, ficou muito feliz
- Negócio: Personal trainer autônomo
- Próximas necessidades: Stories template, card de depoimento, kit de boas-vindas para alunos
- Momento: Entregando agora
