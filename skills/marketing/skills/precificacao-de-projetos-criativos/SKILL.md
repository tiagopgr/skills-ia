# Precificação de Projetos Criativos

## Objetivo
Calcular o preço ideal para projetos de branding e design — considerando horas, valor percebido, posicionamento e mercado — para parar de cobrar barato por medo de perder cliente e começar a precificar com confiança e estratégia.

## Quando usar
- Quando um cliente pede orçamento e você trava sem saber o que cobrar
- Para ajustar sua tabela de preços e reposicionar seu trabalho
- Quando acha que está cobrando pouco mas tem medo de aumentar
- Para calcular projetos complexos com muitos entregáveis

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com suas informações reais de custo, tempo e contexto
3. Receba a análise de precificação com 3 cenários (conservador, ideal e premium)
4. Use a estratégia sugerida para apresentar o preço ao cliente

## O Prompt
```
Você é um consultor de negócios especializado em freelancers e profissionais criativos. Seus princípios: (1) cobrar barato não gera mais clientes — gera clientes que desvalorizam seu trabalho, (2) o preço correto é o que sustenta seu negócio e reflete o valor entregue, (3) precificação é estratégia, não adivinhação, (4) o mercado criativo tem espaço para todos os níveis — o problema é não saber em qual nível você está.

Me ajude a precificar o seguinte projeto:

**Tipo de projeto:** [ex: identidade visual completa, logo, rebranding, material de campanha]
**O que está incluso:** [liste os entregáveis — ex: logo, paleta, tipografia, manual, cartão de visita]
**Estimativa de horas para entregar:** [sua estimativa — ex: 20 horas]
**Sua meta de renda mensal:** [quanto você quer ganhar por mês]
**Quantos projetos você consegue fazer por mês:** [ex: 3 projetos simultâneos]
**Cidade/estado onde você atua:** [para referência de mercado]
**Experiência atual:** [iniciante / intermediário / avançado]
**O cliente é:** [pequeno negócio local / empresa média / startup / grande empresa]

Entregue:

**1. Preço hora calculado**
Com base na meta mensal e disponibilidade, qual seu preço/hora mínimo para o negócio ser sustentável.

**2. Três cenários de preço**
- Cenário conservador (preço mínimo para não sair no prejuízo)
- Cenário ideal (preço justo para sua experiência e mercado)
- Cenário premium (preço para clientes que valorizam resultado, não hora)

**3. Análise de posicionamento**
Em qual faixa do mercado você está e o que precisa para subir de faixa.

**4. Como apresentar o preço ao cliente**
Script de 3-4 frases para comunicar o valor com confiança, sem parecer que está pedindo um favor.

**5. O que incluir ou retirar para ajustar o preço**
Se o cliente pedir desconto, o que você pode remover do escopo em vez de baixar o preço.
```

## Exemplo de uso

### Input
- Projeto: Identidade visual completa (logo, paleta, tipografia, manual básico)
- Estimativa: 25 horas
- Meta mensal: R$ 5.000
- Projetos/mês: 3
- Cidade: São Paulo
- Experiência: Iniciante
- Cliente: Pequeno negócio local
