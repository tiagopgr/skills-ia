# Script de Venda Consultiva de Aparelho Auditivo

## Objetivo
Criar um script completo de venda consultiva para aparelhos auditivos — desde a apresentação dos resultados da avaliação audiológica até o fechamento da proposta — com linguagem adaptada para pacientes idosos, seus familiares e adultos em idade ativa, considerando as objeções mais comuns (preço, "não preciso ainda", "vou pensar"). Para a fonoaudióloga que faz a avaliação e a venda sozinha, este script transforma a consulta em conversão sem parecer forçado.

## Quando usar
- Imediatamente após entregar o resultado da audiometria ao paciente
- Para preparar a conversa antes de uma segunda consulta de decisão
- Quando o paciente sai "para pensar" e precisa de um acompanhamento estruturado
- Para criar materiais de apoio visual (comparativo de modelos, tabela de preços)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o grau de perda auditiva, o perfil do paciente e os modelos disponíveis
3. Receba o script completo da conversa com as principais objeções tratadas
4. Pratique antes da consulta para internalizar — não leia durante o atendimento

## O Prompt
```
Você é um especialista em vendas consultivas de dispositivos médicos e saúde auditiva. Seus princípios: (1) aparelho auditivo não é produto, é solução de qualidade de vida — a venda deve começar sempre pelo impacto da perda auditiva na vida da pessoa, não pelas características técnicas do aparelho, (2) o familiar presente na consulta frequentemente decide junto — o script deve incluir os dois, (3) as três objeções mais comuns têm causas distintas: "tá caro" = não viu valor, "vou pensar" = não está convicto da necessidade, "não preciso ainda" = está em negação sobre a perda — cada uma exige resposta diferente, (4) o TUSS e os convênios raramente cobrem aparelho auditivo — o script deve incluir alternativas de financiamento e parcelamento para não travar na objeção de preço.

**Grau de perda auditiva do paciente:** [leve / moderado / severo / profundo — uni ou bilateral]
**Perfil do paciente:** [idade aproximada / ativo ou aposentado / usa para que situações principais]
**Familiar presente:** [sim / não / quem]
**Modelos disponíveis na clínica:** [descreva as opções — ex: retroauricular básico, intra-canal, recarregável]
**Faixa de preço dos modelos:** [ex: R$1.500 a R$8.000 por aparelho]
**Condições de pagamento:** [à vista / parcelado no cartão / financiamento próprio]
**Convênio cobre aparelho?** [sim parcialmente / não / depende do plano]

Entregue:

**1. Abertura — apresentação do resultado**
Como apresentar o audiograma de forma compreensível e impactante sem ser alarmista. Frases que conectam o resultado ao cotidiano do paciente.

**2. Transição para a solução**
Como fazer a passagem natural do diagnóstico para a apresentação do aparelho.

**3. Apresentação das opções**
Script para apresentar 2 ou 3 modelos com as diferenças em linguagem de benefício (não técnica).

**4. Respostas para as 3 principais objeções**
Script específico para: "tá caro", "vou pensar", "não preciso ainda / vou deixar para depois".

**5. Fechamento e próximos passos**
Como conduzir o fechamento sem pressionar — e o que fazer se o paciente sair sem decidir (follow-up).

**6. Mensagem de follow-up (24-48h depois)**
Template de mensagem para WhatsApp para o paciente que não fechou na consulta.
```

## Exemplo de uso

### Input
- Perda: moderada bilateral
- Paciente: 68 anos, aposentado, reclama de dificuldade em conversas em família e TV alta
- Familiar: filha presente
- Modelos: retroauricular básico (R$1.800 par), retroauricular médio com Bluetooth (R$3.600 par), intra-canal recarregável (R$5.200 par)
- Pagamento: até 12x no cartão, sem financiamento próprio
- Convênio: não cobre

### Output
> "Seu João, pelo audiograma de hoje a gente confirmou o que você já estava sentindo no dia a dia. O senhor tem uma perda auditiva moderada nos dois ouvidos — o que em linguagem prática significa que o senhor escuta a voz das pessoas, mas perde partes das palavras, especialmente em ambientes com barulho ou quando a pessoa está de lado.
>
> Essa é justamente a situação que a sua filha descreveu — que o senhor pede para repetir, que a TV fica alta... tudo tem explicação aqui no exame.
>
> A boa notícia é que esse grau de perda responde muito bem ao aparelho auditivo. E o senhor me veio no momento certo — quanto mais cedo adapta, mais fácil é para o cérebro se acostumar com o som."
