# Copy para Landing Page de Serviço Contábil

## Objetivo
Criar a copy completa de uma landing page para serviços contábeis — do headline principal ao CTA final — com estrutura persuasiva que qualifica o visitante, apresenta o valor do serviço, elimina objeções e converte em lead ou agendamento, para que o escritório tenha uma página de vendas profissional que trabalha 24h por dia mesmo quando o dono está atendendo clientes.

## Quando usar
- Para criar a página de vendas de um serviço novo (BPO, consultoria tributária, pacote premium)
- Quando a página atual do escritório tem tráfego mas não gera contatos
- Para criar uma página específica para uma campanha de tráfego pago
- Ao reformular o site do escritório com foco em conversão

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o serviço, o público e o objetivo da página
3. Receba a copy completa por seção, pronta para passar ao designer ou montar no Webflow/Wix
4. Teste com tráfego real e ajuste com base nos dados de conversão

## O Prompt
```
Você é um copywriter especializado em landing pages para serviços B2B e contábeis. Seus princípios: (1) headline de landing page contábil não vende serviço — vende o resultado que o empresário quer ter, (2) a estrutura que converte é: dor identificada → solução específica → prova → oferta → CTA — nessa ordem, (3) objeção não declarada mata a conversão — toda landing page precisa antecipar e responder as 3 principais dúvidas do visitante, (4) CTA específico converte mais — "Quero meu diagnóstico gratuito" bate "Entre em contato".

**Serviço:** [ex: BPO financeiro / consultoria tributária / contabilidade para e-commerce]
**Público-alvo:** [ex: donos de e-commerce / prestadores de serviço / empresas em crescimento]
**Principal dor do público:** [ex: não sabe se está lucrando / paga imposto demais / contador não responde]
**Principal resultado entregue:** [ex: DRE e fluxo de caixa todo mês / redução de imposto dentro da lei]
**Prova social disponível:** [ex: X clientes ativos / Y anos de experiência / cases de resultado]
**Oferta de entrada:** [ex: diagnóstico gratuito / consulta de 30 min / análise tributária grátis]
**CTA:** [ex: agendar / preencher formulário / chamar no WhatsApp]

Entregue:

**1. Headline + subheadline**
3 opções de headline com subheadline para teste A/B.

**2. Seção de dor (acima da dobra)**
Texto que identifica o problema do visitante e cria identificação imediata.

**3. Seção de solução**
Como o serviço resolve a dor — em linguagem de resultado, não de processo.

**4. Seção de prova social**
Estrutura de depoimentos e números — com sugestão de formato para cada tipo de prova disponível.

**5. Seção de objeções (FAQ)**
4 perguntas e respostas que eliminam as principais dúvidas antes do CTA.

**6. CTA final**
Texto do botão + frase de suporte + garantia (se houver).
```

## Exemplo de uso

### Input
- Serviço: BPO Financeiro para PMEs
- Público: Donos de PMEs com faturamento R$ 500k–R$ 5M
- Dor: Não sabe se está lucrando, toma decisão no escuro
- Resultado: DRE + fluxo de caixa + reunião mensal de análise
- Prova: 35 clientes ativos, 7 anos de experiência
- Oferta: Diagnóstico financeiro gratuito de 45 min
- CTA: Agendar pelo WhatsApp

### Output
"Você sabe exatamente quanto seu negócio lucrou este mês?"
*Subheadline:* "Enquanto a maioria dos donos de empresa descobre o resultado no final do ano — nossos clientes tomam decisão com número na mão todo mês."
