# Gerador de Conceito de Marca

## Objetivo
Criar o conceito estratégico completo de uma marca — arquétipo, personalidade, posicionamento, tom de voz e direção criativa — antes de abrir qualquer software de design, para parar de criar logos no achismo e começar a tomar decisões visuais com fundamento estratégico que o cliente entende e aprova.

## Quando usar
- No início de qualquer projeto de branding ou identidade visual
- Quando o cliente não sabe o que quer e você precisa construir junto com ele
- Para justificar suas escolhas criativas de forma profissional
- Para criar uma apresentação de conceito antes da identidade visual

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com as informações do briefing do cliente
3. Receba o conceito estratégico completo da marca
4. Use como base para criar a identidade visual e para apresentar ao cliente

## O Prompt
```
Você é um estrategista de branding com especialização em posicionamento e arquétipos de marca. Seus princípios: (1) design sem estratégia é decoração, (2) o conceito de marca deve ser aprovado antes de qualquer pixel ser criado, (3) o arquétipo é a bússola que guia todas as decisões visuais e verbais, (4) o conceito precisa ser simples o suficiente para o cliente entender e forte o suficiente para o designer seguir.

Crie o conceito estratégico completo da marca com base nessas informações do briefing:

**Nome da empresa/marca:** [nome]
**O que a empresa faz:** [descrição do negócio]
**Público-alvo:** [quem são os clientes]
**Principal diferencial:** [o que a empresa faz que os concorrentes não fazem bem]
**Adjetivos que o cliente usou para descrever a marca:** [palavras que o cliente disse — ex: moderna, acolhedora, séria]
**Marcas que o cliente admira:** [referências citadas]
**Marcas que o cliente NÃO quer parecer:** [o que deve ser evitado]

Entregue:

**1. Essência da marca (1 frase)**
A ideia central que tudo na marca deve comunicar. A "alma" da marca em uma frase.

**2. Arquétipo principal (e secundário se houver)**
Qual dos 12 arquétipos de Jung melhor representa esta marca, com explicação de por que e como ele se manifesta neste negócio específico.

**3. Personalidade da marca (5 atributos)**
Cinco adjetivos que definem o caráter da marca, com uma frase explicando como cada um se expressa na comunicação.

**4. Tom de voz**
Como a marca fala: 3 exemplos de como ela se comunicaria (e 3 exemplos do que ela NUNCA diria).

**5. Direção criativa visual**
Orientações para: paleta de cores (com justificativa estratégica), estilo tipográfico, estilo de fotografia/ilustração e referências visuais alinhadas ao arquétipo.

**6. Conceito criativo (tagline interna)**
Frase-conceito interna que guia o designer — pode ser diferente da tagline pública.
```

## Exemplo de uso

### Input
- Marca: Studio Bloom
- O que faz: Estúdio de pilates e bem-estar feminino
- Público: Mulheres de 30-50 anos, classe B/A, que buscam saúde e autocuidado
- Diferencial: Ambiente acolhedor, turmas pequenas, atenção individualizada
- Adjetivos do cliente: acolhedora, elegante, tranquila
- Admira: Lululemon, Reserva
- Não quer parecer: clínica médica, academia popular
