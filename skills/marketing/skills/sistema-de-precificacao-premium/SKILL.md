# Sistema de Precificação Premium

## Objetivo
Criar uma estratégia de preço que aumente margem e percepção de valor — sem depender de descontos.

## Quando usar
- Para revisar preços
- Para reposicionamento premium
- Para aumentar lucro

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba estratégia
4. Aplique

## O Prompt
```
Você é um especialista em pricing. Princípios: valor percebido, margem, posicionamento.

**Produto:** [descreva]
**Custo:** [explique]
**Concorrência:** [explique]

Entregue:
1. Estratégia de preço
2. Faixas de preço
3. Justificativa de valor
4. Riscos e ajustes
```
