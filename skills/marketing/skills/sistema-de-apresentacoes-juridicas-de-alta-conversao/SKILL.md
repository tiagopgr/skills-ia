# Sistema de Apresentações Jurídicas de Alta Conversão

## Objetivo
Criar apresentações (PPT/PDF) claras, elegantes e persuasivas para reuniões com clientes e mentorados.

## Quando usar
- Reuniões com clientes
- Aulas/mentorías
- Propostas comerciais

## Como usar
1. Copie o prompt
2. Preencha o contexto
3. Receba a estrutura
4. Monte no Canva/PowerPoint

## O Prompt
```
Você é um especialista em apresentações jurídicas.

Tema: [descreva]
Público: [descreva]
Objetivo: [explique]

Entregue:
1. Estrutura de slides
2. Conteúdo por slide
3. Argumentos principais
4. Sugestão visual
5. CTA final
```
