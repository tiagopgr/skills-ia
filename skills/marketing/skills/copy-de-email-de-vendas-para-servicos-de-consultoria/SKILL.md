# Copy de Email de Vendas para Serviços de Consultoria

## Objetivo
Criar emails de venda para serviços de consultoria, coaching e mentoria — com abordagem consultiva, linguagem de alto nível e estrutura de direct response — que geram respostas e agendamentos reais sem soar como email de disparo em massa, sem pressão excessiva e sem destoar do posicionamento de especialista.

## Quando usar
- Para enviar para leads que demonstraram interesse mas não agendaram call
- Para reativar contatos que participaram de uma aula gratuita, webinar ou evento
- Para fazer uma oferta direta de serviço para lista de email qualificada
- Quando quiser criar uma sequência de nutrição que termina em convite para call
- Para acompanhar uma proposta enviada que não recebeu resposta

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto: para quem vai mandar, o serviço ofertado e qual foi o último contato com esse lead
3. Receba os emails completos com assunto, copy e CTA
4. Adapte com seu estilo antes de enviar — emails de serviço de alto valor precisam soar como você

## O Prompt
```
Você é uma especialista em email marketing para consultores e especialistas de alto valor. Seus princípios: (1) email de serviço premium não pode parecer automação — precisa soar como um email escrito por uma pessoa, para uma pessoa, (2) a primeira linha do email é lida no preview — precisa criar curiosidade ou reconhecimento imediato sem revelar tudo, (3) email de consultoria vende uma conversa, não um produto — o objetivo é o agendamento, não a compra imediata, (4) menos urgência artificial, mais relevância real — a razão para agir agora precisa ser verdadeira.

Crie os emails com os seguintes dados:

**Serviço ofertado:** [consultoria / coaching / mentoria — com breve descrição]
**Ticket:** [valor — para calibrar o tom e a abordagem]
**Para quem vai mandar:** [ex: leads da lista que se inscreveram para uma aula gratuita / contatos que pediram proposta mas não responderam / lista fria qualificada]
**Último contato com esse lead:** [ex: assistiram a uma live há 2 semanas / receberam proposta e não responderam / nunca houve contato — abordagem fria]
**O principal problema que o lead enfrenta:** [o que o leva a buscar ajuda]
**Por que agora é um bom momento para agir:** [motivo real de urgência ou relevância]
**CTA desejado:** [agendar call / responder ao email / preencher formulário]

Entregue a sequência completa:

**Email 1 — Abertura de valor**
Assunto + corpo completo: Começa com insight ou pergunta relevante para o problema do lead — não com oferta. Termina com CTA suave.

**Email 2 — Caso de resultado (enviado 3 dias depois)**
Assunto + corpo: História de cliente (anônimo) com resultado concreto. O lead se vê no personagem. CTA para call.

**Email 3 — Pergunta direta (enviado 5 dias depois)**
Assunto + corpo: Email curto, pergunta direta sobre o problema do lead. CTA de resposta ao email — baixíssima fricção.

**Email 4 — Último contato (enviado 7-10 dias depois)**
Assunto + corpo: "Vou parar de incomodar" — fecha o ciclo com leveza, deixa a porta aberta, sem pressão.

Para cada email:
- Assunto (máx 50 caracteres, sem clickbait)
- Pré-header (60-80 caracteres)
- Corpo completo
- CTA
- Tempo de envio após o anterior
```

## Exemplo de uso

### Input
- Serviço: Consultoria de marketing de 3 meses para coaches (R$4.500)
- Para quem: Pessoas que se inscreveram para uma aula gratuita sobre "como atrair clientes com conteúdo" — assistiram mas não tomaram ação
- Último contato: Aula gratuita há 5 dias
- Problema do lead: Tem audiência mas não converte em clientes
- Motivo de urgência: Está abrindo apenas 3 vagas para o próximo trimestre
- CTA: Agendar call de diagnóstico gratuita (30 min)

### Output
Assunto: "Uma pergunta sobre a aula de quinta"
Pré-header: "Você chegou até aqui — isso já diz algo sobre onde está"

Corpo: "Oi [nome],

Você participou da aula sobre conteúdo e clientes na quinta-feira.

Quero te fazer uma pergunta direta: no final da aula, qual foi o pensamento que ficou?

Pergunto porque as pessoas que ficam até o final geralmente estão num ponto específico — sabem que o conteúdo que fazem tem qualidade, mas algo na conversão não está funcionando.

Se for esse o caso, a aula provavelmente te deu clareza sobre o problema — mas não a solução completa. Faz sentido?

Nos próximos dias vou compartilhar mais sobre como coaches na sua situação resolveram exatamente isso. Mas se quiser conversar antes, é só responder esse email.

— Sara"

CTA: Responder ao email
Envio: 5 dias após a aula
