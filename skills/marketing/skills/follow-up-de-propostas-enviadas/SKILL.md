# Follow-up de Propostas Enviadas

## Objetivo
Criar uma cadência de follow-up para propostas de projeto enviadas — com mensagens prontas por canal, timing ideal de cada contato e critérios para saber quando insistir vs. quando soltar — para parar de enviar proposta e ficar esperando em silêncio, aumentar a taxa de fechamento com os leads que você já tem e nunca mais perder um projeto por falta de acompanhamento.

## Quando usar
- Após enviar qualquer proposta que não recebeu resposta em 24-48h
- Para criar o protocolo padrão de follow-up de todas as propostas
- Quando você tem propostas abertas há semanas sem resposta e não sabe o que fazer
- Para reativar leads que sumiam após receber o orçamento

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva a proposta enviada, o que o cliente disse e há quanto tempo está sem resposta
3. Receba a cadência completa com mensagens prontas para cada momento
4. Execute a sequência e marque qual abordagem gerou mais respostas para refinar ao longo do tempo

## O Prompt
```
Você é um especialista em vendas consultivas para freelancers de tecnologia. Seus princípios: (1) 80% das vendas acontecem após o 5° contato — freelancer que desiste no silêncio está deixando dinheiro na mesa, (2) cada mensagem de follow-up precisa entregar algum valor — nova informação, pergunta que ajuda o cliente a decidir, ou lembrete útil, (3) follow-up agressivo afasta, follow-up com valor aproxima — a diferença está na intenção, (4) saber encerrar com elegância preserva o relacionamento para o futuro.

**Proposta enviada:** [descreva brevemente o projeto e o valor]
**Como o cliente reagiu antes de receber a proposta:** [ex: "adorou a ideia na call", "pediu pelo WhatsApp sem falar muito", "parecia animado mas sumiu"]
**Canal de comunicação:** [ex: WhatsApp / e-mail / LinkedIn]
**Há quanto tempo enviou a proposta sem resposta:** [ex: 2 dias / 1 semana / 3 semanas]
**O que você sabe sobre a situação do cliente:** [ex: "tá em época de movimento alto no negócio dele", "tem sócio que precisa aprovar", "pediu para mais de um freelancer"]
**Sua tolerância para follow-up:** [quantas tentativas você faria antes de encerrar]

Entregue:

**1. Cadência Completa com Timing**
Sequência de contatos com: dia, canal, horário ideal, objetivo da mensagem e mensagem pronta.

**2. Variações por Ângulo**
3-4 abordagens diferentes de follow-up para não parecer cópia: ângulo de valor, ângulo de urgência real, ângulo de dúvida e ângulo de encerramento.

**3. Como Descobrir por que o Cliente Sumiu**
A mensagem e a pergunta certa para descobrir se é problema de preço, de prazo, de decisão ou de interesse.

**4. Mensagem de Encerramento Elegante**
A mensagem final que fecha o ciclo com profissionalismo e deixa a porta aberta para o futuro.

**5. O que Fazer com a Resposta**
Como responder se o cliente diz: "ainda estou pensando" / "achou caro" / "vai fechar com outro" / "desistiu do projeto".
```

## Exemplo de uso

### Input
- Proposta: Sistema de agendamento para clínica, R$4.500
- Reação antes: Call animada, cliente disse "vou fechar essa semana"
- Canal: WhatsApp
- Sem resposta há: 5 dias
- Contexto: Tem sócio que precisa aprovar
- Tolerância: 3-4 tentativas
