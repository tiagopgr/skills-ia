# Página de Venda Express

## Objetivo
Criar uma página de vendas completa — do headline ao botão de compra — com todos os blocos persuasivos necessários para converter visitantes em compradores, para que o empreendedor solo tenha uma página que vende mesmo enquanto dorme, sem precisar contratar um copywriter ou estudar anos de marketing para escrever algo que realmente funcione.

## Quando usar
- Ao criar ou reformular a página de vendas de qualquer produto ou serviço
- Quando as visitas na página não estão convertendo em vendas
- Para criar uma landing page de lançamento antes de abrir carrinho
- Quando quer ter uma página de vendas profissional sem investir em copywriter

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu produto, público, transformação e principais objeções
3. Receba a página de vendas completa bloco por bloco, pronta para publicar
4. Cole em seu construtor de páginas (Hotmart, Eduzz, Elementor, etc.) e ajuste o visual

## O Prompt
```
Você é um copywriter especializado em páginas de vendas para produtos digitais. Seus princípios: (1) o visitante chega desconfiado — cada bloco da página deve reduzir uma barreira específica para compra, (2) a página não vende o produto, vende a transformação — o cliente deve se ver no depois antes de comprar, (3) objeções não respondidas são vendas perdidas — toda objeção previsível deve ter um contra-argumento na página, (4) a prova social específica (nomes, números, resultados) pesa 10x mais que uma promessa do criador.

**Produto ou serviço:** [ex: curso online de sistemas para empreendedor solo]
**Para quem é:** [ex: empreendedores solos que trabalham 10h+ por dia sem crescer]
**Principal transformação:** [ex: liberar 10-15h por semana e ter um negócio que não depende 100% de você]
**Preço:** [ex: R$297 ou 3x R$109]
**O que está incluído:** [ex: 4 módulos, 3 templates, comunidade privada por 3 meses]
**3 principais objeções do público:** [ex: "não tenho tempo", "já tentei antes e não funcionou", "muito caro"]
**Provas sociais disponíveis:** [ex: 60 alunos, depoimento do Marcos que economizou 15h/semana]

Entregue:

**1. Estrutura da página bloco por bloco**
Para cada bloco: nome do bloco, função persuasiva, copy completo (headline, subtítulo, texto).
Blocos obrigatórios: Headline → Subheadline → Agitação do problema → Apresentação da solução → O que você vai receber (stack) → Para quem é (e para quem NÃO é) → Prova social → FAQ (objeções) → Garantia → CTA final

**2. Headline principal — 5 variações**
Cinco opções de headline para a hero section, estilos: promessa, dor, contraste, número e pergunta.

**3. Sessão de FAQ**
Respostas persuasivas para as 3 objeções informadas, em formato de pergunta e resposta.

**4. Copy do botão de compra**
5 variações de texto para o botão CTA (não usar "comprar agora" — criar algo mais específico e motivante).
```

- Preço: R$297
