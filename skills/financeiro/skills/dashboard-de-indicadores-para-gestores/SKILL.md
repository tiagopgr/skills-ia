# Dashboard de Indicadores para Gestores

## Objetivo
Criar a estrutura completa de um dashboard de KPIs — com os indicadores certos para cada área do negócio, fórmulas de cálculo e frequência de atualização — para que gestores de pequenas equipes acompanhem o desempenho do negócio de forma visual e rápida, sem precisar de um analista ou sistema caro. O dashboard certo transforma dados dispersos em clareza para decisão.

## Quando usar
- Para criar o primeiro painel de indicadores do negócio e sair do acompanhamento por intuição
- Ao querer ter visibilidade de financeiro, operação e equipe em um só lugar
- Para definir quais métricas realmente importam e parar de acompanhar tudo ao mesmo tempo
- Quando quiser apresentar o desempenho do negócio para sócios com dados organizados

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe as áreas do negócio, os dados que você já tem disponíveis e o que mais precisa monitorar
3. Receba a estrutura completa com KPIs, fórmulas e modelo de dashboard para montar em planilha
4. Monte no Google Sheets ou Excel e atualize semanalmente com a equipe

## O Prompt
```
Você é um especialista em business intelligence e gestão por indicadores para pequenas e médias empresas. Seus princípios: (1) menos é mais em dashboard — 5 indicadores acompanhados com atenção valem mais do que 30 indicadores que ninguém olha, (2) cada KPI deve ter um dono (quem é responsável por mover esse número), uma meta e uma frequência de atualização — sem isso, é decoração, (3) indicadores lagging (resultado passado) precisam ser balanceados com indicadores leading (sinais antecipados) — um mostra onde você chegou, o outro mostra para onde está indo, (4) dashboard que ninguém atualiza é porque é difícil demais ou não foi adotado culturalmente — simplifique ao máximo.

Crie a estrutura do dashboard de indicadores com base nas informações abaixo:

**Áreas do negócio a monitorar:** [ex: financeiro / vendas / operação / equipe / clientes]
**Principais objetivos do negócio agora:** [ex: crescer receita / reduzir custos / melhorar qualidade / aumentar clientes]
**Dados que você já tem disponíveis:** [ex: faturamento mensal / número de clientes / horas trabalhadas / etc.]
**Tamanho da equipe:** [quantas pessoas e funções principais]
**Ferramentas que usa:** [ex: planilha / ERP / CRM / sistema próprio]
**Frequência de revisão desejada:** [semanal / quinzenal / mensal]

Entregue:

**1. KPIs recomendados por área**
Para cada área informada: lista de 3-5 KPIs com nome, o que mede, fórmula de cálculo, frequência e referência de meta (se houver benchmark disponível).

**2. Os 5 KPIs mais críticos (North Star)**
Os 5 indicadores que, se monitorados, dão uma visão clara da saúde do negócio. Justificativa para cada escolha.

**3. Estrutura do dashboard**
Layout recomendado: quais KPIs em destaque (visão rápida), quais em tabela (detalhe), quais em gráfico (tendência).

**4. Modelo de tabela para planilha**
Template pronto com colunas: KPI | Meta | Realizado | Variação | Status (🟢🟡🔴) | Responsável | Última atualização.

**5. Rotina de atualização**
Quem atualiza o quê, quando e em quanto tempo — para que o dashboard esteja sempre atual.

**6. Protocolo de alerta**
Quando um indicador entra em zona de atenção (amarelo) ou zona de risco (vermelho), qual é a ação imediata.
```

## Exemplo de uso

### Input
- Áreas: Financeiro + Vendas + Equipe
- Objetivos: Aumentar receita 20% e reduzir custo com equipe 15%
- Dados disponíveis: Faturamento mensal, horas trabalhadas por pessoa, número de propostas enviadas/fechadas, despesas por categoria
- Equipe: 4 pessoas (1 comercial, 2 operacionais, 1 administrativo)
- Ferramenta: Google Sheets
- Frequência: Semanal
