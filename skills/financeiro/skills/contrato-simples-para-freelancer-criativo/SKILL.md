# Contrato Simples para Freelancer Criativo

## Objetivo
Gerar um contrato de prestação de serviços criativos — claro, direto e sem juridiquês desnecessário — para proteger o freelancer de branding e design de calotes, revisões infinitas, uso indevido de material e desentendimentos sobre escopo, sem precisar de advogado para começar.

## Quando usar
- Antes de iniciar qualquer projeto pago de branding ou design
- Quando o cliente diz "pode ir fazendo que a gente acerta depois"
- Para ter segurança jurídica básica sem pagar por um contrato personalizado
- Para adaptar para diferentes tipos de projeto

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do projeto e do cliente
3. Receba o contrato completo pronto para adaptar e assinar
4. Use o DocuSign, ClickSign ou imprima para assinar

## O Prompt
```
Você é um especialista em contratos para profissionais criativos e freelancers. Seus princípios: (1) um contrato claro previne 90% dos conflitos — a maioria dos problemas vem de expectativas diferentes, não de má-fé, (2) linguagem simples é mais eficaz que juridiquês — cliente que entende o que assinou respeita mais, (3) o contrato protege os dois lados, não só o prestador, (4) escopo detalhado é a cláusula mais importante de qualquer contrato criativo.

Gere um contrato de prestação de serviços para o seguinte projeto:

**Nome do prestador (você):** [seu nome completo ou razão social]
**CPF ou CNPJ do prestador:** [seu documento]
**Nome do cliente:** [nome completo ou razão social]
**CPF ou CNPJ do cliente:** [documento do cliente]
**Serviço contratado:** [ex: criação de identidade visual completa]
**Entregáveis (escopo detalhado):** [liste tudo que está incluso]
**Valor total:** [valor em reais]
**Forma de pagamento:** [ex: 50% entrada + 50% entrega]
**Prazo de entrega:** [data ou número de dias úteis]
**Número de rodadas de revisão inclusas:** [ex: 2 rodadas]
**O que NÃO está incluso:** [o que o cliente não pode pedir sem pagar a mais]

Entregue:

**1. Contrato completo**
Com as seguintes cláusulas em linguagem simples:
- Identificação das partes
- Objeto do contrato (escopo detalhado)
- Valor e forma de pagamento
- Prazo de entrega e condições
- Revisões (quantas estão inclusas e o que acontece se ultrapassar)
- Propriedade intelectual (quando os direitos são transferidos)
- Cancelamento e reembolso
- Aprovação e aceite final
- Foro

**2. E-mail de encaminhamento do contrato**
Texto curto e profissional para enviar ao cliente junto com o contrato.

**3. Cláusula extra: atraso de pagamento**
Texto específico sobre multa por atraso, para adicionar ao contrato se quiser.
```

## Exemplo de uso

### Input
- Prestador: Gui Furtado, CPF 000.000.000-00
- Cliente: Café Raiz LTDA, CNPJ 00.000.000/0001-00
- Serviço: Identidade visual completa
- Entregáveis: Logo em 3 versões, paleta de cores, tipografia selecionada, manual de marca básico, arquivos finais em AI, PDF e PNG
- Valor: R$ 2.500
- Pagamento: 50% na assinatura + 50% na entrega
- Prazo: 21 dias úteis após pagamento da entrada
- Revisões: 2 rodadas inclusas
- Não incluso: aplicações em materiais físicos, social media, animações
