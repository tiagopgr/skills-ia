# Template de Contratos e Documentos Prontos para Revisão

## Objetivo
Criar contratos e documentos estruturados com rapidez — deixando para a revisão final apenas os ajustes críticos, sem precisar começar tudo do zero.

## Quando usar
- Ao preparar contratos recorrentes
- Para ganhar velocidade na produção documental
- Quando quer reduzir tempo de elaboração e manter padrão
- Para organizar uma base reaproveitável de documentos

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tipo de documento e o contexto
3. Receba a minuta ou estrutura pronta para revisão
4. Ajuste os pontos variáveis e finalize

## O Prompt
```
Você é um especialista em estruturação de contratos e documentos empresariais. Seus princípios: (1) documento bom reduz dúvida, (2) estrutura padronizada economiza tempo, (3) clareza evita retrabalho, (4) revisão final precisa focar só no que importa.

**Tipo de documento:** [ex: contrato, proposta, termo, aditivo]
**Contexto:** [explique]
**Partes envolvidas:** [descreva]
**Objetivo do documento:** [explique]
**Condições principais:** [prazos, valores, obrigações, restrições]
**Pontos sensíveis:** [explique]

Entregue:

**1. Estrutura completa do documento**
Mostre as seções em ordem lógica.

**2. Texto-base pronto para adaptação**
Redija uma versão utilizável da minuta.

**3. Campos que precisam ser personalizados**
Destaque o que muda em cada caso.

**4. Pontos que exigem revisão jurídica ou estratégica final**
Mostre o que precisa de atenção antes do uso.

**5. Checklist final de conferência**
Crie uma lista simples para revisar antes de enviar.

**6. Sugestão de padronização**
Explique como transformar isso em modelo recorrente.
```

## Exemplo de uso

### Input
- Contrato recorrente
- Pouco tempo para redigir do zero
- Objetivo de revisar só o essencial

### Output
Documento estruturado, campos variáveis, pontos sensíveis e checklist final.
