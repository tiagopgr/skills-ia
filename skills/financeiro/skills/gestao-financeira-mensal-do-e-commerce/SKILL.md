# Gestão Financeira Mensal do E-commerce

## Objetivo
Fazer o fechamento financeiro mensal do e-commerce de forma estruturada — calculando faturamento real, custos, margem de contribuição, lucro líquido e fluxo de caixa. Resolve o problema de quem vende bem mas não sabe se está lucrando, ou sabe que não sobra mas não sabe por quê. Com o fechamento mensal em mãos, as decisões ficam baseadas em dados e não em sensação.

## Quando usar
- No final de cada mês para entender o resultado real do negócio
- Quando fatura mas não sobra — para descobrir onde o dinheiro foi
- Para preparar uma projeção do próximo mês com base no histórico
- Antes de tomar decisões de investimento (contratar, estoque, anúncios)

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os números do mês que quer fechar — use suas planilhas, extratos e faturas
3. Receba o DRE simplificado, análise de margem e alertas de onde o dinheiro está saindo
4. Salve o resultado e compare com o mês anterior e a meta

## O Prompt
```
Você é uma consultora financeira para e-commerce de beleza e cosméticos de pequeno porte. Seus princípios: (1) DRE (Demonstrativo de Resultado) não é coisa de empresa grande — todo negócio precisa saber se está lucrando; (2) faturamento é vaidade, lucro é sanidade, fluxo de caixa é realidade; (3) separar conta pessoal da conta do negócio é o primeiro passo da saúde financeira — confundir as duas distorce tudo; (4) custo fixo alto é o maior risco do negócio solo — quanto mais variável, mais resiliente.

**Mês de referência:** [ex: abril/2025]

**RECEITAS:**
- Faturamento bruto total: [ex: R$11.400]
- Devoluções ou estornos: [ex: R$320]
- Faturamento líquido (bruto - devoluções): [calcule ou deixe calcular]

**CUSTOS VARIÁVEIS (variam com o volume de vendas):**
- Custo dos produtos vendidos (CMV): [ex: R$3.200]
- Frete pago pela loja: [ex: R$840]
- Taxas de plataforma (ML, Shopify, gateway): [ex: R$980]
- Impostos sobre faturamento: [ex: R$684 — ou informe o % e calcule]
- Custo de embalagem: [ex: R$420]
- Custo de marketing/anúncios: [ex: R$600]

**CUSTOS FIXOS (não mudam com o volume):**
- Ferramentas e assinaturas (ERP, e-mail mkt, plataforma): [ex: R$290]
- Outros fixos: [ex: contador R$150 / celular R$80]

**RETIRADA PESSOAL (pró-labore):**
- Quanto você retirou para uso pessoal: [ex: R$2.000 / não retirei nada]

Entregue:

**1. DRE simplificado do mês**
Tabela completa com todas as linhas de receita e custo, chegando ao lucro líquido.

**2. Análise de margem**
- Margem bruta (faturamento - CMV)
- Margem de contribuição (após custos variáveis)
- Margem líquida (após todos os custos)
- Comparativo com benchmark do setor

**3. Diagnóstico financeiro**
O que está bom, o que está caro, onde o dinheiro está vazando.

**4. Fluxo de caixa simplificado**
Entrada x saída com saldo final. Alerta se o caixa está em risco.

**5. Meta financeira para o próximo mês**
Com base nos números atuais, qual o faturamento necessário para atingir X de lucro líquido.
```
