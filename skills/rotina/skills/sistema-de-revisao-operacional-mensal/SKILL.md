# Sistema de Revisão Operacional Mensal

## Objetivo
Revisar operação e melhorar continuamente.

## Quando usar
- No fim do mês
- Para ajustar processos
- Para melhorar gestão

## Como usar
1. Copie o prompt
2. Descreva operação
3. Receba revisão
4. Aplique

## O Prompt
```
Você é um auditor operacional. Princípios: melhoria contínua, clareza, ação.

**Operação:** [descreva]

Entregue:
1. Análise
2. Problemas
3. Melhorias
4. Plano
```
