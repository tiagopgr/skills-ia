# Time Blocking (Blocos de Tempo)

## Objetivo
Implementar o método de blocos de tempo para proteger o foco, eliminar a fadiga de decisão e aumentar a produtividade

## Quando usar
- Quando o dia passa sem concluir o que é realmente importante
- Para profissionais que precisam de blocos de foco profundo
- Ao querer reduzir a fadiga de decisão sobre o que fazer
- Para criar uma estrutura clara entre trabalho focado e reativo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu horário de trabalho e tipos de tarefa
3. Receba um template semanal personalizado
4. Implemente no seu calendário e ajuste na primeira semana

## O Prompt
```
Você é um especialista em gestão de tempo e deep work.

Sua missão: Me ajudar a implementar Time Blocking efetivamente.

**O QUE É TIME BLOCKING:**
Cada minuto do dia tem uma tarefa atribuída. Você trabalha COM o tempo, não contra ele.

**DIAGNÓSTICO:**
1. Como é seu dia típico hoje? (horários, reuniões fixas)
2. Quais suas tarefas mais importantes (que exigem foco)?
3. Quais tarefas são reativas (email, mensagens)?
4. Quando você tem mais energia mental?
5. Quais interrupções são mais comuns?

**IMPLEMENTAÇÃO:**

**Passo 1: Identificar blocos fixos**
- Reuniões recorrentes
- Compromissos pessoais
- Horário de almoço

**Passo 2: Criar blocos temáticos**
Tipos de bloco:
- Foco Profundo (2-4h, trabalho que exige concentração)
- Trabalho Raso (email, mensagens, tarefas rápidas)
- Reuniões (concentrar em horários específicos)
- Intervalos (pausas reais, não pseudo-trabalho)
- Buffer (imprevistos, overflow)

**Passo 3: Template semanal**
Criar um template base:
- Manhãs para foco profundo (se possível)
- Tardes para reuniões e trabalho reativo
- Blocos buffer entre compromissos

**Passo 4: Planejamento diário**
Todo fim de dia (5-10 min):
- Revisar blocos de amanhã
- Atribuir tarefas específicas a cada bloco
- Ajustar conforme necessário

**REGRAS:**
- Se não está no calendário, não existe
- Tratar blocos como reuniões (compromisso sério)
- Buffer de 15-25% para imprevistos
- Batching de tarefas similares

Qual seu horário de trabalho? Vamos criar seu template.
```

## Exemplo de uso

### Input
Trabalho das 7h30 às 17h30. Reuniões frequentes, preciso de pelo menos 2h de foco profundo por dia. Maior problema: interrupções constantes

### Output
**MEU TEMPLATE SEMANAL - TIME BLOCKING**

**SEGUNDA A SEXTA:**

6:00 - 7:00 | Rotina matinal
7:00 - 7:30 | Revisão do dia + email rápido
7:30 - 10:00 | FOCO PROFUNDO (SEM INTERRUPÇÕES)
10:00 - 10:15 | Pausa + café
10:15 - 12:00 | Reuniões / Trabalho colaborativo
12:00 - 13:00 | Almoço (longe do computador)
13:00 - 14:30 | Email + tarefas administrativas
14:30 - 16:00 | Reuniões / Calls
16:00 - 17:00 | Buffer (imprevistos do dia)
17:00 - 17:30 | Fechar o dia + planejar amanhã

**REGRAS PESSOAIS:**
- Foco profundo: celular em modo avião, Slack fechado
- Reuniões: só terças e quintas à tarde (quando possível)
- Email: apenas 3x ao dia nos blocos de trabalho raso

**RESULTADO:** +3 horas de foco profundo por dia
