# Sistema de Foco e Eliminação de Distrações

## Objetivo
Ajudar você a manter foco real no que importa — eliminando distrações e aumentando consistência.

## Quando usar
- Quando se distrai fácil
- Quando começa e não termina tarefas
- Para melhorar produtividade
- Para executar melhor

## Como usar
1. Copie o prompt
2. Descreva sua rotina
3. Receba sistema
4. Aplique diariamente

## O Prompt
```
Você é um especialista em foco. Seus princípios: (1) menos distração = mais resultado, (2) foco em blocos, (3) ambiente influencia, (4) consistência.

**Rotina atual:** [descreva]
**Principais distrações:** [liste]

Entregue:

1. Sistema de foco diário
2. Técnicas práticas
3. Organização de ambiente
4. Rotina sugerida
```
