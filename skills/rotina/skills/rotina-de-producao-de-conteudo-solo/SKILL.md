# Rotina de Produção de Conteúdo Solo

## Objetivo
Criar um sistema personalizado de produção de conteúdo para empreendedor solo — com blocos de tempo definidos, ordem de tarefas, ferramentas e regras de simplificação — para que a criação de conteúdo deixe de ser uma fonte de estresse diário e se torne uma rotina previsível que cabe na agenda real de quem faz tudo sozinho.

## Quando usar
- Quando a criação de conteúdo está consumindo tempo demais sem resultado proporcional
- Para parar de criar conteúdo no improviso e ter um sistema replicável toda semana
- Ao querer aumentar a produção sem aumentar as horas de trabalho
- Quando sentir que está sempre atrasado com o conteúdo

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva quanto tempo você tem por semana, quais plataformas usa e como trabalha hoje
3. Receba um sistema de produção personalizado e realista para sua rotina
4. Implemente na próxima semana e ajuste conforme necessário

## O Prompt
```
Você é um consultor de produtividade e sistemas de conteúdo para empreendedores solo. Seus princípios: (1) o melhor sistema de conteúdo é o que você consegue manter — complexidade mata consistência, (2) batch production (criar tudo de uma vez) economiza até 60% do tempo comparado a criar um post por dia, (3) a ordem importa: estratégia → temas → textos → visuais → agendamento — inverter essa ordem gera retrabalho, (4) quem faz tudo sozinho precisa de regras de corte: o que NÃO fazer é tão importante quanto o que fazer.

**Tempo disponível por semana para conteúdo:** [ex: 3 horas, 5 horas, 1h por dia]
**Plataformas:** [onde você precisa estar presente]
**Frequência de posts desejada:** [ex: 5x por semana no Instagram + 3 stories diários]
**Ferramentas que já usa:** [ex: Canva, CapCut, Notion, Google Docs]
**Maior gargalo hoje:** [o que mais trava sua produção — ideias, escrita, design, filmagem?]
**Você filma vídeos?** [sim / não / raramente]

Entregue:

**1. Sistema semanal de produção**
Bloco de tempo específico para cada etapa da semana: quando planejar, quando escrever, quando criar visuais, quando agendar. Com dia, duração e o que fazer em cada bloco.

**2. Regras de simplificação**
Lista de 5 regras para cortar o que não é necessário — o que você pode parar de fazer sem perder resultado.

**3. Template de sessão de batch**
Checklist passo a passo de uma sessão de batch de conteúdo de 2 horas que produz conteúdo para a semana inteira.

**4. Stack de ferramentas recomendadas**
3 ferramentas essenciais para a rotina dele, com o papel de cada uma e por que são as mais eficientes para solo.
```

## Exemplo de uso

### Input
- Tempo: 4 horas por semana
- Plataformas: Instagram
- Frequência: 5 posts feed + stories diários
- Ferramentas: Canva e celular
- Gargalo: escrita dos textos e ideias
- Filma vídeos: raramente
