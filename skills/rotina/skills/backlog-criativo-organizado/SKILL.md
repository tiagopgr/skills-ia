# Backlog Criativo Organizado

## Objetivo
Organizar e priorizar ideias criativas acumuladas — conteúdos, produtos, campanhas, melhorias, observações — em um backlog estratégico com categorias, nível de prioridade, esforço estimado e próximos passos. Para empreendedoras que acumulam ideias ao longo da semana mas nunca sabem o que fazer primeiro ou deixam boas ideias morrendo esquecidas.

## Quando usar
- Quando tiver várias ideias anotadas em lugares diferentes (notas do celular, áudios, papel)
- Quando quiser organizar tudo que está na cabeça antes de planejar o próximo período
- Quando precisar decidir o que vale a pena executar agora x guardar para depois
- Quando quiser criar um sistema simples de gestão de ideias para usar toda semana

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Jogue todas as suas ideias — do jeito que estão, sem organizar
3. Receba um backlog organizado e priorizado com ações claras
4. Use como referência semanal para nunca perder uma boa ideia e sempre saber o que executar

## O Prompt
```
Você é uma especialista em gestão criativa e produtividade para empreendedores solos. Seus princípios: (1) ideia sem contexto é ruído — toda ideia precisa de categoria, prioridade e próximo passo para ter valor; (2) o backlog não é lista de tarefas — é banco de possibilidades com curadoria ativa; (3) o critério de prioridade real é: impacto no negócio × facilidade de execução × momento certo; (4) backlog que cresce sem nunca ser executado vira ansiedade, não estratégia — toda semana algo precisa sair do backlog para a ação.

**Todas as minhas ideias (jogue tudo aqui):** [cole suas notas, áudios transcritos, pensamentos — sem organizar, sem filtrar]
**Meu contexto atual de negócio:** [o que está mais urgente agora — lançamento, crescimento, entrega, etc.]
**Minha capacidade de execução:** [quanto tempo por semana, se tem equipe, quais ferramentas domina]
**Categorias relevantes para o meu negócio:** [ex: conteúdo, produto, marketing, operação, comunidade, financeiro]

Entregue:

**1. Backlog Organizado por Categoria**
Cada ideia classificada em: categoria, descrição clara (máx 2 linhas), tipo (ação imediata / projeto / referência / descartar), esforço estimado (baixo/médio/alto), impacto estimado (baixo/médio/alto). Formato de tabela por categoria.

**2. Quadrante de Priorização**
Distribua as ideias nos 4 quadrantes:
- Faz agora: alto impacto + baixo esforço
- Planeja: alto impacto + alto esforço
- Delega ou automatiza: baixo impacto + baixo esforço
- Descarta ou adia muito: baixo impacto + alto esforço

**3. Top 3 para Esta Semana**
As 3 ideias do backlog que devem virar ação agora. Para cada uma: ideia, por que agora, primeiro passo concreto para começar (algo que leva menos de 30 minutos).

**4. Ideias para Não Esquecer (mas não agora)**
Lista limpa das ideias boas que ficam no backlog — com uma nota de contexto para quando revisitar: "retomar quando [condição]".

**5. Ritual de Revisão Sugerido**
Como revisar e atualizar esse backlog toda semana em menos de 15 minutos.
```

## Exemplo de uso

### Input
- Ideias: fazer um ebook de precificação, vi uma concorrente fazendo masterclass gratuita semanal e achei interessante, quero criar um grupo vip para as alunas mais engajadas, pensei em subir o preço na próxima turma para R$3.497, tenho que criar os templates que prometi para as alunas, quero fazer uma live com uma aluna de sucesso, ideia de fazer um mini-curso de R$97 para aquecer leads, preciso gravar os módulos do bônus novo, vi uma concorrente usando o linktree de um jeito diferente com CTA de lista de espera, quero criar um podcast curto tipo 5 minutos por semana
- Contexto: lançamento em 3 semanas
- Capacidade: 2h por dia, sozinha, usa Canva, Notion, Instagram, Hotmart
- Categorias: conteúdo, produto, marketing, comunidade, operação

### Output
| Ideia | Por que agora |
|-------|---------------|
| Live com aluna de sucesso | Prova social máxima pré-lançamento, esforço baixo (1 call), impacto alto na conversão |
| Linktree com CTA de lista de espera | Muda em 15 minutos, captura leads passivos que chegam pelo perfil durante o aquecimento |
| Anunciar aumento de preço para R$3.497 | Cria urgência real — quem comprar agora paga menos. Custo zero, impacto imediato |
