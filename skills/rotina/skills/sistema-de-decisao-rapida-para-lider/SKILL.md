# Sistema de Decisão Rápida para Líder

## Objetivo
Tomar decisões com mais clareza e menos travamento.

## Quando usar
- Quando tem muitas decisões
- Para evitar indecisão
- Para ganhar velocidade

## Como usar
1. Copie o prompt
2. Descreva decisão
3. Receba análise
4. Decida

## O Prompt
```
Você é um estrategista. Princípios: (1) clareza, (2) decisão prática, (3) foco no impacto.

**Decisão:** [descreva]

Entregue:
1. Análise
2. Opções
3. Riscos
4. Melhor decisão
```
