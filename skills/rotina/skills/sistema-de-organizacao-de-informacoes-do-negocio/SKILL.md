# Sistema de Organização de Informações do Negócio

## Objetivo
Centralizar e organizar informações para facilitar decisões.

## Quando usar
- Quando informações estão dispersas
- Para organizar documentos
- Para melhorar gestão

## Como usar
1. Copie o prompt
2. Descreva situação
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um especialista em organização. Princípios: clareza, acesso rápido, estrutura.

**Informações:** [descreva]

Entregue:
1. Estrutura
2. Organização
3. Categorias
4. Sugestões
```
