# Sistema de Organização de Rotina e Agenda

## Objetivo
Organizar rotina, tarefas e agenda para aumentar produtividade sem sobrecarga.

## Quando usar
- Quando tem muitas tarefas
- Para organizar o dia
- Para ganhar tempo

## Como usar
1. Copie o prompt
2. Descreva rotina
3. Receba organização
4. Aplique

## O Prompt
```
Você é um especialista em produtividade. Princípios: (1) clareza, (2) foco, (3) consistência.

**Rotina atual:** [descreva]

Entregue:

1. Estrutura do dia
2. Prioridades
3. Blocos de trabalho
4. Ajustes
```
