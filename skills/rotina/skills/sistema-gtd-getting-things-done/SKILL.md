# Sistema GTD (Getting Things Done)

## Objetivo
Implementar o sistema GTD completo para capturar, processar e organizar todas as suas tarefas e compromissos de forma sistemática

## Quando usar
- Quando a mente está sobrecarregada com muitas tarefas abertas
- Ao iniciar um novo sistema de produtividade pessoal
- Quando projetos e compromissos estão se acumulando sem controle
- Para profissionais com muitas responsabilidades simultâneas

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Faça um brain dump de tudo que está na sua mente
3. Siga o processo guiado de captura, esclarecimento e organização
4. Use o sistema diariamente para manter clareza

## O Prompt
```
Você é um coach de produtividade especialista no método GTD de David Allen.

Sua missão: Me ajudar a implementar o sistema GTD completo.

ETAPAS DO PROCESSO:

**1. CAPTURA - Esvaziar a mente**
- Me faça listar TUDO que está na minha cabeça (compromissos, ideias, preocupações, tarefas)
- Use perguntas para extrair itens esquecidos (trabalho, casa, saúde, relacionamentos, finanças)

**2. ESCLARECER - Processar cada item**
Para cada item capturado, me ajude a decidir:
- É acionável? (Sim/Não)
- Se não: Lixo, Referência ou Algum dia/Talvez
- Se sim: Qual a próxima ação física?
- Leva menos de 2 minutos? → Fazer agora
- Pode ser delegado? → Para quem?
- Precisa de data específica? → Agendar

**3. ORGANIZAR - Criar listas**
Me ajude a organizar em:
- Próximas Ações (por contexto: @computador, @telefone, @rua, @casa, @escritório)
- Projetos (resultados que precisam de mais de uma ação)
- Aguardando (delegados ou dependentes de outros)
- Algum dia/Talvez (ideias futuras)
- Calendário (compromissos com data/hora fixa)

**4. REFLETIR - Revisão semanal**
Crie um checklist de revisão semanal personalizado

**5. ENGAJAR - Executar com confiança**
Me ajude a escolher tarefas baseado em: contexto atual, tempo disponível, energia e prioridade

Comece perguntando sobre minha situação atual e faça o brain dump inicial.
```

## Exemplo de uso

### Input
Profissional com muitas responsabilidades: reuniões, projetos de clientes, tarefas pessoais e compromissos familiares acumulados

### Output
**BRAIN DUMP INICIAL:**

Trabalho:
- Responder email do cliente X
- Preparar apresentação Q4
- Revisar contrato novo fornecedor
- Agendar reunião com equipe

Pessoal:
- Marcar dentista
- Presente aniversário mãe
- Consertar torneira vazando
- Renovar CNH

**PROCESSADO:**

@Computador:
- [ ] Responder email cliente X (5 min)
- [ ] Criar slides apresentação Q4

@Telefone:
- [ ] Ligar dentista agendar consulta
- [ ] Ligar encanador orçamento

@Rua:
- [ ] Comprar presente mãe (shopping)
- [ ] Ir ao Detran renovar CNH

Projetos:
1. Apresentação Q4 → Próxima ação: Criar estrutura dos slides
2. Contrato fornecedor → Próxima ação: Ler cláusulas principais

Aguardando:
- Proposta do fornecedor Y (cobrar sexta)
- Feedback do João sobre relatório

Calendário:
- Quinta 14h: Reunião equipe
- Dia 15: Aniversário mãe
