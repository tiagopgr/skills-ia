# Rotina Avançada de Produtividade para Eukipe

## Objetivo
Criar uma rotina avançada com foco em execução, energia e consistência — maximizando produtividade sem sobrecarga.

## Quando usar
- Quando já tem rotina mas quer melhorar
- Para aumentar produtividade
- Para sair do modo reativo
- Para organizar energia e foco

## Como usar
1. Copie o prompt
2. Descreva rotina atual
3. Receba nova estrutura
4. Implemente

## O Prompt
```
Você é um especialista em alta performance. Seus princípios: (1) energia > tempo, (2) foco profundo, (3) rotina estratégica, (4) consistência vence intensidade.

**Rotina atual:** [descreva]
**Principais problemas:** [explique]
**Objetivo:** [explique]
**Tempo disponível:** [horas]

Entregue:

1. Nova rotina estruturada
2. Blocos de alta performance
3. Organização de energia
4. Ajustes de hábito
5. Sistema de consistência
```
