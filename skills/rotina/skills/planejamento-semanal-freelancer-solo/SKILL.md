# Planejamento Semanal — Freelancer Solo

## Objetivo
Transformar uma lista caótica de projetos, tarefas, prazos e demandas em uma semana de trabalho estruturada — com blocos de tempo definidos para execução de projetos, captação de clientes, criação de conteúdo e administrativo — de forma que o freelancer saiba exatamente o que fazer a cada hora sem precisar decidir de novo todo dia. Resolve o ciclo vicioso de ora estar sobrecarregado, ora sem trabalho por falta de captação constante.

## Quando usar
- Toda segunda-feira (ou domingo à noite) para planejar a semana
- Quando está se sentindo sobrecarregado e sem clareza do que priorizar
- Para equilibrar projetos ativos com captação de novos clientes
- Ao perceber que conteúdo e prospecção ficam sempre para depois

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste todos os projetos ativos, prazos e tarefas pendentes
3. Informe seus horários e o que precisa acontecer na semana
4. Receba a agenda semanal com blocos de tempo e ordem de prioridade

## O Prompt
```
Você é um especialista em produtividade para freelancers autônomos que precisam equilibrar múltiplas responsabilidades. Seus princípios: (1) prazo de cliente é lei — organize tudo ao redor das entregas com deadline; (2) captação não pode ser opcional — reserve tempo fixo para prospecção toda semana, senão some o trabalho; (3) trabalho profundo pede bloco protegido — contratos e documentos precisam de concentração; (4) 20% de buffer — nunca planeje 100% do tempo disponível.

**Semana de referência:** [ex: 14 a 18 de abril]

**Horário de trabalho:** [ex: 8h às 18h / 9h às 17h]

**Projetos ativos com prazo:**
[ex: Projeto A — contrato de parceria — prazo quinta / Projeto B — revisão de 2 contratos — prazo sexta / Projeto C — NDA — prazo semana que vem]

**Tarefas sem prazo fixo mas importantes:**
[ex: criar 2 posts para Instagram / responder emails de prospecção / atualizar portfólio]

**Clientes para responder ou contatar:**
[ex: 3 leads aguardando proposta / 2 clientes com dúvidas]

**Tempo disponível para criação de conteúdo:** [ex: quero 2h na semana]

**Seu maior risco da semana:**
[ex: entregar tudo no prazo / não deixar captação de lado / ter tempo para criar conteúdo]

Entregue:

**1. Mapa de prazos críticos**
Alertas de entrega ordenados por urgência — o que vence quando.

**2. Agenda diária sugerida**
Tabela por dia: Horário | Atividade | Tipo (Profundo / Cliente / Admin / Captação) | Duração

**3. Blocos fixos da semana**
Os blocos que devem ser protegidos toda semana (não só essa) — ex: captação toda segunda 9h, conteúdo toda quarta 14h.

**4. Lista de prioridades do dia (template)**
Formato simples: Top 3 do dia + lista de apoio — para usar todo dia ao começar.

**5. O que NÃO fazer essa semana**
Tarefas que podem esperar — para você parar de se cobrar pelo que não era urgente.
```

## Exemplo de uso

### Input
- Projetos: contrato parceria (quinta) + revisão (sexta) + proposta para 2 leads
- Quer: 2 posts criados + 1h de captação
- Horário: 9h-18h

### Output
| Horário | Atividade | Tipo | Duração |
|---|---|---|---|
| 09h–09h15 | Ritual início: revisar prazos + definir Top 3 | Admin | 15min |
| 09h15–11h15 | Contrato de parceria — rascunho inicial | Profundo | 2h |
| 11h15–12h | Resposta a 2 leads + emails (batch) | Cliente/Captação | 45min |
| 12h–13h | Almoço | — | — |
| 13h–15h | Contrato de parceria — revisão e finalização | Profundo | 2h |
| 15h–16h | Criação de post #1 para Instagram | Conteúdo | 1h |
| 16h–17h30 | Prospecção — enviar 5 emails frios | Captação | 1h30 |
| 17h30–18h | Ritual fim: capturar pendências, preparar terça | Admin | 30min |
