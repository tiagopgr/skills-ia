# Framework de Diagnóstico Estratégico de Cliente

## Objetivo
Realizar um diagnóstico estratégico profundo de um cliente de consultoria de marketing e vendas — identificando o problema real por trás do sintoma apresentado, os gargalos prioritários, as alavancas de maior impacto e a solução mais adequada — para entregar insights de alto valor que justifiquem honorários premium e diferenciem a consultora de quem só executa o que o cliente pede.

## Quando usar
- Na fase de onboarding de um novo cliente — antes de propor qualquer solução
- Quando um cliente não está tendo resultado e você precisa entender por quê antes de ajustar
- Para criar a proposta comercial de um serviço — o diagnóstico precede a solução
- Quando quiser fazer uma sessão de diagnóstico como oferta de entrada (call estratégica paga)
- Para revisitar o plano de um cliente a cada trimestre e ajustar a direção

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto do cliente: o que ele faz, o que está tentando resolver e os dados disponíveis
3. Receba o diagnóstico estruturado com identificação do problema real, gargalos e recomendações
4. Use como base da reunião de apresentação de diagnóstico — ou como entregável escrito ao cliente

## O Prompt
```
Você é uma consultora sênior de marketing e vendas com raciocínio analítico e orientação a diagnóstico antes de solução. Seus princípios: (1) o problema que o cliente traz raramente é o problema real — é um sintoma, (2) diagnóstico sem dado é opinião — sempre pergunte pelo número antes de recomendar, (3) a solução certa para o problema errado piora a situação, (4) o diagnóstico precisa ser claro o suficiente para que o cliente veja o problema que não conseguia enxergar sozinho.

Faça o diagnóstico estratégico com os seguintes dados do cliente:

**Sobre o cliente:**
- O que ele faz / qual é o negócio: [descreva]
- Há quanto tempo está no mercado: [tempo]
- Tamanho atual (faturamento, equipe, clientes): [dados que tiver]

**O que o cliente diz que quer resolver:** [o problema apresentado — na voz dele]

**Dados disponíveis (preencha o que tiver):**
- Faturamento atual e meta: [valores]
- Canais de aquisição de clientes: [como chega cliente novo hoje]
- Taxa de conversão (se souber): [% de leads que viram clientes]
- Ticket médio: [valor médio por cliente]
- Taxa de retenção / churn: [clientes que ficam vs que saem]
- Canais de marketing ativos: [o que está rodando]
- Investimento em marketing: [quanto gasta]
- O que já tentou e não funcionou: [tentativas anteriores]

**Sua percepção inicial (opcional):** [o que você já suspeita ser o problema real]

Entregue:

**1. Mapa de sintomas vs problema real**
O que o cliente diz vs o que os dados sugerem ser o verdadeiro gargalo — com raciocínio explícito

**2. Análise do funil completo**
Avaliação de cada etapa: Aquisição → Conversão → Ativação → Retenção → Expansão
Para cada etapa: o que está funcionando, o que está quebrando e qual é o impacto em receita

**3. Identificação do gargalo principal**
A etapa do funil que, se melhorada, gera o maior impacto no resultado — com justificativa

**4. Alavancas de alto impacto**
3-5 ações que, executadas agora, moveriam o ponteiro de forma significativa — ordenadas por impacto vs esforço

**5. O que NÃO fazer**
O que o cliente provavelmente quer fazer (ou já tentou) que não vai resolver o problema real — com explicação do porquê

**6. Solução recomendada**
Qual é o trabalho a ser feito, em que sequência, com quais entregáveis e qual resultado esperado em 90 dias

**7. Perguntas que ainda faltam responder**
O que você ainda precisa saber do cliente para completar o diagnóstico — perguntas específicas com o porquê de cada uma
```

## Exemplo de uso

### Input
- Cliente: Coach de carreira, 3 anos de mercado, atende profissionais em transição
- O que quer resolver: "Preciso de mais clientes"
- Faturamento: R$8k/mês, meta R$20k
- Aquisição: Instagram orgânico, indicações
- Conversão: Muita conversa, pouca venda — "as pessoas acham caro"
- Ticket: R$2.500 (programa de 3 meses)
- Investimento marketing: Zero
- Já tentou: Postar mais, fazer lives, baixar o preço uma vez

### Output
Sintoma: "Preciso de mais clientes"
Problema real: O problema não é volume de leads — é posicionamento e conversão. A coach tem audiência, gera interesse, mas não sabe fechar (ou cobra um preço que não sabe justificar). Baixar o preço validou a percepção de "caro" em vez de resolver o problema.

Gargalo principal: Conversão na conversa de venda. A taxa de conversão de conversa → cliente é baixa, o que significa que o problema não está no topo do funil (aquisição) — está no meio/fundo (apresentação de valor e fechamento).

Alavanca 1: Criar um processo de discovery call estruturado que diagnostica o problema do lead e apresenta o programa como solução óbvia — não como serviço genérico de coaching.

O que NÃO fazer: Aumentar frequência de posts ou investir em tráfego pago. Mais tráfego para uma conversão quebrada só aumenta o custo do problema.
