# Guia Passo a Passo para Usar o Claude no Trabalho Jurídico

## Objetivo
Mostrar de forma simples e prática como usar o Claude para acelerar o trabalho jurídico do Procurador — desde a triagem da planilha diária até a elaboração de peças — com exemplos concretos, sem jargão técnico e com passo a passo que funciona mesmo para quem está começando agora com IA.

## Quando usar
- Quando quiser aprender a usar o Claude de forma eficiente no trabalho jurídico
- Quando travar em como fazer uma pergunta ou como passar uma informação para a IA
- Quando quiser criar um fluxo de trabalho consistente com o Claude
- Quando quiser ensinar alguém da equipe a usar a ferramenta
- Como referência rápida nos primeiros meses de uso

## Como usar
1. Este arquivo funciona diferente dos outros — ele é um guia para consulta, não um prompt para copiar. Leia ele como referência e use os exemplos como modelo para suas próprias perguntas ao Claude.
