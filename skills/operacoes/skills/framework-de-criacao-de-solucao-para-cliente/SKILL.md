# Framework de Criação de Solução para Cliente

## Objetivo
Criar soluções de marketing e vendas personalizadas para clientes de consultoria — usando um framework de raciocínio estratégico que vai do diagnóstico ao plano de ação — para que a consultora possa montar propostas únicas, específicas e de alto valor para cada cliente sem começar do zero toda vez, entregando o que nenhum template genérico entregaria.

## Quando usar
- Ao construir o plano de trabalho após a discovery call
- Para montar a estratégia de marketing ou vendas que vai guiar os 3-6 meses de consultoria
- Quando precisar criar uma solução personalizada como parte de um entregável de consultoria
- Para estruturar o raciocínio antes de uma reunião de apresentação de estratégia ao cliente
- Quando quiser elevar a qualidade e profundidade das entregas sem aumentar o tempo de trabalho

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o diagnóstico do cliente, os objetivos, as restrições e o contexto do mercado
3. Receba a solução estruturada com raciocínio explícito, priorização e plano de ação
4. Apresente ao cliente mostrando o raciocínio por trás de cada decisão — isso é o que diferencia consultoria de execução

## O Prompt
```
Você é uma consultora estratégica de marketing e vendas que pensa antes de executar. Seus princípios: (1) solução boa começa com o problema certo — se o diagnóstico está errado, a solução não funciona mesmo bem executada, (2) toda estratégia precisa de sequência lógica — o que vem antes habilita o que vem depois, (3) a solução precisa respeitar as restrições reais do cliente — de orçamento, de tempo, de equipe e de maturidade de mercado, (4) mostrar o raciocínio é mais valioso que mostrar o plano — o cliente aprende a tomar decisões, não apenas a executar tarefas.

Crie a solução personalizada com os seguintes dados do cliente:

**Perfil do cliente:**
- O que faz e há quanto tempo: [descrição]
- Tamanho atual (faturamento, equipe, clientes): [dados]
- Maturidade digital (presença online, audiência, ferramentas): [descrição]

**Diagnóstico (o problema real identificado):**
[Descreva o problema central — não o sintoma apresentado, mas o que você diagnosticou como causa raiz]

**Objetivo do cliente nos próximos 90-180 dias:** [meta específica e mensurável]

**Restrições:**
- Orçamento disponível para marketing: [valor]
- Horas disponíveis por semana para executar: [horas]
- Tem equipe ou é solo: [descrição]
- O que já tentou e não funcionou: [tentativas anteriores]

**Contexto competitivo:** [resumo do mercado — saturação, como os concorrentes se comunicam]

Entregue:

**1. Premissas estratégicas**
As 3-5 crenças sobre marketing e vendas que fundamentam a solução — o cliente precisa entender e concordar com essas premissas para a estratégia funcionar

**2. O que NÃO vai fazer e por quê**
Lista de ações que parecem óbvias mas que você está descartando conscientemente — com o raciocínio de cada exclusão

**3. A estratégia principal (a aposta central)**
A alavanca estratégica principal — o que, se funcionar, resolve 80% do problema. Uma coisa, não uma lista.

**4. Plano de ação em fases (90 dias)**
Fase 1 (mês 1): Fundação — o que precisa existir antes de qualquer coisa
Fase 2 (mês 2): Tração — o que colocar para rodar sobre a fundação
Fase 3 (mês 3): Escala — o que amplificar com base no que funcionou

Para cada fase: ações específicas, responsável (cliente ou consultora), indicador de sucesso

**5. Sequência de decisões**
O caminho crítico — o que precisa acontecer em cada semana para o plano funcionar. O que, se atrasar, compromete tudo.

**6. Indicadores de progresso (não de resultado)**
O que acompanhar toda semana para saber se estão no caminho certo — antes que os resultados finais apareçam

**7. Cenários**
Se tudo correr bem: o que o cliente terá em 90 dias
Se as primeiras ações não derem resultado: o que ajustar e quando tomar essa decisão
```

## Exemplo de uso

### Input
- Cliente: Coach de carreira, solo, 3 anos de mercado, fatura R$8k/mês
- Diagnóstico: Posicionamento amplo demais + processo de venda inexistente (vende só por indicação)
- Objetivo: Chegar a R$20k/mês em 6 meses
- Restrições: Sem verba para tráfego pago / 6h/semana para marketing / solo
- Já tentou: Postar todo dia (2 meses, abandonou), fazer lives (engajamento mas sem clientes)
- Mercado: Alta concorrência, coaches com 50-200k seguidores disputando o mesmo público

### Output
- Tráfego pago: Sem margem e sem processo de venda validado, ads vão queimar orçamento em leads que não convertem. Descartado até mês 3.
- Aumentar frequência de posts: O problema não é alcance — é conversão. Mais posts com posicionamento difuso = mais ruído.
- Lives: Geram engajamento mas não qualificam intenção de compra. Desgasta sem retorno proporcional. Volta no mês 2 com formato diferente.

Estratégia principal: Redesenhar o posicionamento para um público ultra-específico + criar processo de venda via LinkedIn orgânico (onde o público está, com menos concorrência). Uma coisa só, bem feita.
