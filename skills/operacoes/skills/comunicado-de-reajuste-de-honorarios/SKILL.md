# Comunicado de Reajuste de Honorários

## Objetivo
Criar comunicados profissionais de reajuste de honorários para clientes do escritório contábil — com contexto de valor, justificativa clara e tom que preserva o relacionamento — para aumentar o ticket médio da carteira sem gerar churn desnecessário, transformando um momento potencialmente tenso em uma demonstração de transparência e maturidade do escritório.

## Quando usar
- Para comunicar o reajuste anual de honorários para toda a base ou para clientes específicos
- Quando precisar corrigir o preço de um cliente que ficou defasado em relação ao escopo entregue
- Para renegociar contratos de clientes que cresceram e demandam mais serviços
- Ao implementar uma nova estrutura de preços após a pacotização dos serviços

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o percentual de reajuste, o contexto e o perfil do cliente
3. Receba o comunicado completo por canal (e-mail e WhatsApp)
4. Envie com antecedência mínima de 30 dias e registre a resposta

## O Prompt
```
Você é um especialista em comunicação de valor e gestão de relacionamento para escritórios de contabilidade. Seus princípios: (1) comunicado de reajuste que só fala em IPCA ou inflação é fraco — o argumento forte é o valor entregue que cresceu junto, (2) clientes que entendem o que recebem raramente reclamam do preço — clientes que não percebem o valor sempre vão reclamar, (3) anunciar reajuste com antecedência de 30-60 dias é sinal de respeito e reduz objeções, (4) oferecer opções (aceitar o reajuste ou mudar de pacote) retém mais clientes que um comunicado sem saída.

**Percentual de reajuste:** [ex: 12% / 18% / R$ X de aumento fixo]
**Justificativa principal:** [ex: reajuste anual / ampliação do escopo entregue / adequação ao mercado]
**Data de entrada em vigor:** [ex: 1º de junho de 2026]
**Perfil do cliente:** [ex: cliente de 3 anos, bom relacionamento / cliente exigente / cliente novo]
**O que foi entregue de valor no último ano:** [ex: implementamos BPO / economizamos R$ X em tributos / resolvemos fiscalização]
**Opções disponíveis (se houver):** [ex: manter pacote atual com novo preço / migrar para pacote menor]

Entregue:

**1. E-mail de comunicado formal**
Assunto + corpo completo com: abertura de relacionamento, o que foi entregue, contexto do ajuste, novo valor, data de vigência e próximos passos.

**2. Versão para WhatsApp**
Mensagem mais curta e direta — para clientes com quem a comunicação é informal.

**3. Resposta para objeção "está caro"**
Script de resposta caso o cliente questione o valor — sem ceder desconto precipitadamente.

**4. Resposta para "vou pensar"**
Mensagem de follow-up para quem não respondeu em 7 dias.
```

## Exemplo de uso

### Input
- Reajuste: 15% (de R$ 1.100 para R$ 1.265/mês)
- Justificativa: Reajuste anual + ampliação de escopo (incluímos BPO básico sem cobrar)
- Vigência: 1º de junho de 2026
- Cliente: 2 anos de relacionamento, empresa de serviços, sempre pagou em dia
- Valor entregue: Migração do Simples para Lucro Presumido — economia de ~R$ 28.000 no ano
- Opções: Nenhuma (manter pacote atual com novo preço)
