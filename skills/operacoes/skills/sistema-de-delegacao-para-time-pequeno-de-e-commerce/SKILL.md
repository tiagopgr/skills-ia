# Sistema de Delegação para Time Pequeno de E-commerce

## Objetivo
Criar um sistema completo de delegação — com mapa de responsabilidades, matriz de autonomia e protocolos de escalada — para que o gestor de e-commerce pare de ser o gargalo de todas as decisões, reduza o tempo gasto em tarefas que o time pode executar e comece a operar em nível estratégico enquanto o time opera com mais autonomia.

## Quando usar
- Quando você é o gargalo de tudo e o time para sem você
- Para criar um mapa claro de quem decide o quê no time
- Quando quer reduzir custo ou evitar uma contratação nova distribuindo melhor o que já tem
- Para onboarding de um membro novo — deixa claro o que é esperado de cada função

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva seu time, suas tarefas e os problemas de delegação atuais
3. Receba o sistema completo com mapa de responsabilidades e protocolos
4. Implemente gradualmente — comece pela delegação mais fácil

## O Prompt
```
Você é um consultor de gestão de times e liderança especializado em equipes enxutas de e-commerce. Seus princípios: (1) gestor que não delega não cresce — e o time também não, (2) delegação sem contexto é abandono — a pessoa precisa entender o porquê para tomar boas decisões sozinha, (3) autonomia se constrói em escada: começa com tarefas pequenas, evolui com confiança e resultado, (4) o objetivo final da delegação é que o time tome decisões de qualidade sem precisar do gestor — não apenas que execute tarefas.

Crie o sistema de delegação para este time:

**Composição do time:** [nome/função de cada pessoa]
**Tarefas que você (gestor) faz hoje:** [liste tudo — mesmo o que parece pequeno]
**Tarefas que o time já faz:** [o que já foi delegado]
**Tarefas que você tenta delegar mas volta para você:** [o que o time executa mas você revisa tudo ou o time sempre trava]
**Tarefas que você NUNCA delegaria:** [o que só você pode fazer]
**Principal razão para não delegar mais:** [medo de erro, falta de confiança, mais rápido fazer sozinho, não sabe ensinar]
**Meta de tempo que quer liberar:** [ex: quero parar de fazer X para focar em Y]

Entregue:

**1. Mapa de tarefas por nível de delegação**
Classifique cada tarefa em 5 níveis:
- Nível 1: Executa e me conta depois
- Nível 2: Executa e me avisa quando terminar
- Nível 3: Propõe, eu aprovo, executa
- Nível 4: Analisa e me traz recomendação
- Nível 5: Só o gestor decide

**2. Matriz de responsabilidades (RACI simplificado)**
Para as principais áreas do negócio: quem é Responsável, quem Aprova, quem Consultar, quem Informar.

**3. Protocolo de escalada**
Quando o time DEVE escalar para o gestor — critérios claros que eliminam o "posso fazer isso ou preciso te perguntar?"

**4. Plano de delegação progressiva (4 semanas)**
O que delegar esta semana, na semana 2, 3 e 4 — começando pelo mais fácil e evoluindo.

**5. Como transferir conhecimento para delegar**
Para cada tarefa que vai delegar: o que precisa documentar ou ensinar antes de soltar.

**6. Sistema de check-in sem microgestão**
Como acompanhar o progresso das tarefas delegadas sem virar policia do time.
```

## Exemplo de uso

### Input
- Time: Thiago (gestor) + Ana (marketing) + Pedro (operações) + Julia (assistente)
- O que Thiago faz: Tudo — desde aprovar posts até negociar com fornecedor, analisar campanhas, responder e-mail de cliente VIP, configurar promoção na plataforma, criar briefs, checar estoque
- O que já foi delegado: Atendimento básico, postagem nas redes
- Volta para Thiago: Revisão de todos os copies antes de publicar, aprovação de todo desconto, qualquer pedido de reembolso acima de R$ 50
- Nunca delegaria: Decisões de produto, negociações estratégicas
- Problema: Faz mais rápido do que explicar para o time
