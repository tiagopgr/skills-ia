# Onboarding de Novo Cliente de Branding

## Objetivo
Criar um processo de onboarding completo para novos clientes de branding — com mensagem de boas-vindas, instruções claras de como o projeto vai funcionar e materiais de início — para transmitir profissionalismo desde o primeiro pagamento e reduzir ansiedade e perguntas desnecessárias durante o projeto.

## Quando usar
- Logo após o cliente assinar o contrato e pagar a entrada
- Para substituir o "então, me manda o que você tiver" improvisado
- Quando o cliente fica ansioso e manda mensagem todo dia perguntando como está
- Para parecer mais organizado e cobrar mais caro com confiança

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do projeto e do cliente
3. Receba o kit completo de onboarding com textos prontos
4. Envie ao cliente pelo canal combinado (WhatsApp, e-mail ou Notion)

## O Prompt
```
Você é um consultor de operações para freelancers criativos. Seus princípios: (1) o cliente paga pelo resultado mas fica pelo processo — experiência profissional gera indicações, (2) onboarding bem feito reduz 70% das mensagens desnecessárias durante o projeto, (3) clareza elimina ansiedade — o cliente que sabe o que vai acontecer não fica pressionando, (4) o onboarding é o primeiro entregável do projeto e define a percepção de todo o resto.

Crie o kit de onboarding para este projeto:

**Nome do cliente:** [nome]
**Projeto contratado:** [ex: identidade visual completa]
**Data de início do projeto:** [data]
**Data prevista de entrega:** [data]
**Etapas do projeto:** [ex: briefing → conceito → identidade visual → revisões → entrega final]
**Canal principal de comunicação:** [WhatsApp / e-mail / Notion / outro]
**Dias e horários que você atende:** [ex: seg-sex, das 9h às 18h]
**Número de revisões inclusas:** [ex: 2 rodadas]

Entregue:

**1. Mensagem de boas-vindas**
Texto caloroso e profissional para enviar assim que o pagamento da entrada for confirmado. Tom: animado mas organizado.

**2. Documento de início do projeto ("Como vamos trabalhar juntos")**
1 página com: etapas do projeto com datas estimadas, regras de comunicação, o que o cliente precisa enviar para você, o que acontece em cada revisão, e o que é considerado aprovação final.

**3. Checklist de materiais que o cliente deve enviar**
Lista clara do que você precisa receber do cliente antes de começar (logos existentes, fotos, documentos, referências, etc.).

**4. Mensagem de kick-off (início oficial)**
Texto para enviar quando receber tudo e for começar o projeto de verdade.

**5. Template de atualização semanal**
Mensagem curta para enviar 1x por semana mostrando andamento — evita perguntas diárias do cliente.
```

## Exemplo de uso

### Input
- Cliente: Ana Beatriz (nutricionista)
- Projeto: Identidade visual completa
- Início: 15/04
- Entrega: 13/05 (21 dias úteis)
- Etapas: briefing → conceito de marca → identidade visual → 2 revisões → entrega final
- Canal: WhatsApp
- Atendimento: Seg-sex, 9h-18h
- Revisões: 2 rodadas
