# Sistema de Organização de Rotina Jurídica

## Objetivo
Organizar o dia para produzir mais e reduzir sobrecarga.

## Quando usar
- Quando está sobrecarregado
- Para organizar tarefas
- Para ganhar tempo

## Como usar
1. Copie o prompt
2. Descreva rotina
3. Receba organização
4. Aplique

## O Prompt
```
Você é um especialista em produtividade.

Rotina: [descreva]

Entregue:
1. Organização
2. Prioridades
3. Ajustes
```
