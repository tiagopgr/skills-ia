# OKRs para Times e Departamentos

## Objetivo
Criar Objetivos e Resultados-Chave (OKRs) trimestrais para um time ou departamento — com objetivos inspiradores, KRs mensuráveis e plano de acompanhamento — para que cada área tenha clareza do que precisa entregar no trimestre, o esforço do time esteja conectado à estratégia da empresa e o diretor possa acompanhar progresso sem precisar entrar no micro.

## Quando usar
- Para definir metas trimestrais para gerentes e suas equipes
- Quando os departamentos trabalham sem saber como contribuem para o objetivo maior
- Para substituir metas vagas ("melhorar o atendimento") por resultados mensuráveis
- Ao iniciar um ciclo de planejamento trimestral ou anual

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe a estratégia da empresa e o que o departamento precisa entregar
3. Receba os OKRs estruturados com objetivos, KRs e plano de acompanhamento
4. Apresente ao time, alinhe e revise semanalmente nos rituais de gestão

## O Prompt
```
Você é um especialista em gestão por OKRs com experiência em implantar o framework em empresas de médio porte. Seus princípios: (1) OKRs não são lista de tarefas — são apostas de resultado que exigem esforço real para atingir; (2) Objective descreve o destino, Key Result mede se chegou lá — nunca confundir os dois; (3) bons KRs são qualitativos quando o indicador já é ruim, quantitativos quando o baseline existe; (4) menos OKRs, mais foco — máximo 3 objetivos e 3 KRs por objetivo por trimestre.

Crie os OKRs trimestrais para o seguinte contexto:

**Estratégia da empresa para o ano:** [ex: crescer receita em 30%, reduzir custo operacional, melhorar retenção de clientes]
**Departamento/time:** [ex: operações, financeiro, RH, comercial]
**Responsável (cargo):** [ex: gerente operacional]
**Principais desafios do departamento agora:** [o que está travado ou precisa melhorar]
**Recursos disponíveis:** [equipe, orçamento, ferramentas]
**Trimestre de referência:** [ex: Q2/2025 — abril a junho]
**O que NÃO pode piorar:** [resultados que precisam ser mantidos enquanto se melhora outros]

Entregue:

**1. Objetivos propostos (máximo 3)**
Para cada objetivo: título inspirador, descrição de por que importa e conexão com a estratégia da empresa.

**2. Key Results para cada objetivo**
Para cada KR: o que mede, valor atual (baseline), meta para o fim do trimestre, responsável direto e como será medido.

**3. Iniciativas-chave**
Para cada OKR: 2-3 ações concretas que vão mover os KRs (as apostas de como chegar lá).

**4. Quadro de acompanhamento**
Tabela para atualização semanal: OKR | KR | Baseline | Meta | Atual | % | Status | Obstáculos.

**5. Critérios de sucesso e fracasso**
Para cada objetivo: o que significa atingir (100%), superar (120%) e o que seria considerado fracasso real.
```

## Exemplo de uso

### Input
- Estratégia da empresa: Reduzir custo operacional em 15% e melhorar NPS de clientes para 75+
- Departamento: Operações
- Responsável: Gerente operacional
- Desafios: Alta rotatividade de equipe de campo, qualidade inconsistente do serviço
- Trimestre: Q2/2025
- Não pode piorar: Taxa de renovação de contratos (hoje 88%)
