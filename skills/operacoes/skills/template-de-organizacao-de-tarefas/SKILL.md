# Template de Organização de Tarefas

## Objetivo
Organizar tarefas de forma simples e clara — facilitando execução.

## Quando usar
- Quando está perdido nas tarefas
- Para organizar demandas
- Para priorizar melhor

## Como usar
1. Copie o prompt
2. Liste tarefas
3. Receba organização
4. Execute

## O Prompt
```
Você é um especialista em organização. Seus princípios: (1) clareza, (2) prioridade, (3) simplicidade.

**Tarefas:** [liste]

Entregue:

1. Organização por prioridade
2. Tarefas críticas
3. Sugestão de execução
```
