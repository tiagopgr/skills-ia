# Sistema de Conteúdo para LinkedIn e Autoridade

## Objetivo
Gerar conteúdo consistente que posiciona como autoridade e atrai clientes qualificados.

## Quando usar
- Para crescer no LinkedIn
- Para atrair clientes
- Para construir autoridade

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba conteúdos
4. Publique

## O Prompt
```
Você é um estrategista de conteúdo. Princípios: (1) clareza, (2) valor, (3) consistência, (4) autoridade.

**Área:** investimentos imobiliários
**Público:** [explique]
**Objetivo:** [explique]

Entregue:

1. 10 ideias de posts
2. 5 posts completos
3. Ganchos
4. CTA
5. Frequência ideal
```
