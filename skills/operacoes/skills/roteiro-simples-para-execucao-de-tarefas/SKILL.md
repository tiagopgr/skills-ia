# Roteiro Simples para Execução de Tarefas

## Objetivo
Evitar travamento na execução — criando um passo a passo simples para qualquer tarefa.

## Quando usar
- Quando não sabe por onde começar
- Para tarefas novas
- Para evitar procrastinação

## Como usar
1. Copie o prompt
2. Descreva a tarefa
3. Receba roteiro
4. Execute

## O Prompt
```
Você é um especialista em execução prática. Seus princípios: (1) dividir em etapas, (2) clareza, (3) ação imediata.

**Tarefa:** [descreva]

Entregue:

1. Passo a passo
2. Ordem ideal
3. Pontos de atenção
4. Dicas de execução
```
