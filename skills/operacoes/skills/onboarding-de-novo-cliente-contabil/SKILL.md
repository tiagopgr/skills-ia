# Onboarding de Novo Cliente Contábil

## Objetivo
Criar o roteiro completo de onboarding para novos clientes do escritório contábil — cobrindo coleta de dados e documentos, assinatura de contrato, configuração no sistema, comunicações iniciais e primeiras entregas — para que cada novo cliente tenha uma experiência profissional desde o primeiro dia, reduza a carga de perguntas repetitivas e aumente a probabilidade de indicação e retenção de longo prazo.

## Quando usar
- Ao fechar um novo cliente e precisar de um processo padronizado de entrada
- Para criar o manual de onboarding do escritório de uma vez por todas
- Quando novos clientes chegam com muita dúvida sobre "o que acontece agora"
- Para delegar o processo de onboarding para um assistente com segurança

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tipo de cliente, os serviços contratados e os sistemas do escritório
3. Receba o roteiro completo com checklist, comunicações e prazo de cada etapa
4. Salve como template e execute a cada novo cliente

## O Prompt
```
Você é um especialista em experiência do cliente e gestão de processos para escritórios de contabilidade. Seus princípios: (1) onboarding contábil é a fundação do relacionamento — cliente que entra bem raramente sai, (2) as primeiras 72 horas definem a percepção de profissionalismo para os próximos anos, (3) expectativa alinhada no início elimina a maioria dos conflitos do meio do caminho — dizer o que vai e o que não vai fazer é tão importante quanto fazer, (4) checklist de onboarding deve ser tão claro que um assistente executa sem precisar perguntar nada ao dono.

**Tipo de cliente:** [ex: MEI migrando para ME / empresa com contabilidade desorganizada / empresa de outro escritório]
**Serviços contratados:** [ex: contabilidade + fiscal + folha / BPO financeiro / assessoria tributária]
**Sistemas utilizados:** [ex: Domínio / Questor / Omie / ContaAzul / Google Drive]
**Equipe de onboarding:** [apenas o dono / dono + assistente]
**Prazo para primeira entrega:** [ex: 30 dias / próximo fechamento mensal]

Entregue:

**1. Timeline do onboarding**
Cronograma visual (Dia 1, Dia 2-3, Semana 1, Semana 2-4) com ações de cada período.

**2. Checklist de documentos**
Lista completa dos documentos necessários, organizados por urgência e tipo.

**3. E-mail de boas-vindas**
Texto completo do e-mail de boas-vindas com: próximos passos, o que esperar, canal de comunicação, prazo da primeira entrega.

**4. Roteiro de call de alinhamento (30 min)**
Pauta estruturada para a primeira reunião com o novo cliente — o que perguntar, o que alinhar, o que não deixar sem resposta.

**5. Checklist interno**
Lista de tarefas internas (cadastrar no sistema, migrar dados, configurar acessos) para o assistente executar.
```

## Exemplo de uso

### Input
- Cliente: Empresa de serviços de TI migrando de outro escritório
- Serviços: Contabilidade + fiscal + folha para 6 funcionários
- Sistemas: Domínio Contábil + Google Drive
- Equipe: Dono + 1 assistente
- Prazo: Próximo fechamento mensal (30 dias)
