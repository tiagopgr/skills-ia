# Planejamento Mensal do Negócio de Consultoria

## Objetivo
Criar o planejamento mensal do próprio negócio de consultoria — com metas de faturamento, gestão de pipeline, prioridades de entrega para clientes ativos, agenda de captação e indicadores de acompanhamento — para que o consultor aplique em si mesmo o que entrega para os clientes e pare de improvisar a gestão do próprio negócio.

## Quando usar
- No início de cada mês para organizar o que vai ser executado
- Quando quiser ter clareza sobre de onde virá o faturamento do mês
- Quando precisar equilibrar o tempo entre entregar para clientes ativos e captar novos
- Quando quiser criar uma rotina de planejamento e revisão mensal consistente
- Quando sentir que está correndo sem saber se está avançando

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com os dados do seu negócio — clientes ativos, pipeline, meta do mês
3. Receba o plano mensal estruturado com semanas, prioridades e indicadores
4. Use como guia de execução e revise semanalmente

## O Prompt
```
Você é um consultor de gestão especializado em negócios de consultoria, coaching e serviços profissionais. Seus princípios: (1) consultor que não planeja o próprio negócio tem dificuldade de credibilidade para cobrar planejamento do cliente, (2) faturamento de consultoria vem de: renovações + expansões + novos clientes — monitorar os três separadamente é essencial, (3) tempo de entrega e tempo de captação são diferentes e precisam de proteção na agenda, (4) consultor que só faz delivery sem captar fica sem cliente quando um termina — pipeline é segurança.

Monte meu planejamento mensal de consultoria com as seguintes informações:

**Mês de referência:** [ex: Maio 2025]
**Meta de faturamento do mês:** [R$ desejado]
**Clientes ativos e seus valores mensais:** [liste: cliente + valor + fase da consultoria]
**Clientes com renovação ou encerramento previsto este mês:** [quem renova, quem encerra]
**Pipeline de novos clientes:** [prospects em negociação + em que fase + valor estimado]
**Entregas comprometidas este mês:** [o que precisa entregar para cada cliente]
**Capacidade disponível para novos clientes:** [quantas horas ou slots tenho livres]
**Ação de captação planejada:** [ex: LinkedIn / networking / indicações / webinar]
**Pendências do mês anterior:** [o que ficou pra trás]
**Objetivo pessoal do mês:** [algo além do faturamento — aprendizado, sistema, produto]

Entregue:

**1. Diagnóstico financeiro do mês**
Quanto está garantido (contratos ativos) vs quanto precisa gerar (novos fechamentos) para atingir a meta.

**2. Plano semanal por semana**
4 semanas com:
- Foco principal
- Entregas prioritárias para clientes
- Ações de captação/comercial
- Tarefas de gestão do negócio

**3. Plano de pipeline**
Para cada prospect: próximo passo, prazo, probabilidade e valor — e o que fazer para avançar.

**4. Agenda de entregas**
O que cada cliente recebe este mês e até quando — para não perder nada.

**5. Indicadores para acompanhar semanalmente**
O que medir toda semana para saber se o mês está no trilho.
```

## Exemplo de uso

### Input
- Mês: Maio 2025
- Meta: R$22.000
- Clientes ativos: Empresa A R$3.800/mês (mês 3 de 6) / Empresa B R$4.500/mês (mês 5 de 6 — renova?) / Empresa C R$2.200/mês (contrato aberto)
- Pipeline: Prospect 1 em proposta enviada R$4.000/mês / Prospect 2 em reunião marcada R$3.500/mês
- Entregas comprometidas: Relatório mensal A e B, diagnóstico C, plano 90 dias para novo cliente se fechar
- Capacidade: Tenho slot para 1 cliente novo
- Captação: LinkedIn 3x semana + 2 indicações a ativar
- Meta pessoal: Gravar um webinar de captação

### Output
"MAIO — DIAGNÓSTICO FINANCEIRO
✅ Garantido: R$10.500 (clientes A + B + C)
❓ Dependente de renovação: R$4.500 (Empresa B — mês 5, precisa de conversa de renovação esta semana)
🎯 Precisa gerar: R$11.500 em novos fechamentos para bater a meta de R$22.000
📥 Pipeline disponível: R$7.500 potencial (Prospect 1 + 2)
→ Se fechar os 2 prospects + renovar B: R$22.500 — meta atingida
→ Se fechar só 1 prospect + renovar B: R$18.000 — falta R$4.000"
