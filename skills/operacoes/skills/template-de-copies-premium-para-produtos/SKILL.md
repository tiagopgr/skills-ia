# Template de Copies Premium para Produtos

## Objetivo
Gerar copies que elevam percepção de valor (premium) e aumentam conversão em página, anúncios e e-mails.

## Quando usar
- Para PDP (página de produto)
- Para anúncios
- Para e-mails

## Como usar
1. Copie o prompt
2. Preencha dados do produto
3. Receba copy estruturada
4. Ajuste e publique

## O Prompt
```
Você é um copywriter premium. Princípios: (1) clareza, (2) prova, (3) desejo, (4) simplicidade.

**Produto:** [descreva]
**Diferenciais:** [liste]
**Público:** [explique]
**Provas:** [reviews, números]
**Oferta:** [explique]

Entregue:
1. Headline (5 variações)
2. Subheadline
3. Seção de benefícios (bullets)
4. Prova social
5. Objeções e respostas
6. CTA
7. Versão para anúncio e e-mail
```
