# Checklist de Onboarding de Cliente

## Objetivo
Gerar um checklist e roteiro completo de onboarding para novos clientes do escritório — cobrindo coleta de documentos, alinhamento de expectativas, assinatura de contrato e primeiras comunicações — para padronizar o processo de entrada de clientes, eliminar retrabalho, reduzir dúvidas repetitivas e criar uma experiência profissional desde o primeiro dia.

## Quando usar
- Ao fechar um novo cliente e querer garantir que nenhum passo seja esquecido
- Para criar o processo padrão de onboarding do escritório de uma vez por todas
- Quando clientes novos chegam com muitas dúvidas sobre "o que acontece agora"
- Para delegar parte do onboarding para um estagiário ou assistente com segurança

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tipo de serviço e as etapas típicas do caso
3. Receba o checklist completo e o roteiro de comunicação inicial
4. Salve como template e use a cada novo cliente

## O Prompt
```
Você é um especialista em gestão de processos para escritórios de advocacia individual. Seus princípios: (1) cliente bem onboardado reclama menos, paga mais rápido e indica mais, (2) expectativa alinhada no início evita conflito no meio — tudo que pode dar errado deve ser antecipado, (3) checklist é um produto do escritório — deve ser tão profissional quanto qualquer petição, (4) comunicação proativa nas primeiras 48h define a percepção do cliente sobre o escritório para sempre.

**Tipo de caso/serviço:** [ex: ação trabalhista / assessoria mensal / divórcio / abertura de empresa]
**Duração estimada do caso:** [ex: 6 meses / indeterminado / até 30 dias]
**Documentos que precisará do cliente:** [liste os documentos típicos para esse tipo de caso]
**Canais de comunicação do escritório:** [ex: WhatsApp + e-mail / apenas e-mail / plataforma própria]
**Perfil do cliente típico:** [ex: pessoa física leiga / empresário / profissional liberal]

Entregue:

**1. Checklist de documentos**
Lista completa dos documentos necessários, organizados por prioridade (urgente / pode aguardar).

**2. Roteiro das primeiras 48h**
Sequência de ações e comunicações a fazer nas primeiras 48h após fechar o contrato — com textos prontos para cada comunicação.

**3. E-mail de boas-vindas**
E-mail padrão de boas-vindas ao novo cliente com: o que acontece agora, como funciona a comunicação, prazo para próximo contato.

**4. FAQ de onboarding**
As 5 perguntas mais comuns de clientes novos — com respostas prontas para copiar e enviar.

**5. Checklist interno do advogado**
Lista de ações internas (cadastrar no sistema, protocolar, abrir pasta física/digital, etc.) para nenhum passo ser esquecido.
```

## Exemplo de uso

### Input
- Caso: Ação trabalhista (trabalhador demitido sem receber verbas rescisórias)
- Duração: 12-18 meses
- Documentos: CTPS, contrato de trabalho, holerites, comprovante de demissão, extrato FGTS
- Canais: WhatsApp + e-mail
- Perfil: Pessoa física, primeira experiência com processo judicial
