# Estrutura de Controle de Entregas

## Objetivo
Garantir que entregas sejam feitas com qualidade e prazo.

## Quando usar
- Para controlar prazos
- Para organizar entregas
- Para evitar atrasos

## Como usar
1. Copie o prompt
2. Liste entregas
3. Receba controle
4. Aplique

## O Prompt
```
Você é um gestor operacional. Princípios: controle, clareza, consistência.

**Entregas:** [liste]

Entregue:
1. Sistema de controle
2. Acompanhamento
3. Alertas
4. Ajustes
```
