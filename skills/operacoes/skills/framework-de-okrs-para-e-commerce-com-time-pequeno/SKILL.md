# Framework de OKRs para E-commerce com Time Pequeno

## Objetivo
Criar OKRs (Objectives and Key Results) trimestrais para o e-commerce — com objetivos inspiradores, resultados-chave mensuráveis e iniciativas priorizadas — para alinhar o time em torno do que realmente importa, eliminar a dispersão de esforço e criar uma cultura de accountability sem precisar de um sistema complexo de gestão.

## Quando usar
- No início de cada trimestre para definir as metas do time
- Quando o time trabalha muito mas sem direção unificada
- Para criar clareza de prioridade que o time consulta sem precisar perguntar ao gestor
- Para substituir a lista infinita de tarefas por um norte estratégico compartilhado

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe os objetivos de negócio e os dados do trimestre anterior
3. Receba os OKRs completos com iniciativas e cadência de revisão
4. Apresente ao time e revise mensalmente

## O Prompt
```
Você é um consultor de estratégia e gestão por objetivos especializado em e-commerce e times enxutos. Seus princípios: (1) OKR bom tem no máximo 3 objetivos e 3-5 KRs por objetivo — mais que isso vira decoração, (2) Key Result deve ser mensurável e binário: ou atingiu ou não atingiu — sem "em progresso" ou "parcialmente", (3) objetivo inspira, Key Result comprova — confundir os dois é o erro mais comum, (4) para times pequenos, OKR sem iniciativas vinculadas é meta sem caminho.

Crie os OKRs trimestrais para o e-commerce com as seguintes informações:

**Resultado do trimestre anterior:**
- Faturamento: [valor e variação]
- Conversão: [%]
- CAC: [valor]
- NPS ou satisfação: [se tiver]
- O que não funcionou: [principais falhas]

**Contexto do próximo trimestre:**
- Período: [ex: Q2 2026 — Abril a Junho]
- Sazonalidade relevante: [datas comemorativas, períodos de pico ou baixa]
- Novidades planejadas: [lançamentos, campanhas, mudanças]
- Restrições: [budget, time, dependências]

**Ambições do negócio para o trimestre:**
- Crescimento de faturamento desejado: [% ou valor absoluto]
- Outros objetivos estratégicos: [o que mais importa além de faturamento]

**Composição do time:** [quantas pessoas e funções]

Entregue:

**1. OKRs do trimestre (máximo 3 objetivos)**
Para cada objetivo:
- **Objective:** frase inspiradora que define o que quer conquistar
- **Key Results (3-4):** métricas específicas e mensuráveis que provam que o objetivo foi atingido
- **Iniciativas:** as 2-3 ações principais que vão mover cada KR

**2. OKRs individuais por membro do time (se aplicável)**
Como desdobrar os OKRs do negócio em contribuições individuais de cada pessoa.

**3. Cadência de revisão**
Quando e como revisar os OKRs durante o trimestre — check-in semanal, revisão mensal, retrospectiva trimestral.

**4. Critério de sucesso ao final do trimestre**
O que significa "trimestre bem-sucedido" — com % de atingimento esperado e o que fazer com cada faixa de resultado.

**5. O que não fazer este trimestre (anti-OKRs)**
As iniciativas que foram consideradas mas conscientemente descartadas — para o time saber o que não priorizar.
```

## Exemplo de uso

### Input
- Q1 resultado: Faturamento R$ 280k (meta era R$ 320k) / Conversão 1,3% / CAC R$ 45 / Problema: lançamento de novo produto não performou
- Q2 período: Abril-Junho / Dia das Mães (maio) é pico da categoria
- Novidades: Relançamento do produto que não performou + campanha de Dia das Mães
- Ambição: R$ 380k no trimestre / melhorar conversão para 1,7%
- Time: Thiago + 2 analistas + 1 assistente
