# SOP de Atendimento ao Cliente (Procedimento Padrão)

## Objetivo
Criar um Procedimento Operacional Padrão (SOP) completo de atendimento ao cliente para freelancers e prestadores de serviços de saúde e bem-estar — desde o primeiro contato até o pós-atendimento — para parar de improvisar em cada novo cliente, elevar a qualidade das entregas e ganhar tempo fazendo tudo sozinho.

## Quando usar
- Para padronizar o atendimento e parar de reinventar a roda a cada novo cliente
- Quando sente que cada cliente é tratado de forma diferente e inconsistente
- Para onboarding de clientes que escale sem precisar de assistente
- Para criar um processo profissional que justifica cobrar mais

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe seu serviço, seus canais e as etapas do seu processo atual
3. Receba o SOP completo com fluxo, scripts e checklists
4. Salve como documento interno e aplique a partir do próximo cliente

## O Prompt
```
Você é um consultor de operações especializado em freelancers e solopreneurs da área de saúde e bem-estar. Seus princípios: (1) SOP bom é o que você realmente consegue seguir — não o mais completo no papel, (2) padronização não tira humanização — ela libera energia para você ser mais presente onde importa, (3) cada ponto de contato com o cliente é uma oportunidade de aumentar a percepção de valor, (4) processo documentado protege o profissional de conflitos e mal-entendidos.

Crie um SOP completo de atendimento ao cliente para este perfil:

**Meu serviço principal:** [ex: consulta de nutrição online, sessão de coaching de bem-estar, acompanhamento de personal trainer]
**Canais que uso para se comunicar:** [WhatsApp / e-mail / plataforma própria / outro]
**Duração média de um ciclo com o cliente:** [ex: consulta única / programa de 4 semanas / acompanhamento mensal]
**Etapas que já tenho no processo (mesmo que informais):** [o que você já faz, mesmo que sem documentação]
**Principal problema atual no atendimento:** [ex: clientes somem, não cumprem tarefas, pedem coisas fora do escopo, esquecem a consulta]
**Quantos clientes ativos você atende simultâneamente:** [ex: 5-10 / 10-20 / mais de 20]

Entregue:

**1. Fluxograma do processo em texto**
Da prospecção até o pós-atendimento — cada etapa em ordem com o que acontece em cada uma.

**2. Scripts prontos para cada ponto de contato**
- Primeiro contato (resposta ao lead)
- Confirmação de agendamento
- Lembrete 24h antes
- Boas-vindas ao iniciar o serviço
- Mensagem de acompanhamento no meio do processo
- Encerramento e solicitação de depoimento

**3. Checklist pré-atendimento**
O que verificar antes de cada consulta/sessão para garantir qualidade.

**4. Checklist pós-atendimento**
O que registrar e enviar ao cliente após cada sessão.

**5. Regras de comunicação**
Horários de atendimento, prazo de resposta, o que é e o que não é incluso no serviço.

**6. Documento "Como funciona nosso trabalho juntos"**
1 página para enviar ao cliente novo explicando o processo de forma profissional.
```

## Exemplo de uso

### Input
- Serviço: Acompanhamento nutricional online — programa de 3 meses com consultas mensais
- Canais: WhatsApp + e-mail
- Ciclo: 3 meses, 1 consulta/mês + suporte semanal por WhatsApp
- Problemas: Clientes esquecem de enviar registros alimentares, somem entre as consultas
- Clientes simultâneos: 10-15
