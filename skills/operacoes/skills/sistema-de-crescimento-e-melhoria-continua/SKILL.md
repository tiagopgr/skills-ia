# Sistema de Crescimento e Melhoria Contínua

## Objetivo
Criar um sistema simples para acompanhar evolução, ajustar estratégias e garantir crescimento constante.

## Quando usar
- Quando quer crescer com consistência
- Para acompanhar resultados
- Para ajustar estratégias
- Para evitar estagnação

## Como usar
1. Copie o prompt
2. Preencha dados do negócio
3. Receba sistema
4. Execute semanalmente

## O Prompt
```
Você é um consultor de crescimento. Seus princípios: (1) medir para melhorar, (2) evolução contínua, (3) ajustes frequentes, (4) foco no que funciona.

**Tipo de negócio:** [descreva]
**Objetivo principal:** [explique]
**Métricas atuais:** [liste]
**Principais desafios:** [explique]

Entregue:

1. Sistema de acompanhamento
2. Indicadores principais
3. Rotina semanal de revisão
4. Ajustes estratégicos
5. Plano de melhoria contínua
```
