# Automação de Processos com o Claude

## Objetivo
Criar um sistema de automação de processos do negócio usando o Claude como assistente central — com prompts encadeados, instruções reutilizáveis e fluxos de trabalho documentados — para transformar o Claude de "ferramenta ocasional" em "segundo cérebro do negócio" que executa tarefas repetitivas com padrão profissional e velocidade 10x maior.

## Quando usar
- Para mapear quais processos do seu negócio o Claude pode executar ou acelerar
- Quando usa o Claude mas sente que não está aproveitando o potencial
- Para criar um playbook de prompts que qualquer pessoa (ou você no futuro) pode usar
- Para reduzir o tempo gasto em tarefas repetitivas sem contratar ninguém

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva suas tarefas repetitivas e o fluxo do seu negócio
3. Receba o mapa de automação com prompts prontos para cada processo
4. Salve os prompts no Notion e use como biblioteca de automação

## O Prompt
```
Você é um consultor de produtividade e automação especializado em usar modelos de linguagem (LLMs como o Claude) para automatizar processos de marketing digital e gestão de negócios. Seus princípios: (1) LLM bom é o que executa com contexto — quanto mais contexto você dá, melhor o resultado, (2) prompt reutilizável é um ativo do negócio — vale mais do que qualquer template de planilha, (3) automatizar tarefas repetitivas libera energia para tarefas criativas e estratégicas, (4) o melhor ponto de entrada da automação é a tarefa mais frequente e mais chata.

Crie o sistema de automação de processos com o Claude para este negócio:

**Meu negócio:** [ex: freelancer de marketing digital — gestão de tráfego e consultoria]
**Tarefas que faço toda semana (liste tudo):** [ex: responder leads, escrever relatório, criar copies, responder e-mail de cliente]
**Tarefas que mais odeio fazer:** [o que procrastina mais]
**Tarefas que mais consomem tempo:** [onde passa mais horas]
**Ferramentas que já usa:** [Google Docs, Notion, Canva, Gmail, WhatsApp, etc.]
**Experiência atual com o Claude:** [uso básico / intermediário / já uso prompts personalizados]

Entregue:

**1. Mapa de automação por tarefa**
Para cada tarefa identificada: pode ser automatizada total ou parcialmente? Qual é o ganho de tempo estimado?

**2. Top 5 processos para automatizar agora (por impacto × facilidade)**
As 5 automações com maior retorno imediato — com o prompt completo para cada uma.

**3. Prompts prontos para uso imediato**
Para cada uma das top 5 automações: o prompt completo e documentado, com campos personalizáveis em [colchetes], que você pode usar agora.

**4. Como criar prompts de sistema (instruções permanentes)**
Como configurar o Claude com um "prompt de sistema" que define quem você é e como ele deve sempre te responder — eliminando a necessidade de recontextualizar em cada conversa.

**5. Biblioteca de prompts recomendada (organizada por categoria)**
Lista de prompts organizados por área: Vendas, Conteúdo, Clientes, Operações, Financeiro.

**6. Fluxo de trabalho semanal com o Claude**
Um ritual semanal usando o Claude para as tarefas-chave do negócio — com quando usar e o que pedir em cada momento.

**7. O que o Claude não consegue (ainda) automatizar**
Tarefas que parecem automáticas mas exigem julgamento humano — para não criar expectativa errada.
```

## Exemplo de uso

### Input
- Negócio: Freelancer de marketing digital (tráfego pago + consultoria)
- Tarefas semanais: Responder leads no Instagram, escrever relatório mensal, criar copies para anúncios, fazer ata de reuniões, responder e-mails de clientes, montar proposta comercial
- Mais odeia: Escrever relatório mensal e responder e-mails de cobrança
- Mais consome tempo: Montar proposta do zero + criar copies para cada cliente
- Ferramentas: Notion, Gmail, Google Docs, WhatsApp
- Experiência com Claude: Uso básico — peço coisas simples mas não uso prompts estruturados
