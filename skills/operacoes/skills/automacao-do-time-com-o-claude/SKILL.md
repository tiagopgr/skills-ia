# Automação do Time com o Claude

## Objetivo
Criar um sistema completo de automação de processos do time de e-commerce usando o Claude — com prompts documentados por função, fluxos de trabalho assistidos por IA e playbook de uso — para reduzir custo com mão de obra, aumentar a produtividade do time existente e manter padrão de qualidade sem depender de mais pessoas.

## Quando usar
- Para mapear quais processos do time podem ser parcial ou totalmente automatizados com IA
- Quando pensa em contratar mas quer primeiro ver se dá para fazer mais com quem já tem
- Para criar um playbook que o time usa de forma autônoma — sem precisar do gestor para cada prompt
- Para reduzir custo operacional sem reduzir qualidade de entrega

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva os processos do seu time e onde está o maior desperdício de tempo
3. Receba o mapa de automação com prompts prontos por função
4. Salve no Notion e treine o time para usar de forma autônoma

## O Prompt
```
Você é um consultor de produtividade e automação com IA especializado em times de e-commerce. Seus princípios: (1) IA não substitui o time — amplifica a capacidade do time existente, (2) processo mal documentado não se beneficia de automação — documentar antes de automatizar, (3) o maior retorno de automação está nas tarefas repetitivas de alta frequência, não nas tarefas complexas de baixa frequência, (4) time que usa IA bem não precisa de mais gente — precisa de processos melhores e ferramentas certas.

Crie o sistema de automação do time com IA para este contexto:

**Composição do time:** [quem são as pessoas e o que cada uma faz]
**Processos mais repetitivos por função:**
- [Função 1]: [liste as tarefas repetitivas]
- [Função 2]: [liste as tarefas repetitivas]
- [Função 3]: [liste as tarefas repetitivas]
**Tarefas que mais consomem tempo sem agregar muito valor estratégico:** [o trabalho braçal]
**Ferramentas que o time já usa:** [e-commerce, marketing, comunicação, análise]
**Nível de experiência do time com IA:** [não usa / usa superficialmente / usa com alguma frequência]
**Meta de economia de tempo:** [ex: reduzir 30% do tempo em tarefas repetitivas]

Entregue:

**1. Mapa de automação por função**
Para cada membro do time: lista de tarefas que podem ser automatizadas, nível de automação (total / parcial / assistida) e ganho estimado de tempo.

**2. Prompts prontos por função (biblioteca do time)**
Para as 3-4 automações mais impactantes por função: o prompt completo documentado e pronto para usar.

**3. Fluxo de trabalho assistido por IA (para processos chave)**
Para os processos mais complexos: como o time usa o Claude em cada etapa sem substituir o julgamento humano.

**4. Playbook de onboarding de IA para o time**
Como apresentar e treinar o time para usar o Claude de forma produtiva — incluindo o que pedir, o que revisar sempre e o que nunca publicar sem revisão.

**5. Estimativa de economia de custo**
Cálculo de horas economizadas por semana × custo hora do time = economia mensal em mão de obra ou capacidade liberada.

**6. O que a IA não consegue fazer (para o time não confiar cegamente)**
Lista de tarefas onde o julgamento humano é insubstituível — para o time saber quando não usar o Claude.
```

## Exemplo de uso

### Input
- Time: Thiago (gestor) + Ana (marketing) + Pedro (operações) + Julia (assistente)
- Processos repetitivos de Ana: Criar legendas para posts, escrever e-mails de campanha, adaptar copy de anúncio para diferentes formatos, criar briefings de criativos
- Processos repetitivos de Pedro: Responder e-mails de clientes (reclamação, rastreio, troca), criar relatórios de estoque, redigir comunicados de atraso
- Processos repetitivos de Julia: Transcrever reuniões, organizar tarefas do time, redigir atas
- Nível IA do time: Usam superficialmente, sem prompts estruturados
- Meta: Reduzir 40% do tempo em tarefas repetitivas de conteúdo e atendimento
