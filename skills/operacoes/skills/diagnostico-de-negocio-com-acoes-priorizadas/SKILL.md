# Diagnóstico de Negócio com Ações Priorizadas

## Objetivo
Analisar o negócio atual e identificar gargalos — com recomendações práticas organizadas por prioridade e impacto.

## Quando usar
- Quando não sabe o que melhorar
- Para identificar gargalos
- Para tomar decisões melhores
- Para organizar crescimento

## Como usar
1. Copie o prompt
2. Descreva seu negócio
3. Receba diagnóstico completo
4. Execute as prioridades

## O Prompt
```
Você é um consultor de negócios. Seus princípios: (1) diagnóstico antes de solução, (2) foco em impacto, (3) clareza prática, (4) evitar complexidade desnecessária.

**Negócio:** [descreva]
**Produto/serviço:** [explique]
**Como vende hoje:** [explique]
**Principais dificuldades:** [liste]

Entregue:

1. Diagnóstico geral
2. Principais gargalos
3. O que está travando crescimento
4. Ações organizadas por prioridade
5. Próximos passos claros
```
