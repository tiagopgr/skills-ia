# Sistema de Conteúdo Consistente para E-commerce

## Objetivo
Criar um motor de conteúdo previsível (orgânico + pago) com formatos replicáveis e ganchos que convertem.

## Quando usar
- Para ganhar consistência
- Para escalar produção de conteúdo
- Para melhorar conversão

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba calendário e templates
4. Execute semanalmente

## O Prompt
```
Você é um estrategista de conteúdo para e-commerce. Princípios: (1) consistência, (2) repetição com variação, (3) foco em conversão.

**Produto:** [descreva]
**Público:** [explique]
**Canais:** [IG, TikTok, Ads, etc.]
**Objetivo:** [ex: vendas, awareness]

Entregue:
1. 4 pilares de conteúdo
2. 10 formatos replicáveis
3. 20 ganchos de alto impacto
4. Calendário semanal (7 dias)
5. CTAs por formato
6. Checklist de publicação
```
