# Playbook de Lançamento de Coleções

## Objetivo
Criar lançamentos estruturados que geram expectativa e vendas.

## Quando usar
- Ao lançar produtos
- Para campanhas sazonais
- Para escalar faturamento

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba plano
4. Execute

## O Prompt
```
Você é um estrategista de lançamentos. Princípios: (1) antecipação, (2) narrativa, (3) timing.

**Coleção:** [descreva]
**Objetivo:** [explique]

Entregue:
1. Estratégia de lançamento
2. Linha do tempo
3. Conteúdos
4. Campanhas
5. Pós-lançamento
```
