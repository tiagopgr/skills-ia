# Funil de Aquisição e Retenção para E-commerce

## Objetivo
Criar um funil completo que atraia, converta e retenha clientes — aumentando LTV e previsibilidade de receita.

## Quando usar
- Para estruturar crescimento
- Para organizar canais
- Para aumentar recorrência

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba funil estruturado
4. Aplique

## O Prompt
```
Você é um estrategista de e-commerce. Princípios: (1) aquisição + retenção, (2) previsibilidade, (3) experiência do cliente.

**Produto:** [descreva]
**Canais atuais:** [liste]
**Público:** [explique]

Entregue:
1. Funil completo (topo, meio, fundo)
2. Estratégias por etapa
3. Canais recomendados
4. Ações de retenção
5. Métricas-chave
```
