# Estrutura de Programa de Mentoria ou Consultoria

## Objetivo
Criar a estrutura completa de um programa de mentoria, consultoria ou treinamento — com módulos, sessões, entregas, metodologia e jornada do cliente — para que especialistas transformem seu conhecimento em um produto estruturado, escalável e fácil de comunicar. Programa bem estruturado vende mais porque o cliente entende o que vai receber, e entrega mais resultado porque o especialista sabe o que fazer em cada etapa.

## Quando usar
- Para criar um novo programa de mentoria ou consultoria do zero
- Ao querer estruturar o que você já faz de forma intuitiva em um método com nome e etapas claras
- Para criar uma trilha de treinamento corporativo que possa ser replicado para diferentes clientes
- Quando quiser cobrar mais pelo seu serviço justificando o valor com uma entrega estruturada

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o tema do programa, o público, o resultado prometido e o formato desejado
3. Receba a estrutura completa com módulos, sessões, entregas e metodologia
4. Use como base para criar os materiais, a página de vendas e o onboarding dos clientes

## O Prompt
```
Você é um designer instrucional e estrategista de produtos educacionais para coaches, consultores e especialistas. Seus princípios: (1) programa bom começa pelo resultado do cliente — cada módulo deve ser uma etapa da jornada de transformação, não um bloco de conteúdo, (2) o nome do programa e de cada módulo precisa comunicar progresso — o cliente deve conseguir ver a jornada antes de começar, (3) sessão sem entregável concreto é sessão que o cliente esquece — cada encontro deve terminar com algo que o cliente leva: insight, ferramenta, plano ou ação, (4) programa que o especialista consegue explicar em 3 frases já está bem estruturado — se precisar de 10 frases, está complexo demais.

Crie a estrutura do programa com base nas informações abaixo:

**Tipo de programa:** [mentoria individual / mentoria em grupo / consultoria / treinamento corporativo / programa online]
**Especialidade / tema central:** [o que você ensina ou facilita]
**Público:** [para quem — perfil, nível, contexto]
**Resultado prometido:** [transformação concreta ao final do programa]
**Duração:** [ex: 3 meses / 6 semanas / 1 dia intensivo]
**Formato de entrega:** [sessões ao vivo / assíncrono / presencial / híbrido]
**Frequência:** [semanal / quinzenal / mensal / intensivo]
**Ticket do programa:** [valor — influencia a profundidade e os entregáveis]

Entregue:

**1. Nome e tagline do programa**
3 opções de nome com ângulo diferente + 1 tagline para cada.

**2. Promessa de transformação**
A frase que resume a jornada do cliente: "De [ponto A] para [ponto B] em [prazo]."

**3. Estrutura modular completa**
Para cada módulo/fase:
- Nome do módulo e objetivo de transformação
- Sessões incluídas (com título e o que será abordado)
- Entregável concreto ao fim do módulo (ferramenta, plano, decisão, ação)
- Resultado parcial que o cliente sente nesta fase

**4. Metodologia**
Como você conduz o programa — seu método em 5-7 passos, com nome para cada etapa se possível.

**5. O que está incluso**
Lista completa de entregas: sessões, materiais, ferramentas, suporte, bônus.

**6. Jornada do cliente**
Linha do tempo visual (em texto): o que o cliente faz/sente/recebe em cada semana do programa.
```

## Exemplo de uso

### Input
- Tipo: Mentoria em grupo
- Especialidade: Desenvolvimento de liderança e gestão de equipes
- Público: Gestores de nível médio em empresas de 50-300 funcionários
- Resultado: Liderar equipes de alta performance com clareza, confiança e menos conflito
- Duração: 3 meses
- Formato: Sessões ao vivo semanais + material assíncrono
- Frequência: Semanal (90 min/sessão)
- Ticket: R$3.500/pessoa

### Output
1.
