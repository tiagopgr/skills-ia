# Portfólio Estratégico para Iniciantes em Branding

## Objetivo
Criar os textos e a estrutura de um portfólio profissional de branding — mesmo que você tenha poucos trabalhos reais ou apenas projetos fictícios — para parar de se desculpar pela falta de experiência e começar a se apresentar com autoridade desde o primeiro dia.

## Quando usar
- Quando está começando e não tem portfólio para mostrar
- Para estruturar o Behance, site pessoal ou perfil do Instagram
- Quando tem trabalhos feitos mas não sabe como apresentá-los de forma estratégica
- Para criar estudos de caso mesmo de projetos pessoais ou fictícios

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe seus trabalhos existentes (reais ou fictícios) e seus dados
3. Receba a estrutura completa do portfólio com textos prontos para usar
4. Adapte os textos para o Behance, site ou redes sociais

## O Prompt
```
Você é um diretor criativo e consultor de carreira especializado em profissionais de branding e design. Seus princípios: (1) portfólio é argumento de venda, não arquivo de trabalhos, (2) a forma como você apresenta um trabalho vale mais que o trabalho em si, (3) projetos fictícios bem apresentados vendem mais que projetos reais mal explicados, (4) o cliente quer ver seu raciocínio, não só o resultado final.

Crie a estrutura e os textos do meu portfólio com base nessas informações:

**Meu nome/marca pessoal:** [seu nome ou nome do estúdio]
**Especialidade principal:** [ex: identidade visual para pequenos negócios, branding para saúde e bem-estar]
**Público que quero atender:** [ex: pequenos empreendedores locais, startups, profissionais de saúde]
**Trabalhos que tenho (reais ou fictícios):** [liste de 1 a 5, com nome do projeto e o que foi feito]
**Diferenciais ou forma de trabalhar:** [algo que você faz diferente — mesmo que pequeno]
**Onde o portfólio vai ficar:** [Behance / Instagram / site próprio / PDF]

Entregue:

**1. Bio profissional (versão curta — 2 linhas)**
Para usar no Instagram, Behance e LinkedIn.

**2. Bio profissional (versão longa — 1 parágrafo)**
Para o site ou apresentação pessoal.

**3. Estrutura de apresentação para cada projeto**
Template de estudo de caso com: contexto do cliente, problema, processo criativo, solução e resultado. Preencha com os trabalhos informados.

**4. Headline do portfólio**
Frase principal que aparece no topo do site ou perfil — que comunica quem você atende e o que você entrega.

**5. Chamada para contato (CTA)**
Texto para o botão ou frase final que convida o visitante a entrar em contato.

**6. Dica de como compensar a falta de cases reais**
3 estratégias práticas para construir portfólio sem precisar ter clientes reais.
```

## Exemplo de uso

### Input
- Nome: Gui Furtado
- Especialidade: Identidade visual para pequenos negócios
- Público: Empreendedores iniciantes e profissionais autônomos
- Trabalhos: 1 projeto fictício (Café Raiz — cafeteria artesanal), 1 projeto pessoal (minha própria marca)
- Diferencial: Entrego rápido e explico cada decisão criativa para o cliente
- Onde: Instagram + Behance
