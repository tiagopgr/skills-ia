# Programa de Mentoria para Donos de Imobiliária

## Objetivo
Estruturar um programa completo de mentoria para donos de imobiliária — com diagnóstico inicial, trilha de sessões por tema, entregáveis por fase e ferramentas práticas para cada encontro — para que o mentor entregue uma experiência estruturada e de alto valor, e o mentorado saia de cada sessão com clareza sobre o próximo passo do seu negócio.

## Quando usar
- Para estruturar o produto de mentoria que você vai vender ou já vende
- Quando quer padronizar a experiência de diferentes mentorados
- Para criar a trilha de conteúdo de um programa de mentoria em grupo
- Ao lançar um novo formato de programa de desenvolvimento para donos de imobiliária

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o perfil do mentorado ideal, a duração do programa e os temas que você domina
3. Receba a estrutura completa do programa com sessões, diagnóstico e materiais por fase
4. Use como base para criar a proposta comercial e a entrega do programa

## O Prompt
```
Você é um arquiteto de programas de mentoria para empreendedores do setor imobiliário. Seus princípios: (1) mentoria sem diagnóstico é conselho genérico — o primeiro passo é sempre entender onde o mentorado está de verdade, (2) cada sessão precisa de entregável concreto — o mentorado não paga para ter insight, paga para transformar o negócio, (3) a trilha deve ter sequência lógica: primeiro estrutura, depois crescimento — construir em cima de base frágil não funciona, (4) o mentor de imobiliária mais eficaz é o que transfere sistema, não só experiência.

**Perfil do mentorado ideal:** [ex: dono de imobiliária com 1-5 anos de operação, equipe pequena, fatura R$30k-150k/mês]
**Duração do programa:** [ex: 3 meses / 6 meses / 12 encontros]
**Frequência dos encontros:** [ex: semanal / quinzenal / mensal]
**Principais transformações que você promete:** [ex: organizar processos, dobrar faturamento, montar equipe de alta performance]
**Seus maiores diferenciais como mentor:** [ex: já escalou imobiliária própria, especialista em processos, expertise em marketing digital imobiliário]
**Formato:** [ex: 1:1 online / grupo / presencial]

Entregue:

**1. Diagnóstico Inicial (Sessão 0)**
Estrutura completa do diagnóstico de entrada: as 10-15 perguntas que mapeiam a real situação do negócio, o que analisar nos dados e como formatar o resultado para o mentorado.

**2. Trilha de Sessões**
Cronograma de todas as sessões com: número, tema, objetivo, conteúdo principal e entregável que o mentorado sai com.

**3. Material de Cada Fase**
Por fase do programa (ex: base / crescimento / escala): qual o foco, quais ferramentas usar, quais templates entregar.

**4. Proposta Comercial do Programa**
Estrutura da apresentação do programa para vender: o que está incluso, a transformação prometida e o investimento.

**5. Critérios de Sucesso por Sessão**
Como o mentor sabe que o mentorado absorveu e aplicou — o que checar no início de cada sessão.
```

## Exemplo de uso

### Input
- Mentorado ideal: Dono de imobiliária pequena, 2-4 anos de mercado, equipe de 3-6 pessoas, fatura R$40k-100k/mês
- Duração: 3 meses (12 sessões quinzenais)
- Promessas: Processos estruturados, equipe funcionando sem o dono em tudo, crescimento de 30% no faturamento
- Diferenciais do mentor: Escalou imobiliária própria de 2 para 18 pessoas, especialista em processos e marketing
- Formato: 1:1 online, 1h por sessão
