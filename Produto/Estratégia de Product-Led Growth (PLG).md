# Estratégia de Product-Led Growth (PLG)

## Objetivo
Criar a estratégia de Product-Led Growth para um produto digital — onde o próprio produto é o principal motor de aquisição, conversão e expansão — definindo o modelo de free/trial, loops de crescimento, momento aha! e métricas.

## Quando usar
- Ao definir a estratégia de go-to-market de um SaaS/app
- Para migrar de modelo sales-led para product-led
- Quando o CAC está alto e quer reduzir dependência de vendas
- Em planejamento de crescimento para produto digital

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o produto, público e modelo atual
3. Receba a estratégia PLG completa
4. Implemente em fases, começando pelo modelo de free/trial

## O Prompt
```
Você é um estrategista de Product-Led Growth que sabe que PLG não é apenas "ter um plano free" — é uma filosofia onde o produto faz o trabalho de marketing e vendas. Seus princípios: (1) o usuário deve experimentar valor ANTES de pagar, (2) o produto deve ter loops de viralidade naturais, (3) a expansão acontece de baixo para cima.

Crie a estratégia de PLG para:

**Produto:** [descreva o que faz]
**Público-alvo:** [quem usa — indivíduo, equipe, empresa]
**Modelo atual:** [sales-led, marketing-led, já product-led]
**Pricing atual:** [planos e preços]
**Tem versão free/trial?** [sim/não — se sim, qual modelo]
**Momento "aha!":** [quando o usuário experimenta valor pela primeira vez]
**Viralidade natural:** [o produto é usado com outros? compartilhável?]

Entregue: Modelo free/trial recomendado, Fluxo de ativação (signup → aha!), Loops de crescimento (viral, content, network), Product-Qualified Leads (PQLs), Expansão (Land & Expand), Métricas PLG, Roadmap de implementação.
```

## Exemplo de uso

### Input
Produto: Ferramenta de design colaborativo (tipo Canva para equipes de marketing)
Público: Equipes de marketing em empresas de 10-200 funcionários
Modelo atual: 14 dias de trial, depois venda por sales
Momento aha!: Quando 2+ pessoas editam um design ao mesmo tempo
Viralidade: Designs são compartilhados para aprovação

### Output
Modelo recomendado: Freemium com limite de uso (não free trial por tempo)

Free: Até 3 usuários, 5 projetos, templates básicos, exportação com marca d'água
Pro (R$29/user/mês): Ilimitado, sem marca d'água, templates premium, brand kit
Enterprise (R$59/user/mês): SSO, admin controls, analytics, API

Por quê freemium: O momento aha! requer colaboração. Trial de 14 dias é pouco tempo para que o hábito se forme.

Viral loop: Usuário free compartilha design → Pessoa cria conta free para comentar → Começa a usar → Convida equipe.

PQL: Usuário free que convidou 2+ pessoas E criou 3+ projetos em 30 dias = pronto para upgrade.
