# Pesquisa de Keywords (Seed > Long Tail)

## Objetivo
Realizar uma pesquisa de palavras-chave completa e estratégica — partindo de keywords semente até long tails específicas — classificando por intenção de busca, dificuldade e potencial de conversão para criar um plano de conteúdo que gere tráfego qualificado.

## Quando usar
- Ao planejar estratégia de conteúdo para blog/site
- Para encontrar oportunidades de keywords com pouca concorrência
- Antes de criar conteúdo (validar se há demanda)
- Em planejamento de SEO trimestral/anual

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Forneça as keywords semente e contexto do negócio
3. Receba a pesquisa completa com clusters organizados
4. Priorize e crie conteúdo para as mais estratégicas

## O Prompt
```
Você é um especialista em keyword research que sabe que a pesquisa de palavras-chave não é só listar termos com volume — é entender a intenção por trás de cada busca e conectar ao funil de vendas.

Faça a pesquisa de keywords para:

**Negócio:** [o que a empresa faz]
**Site/Blog:** [URL se tiver]
**Autoridade do domínio:** [baixa (site novo), média, alta]
**Público-alvo:** [quem busca no Google]
**Keywords semente (3-5):** [os termos principais do seu nicho]
**Objetivo:** [tráfego, leads, vendas, autoridade]
**Concorrentes principais:** [URLs de concorrentes que ranqueiam bem]

Entregue: Expansão de keywords (cauda média, long tail, perguntas, comerciais, comparativas), Classificação por intenção e dificuldade, Clusters de conteúdo (pilar + apoio), Análise de oportunidades (quick wins, content gaps), Plano de conteúdo priorizado, Keywords negativas.
```

## Exemplo de uso

### Input
Negócio: Escritório de contabilidade digital
Keywords semente: "contabilidade", "contador online", "abrir empresa"
Público: MEIs e donos de pequenas empresas
Objetivo: Gerar leads (cadastros no site)
Autoridade: Baixa (blog novo)

### Output
Quick wins:
1. "quanto custa um contador para MEI" — Volume médio, dificuldade baixa, intenção comercial
2. "como mudar de MEI para ME" — Volume médio, dificuldade baixa
3. "MEI pode emitir nota fiscal de serviço" — Volume alto, featured snippet disponível

Cluster "MEI":
- Pilar: "Guia Completo do MEI: Tudo que Você Precisa Saber"
- Apoio: "como abrir MEI", "MEI pode ter funcionário", "limite de faturamento MEI"
