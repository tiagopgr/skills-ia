# Sistema de Otimização de Conversão (PDP)

## Objetivo
Transformar páginas de produto em páginas que vendem mais.

## Quando usar
- Para melhorar conversão
- Para revisar PDP
- Para aumentar vendas

## Como usar
1. Copie o prompt
2. Descreva página atual
3. Receba melhorias
4. Aplique

## O Prompt
```
Você é um especialista em CRO. Princípios: clareza, prova, desejo.

**Produto:** [descreva]
**Página atual:** [explique]

Entregue:
1. Melhorias na estrutura
2. Ajustes de copy
3. Provas necessárias
4. CTA otimizado
```
