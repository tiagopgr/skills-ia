# Auditoria de SEO On-Page

## Objetivo
Fazer uma auditoria completa de SEO on-page de uma página específica — analisando título, meta description, headings, conteúdo, imagens, links internos e dados estruturados — com diagnóstico e recomendações práticas para melhorar o ranking no Google.

## Quando usar
- Para otimizar páginas que não estão ranqueando bem
- Antes de publicar conteúdo novo (checklist de SEO)
- Em auditorias regulares das páginas mais importantes
- Quando uma página perde posição no Google

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o conteúdo da página ou forneça a URL
3. Receba a auditoria com score e recomendações
4. Implemente as correções por prioridade

## O Prompt
```
Você é um especialista em SEO on-page que audita páginas com o rigor de um consultor e a praticidade de quem implementa. Você sabe que SEO on-page é a base — sem ele, nem o melhor conteúdo ranqueia.

Audite a seguinte página:

**URL ou conteúdo da página:** [cole o conteúdo completo ou descreva]
**Keyword principal:** [a palavra-chave que quer ranquear]
**Keywords secundárias:** [2-3 variações]
**Posição atual no Google:** [se souber]
**Tipo de página:** [artigo de blog, página de produto, landing page]
**Concorrentes no top 3:** [URLs dos concorrentes que estão ranqueando]

Audite: Title Tag, Meta Description, URL (Slug), Heading Structure (H1-H6), Conteúdo, Imagens, Links internos, Schema Markup.

Entregue: Score geral (0-100), Análise de cada elemento com ✅/⚠️/❌, Plano de ação priorizado (Top 10 correções).
```

## Exemplo de uso

### Input
Página: Artigo "Como fazer bolo de cenoura"
Keyword: "bolo de cenoura"
Posição atual: 15° (página 2)
Title atual: "Receita de Bolo de Cenoura | Blog da Maria"
Conteúdo: 600 palavras, sem imagens otimizadas

### Output
Score: 42/100

Top 3 problemas:
1. **Conteúdo curto** (600 palavras vs 1.800+ dos top 3). Ampliar com dicas, variações, erros comuns, FAQ. Impacto: ALTO.
2. **Title genérico** — não diferencia de milhares de resultados. Sugestão: "Bolo de Cenoura Fofinho: Receita Testada (Pronta em 40 Min)"
3. **Zero links internos** apontando para esta página. Adicionar links de 5 artigos relacionados.

Quick win: Adicionar schema de Recipe (rich snippet com tempo de preparo, avaliação).
