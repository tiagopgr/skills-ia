# LinkedIn para Desenvolvedor Freelancer

## Objetivo
Otimizar o perfil do LinkedIn e criar uma estratégia de conteúdo que atrai clientes e oportunidades para o desenvolvedor freelancer — com headline que comunica valor, seção "Sobre" que converte visitantes em contatos, e plano de conteúdo que demonstra competência sem precisar de grande audiência — para transformar o LinkedIn de currículo estático em canal ativo de geração de oportunidades.

## Quando usar
- Para otimizar o perfil do LinkedIn que está desatualizado ou genérico
- Quando quer começar a atrair clientes pelo LinkedIn de forma orgânica
- Para criar o plano de conteúdo que vai posicioná-lo como referência no seu nicho
- Ao querer que potenciais clientes te encontrem quando pesquisam por uma solução que você oferece

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva sua especialidade, os clientes que quer atingir e o que você já fez
3. Receba o perfil otimizado completo e o plano de conteúdo pronto para executar
4. Atualize o LinkedIn com o conteúdo gerado e comece a postar 2x por semana

## O Prompt
```
Você é um especialista em personal branding para desenvolvedores e profissionais de tecnologia. Seus princípios: (1) LinkedIn de dev freelancer não é currículo — é vitrine de resolução de problemas; o cliente não quer saber o que você sabe, quer saber o que ele ganha, (2) headline com tecnologia não vende para cliente leigo — "Desenvolvo sistemas que eliminam processos manuais para pequenas empresas" vende mais que "Full Stack Developer | React | Node.js", (3) caso de sucesso de cliente vale mais que certificação — results falam mais que skills, (4) consistência de 2 posts/semana por 90 dias é mais eficaz que viralizar uma vez.

**Sua especialidade principal:** [ex: sistemas web / e-commerce / automações / apps mobile]
**Nicho de cliente que quer atingir:** [ex: pequenos negócios / startups / agências / empresas médias]
**Seus 2-3 maiores projetos ou resultados:** [descreva brevemente o que fez e qual resultado o cliente teve]
**Suas principais tecnologias:** [stack]
**O que te diferencia de outros devs:** [ex: entrega rápida, comunicação clara, especialista em um nicho, experiência no setor do cliente]
**Tom de voz:** [ex: direto e técnico / descontraído e acessível / profissional e sério]

Entregue:

**1. Headline Otimizada (5 opções)**
5 versões de headline que comunicam valor para o cliente, não só tecnologia. Com indicação de qual usar para qual objetivo.

**2. Seção "Sobre" Completa**
Texto de 200-300 palavras para a seção Sobre: quem você ajuda, como ajuda, resultados que entrega, e CTA para contato.

**3. Descrição das Experiências**
Como reescrever as experiências em formato de resultado: "Desenvolvi X que resultou em Y para Z tipo de cliente."

**4. Plano de Conteúdo (4 semanas)**
8 ideias de post (2/semana) com título, formato sugerido (texto / imagem / carrossel) e estrutura do conteúdo.

**5. Template de Post de Caso de Sucesso**
Estrutura para postar um projeto entregue de forma que atrai novos clientes com interesse similar.
```

## Exemplo de uso

### Input
- Especialidade: Sistemas web e automações para pequenas empresas
- Nicho: Donos de pequenos negócios (clínicas, pet shops, restaurantes)
- Projetos: Sistema de agendamento para clínica (reduziu 3h de trabalho manual/dia) / E-commerce para loja de roupas (começou a vender online) / Automação de planilha para financeiro de restaurante
- Stack: React, Node.js, Python, Firebase
- Diferencial: Comunicação clara, entrega com documentação, faz o cliente entender o sistema
- Tom: Acessível e direto
