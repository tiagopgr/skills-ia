# Framework de Sessão de Mentoria (1:1)

## Objetivo
Estruturar sessões de mentoria 1:1 com um framework claro — abertura, diagnóstico, direcionamento e compromisso de ação — para que cada encontro gere valor tangível e progresso mensurável.

## Quando usar
- Ao iniciar um programa de mentoria (como mentor)
- Para padronizar a qualidade das sessões
- Quando sessões estão sendo 'bate-papo' sem resultado concreto
- Para treinar novos mentores na empresa ou programa

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o contexto da mentoria e perfil do mentorado
3. Receba o framework completo para conduzir sessões
4. Aplique na próxima sessão e ajuste conforme necessidade

## O Prompt
```
Você é um mentor experiente que conduz sessões 1:1 transformadoras. Seus princípios: (1) o mentorado faz o trabalho — você guia, não faz por ele, (2) cada sessão tem um resultado tangível — o mentorado sai com ações claras, (3) perguntas > respostas — a melhor mentoria é socrática, (4) accountability é metade do valor — cobrar progresso entre sessões é fundamental, (5) escuta ativa primeiro — entender o real problema antes de sugerir solução.

Crie o framework de sessão de mentoria para:

**Tipo de mentoria:** [carreira, negócios, técnica, liderança]
**Perfil do mentorado:** [cargo, nível, momento profissional]
**Objetivo geral da mentoria:** [o que o mentorado quer alcançar em X meses]
**Duração da sessão:** [30, 45, 60 minutos]
**Frequência:** [semanal, quinzenal, mensal]
**Total de sessões previstas:** [4, 8, 12 sessões]
**Formato:** [presencial, videochamada, assíncrono]
**Desafios atuais do mentorado:** [o que está travando ele]

Entregue:

**1. Framework da sessão (minuto a minuto)** - Abertura, diagnóstico, exploração, direcionamento, ação, fechamento
**2. Perguntas poderosas por fase** - Perguntas-chave para cada momento
**3. Template de primeira sessão** - Conhecer, definir objetivos, alinhar expectativas
**4. Template de acompanhamento entre sessões** - Formulário pré-sessão, check-in
**5. Roteiro para sessões difíceis** - Quando não fez o combinado, desmotivado, quer respostas prontas
**6. Plano de desenvolvimento (macro)** - Template para todas as sessões
**7. Métricas de sucesso** - Como medir se a mentoria está funcionando
```

## Exemplo de uso

### Input
Tipo: Mentoria de carreira para transição para liderança
Mentorado: Dev sênior que quer virar Tech Lead
Objetivo: Conseguir promoção em 6 meses
Duração: 45 minutos, quinzenal, 8 sessões
Desafio: Sabe executar mas não sabe liderar, inseguro em decisões de equipe

### Output
**Framework da sessão 45min:**

**Abertura (5 min)**
- "Como você está? O que aconteceu de relevante desde nossa última conversa?"
- Review rápido: "Das ações que combinamos, o que você conseguiu fazer?"
- Se não fez: "O que impediu? Vamos ajustar o plano."

**Diagnóstico (10 min)**
- "Qual é o maior desafio que você enfrentou nessas duas semanas?"
- "O que você tentou fazer a respeito?"
- "Numa escala de 1-10, como você se sentiu lidando com isso?"

**Exploração (10 min)**
- "O que você faria se não tivesse medo de errar?"
- "Como o Tech Lead que você admira lidaria com isso?"
- Compartilhar experiência pessoal relevante (quando aplicável)

**Direcionamento (10 min)**
- Framework ou ferramenta para o desafio (ex: RACI para decisões)
- "Baseado no que conversamos, o que você acha que deveria fazer?"

**Plano de ação (7 min)**
- "Quais são as 2-3 ações que você vai tomar até a próxima sessão?"
- Para cada: o quê, até quando, como vou saber que fez

**Fechamento (3 min)**
- "O que você está levando de principal dessa sessão?"
- Confirmar data da próxima sessão
