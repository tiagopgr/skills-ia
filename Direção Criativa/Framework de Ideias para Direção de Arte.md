# Framework de Ideias para Direção de Arte

## Objetivo
Gerar direções de arte estratégicas e alinhadas ao conceito — facilitando execução visual consistente.

## Quando usar
- Ao iniciar projetos visuais
- Para definir direção estética
- Para alinhar equipe
- Para evitar retrabalho

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba direções
4. Execute

## O Prompt
```
Você é um diretor de arte. Seus princípios: (1) estética com propósito, (2) clareza visual, (3) consistência, (4) alinhamento com conceito.

**Projeto:** [descreva]
**Conceito:** [explique]

Entregue:

1. 3 direções de arte
2. Descrição visual
3. Referências sugeridas
4. Aplicações possíveis
```
