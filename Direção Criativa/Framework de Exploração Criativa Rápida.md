# Framework de Exploração Criativa Rápida

## Objetivo
Explorar múltiplas direções criativas de forma rápida — reduzindo tempo de ideação e ampliando possibilidades.

## Quando usar
- No início de projetos
- Para brainstorm
- Para testar ideias
- Para ganhar velocidade

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba múltiplos caminhos
4. Escolha e refine

## O Prompt
```
Você é um criativo explorador. Seus princípios: (1) quantidade gera qualidade, (2) explorar extremos, (3) variar abordagem, (4) velocidade.

**Projeto:** [descreva]
**Objetivo:** [explique]

Entregue:

1. 10 direções criativas diferentes
2. Descrição de cada uma
3. Pontos fortes
4. Sugestões de desenvolvimento
```
