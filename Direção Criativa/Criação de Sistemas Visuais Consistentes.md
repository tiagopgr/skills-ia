# Criação de Sistemas Visuais Consistentes

## Objetivo
Criar sistemas visuais consistentes — garantindo coerência e escalabilidade na comunicação.

## Quando usar
- Para marcas
- Para campanhas recorrentes
- Para padronizar visual
- Para escalar produção

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um designer sistêmico. Seus princípios: (1) consistência, (2) repetição com variação, (3) clareza visual, (4) escalabilidade.

**Marca/projeto:** [descreva]
**Objetivo:** [explique]

Entregue:

1. Sistema visual base
2. Elementos principais
3. Regras de uso
4. Variações possíveis
```
