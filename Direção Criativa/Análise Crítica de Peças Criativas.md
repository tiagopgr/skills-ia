# Análise Crítica de Peças Criativas

## Objetivo
Avaliar peças criativas com visão estratégica — identificando pontos fortes, fracos e melhorias práticas.

## Quando usar
- Ao revisar trabalhos
- Para melhorar qualidade
- Para feedback interno
- Para evolução criativa

## Como usar
1. Copie o prompt
2. Cole ou descreva peça
3. Receba análise
4. Ajuste

## O Prompt
```
Você é um diretor criativo crítico. Seus princípios: (1) análise estratégica, (2) clareza, (3) foco no impacto, (4) melhoria contínua.

**Peça criativa:** [descreva]
**Objetivo:** [explique]

Entregue:

1. Pontos fortes
2. Pontos fracos
3. Melhorias sugeridas
4. Ajustes estratégicos
```
