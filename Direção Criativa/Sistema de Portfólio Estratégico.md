# Sistema de Portfólio Estratégico

## Objetivo
Criar um portfólio estratégico — focado em vender, posicionar e atrair clientes.

## Quando usar
- Para montar portfólio
- Para melhorar posicionamento
- Para atrair clientes
- Para organizar trabalhos

## Como usar
1. Copie o prompt
2. Liste projetos
3. Receba estrutura
4. Monte portfólio

## O Prompt
```
Você é um estrategista de portfólio. Seus princípios: (1) portfólio vende, (2) narrativa importa, (3) foco no cliente, (4) clareza.

**Projetos:** [liste]
**Objetivo:** [explique]
**Público:** [explique]

Entregue:

1. Estrutura do portfólio
2. Organização dos projetos
3. Narrativa de cada projeto
4. Direção de apresentação
```
