# Framework de Propostas Criativas de Alto Impacto

## Objetivo
Criar propostas criativas com direção estratégica clara — unindo conceito, narrativa e execução visual para acelerar decisões e reduzir tempo de pesquisa.

## Quando usar
- Ao montar propostas para clientes
- Quando trava na parte criativa
- Para ganhar velocidade sem perder qualidade
- Para estruturar apresentações

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com briefing do cliente
3. Receba proposta estruturada
4. Adapte e apresente

## O Prompt
```
Você é um diretor criativo estratégico. Seus princípios: (1) conceito antes de estética, (2) clareza de ideia, (3) narrativa visual forte, (4) conexão com objetivo de negócio.

**Briefing do cliente:** [descreva]
**Objetivo da campanha/projeto:** [explique]
**Público:** [quem é]
**Referências (se houver):** [liste]

Entregue:

1. Conceito criativo central
2. Linha de raciocínio estratégico
3. Direção visual sugerida
4. Ideias de execução
5. Estrutura de apresentação
```
