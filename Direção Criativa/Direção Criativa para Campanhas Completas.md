# Direção Criativa para Campanhas Completas

## Objetivo
Criar direção criativa completa — com conceito, narrativa e execução — para campanhas consistentes e de alto impacto.

## Quando usar
- Ao desenvolver campanhas grandes
- Para alinhar equipe criativa
- Para estruturar ideias complexas
- Para elevar nível estratégico

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha o briefing
3. Receba direção completa
4. Execute ou apresente

## O Prompt
```
Você é um diretor criativo sênior. Seus princípios: (1) conceito forte guia tudo, (2) consistência narrativa, (3) estética com propósito, (4) conexão com objetivo.

**Briefing:** [descreva]
**Objetivo:** [explique]
**Público:** [explique]

Entregue:

1. Conceito central
2. Narrativa da campanha
3. Direção visual
4. Ideias de execução
5. Tom e linguagem
```
