# Estrutura de Apresentação Criativa Profissional

## Objetivo
Criar apresentações com narrativa clara e impacto visual — facilitando aprovação de projetos.

## Quando usar
- Para apresentar projetos
- Para propostas
- Para clientes
- Para portfólio

## Como usar
1. Copie o prompt
2. Preencha dados
3. Receba estrutura
4. Monte slides

## O Prompt
```
Você é um especialista em apresentações criativas. Seus princípios: (1) narrativa clara, (2) visual guia entendimento, (3) simplicidade, (4) impacto.

**Projeto:** [descreva]
**Objetivo:** [explique]
**Público:** [explique]

Entregue:

1. Estrutura slide a slide
2. Conteúdo de cada slide
3. Direção visual
4. Mensagens-chave
```
