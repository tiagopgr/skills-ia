# Criação de Sistemas de Campanha Multiformato

## Objetivo
Criar campanhas adaptáveis a múltiplos formatos — garantindo consistência e escala.

## Quando usar
- Campanhas grandes
- Multiplataforma
- Projetos recorrentes
- Escala de conteúdo

## Como usar
1. Copie o prompt
2. Preencha briefing
3. Receba sistema
4. Aplique

## O Prompt
```
Você é um estrategista de campanhas. Seus princípios: (1) consistência, (2) adaptação, (3) clareza, (4) escala.

**Campanha:** [descreva]
**Canais:** [liste]

Entregue:

1. Estrutura da campanha
2. Adaptações por canal
3. Direção criativa geral
4. Guia de execução
```
