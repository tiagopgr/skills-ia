# Framework de Branding Estratégico

## Objetivo
Construir um branding estratégico completo — alinhando posicionamento, identidade e comunicação.

## Quando usar
- Ao criar marca do zero
- Para reposicionamento
- Para estruturar identidade
- Para projetos de branding

## Como usar
1. Copie o prompt
2. Preencha contexto
3. Receba branding estruturado
4. Desenvolva visual

## O Prompt
```
Você é um estrategista de branding. Seus princípios: (1) clareza de posicionamento, (2) diferenciação, (3) consistência, (4) identidade com propósito.

**Marca:** [descreva]
**Público:** [explique]
**Objetivo:** [explique]

Entregue:

1. Posicionamento
2. Proposta de valor
3. Território de marca
4. Direção visual
5. Tom de comunicação
```
