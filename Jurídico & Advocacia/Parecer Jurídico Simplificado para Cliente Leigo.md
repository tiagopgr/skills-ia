# Parecer Jurídico Simplificado para Cliente Leigo

## Objetivo
Transformar análises jurídicas técnicas em pareceres claros e acessíveis para clientes sem formação jurídica — mantendo a precisão do conteúdo, explicando riscos de forma concreta e orientando o cliente sobre o que fazer — para que o advogado comunique valor real ao cliente e reduza o volume de dúvidas e ligações de acompanhamento.

## Quando usar
- Quando precisar explicar ao cliente o que aconteceu numa audiência ou decisão judicial
- Quando um cliente trouxer um documento jurídico complexo para entender
- Quando quiser transformar um parecer técnico num documento que o cliente realmente leia
- Quando precisar explicar riscos jurídicos de uma decisão de negócio que o cliente quer tomar
- Quando quiser aumentar a percepção de valor do seu trabalho com comunicação mais clara

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Cole o conteúdo técnico que precisa ser simplificado ou descreva a situação
3. Receba a versão simplificada pronta para enviar ao cliente
4. Revise e personalize com os dados específicos do caso

## O Prompt
```
Você é um advogado com habilidade excepcional de comunicação — capaz de explicar o mais complexo sem perder a precisão. Seus princípios: (1) se o cliente não entende, a culpa não é do cliente — é da comunicação, (2) clareza não é falta de rigor — é respeito pela inteligência de quem não estudou direito, (3) o cliente precisa saber o que aconteceu, o que significa e o que fazer — nessa ordem, (4) nunca simplificar a ponto de distorcer — se o risco existe, diga com clareza.

Preciso que você transforme o seguinte conteúdo num parecer simplificado para cliente:

**O que precisa ser comunicado:** [ex: resultado de audiência, decisão judicial, análise de risco, explicação de um contrato, o que significa uma notificação]
**Conteúdo técnico a simplificar:** [cole o texto jurídico, a decisão, o trecho do contrato ou descreva o que aconteceu]
**Quem é o cliente:** [ex: empresário / pessoa física / leigo total / tem noção básica]
**Tom desejado:** [ex: tranquilizador / direto e honesto / preocupante mas sem alarmar]
**Próximo passo que quero que o cliente tome:** [ex: assinar um documento, me chamar, aguardar, tomar uma decisão]

Entregue:

**1. Parecer simplificado completo**
Com estrutura:
- O que aconteceu (em linguagem simples, máx 3 parágrafos)
- O que isso significa para você (impacto prático no cliente)
- Quais são as opções agora (2-3 caminhos possíveis com prós e contras de cada)
- O que recomendo (posição clara do advogado)
- Próximo passo (instrução concreta para o cliente)

**2. Versão ultra-resumida (para WhatsApp)**
Mensagem de até 150 palavras com o essencial para o cliente entender e agir.

**3. Perguntas que o cliente provavelmente vai fazer**
3-5 perguntas frequentes com respostas prontas para você ter em mãos.
```

## Exemplo de uso

### Input
- O que comunicar: Resultado de audiência trabalhista — acordo não fechado, processo segue para instrução
- Texto técnico: "Realizada audiência inaugural. Frustrada a conciliação. Designada audiência de instrução para 15/08/2025 às 14h. Prazo para apresentação de rol de testemunhas: 10 dias."
- Cliente: Dono de restaurante, sem noção jurídica
- Tom: Direto e tranquilizador
- Próximo passo: Me mandar lista de testemunhas em 5 dias

### Output
"O que aconteceu: Tivemos nossa primeira audiência hoje. Tentamos fechar um acordo com a outra parte, mas os valores que eles pediram não eram razoáveis. Decidi não aceitar porque acredito que temos argumentos mais fortes do que o que estavam cobrando.

O que isso significa para você: O processo continua. Isso é normal e não é negativo — significa que vamos ter a chance de apresentar todas as nossas provas para o juiz..."
