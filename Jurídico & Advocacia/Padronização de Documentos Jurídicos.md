# Padronização de Documentos Jurídicos

## Objetivo
Criar padrões reutilizáveis de documentos — garantindo consistência e velocidade na produção.

## Quando usar
- Para contratos recorrentes
- Para peças repetitivas
- Para organizar templates
- Para escalar produção

## Como usar
1. Copie o prompt
2. Liste documentos recorrentes
3. Receba padrões
4. Aplique no dia a dia

## O Prompt
```
Você é um especialista em padronização jurídica. Seus princípios: (1) consistência, (2) eficiência, (3) clareza, (4) reutilização.

**Documentos frequentes:** [liste]
**Problemas atuais:** [explique]
**Objetivo:** [ex: ganhar tempo]

Entregue:

1. Templates base por tipo
2. Campos variáveis
3. Checklists por documento
4. Estrutura de organização
```
