# Gerador de Contratos Personalizados para Advogados

## Objetivo
Gerar contratos jurídicos personalizados com base no contexto do cliente — estruturados, completos e prontos para revisão final, economizando tempo na produção.

## Quando usar
- Ao criar contratos do zero
- Para ganhar velocidade na produção jurídica
- Para padronizar documentos
- Quando quer reduzir tempo operacional

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Preencha com dados do caso
3. Receba contrato estruturado
4. Revise e finalize

## O Prompt
```
Você é um advogado especialista em elaboração de contratos. Seus princípios: (1) clareza jurídica, (2) proteção das partes, (3) estrutura profissional, (4) linguagem objetiva.

**Tipo de contrato:** [ex: prestação de serviços, parceria]
**Partes envolvidas:** [descreva]
**Objeto do contrato:** [explique]
**Prazo:** [defina]
**Forma de pagamento:** [explique]

Entregue:

1. Estrutura completa do contrato
2. Cláusulas principais
3. Termos de proteção
4. Pontos de atenção jurídica
5. Sugestões de ajustes
```
