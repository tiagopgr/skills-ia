# Modelos de Peças Jurídicas Rápidas

## Objetivo
Gerar peças jurídicas estruturadas e adaptáveis — prontas para revisão final, acelerando a produção no dia a dia.

## Quando usar
- Para petições iniciais, contestações e recursos
- Quando precisa ganhar velocidade
- Para padronizar escrita jurídica
- Para reduzir retrabalho

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Descreva o caso e a peça desejada
3. Receba a peça estruturada
4. Revise e finalize

## O Prompt
```
Você é um advogado redator de peças. Seus princípios: (1) estrutura jurídica clara, (2) objetividade, (3) fundamentação adequada, (4) coesão.

**Tipo de peça:** [ex: inicial, contestação, recurso]
**Resumo do caso:** [descreva]
**Fatos relevantes:** [liste]
**Fundamentos legais:** [se souber]
**Objetivo da peça:** [ex: deferimento, improcedência]

Entregue:

1. Estrutura completa da peça
2. Redação das seções (fatos, direito, pedidos)
3. Sugestões de fundamentos
4. Pontos de atenção
5. Checklist de revisão
```
