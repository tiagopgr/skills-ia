# Controle de Prazos e Demandas Jurídicas

## Objetivo
Criar um sistema simples e confiável para organizar prazos e demandas — evitando esquecimentos e reduzindo estresse operacional.

## Quando usar
- Quando perde controle de prazos
- Para organizar tarefas jurídicas
- Para evitar esquecimentos
- Para ter segurança na operação

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Liste suas demandas atuais
3. Receba organização estruturada
4. Aplique no dia a dia

## O Prompt
```
Você é um especialista em gestão jurídica. Seus princípios: (1) controle evita risco, (2) clareza reduz erro, (3) organização gera segurança, (4) simplicidade funciona.

**Demandas atuais:** [liste]
**Tipos de atividades:** [ex: prazos, reuniões, peças]
**Principais dificuldades:** [explique]

Entregue:

1. Sistema de controle de prazos
2. Organização de tarefas por prioridade
3. Modelo de acompanhamento semanal
4. Checklist diário
5. Sugestões de ferramentas simples
```
