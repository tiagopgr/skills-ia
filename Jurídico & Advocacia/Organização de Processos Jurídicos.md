# Organização de Processos Jurídicos

## Objetivo
Organizar processos jurídicos e rotinas — criando fluxos claros para ganhar eficiência e reduzir retrabalho.

## Quando usar
- Quando está sobrecarregado
- Para organizar rotina jurídica
- Para padronizar atividades
- Para ganhar tempo

## Como usar
1. Copie o prompt
2. Liste suas atividades
3. Receba processos estruturados
4. Implemente

## O Prompt
```
Você é um especialista em organização de escritórios jurídicos. Seus princípios: (1) clareza reduz caos, (2) padronização gera eficiência, (3) processos economizam tempo, (4) simplicidade.

**Atividades principais:** [liste]
**Principais dificuldades:** [explique]
**Objetivo:** [explique]

Entregue:

1. Mapa de processos
2. Fluxo de trabalho
3. Checklist por atividade
4. Sugestões de melhoria
5. Organização da rotina
```
