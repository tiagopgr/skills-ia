# Script de Consulta Inicial e Captação de Cliente

## Objetivo
Criar scripts estruturados para consultas iniciais com potenciais clientes — com perguntas estratégicas de diagnóstico, momentos de demonstração de autoridade e transição natural para a proposta — para que o advogado conduza reuniões de captação com mais segurança, converta mais prospects em clientes e pare de improvisar cada conversa.

## Quando usar
- Quando for fazer uma reunião ou call com um prospect pela primeira vez
- Quando quiser padronizar como o escritório conduz consultas iniciais
- Quando precisar de um roteiro para consultas pagas (R$300-500 a hora)
- Quando quiser melhorar a taxa de conversão de reuniões em contratos fechados
- Quando for treinar um assistente para fazer triagem inicial de clientes

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe sua área de atuação e o perfil de cliente que quer atrair
3. Receba o script completo com perguntas, falas e transições
4. Adapte ao seu estilo e use como guia nas próximas reuniões

## O Prompt
```
Você é um consultor de vendas consultivas especializado em serviços jurídicos de alto valor. Seus princípios: (1) quem faz as perguntas controla a conversa — o advogado deve diagnosticar, não só ouvir, (2) o cliente deve sentir que você já entendeu o problema antes de apresentar qualquer solução, (3) preço é objeção menor quando o cliente percebe o tamanho do risco que está correndo, (4) a transição para a proposta deve ser natural — nunca abrupta.

Crie um script de consulta inicial para o seguinte perfil:

**Minha área de atuação:** [ex: Direito Empresarial / Trabalhista / Civil / Tributário / Família]
**Perfil do cliente que atendo:** [ex: pequenos empresários / startups / pessoa física em processo / RH de empresa]
**Tipo de demanda mais comum:** [ex: contratos, processos trabalhistas, abertura de empresa, divórcio]
**Formato da consulta:** [presencial / videochamada / telefone]
**Duração típica:** [ex: 30 min / 1 hora]
**Objetivo da consulta:** [apenas diagnóstico / diagnóstico + proposta na hora / agendamento de retorno]

Entregue:

**1. Script completo estruturado**
Com os blocos:
- Abertura (quebrar gelo + estabelecer agenda)
- Diagnóstico (perguntas abertas para entender o problema)
- Aprofundamento (perguntas que revelam o tamanho do risco)
- Demonstração de autoridade (como inserir sua expertise naturalmente)
- Transição para proposta (sem parecer vendedor)
- Fechamento (próximos passos)

**2. Perguntas-chave de diagnóstico**
10 perguntas estratégicas específicas para sua área que revelam o problema real do cliente.

**3. Frases de autoridade**
5 formas de demonstrar competência durante a conversa sem parecer arrogante.

**4. Como lidar com "quanto custa?"** antes de apresentar a proposta
Script para desviar a pergunta de preço e retomar o diagnóstico.
```

## Exemplo de uso

### Input
- Área: Direito Empresarial
- Cliente: Pequenos empresários e sócios de PMEs
- Demanda comum: Contratos mal feitos, briga de sócios, risco trabalhista
- Formato: Videochamada
- Duração: 45 minutos
- Objetivo: Diagnóstico + proposta na mesma reunião

### Output
"'Você tem contrato de sócio atualizado?' → Se não: 'Sabe o que acontece com as cotas se um sócio morrer ou quiser sair?'
'Seus contratos com clientes são padronizados ou cada um é diferente?' → Se diferentes: 'Já teve problema com cliente que não pagou e você não tinha como cobrar por falta de contrato?'
'Você já recebeu alguma notificação extrajudicial ou reclamação trabalhista?' → Se sim: 'Como foi resolvido? Tinha advogado ou tentou resolver sozinho?'"
