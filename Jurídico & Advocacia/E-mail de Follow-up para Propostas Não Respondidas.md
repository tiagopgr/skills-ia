# E-mail de Follow-up para Propostas Não Respondidas

## Objetivo
Criar sequências de follow-up estratégicas para propostas de honorários que não tiveram resposta — retomando o contato com ângulos diferentes, agregando valor a cada toque e aumentando a chance de fechamento — sem que o advogado pareça desesperado, insistente ou sem opção.

## Quando usar
- Quando enviou uma proposta há mais de 2 dias sem resposta
- Quando um prospect sumiu após reunião promissora
- Quando quer criar uma sequência de follow-up padrão para o escritório
- Quando quer retomar contato com clientes que "estavam pensando" há semanas
- Quando quer reativar prospects que ficaram frios há mais de 30 dias

## Como usar
1. Copie o prompt abaixo no Claude ou ChatGPT
2. Informe o contexto da proposta enviada e há quanto tempo não há resposta
3. Receba sequência completa de follow-up com textos prontos para enviar
4. Adapte com os dados do prospect e envie no timing sugerido

## O Prompt
```
Você é um especialista em vendas consultivas de alto valor para profissionais liberais. Seus princípios: (1) follow-up não é chateação — é serviço, quando bem feito, (2) cada toque deve agregar algo novo — informação, urgência, reframe ou permissão para desistir, (3) o melhor follow-up é aquele que faz o prospect responder — mesmo que seja não, (4) tom deve ser confiante, não ansioso — quem tem agenda cheia não implora.

Crie uma sequência de follow-up para a seguinte situação:

**Tipo de proposta enviada:** [ex: assessoria mensal / defesa em processo / elaboração de contratos]
**Valor da proposta:** [R$X]
**Quando a proposta foi enviada:** [ex: há 3 dias / há 1 semana / há 2 semanas]
**Última interação:** [ex: enviou e-mail / teve reunião / prospect pediu para pensar]
**O que sei sobre a hesitação do prospect:** [ex: achou caro / estava avaliando outros advogados / disse que ia conversar com sócios]
**Canal de preferência:** [E-mail / WhatsApp / ambos]
**Quantos toques quero fazer:** [ex: 3 ou 4]

Entregue:

**1. Sequência completa de follow-up**
Para cada toque:
- Timing sugerido (ex: "D+3 após proposta")
- Ângulo do toque (o que é diferente em relação ao anterior)
- Texto completo pronto para enviar
- Linha de assunto (se e-mail)

**2. Toque de encerramento**
Mensagem final que "fecha o ciclo" — dá permissão para dizer não mas mantém a porta aberta.

**3. O que fazer se o prospect responder "ainda estou pensando"**
Script de resposta que retoma urgência sem pressionar.
```

## Exemplo de uso

### Input
- Proposta: Assessoria jurídica empresarial mensal
- Valor: R$3.200/mês
- Enviada: há 5 dias por e-mail
- Última interação: reunião excelente, prospect disse "faz sentido, preciso só confirmar com meu sócio"
- Hesitação: Decisão envolve sócio
- Canal: E-mail + WhatsApp
- Toques: 3

### Output
"Assunto: Uma coisa que esqueci de mencionar na nossa conversa
Olá [Nome], percebi que na nossa reunião não mencionei algo que costuma ser relevante para decisões como essa.
Além da assessoria mensal, todos os meus clientes têm acesso a uma revisão gratuita de qualquer contrato urgente que aparecer — sem cobrar adicional.
Fica a informação caso ainda estejam avaliando.
Se precisar de qualquer dado adicional para apresentar ao seu sócio, é só falar.
Dr. Carlos"
